美国有许多课程都免费上网了。值得一提的是，许多科技巨头也有自己的系列讲座。今天想给大家介绍的是“Google科技讲座” ([Google Tech Talks](http://www.youtube.com/user/GoogleTechTalks))。
本来这些讲座是为Google员工自己举办的。可贵的是，Google把这些视频都放到了YouTube上，让全世界的人都看的到。唯一遗憾的是，中国的
科技人员必须能到墙外去看。幸亏搞科技的人大多不把翻 墙当会事。Google中国在国内也举办了类似的讲座。我不知道是否也有类似的视频。

Google科技讲座已经有太多的片段，而且它的内容涉及面很广，我很难给出一个全面的介绍。下面根据它的“观看次数”和“被喜欢次数”来选择一些做介绍 (这就是社会网络的力量)。

这个系列很少有关于Google自己的产品。它的覆盖范围非常广，绝对是绝无仅有的。演讲者
也是各色各样，有技术人员、管理人员、某方向的专家、企业家、投资者、医生、教授、大学生、尼姑、和尚，甚至人权活动家和小学生。虽然很少有关于
Google自己的产品，但是我们可以通过这些讲座来感觉Google的兴趣和动向。我曾经听人家说，Google要收集全世界的所有信息，包括人的思
想。看了一些Google科技讲座后，笔者觉得这是很有可能的。虽然Google目前还不能读懂人们的大脑，但至少它已经[改变人脑结构](http://www.wenxuecity.com/news/2012/01/25/1613031.html)。Google对能源的兴趣也是显然的。不只是绿色能源，也有核能源。当然核能源弄好了也是很环保的。

不是每个讲座对你都是有吸引力的，但在一千六百多个视频中总会有你喜欢的。

如果你对下面的某个讲座特别感兴趣，请留意在视频右边的“相关视频”。Google常常在某个主题上有好多讲座。事实上，Google有很多讲座系列。它们之间经常有交叉。希望有机会也能介绍其他的讲座。下面开始具体介绍本次的系列。

JavaScript 是网络时代早期的一个语言，长期以来都在受到批评。但近年来 JavaScript 有了新的活力。考虑到Google的领域，我们不难理解Google对 JavaScript 有兴趣。下面是五个关于 JavaScript 的讲座。

**1.** 现在夸JavaScript已经不是什么问题了，因为JavaScript已经发生了本质的变化并越来越多地被人接受。但这个演讲“JavaScript：好的一面”是3年多以前的。也许它是JavaScript从“bad”到“good”的转折点。[Douglas Crockford](http://en.wikipedia.org/wiki/Douglas_Crockford)是JavaScript的先驱之一，目前在雅虎工作。这个讲座在Google的[博客](http://googlecode.blogspot.com/2009/03/doug-crockford-.-good-parts.html)上。这个讲座可以说是演讲者为自己的同名书做的一个广告。

JavaScript: The Good Parts by Doug Crockford (of Yahoo!)

February 27, 2009

[http://youtu.be/hQVTIJBZook](http://youtu.be/hQVTIJBZook)
**2.** 对 JavaScript 
的批评之一是它的速度。但是速度的快慢并不是汇编的问题，因为 JavaScript 没有汇编。JavaScript 
存在於浏览器之中，但浏览器对你的 JavaScript 指令的执行速度没有特别的关注。这就是 JavaScript 
速度慢的原因。尽管如此，我们还是有一些办法让自己的 JavaScript 
程序加速。如果你是网页开发人，不妨学习一下：“加速你的JavaScript”。

Speed Up Your JavaScript by Nicholas C. Zakas

June 4, 2009

[http://youtu.be/mHtdZgou0qU](http://youtu.be/mHtdZgou0qU)

**11.** “游戏化”([Gamification](http://en.wikipedia.org/wiki/Gamification))
 
是一个新词，它的意思是将游戏思维和游戏机制运用到各个领域，来引导用户互动和使用的过程。“乐趣就是未来：精通游戏化”是本次演讲的标题。到现在，大家
可能觉得这是一个关于游戏软件开发的演讲。其实不对。游戏化能在互联网、医疗/健康、教育、金融等领域中影响到用户使用时的心理倾向，进而促进用户的参与
与分享。 简单的说，它可以用来鼓励人们做一些通常被认为“无聊”的事， 例如完成调查、购物或者阅读网页等。它将是未来企业的一个重要部分。
