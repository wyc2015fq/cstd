# 关于一道印度人出的题目的说明

![](http://image.sciencenet.cn/album/201505/02/210047g442e8bt472fd817.jpg)

上面这个题目是我在新浪微博上发的一条。题目说的是，

用“（1，3，5，7，9，11，13，15）”填到三个空格里使得等式 ▢ + ▢  + ▢  = 30 成立。

当时我犹豫了一下，加上了一句话：“大家知道十进制中三个奇数的和不可能是偶数。那怎么办呢？”有了这个提示，读者给出了许多非十进制的解答：有五进制的“3 + 11 + 11 = 30”，七进制的“3 + 11 + 13 = 30”，九进制的“1 + 13 + 15 = 30”，等。有人指出：“在5进制至19进制的奇数进制中都有使其成立的等式”。这些解都很漂亮。这个题目可以帮助人们视线脑筋急转。很不错。

但其实我有些后悔在出这个微博时画蛇添足地写了一句：“大家知道十进制中三个奇数的和不可能是偶数。”因为我并不知道原题的真正答案。那么现在我们再来一次脑筋急转：假如原题就是在十进制里情况又会怎样呢？

答案是：我们就必须允许答题者增加除了那几个奇数以外的什么符号。比如小数点“.”。问的好！为什么不能是小数呢？如果是这样的话，那么我们又可以多出好多解来，比如“9.5 + 9.5 + 11 = 30”。

但这样一来，我们所取的已经不是整数了。

OK，现在我们假定，必须是十进制，而且必须取整数。你还有办法吗？

有人开始乱来了，比如“  + 15 + 15 = 30”，“15 + 9 + 3! = 30”，“ (11-1) + (11-1) + (11-1) = 30”，“7+7+7=30 mod 7”，甚至“11 + 13 + 倒9 = 30”。这些都不算正确的解，因为它们都用到了除了加号之外的其他运算符号：取零，阶乘！，减号-，模mod，倒数。不能增加算符。

正确的解是：“3 ( +1 +9) = 30”。

网友[](http://blog.sciencenet.cn/home.php?mod=space&uid=533418)[汪小龙](http://blog.sciencenet.cn/home.php?mod=space&uid=533418)老师给出了另一个似乎更好的解：“ ( +15 +15 ) = 30”。
