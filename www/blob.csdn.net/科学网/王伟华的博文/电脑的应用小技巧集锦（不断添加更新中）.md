47.在以Word为代表的Office系列中，图形、文本框等非字符元素的位置的微调： 以Word为例，在页面中插入图形或文本框等非字符元素，在页面排版时，经常 为Word固有的所进所干扰，无法达到如意的效果。在使用中，笔者摸索出以下 经验，与大家分享： 
①、在调整某一图片（或其他元素，以下全以图片为代）的位置时，使用鼠标 或方向键将其移动到大致位置时，按住Ctrl键，再用方向键移动，你会发现现 在图片每次移动的间距比原来要小得多了，在Word要求的精度下应当完全达到 您的要求了。 
②、在调整某一图片（或其他元素，以下全以图片为代）的大小时，不使用鼠 标时，图片变化的大小不连续，按住Alt键，再调节，现在其大小就可以任意 调节了.

                                  让你的电脑飞起来
很多朋友在使用电脑时发现机器反应慢，在这里我们可以进行如下操作，对系统进行优化。 
    进入“控制面板(经典视图）/管理工具/服务”或进行"开始/运行/输入services.msc”这里面的东西我们需要进行一下优化。并不是关闭的程序越多系统优化越好。以下是我的一些修改建议。 

   修改方法是将 启动类型 (禁用；自动；手动）这三个选项进行修改。不清楚的先查一下该服务（当光标移动到某项程序时左面窗口桌面有提示）。 

alerter -错误警报器。 (可关闭) 

application layer gateway service -给与第三者网络共享/防火墙支持的服务，有些防火墙/网络共享软件需要。占用1.5mb内存。 (可关闭) 

application management-用于设定，发布和删除软件服务。(我是把它改为手动的） 

automatic updates -windows 自动更新。你需要自动更新吗？(可关闭) 

background intelligent transfer service - 这个服务原是用来实现http1.1服务器之间的信息传输，微软称支持windows更新时断点续传。(建议自动） 

clipbook - 用与局域网电脑来共享 粘贴/剪贴的内容。(可关闭) 


com+Event system　-一些 COM+　软件需要。(检查你的 c:program filesComPlus Applications 目录，没东西可以把这个服务关闭) 

COM+Event system application　-同上 (可关闭) 

COM+system application　-同上 (可关闭，但我是手动) 

COmputer browser -用来浏览局域网电脑的服务，但关了也不影响浏览!(可关闭) 

cryptographic services -windows更新时用来确认windows 文件指纹的。 (可关闭，但我是自动) 

DHCP client-静态IP者需要(xDSL　等)。 （自动） 

Distributed link tracking client-用于局域网更新连接信息，比如在电脑A有个文件，在B做了个连接，如果文件移动了，这个服务将会更新信息。占用4兆内存。 (可关闭) 　　 

Distributed Transaction coordinator-无聊的东西。 (可关闭) 

DNS Client-DNS解析服务。(可关闭，建议自动，BT等软件需要) 

Error reporting service -错误报告器，把windows中错误报告给微软。你想做微软的免费测试员？(可关闭) 

Event Log- 系统日志纪录服务，很有用于查找系统毛病。（关闭这个会让你的电脑启动时间变长，建议自动） 

Fast user switching compatibility-多用户快速切换服务。(可关闭，我是手动) 

help and support -帮助。(可关闭) 

HTTP SSL -安全套接层协议层。（有些服务需要，建议手动） 


Human interface device access-支持'弱智'电脑配件的。比如键盘上调音量的按钮等等。(可关闭) 

IMAPI CD-burning COM service　-xp刻牒服务，用软件就不用了。占用1.6兆内存 (可关闭) 

Indexing service -恐怖的xp减速……(可关闭) 

Internet Connection Firewall(ICF)-----xp防火墙。(不用的话可关闭) 

IPSEC Services-大众用户连边都沾不上。 (可关闭) 

Logical Disk manager　-磁盘管理服务。需要时它会通知你，所以一般关。 (可关闭，但我是手动) 

Logical Disk manager administrative service-同上。 (可关闭，但我是手动) 

messenger -不是MSN，不想被骚扰的话就关。注:妖刺就是利用这个。 (可关闭) 

MS software shadow copy provider-无用，据说是备份用的。没大用。 (可关闭，但我手动) 

Net Logon-登陆Domain Controller用的。(可关闭) 

Netmeeting remote desktop sharing-用netmeeting实现电脑共享。(可关闭) 

Network Connections - 上网/局域网要用的。 （我是手动） 

Network DDE　-和clipbook一起用的。(可关闭) 

Network DDE DSDM　-同上 (可关闭) 

Network Location Awareness-如有网络共享或ICS/ICF可能需要。(服务器端) (可关闭，但我是手动) 

Network Provisioning Service 。（偶是手动） 

NT LM Security support provider-telnet　服务用的。(可关闭) 

NVIDIA Driver Helper service -nvidia 显卡帮助。(可关闭) 

Office Source Engine。（手动） 

Performance logs and alerts－记录机器运行状况而且定时写入日志或发警告，内容可能过于专业，所以。。自己决定。 (不是专业人士就关闭）
