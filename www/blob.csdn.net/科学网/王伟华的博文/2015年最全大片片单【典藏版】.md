**飓风营救3**
上映日期: 2015-01-09(美国)


导演: 奥利维尔·米加顿

主演: 连姆·尼森 / 玛姬·格蕾斯

引进指数：★★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWLwCdibmnOzdQibE7GZ9oSQRo2FGvibE1iaXyso0rYgtHsVUUuuzCPpjNeg/0)

当第一部上映时，制片人吕克·贝松表示只拍一部续集，如今，这成为了一句笑话——就跟吕胖子说他一生只拍十部片一样。本片导演由上一部的奥利维尔·米加顿继续担任，女儿也将回归，剧情讲述连姆·尼森被诬陷杀死至亲，成了被猎杀的对象。

海报上的宣传语：“一切终结于此”，似乎也暗示这第三部将是系列电影的最后一部了。





**骇客交锋**
上映日期: 2015-01-16(美国)

导演: 迈克尔·曼


主演: 克里斯·海姆斯沃斯 / 维奥拉·戴维斯

引进指数：★★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWnyF0TeMtqEPOHA4vQaZqNHxWiaQ3RNr0ibLDMFicWj63XCSb2fmurQ1tQ/0)

从2009年的《公众敌人》至今，迈克尔·曼已经将近四年没有推出任何作品。本片将在延续迈克尔·曼此前《小偷》、《盗火线》等硬派写实风格的同时，注入富有“技术含量”的网络盗窃、网络攻击等新内容。

而对于中国观众来说，最大的吸引力，应该是汤唯与王力宏的加盟，其中汤唯还将与“锤哥”展开一段异国恋，虽然演员表里他们的名字排在倒数——戏份堪忧啊，但冲着“中国女神”与“全世界最性感男人”手牵手数星星这个卖点，应该也足以吸干一批颜控吧。





**第七子**
上映日期: 2015-02-06(美国)

导演：谢尔盖·波德罗夫

主演：本·巴恩斯 / 杰夫·布里吉斯

引进指数：★★★★★（已确定于明年3月引进）

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWoUpaBy30RMWXXUWN6k3hkGU0meKrFQ0iaeDQtcega4gKvBeJiczoyZvw/0)

又改编自全球畅销小说，一部不亚于《魔戒》的视觉系魔幻大片，集结了戛纳、威尼斯以及柏林三料影后朱丽安·摩尔、老戏骨杰夫·布里吉斯出演以及“凯斯宾王子”本·巴恩斯的超强阵容。《300勇士：帝国崛起》原班人马及《哥斯拉》制作团队护航，驱魔人、怪物、女巫、大场面、禁忌爱情，这些关键词光听着就来劲。

虽然此片看上去跟《魔戒》，《哈利波特》差不多，均保持了哥特风高耸、阴森、诡异、神秘、恐怖的特点，但在细节上又融入亚洲、阿拉伯等新元素，使得画面更加唯美，别有一分趣味。

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWKekoQIxiaDZdRIHzW3v66FMQRncZcEh0PxaaAmmobhDtjozND1BN8Yg/0)




各种奇葩妖怪必不可少。





**木星上行**
上映日期: 2015-02-06(美国)

导演: 安迪·沃卓斯基 / 拉娜·沃卓斯基

主演: 查宁·塔图姆 / 米拉·库尼斯

引进指数：★★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWHnvrF9OVsNGSLICZsnrfv5f3hFLrlyDm5khf61ZupIm3SEl8qKic8vQ/0)

故事设定在遥远的未来，那时盛行各种基因改造，人类可以将动物基因与人类基因相结合而获得更强大的力量，而纯粹的人类反而变成社会底层。米拉·库妮斯饰演一个人类女性，由于她拥有和宇宙女王相同的基因，威胁到了宇宙女王的权利，于是被下令追杀。查宁·塔图姆在片中饰演负责追杀米拉·库妮斯的赏金猎人，但最后却爱上了这位人类女子。他的角色设定是“半兽人”，基因中一半是人类，一半是狼，还有一部分白化病的基因。

从已经曝光的预告和海报来看，片中充斥着让人肾上腺素爆棚的混战打斗场面，和气势恢弘的特效，逼真效果让人惊叹。

沃卓斯基姐弟表示，《木星上行》将是一部“比前作《云图》还要宏大的野心之作”，希望能再度像当年的《黑客帝国》一样，“完成电影史上的革新”。





**速度与激情7**
上映日期: 2015-04-03(美国)

导演: 温子仁

主演: 范·迪塞尔 / 保罗·沃克

引进指数：★★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWia6aicWW6eZWBezqjbxBYMts7Zd51QfsIJF7Pm8g4dyc5VDzcEIjPVTA/0)

飚车、枪战、美女，这些都是《速度与激情7》要给你展现的。在本集中，“巨石”强森与新加盟的杰森·斯坦森将上演硬汉对决，泰拳巨星托尼·贾也会亮相。

《速度与激情》第五部的全球票房是6.26亿美元，第六部是7.88亿美元，明年上映的第七部，只会更高。

虽然保罗·沃克离世为该片增添了遗憾。但片方决定由保罗的两个弟弟柯迪·沃克和迦勒·沃克担当他的替身，继续拍摄其未完成的戏份。后期再通过CGI尖端特效技术将保罗的面部与声线融合进去，以保证角色的完整性。

该特效技术在好莱坞亦是首次尝试。如此，观众就可以看到《速度与激情》兄弟帮在这一集继续并肩作战。





**复仇者联盟2：奥创纪元**
上映日期: 2015-05-01(美国)

导演: 乔斯·韦登

主演: 斯嘉丽·约翰逊 / 克里斯·埃文斯

引进指数：★★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHW2Cpru9pibYmoiapJ2sckcuD7O4MGQuY1NPCSDnqyEN7yjKTxGn7V8PWQ/0)

托尼·史塔克想要启动一个搁置的维护世界和平的项目，但搞砸了。地球命悬一刻，复联成员除了需要对付老“朋友”奥创，还得分神应对两个神秘而强大的新“对手”：猩红女巫与快银。

这部凭片名就能卖钱的电影已经不用多说。况且最近还有劲爆消息称，“抖森”饰演的洛基，也会出现在影片中，还有不进电影院的理由么？

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWiaJBBwibIq6t4NDiaUk7E8VKSxAPKJelAzYHmIBrpc3jqIQOrmwA2WZPA/0)

对了，这是钢铁侠为了对付浩克失控而制造的反浩克装甲。





**明日世界**
上映日期: 2015-05-22(美国)


导演：布拉德·伯德

主演：乔治·克鲁尼 / 布丽特妮·罗伯森

引进指数：★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWjVE9f4U9wickghc7bazZ2WwQuLjgWwRMbNsELYiaqfichDaYWzTV5SZLQ/0)

讲述一个开朗乐观、对科学充满好奇心的青少年，与一个对现实世界充满厌倦的前少年天才发明家，共同经历一段充满危险的旅程，去寻找他们共有记忆中的神秘之域：明日之地，并揭开它背后隐藏的秘密的故事。

这是久未有新动作的乔治·克鲁尼新作，从预告片来看，绝对是一部脑洞大开的科幻大作。





**侏罗纪世界**
上映日期: 2015-05-22(美国)

导演：科林·特莱沃若

主演：克里斯·帕拉特 / 杰克·M·约翰森

引进指数：★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHW70KQlJYdkmCicNvQno8SNJmb51GOfCWX9xxt4KN0NZc42DibJXw8yQNg/0)

注意，这不是斯皮尔伯格《侏罗纪公园》的重拍版，而是一部生化危机版的《侏罗纪公园》。影片讲述到6500万年前末，恐龙时代的一次大规模病毒爆发。但在此事发生多个世纪后，病毒因重组恐龙计划，而被意外重新带回世上。

影片由刚刚凭借《银河护卫队》大红，登上“全世界最性感男人”榜单亚军的“星爵”克里斯·帕拉特主演。片中将出现不同种类甚至不同物种的DNA相结合，而制造出的“混血”恐龙——可怕的D-Rex掠食者恐龙。

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWV7ARfyfWLl5ianIibyniaCfGKOu7NpFQmuOqTIaTjL4QOn0XU0zO9icA1g/0)

莫非就是这一头。





**头脑大作战**
上映日期: 2015-06-19(美国)

导演: 彼特·道格特

主演: 艾米·波勒 / 敏迪·卡灵

引进指数：★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHW856ebuhM2ibJjBUcgXTd0JDPBOpS0zFx93ybibZ2L9ze4wbW4QfYibW6g/0)

《头脑大作战》是彼得·道格特继《飞屋环游记》、《怪兽电力公司》后的又一部力作，影片将带领观众深入一个11岁的小姑娘莱利的头脑，将她的五种情感幻化为一个个的角色，演绎他们是如何影响这个小姑娘的。

影片再度展示了皮克斯团队卓越的想象力，今年皮克斯佳作缺席，相信明年不会让影迷失望。





**泰迪熊2**
上映日期: 2015-06-19(美国)

导演: 塞思·麦克法兰

主演: 马克·沃尔伯格 / 阿曼达·塞弗里德

引进指数：想都别想

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWiax2m8JGyE9KStamicbCM5c5hGGYFnCApjmrqfEJYo0KbMu62uV69ib9Q/0)

2012年6月29日的《泰迪熊》收入票房5.4亿，成为R级喜剧票房之王，拍摄续集也是顺水推舟。

《泰迪熊2》中，除了女主角换成金发美女阿曼达·塞弗里德，还将加入两位大牌——连姆·尼森和摩根·弗里曼。导演麦克法兰透露，连姆·尼森的角色可能只是匆匆出场，因为他的档期很紧，而摩根·弗里曼会饰演一名民事诉讼律师，泰迪将成为他的客户。





**终结者：创世纪**
上映日期: 2015-07-01(美国)

导演: 阿兰·泰勒

主演: 艾米莉亚·克拉克 / 阿诺·施瓦辛格

引进指数：★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWWSA1vhTV3J0H6Y0UX4XV6YodKxguibWAbm5ia7rianIyfgUFqyUBlHl4Q/0)

在经历了糟糕的第3部和第4部后，《终结者》系列终于迎来了看似靠谱的第五部。说其靠谱，是因为本片编剧名单里，有该系列鼻祖卡梅隆。

跟前作相比，本片故事有着颠覆性的设定：救世主约翰·康纳的母亲莎拉·康纳在影片中被施瓦辛格扮演的T-800抚养长大。最受关注的点，就是今年已经67岁的施瓦辛格，将回归这部重启作品，继续饰演T-800。





**小黄人**
上映日期: 2015-07-01(美国)

导演: 凯尔·巴尔达 / 皮埃尔·科芬

主演: 迈克尔·基顿 / 桑德拉·布洛克

引进指数：★★★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHW929Teb2ImMqLnS0MedhbRqUOic0atMuk3ayAwtnQ7kZrZa3QSicO3YfA/0)

小黄人从哪里来？在为格鲁服务之前他们是怎么生活的？他们原本就是群居吗？这些问题都会在小黄人的独立电影中得到解答。

在前段时间公布的首款预告中，蠢萌的小黄人已经把影迷们的胃口吊得高高的。结合这部电影此前公布的剧情来看，虽然小黄人是绝对主角，格鲁也不会完全缺席，影片还会讲述他认识三个小女孩之前的“黑历史”，相信也会呈现他是如何跟小黄人组成一个新团队的。





**蚁人**
上映日期: 2015-07-17(美国)

导演: 佩顿·里德

主演: 伊万杰琳·莉莉 / 保罗·路德

引进指数：★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWKlbTl3OHliceezh3dh3VQcmoNq3fZChL8vTiaicibfkWkEGYRcicBTC1HNQ/0)

作为漫威动漫电影宇宙第三阶段开篇之作，保罗·路德饰演二代蚁人斯科特·朗，迈克尔·道格拉斯饰演一代蚁人汉克·皮姆博士，《迷失》女星伊万杰琳·莉莉担任女主。

《蚁人》讲述职业骗子斯科特·朗拥有身体变小力量增大的超能力，他必须接受自己的英雄身份并帮助他的导师——生化学家汉克·皮姆保护他所发明的蚁人战服。在强大的敌人威胁下，皮姆与朗必须精心策划并执行一场惊天骗局来拯救世界。

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWEDfIhPZdlGcV3jd0vOEhTYOzr00VyRJicibXyp2bbyicNib4sn0ibWfDVFA/0)

"特工卡特"也将现身，她是"美队"的情人。





**新神奇四侠**
上映日期: 2015-08-07(美国)

导演: 乔什·特兰克

主演: 凯特·玛拉 / 迈尔斯·特勒

引进指数：★★★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWiaxa2Yqbjeic9SvAjfOfT0JIVmjy5IlWSq1s3fibIkwFDGwcUiasfMUMeg/0)

福克斯已经正式请来奥斯卡最佳改编剧本奖得主（《美丽心灵》），《我是传奇》和《全民超人》的制片人阿齐瓦·加德曼，担任新一部《神奇四侠》的制片人，由《超能失控》导演约舒华·特兰克执导。

《新神奇四侠》演员阵容将大换血，新出现的大反派会由“鼹鼠人”担任。





**八恶人**
上映日期: 2015-09-25(美国)

导演: 昆汀·塔伦蒂诺

主演: 塞缪尔·杰克逊 / 布鲁斯·邓恩

引进指数：★

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHW27a4OMumBO2Mnf5okmv76aQAHHB7cmeMekaHkA7EeFwS97MMCeyJVQ/0)

今年春天《八恶人》剧本泄露事件之后，昆汀·塔伦蒂诺就将剧本重新改写。针对泄露版剧本的结局部分，昆汀已特别做出彻底改动，这让影片尚未成型就多次成为影迷们讨论的焦点话题。

《八恶人》已经锁定了塞缪尔·杰克逊、蒂姆·罗斯、詹妮弗·杰森·李、布鲁斯·邓恩与迈克尔·马德森等多位实力演员出演，最近确定加入的，还有当红男星查宁·塔图姆。

昆汀这样形容这部新作，“与其说灵感来源于西部片，不如说更多受到《大淘金》、《弗吉尼亚人》、《灌木丛》这几部美剧的影响。我很喜欢这些剧集，你得看很久才会发现角色到底是好是坏。我想为何不拍这样一部电影呢？有很多角色，但每一个是大英雄，只是一群不法之徒。让他们因为暴风雪困在一个房间里，再给他们枪支，看到底会发生些什么。”

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWh74bysnwIKjLTicIzJrngiaIQC8n01zibmo0ZNT60GydUEc1p9F2cPZ4A/0)

塞缪尔·杰克逊调皮自拍《八恶人》现场照，昆汀讲戏手舞足蹈。





**花生漫画大电影**
上映日期: 2015-11-06(美国)

导演: 史蒂夫·马蒂诺 / 保罗·费格

引进指数：★★★★★（已确定于2016年2月引进）

![](http://mmbiz.qpic.cn/mmbiz/CVTVTibpRbeqibQJw7V1l0UJ7iadgVI6kHWJCy148KhhhdrXjayiaJUUKgSpGg5PrYZMTuJ0DDL2oonuQ1zkuTvXibA/0)

《花生豆》漫画由查尔斯·舒尔茨于1950年开始在报上连载，直到2000年才结束，是有史以来由一个人创作的最长的故事，也是最流行和最受欢迎的四格漫画。

这次的动画电影，由查尔斯·舒尔茨的儿子克莱格·舒尔茨担任制片人，他坦言，自己比父亲更加注重这些角色的保护，动画片将非常忠实原作。

本片由《霍顿和无名氏》、《冰河世纪4》导演史蒂夫·马蒂诺执导，北美的上映日期2015年11月6日，也正是《花生豆》漫画诞生65周年之际。
