# 独生子女没那么可怕？

**作者：陈奎孚 贾慧**

**一、引言**


    中国一胎化计划生育政策已经施行了三十多年了，短期效应是积极的，至于长期的影响目前已引起舆论、公众和学界各方的关注，多呈否定态度，所以前两年放开二胎，近期又有议论要全面放开。经济学家多讨论“一胎化”导致了劳动力短缺，GDP后劲乏力。但作为教育工作者的KFC,
 
更关心“单兵”素质，即个人成才，以及成为杰出人才的影响因素。毫无疑问，一胎化对儿童的性格影响巨大，传统影响中老大、老二、老三的性格也不一样，那么它对成才是否有显著影响呢？




    诺贝尔奖自从1901年设立颁发以来，成为许许多多优秀科学家的奋斗目标，也成就了众多影响人类历史的科学发现和成果。诺贝尔奖是公认的和最具权威的自然科学最高荣誉奖, 代表着基础科学研究的最高水平。故而本文选择诺贝尔自然科学奖获得者群体为研究对象，来探究独生子女和排行与成才相关性。这对杰出人才的培养有一定借鉴意义。



**二、统计结果**



       对583位诺贝尔自然科学奖获奖者（1901年至2015年物理学、化学、生理或医学奖获得者）的是否为独生子女、兄妹排行等信息进行了梳理和挖掘，如下表所示
|独生子女及排行情况| | | | |
|----|----|----|----|----|
|类型|人数|比例| | |
|独生子女|193|33.1%| | |
|非独生子女|排行老大|69|170|29.2%|
|非排行老大|86| | | |
|资料中未涉及排行|15| | | |
|资料未查到|220|37.7%| | |


    在583名获奖者中，独生子女个数为193，非独生子女个数为170，资料中并未涉及是否为独生子女信息的个数为220。 独生子女所占比例为33.1%，非独生子女所占比例为29.2%。在已知家庭孩子个数情况的363名获奖者中，独生子女所占比例为53.2%，非独生子女所占比例为46.8%。




    针对非独生子女的170名获奖者，对其是否排行老大也做了统计。其中排行老大的获奖者个数为69，排行非老大的获奖者个数为86，资料中未介绍兄姐妹排行情况的个数为15。在已知排行情况的155名获奖者中，排行老大的获奖者所占比例为44.5%，非排行老大的获奖者所占比例为55.5%。




**三、讨论**

    中国一胎化计划生育政策已经施行了三十多年了，短期效应是积极的，至于长期的影响目前已引起舆论、公众和学界各方的关注，多持否定态度，尤其是经济学家特别强调“一胎化”导致了劳动力短缺。




    作为教育工作者的KFC, 关心学生成才问题。




    我们日常经验感觉是成才与性格有关。由于缺少与兄弟姐妹的竞争和合作，普遍认为现在的学生往往欠缺韧性和合作性。而如今国家放开二孩政策以解决中国人口老龄化问题，传统说法中“老大性格”、“老二性格”目前再次迎来形成环境。但是我们诺贝尔奖获奖者的统计数据表明是否有兄弟姐妹，以及即是否为独生子女，对杰出人才成长的影响没有想象的那样大。




    传统文化中老大性格是厚道、负责任、善于交流、情绪稳定等，因此有领袖的魅力。对于中间出生的孩子来讲，他们既没有受到父母像对老大那样的重视，也没有得到对老幺的宠爱。他们往往要处理上下的人际关系，因而可能具有较强的交往和适应能力。而最小的孩子，往往得到父母更多的宠爱和哥姐的帮助，智力开发得比较早。由于经常受到保护，老幺会表现出淘气或乖巧的特点，但老幺往往缺乏毅力，习惯于求助别人。独生子女像老大也像老幺。




    调查前我们设想，独生子女的家庭相比有兄弟姐妹的家庭可能容易缺乏合作沟通、敢于担当等良好的品质，因此我们也是根据这种设想出发进行的一系列相关的调查。但是调查结果却没有像我们想象中那样，在调查的已知是否独生子女情况的363名获奖者中，独生子女所占比例为53.2%，非独生子女所占比例为46.8%。这个数据乍一看似乎是否为独生子女两类情况是不分伯仲。但是在欧美等西方国家人群独生子女的比例并非50%。比如三个家庭分别有一个、两个和三个孩子，平均有两个孩子(这是非常保守的估计,因为在欧美国家的家庭大多超过两个孩子，更详细数据有待进一步查找)。按照上述假定，独生子女的人群比例只有1/6=16.7%,这个比例远远低于独生子女所占诺贝尔奖的比例53.2%。




    按照类似的假设，老大的人群比例为(1+1+1)/(1+2+3)=50%，这与排行老大的获奖者所占比例为44.5%比较接近。因此排行差异对诺贝尔奖的比例影响不是很明显。




    排行为什么没有我们想象的差异那么大？在查阅资料过程中我们发现，在这杰出的583个科学家中每个人的经历都是不可复制的，在读他们每个人的自传时，我们能清晰地感受到文字的对面是一个个博学而语重心长的人们，在用亲切的语言向我们来自世界各地的读者讲述自己平凡的，而又极不平凡的一生。成就都不是一蹴而就的，这些杰出的人们并不是仅仅凭借非凡的天资便轻松登上科学的巅峰，每个人都有着丰富的经历，并在浮沉中不迷失自我，始终知道自己所爱是什么，这才创造出一个又一个的奇迹，推动着人类科学一步步地向前发展。所以是否为独生子女并不是一个人成就多少的决定性因素，一个人的学识和经历才是成就多少的决定性因素。至少从在西方国家的调查结果显示，由于家庭或者学校的教育，他们在学到知识的同时在合作与担当等良好品质方面并未缺失。




    当然上述统计结果对国内当下参考意义仍需慎重。这有多方面的因素。首先，统计数据只有500多人，这与全球的74亿人口相比，是非常非常小的一个群体。其次，即便是在自然科学领域也有很多杰出人才没有获得诺贝尔奖，更何况工程技术和人文领域的杰出人才呢(本文未统计文学、经济学和和平奖)。第三，国内几乎全是独生子女，因此整个大环境是独生子女唯我独尊，而国外的多子女家庭为主导，这种环境会对个别独生子女的性格产生烙印(本文未统计屠呦呦)。第四，已获诺奖人群的儿童时代环境与国内独生子女儿童环境差异很大。




    总之，就诺贝尔奖获奖者的统计数据，是否有兄弟姐妹，以及是否为独生子女，对杰出人才成长的影响没有想象的那样大。这对中国的教育有一定参考意义，但不是绝对的。
