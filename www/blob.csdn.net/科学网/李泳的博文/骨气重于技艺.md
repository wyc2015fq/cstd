# 骨气重于技艺

**——旧札新钞 (31)**





@  李贽批判向秀《思旧赋》（鲁迅在《为了忘却的纪念》里提到过），批他“只说康（嵇康）高才妙技而已”。“夫康之才之技，亦今古所有，但其人品气骨，则古今所稀也。”他还骂向秀在“竹林七贤”里“为最无骨头者”。 



@  看《中国文明之光》，略记裘锡珪先生说甲骨文的几点： 

> 
% 甲骨文并不全是殷墟的，后来发现了时代更早和更晚的甲骨文字；甲骨文也不全是卜辞，还有一般的记事的。  

% 甲骨文并不都先写然后再刻。为方便刻划，甲骨文在笔画上有许多简化，很多圆的都化成方的。  

% 甲骨文不能说是最古的文字。但就现在发现的更古老的字符还说不清是属于哪一种文字的先驱，未必就是汉字。  

% 甲骨文是最早的有系统的文字，但也有着萌芽文字的一些痕迹……


@  Dirac在一篇演讲（见《物理学的方向》）里说，“电子始终有伴随着围绕它的库伦场。每当发射一个电子，便同时发射围绕它的库伦场，这好像给电子穿上一种衣服。”引力场却不是粒子的衣服，而是它游泳的池塘。
