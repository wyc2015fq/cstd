# [量化投资_常见的绩效指标计算公式](https://www.cnblogs.com/noah0532/p/8655116.html)





在证券投资下的一些常见绩效指标的计算：

0　　期货常识：

**最小变动价位**：1元/吨，2元/吨，0.05元/克，0.2点等等。这些是表示观察价格每次按照多少数值进行变动

　　　　　　比如：螺纹钢3370 3369，这样一个点一个点的变动，以此类推。

**交易单位**：10吨/手，....等等。这些是表示你买了1手等于买了10吨。以此类推。

那么，对应每波动一个点赚多少钱呢？

**以沥青为例子，交易单位等于10吨/手，最小变动价位为2元/吨，也就是最小变动一次的盈利为20元，对应每波动一个点赚20/2=10元。**

**以铁矿石为例子，交易单位是100吨/手，最小变动价位为0.5元/吨，那么最小变动一次的盈利为50元，对应每波动一个点赚50/0.5=100元。**

**以PVC为例子，交易单位是5吨/手，最小变动价位为5元/吨，那么最小变动一次的盈利为25元，对应每波动一个点赚25/5=5元。**

**以铁矿石为例子，交易单位是100吨/手，最小变动价位为0.5元/吨，那么最小变动一次的盈利为50元。我现在赚了15.5个点。总共盈利 = 15.5*100 = 1550元。**

1　　**交易总手数**：**盈利手数 + 亏损手数**

备注：开仓然后平仓算交易一次。



2　　**胜率**：盈利的手数 / （盈利手数+亏损手数）* 100 = 胜率(%)

备注：胜率是带率的，因此要有乘以100过程。胜率也叫成功率，也叫盈利比率。

举例：比如一段时间的交易，做多交易：（平仓价格-开仓价格）*  交易单位 - 交易成本 * 手数 > 0表示盈利，反之表示亏损。

　　　　　　　　　　　　    做空交易：（开仓价格-平仓价格）* 交易单位 - 交易成本  * 手数  >0 表示盈利，反之表示亏损。

　　　亏损了4次，盈利了6次，因此胜率为：6 / （6+4） * 100 =  60% 的胜率。



3　　**净利润**：**总获利 - 交易成本**

备注：净利润是表示一个时间段内，总共获得多少扣除交易成本后的实得。

举例：比如一段时间的交易，做多交易：（平仓价格-开仓价格）*  交易单位 - 交易成本  * 手数 > 0表示盈利，反之表示亏损；如果大于0记录为正值，小于0记录为负值。

　　　　　　　　　　　　    做空交易：（开仓价格-平仓价格）* 交易单位 - 交易成本  * 手数 >0 表示盈利，反之表示亏损；如果大于0记录为正值，小于0记录为负值。

　　所以：净利润 = 做多交易累加 + 做空交易累加。

%【公式】交易成本=建仓价格*交易单位*交易手数*成本比率+平仓价格*交易单位*交易手数*成本比率
    CostSeries(i)=OpenPosPrice(i)*TradingUnits*Lots*TradingCost+ClosePosPrice(i)*TradingUnits*Lots*TradingCost;
 
%% 5.2 *****净利*****
    %【公式】多头净利=(平仓价格-建仓价格)*交易单位*手数-建仓价格*交易单位*手数*成本比率-平仓价格*交易单位*手数*成本比率
    if Type(i)==1
        NetMargin(i)=(ClosePosPrice(i)-OpenPosPrice(i))*TradingUnits*Lots-CostSeries(i);
    end
    %【公式】空头净利=(建仓价格-平仓价格)*交易单位*手数-建仓价格*交易单位*手数*成本比率-平仓价格*交易单位*手数*成本比率
    if Type(i)==-1
        NetMargin(i)=(OpenPosPrice(i)-ClosePosPrice(i))*TradingUnits*Lots-CostSeries(i);
    end

5　　**夏普比率**：**S = （r - rf）/ sigma**

备注：rf ：为无风险收益率，r：为收益率，sigma：为波动率。逻辑为投资绩效的关键，在于风险和收益的平衡。夏普比率也表示为超额收益，也叫阿尔法收益。

夏普比率（1）

　　如果某个基金的年化收益率15%，波动率8%，无风险收益率为3%；那么夏普比率为=1.5。

夏普比率（2）

　　长期投资股指　　夏普比率0.3-0.5　　年度亏损概率>30%

　　对冲基金（靠谱）　　夏普比率>1.0　　年度亏损概率<16%

　　对冲基金（常青树）　　夏普比率>2.0　　年度亏损概率<2.5%

举例：由于夏普比率是没有起点的算法。因此常用的夏普比率是按照日来算，按照周算，按照月来算的三种方式。
