3.itoa 将整型数组转化为char 然后string 直接等于char ，用string有很多好处

```
int main()
{
    int n;
    cin >> n;
    int a[n];
    string s ;
    char s1[n];
    for(int i=0;i<n;i++)
    {
        cin >> a[i];
        itoa(a[i],s1+i,10);//1.数字2.字符数组地址3.转换进制
    }
    s = s1;
    cout << s;
    return 0;
}
```

 4.atoi（char）相应的返回int 功能是将字符数组转变为整型数字。

常用技巧：

```
int n = atoi(itoa(num, str, 2));   //先把num转换为二进制的字符串，再把该字符串转换为整数
```
