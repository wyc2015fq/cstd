有N个学生去食堂，可教官规定：必须2人或3人组成一组，求有多少种不同分组的方法。






输入描述 Input Description


一个数，N




输出描述 Output Description


一个数，即答案。




样例输入 Sample Input


6




样例输出 Sample Output


2

```
#include<cstdio>

long long a[10001];

int main()

{

int n,i,x=2,y=3;

scanf("%d",&n);a[2]=a[3]=1;

for(i=4;i<=n;i++)a[i]=a[i-2]+a[i-3];

printf("%ld\n",a[n]);

}
```
