## 阶乘因式分解（一）

时间限制：3000 ms  |  内存限制：65535 KB

难度：2



- 描述

********给定两个数m,n,其中m是一个素数。********

**将n（0<=n<=10000）的阶乘分解质因数，求其中有多少个m。**


- 输入
第一行是一个整数s（0<s<=100)，表示测试数据的组数
随后的s行, 每行有两个整数n，m。- 输出
输出m的个数。- 样例输入
2
100 5
16 2
- 样例输出
24
15


```
#include <bits/stdc++.h>
using namespace std;
typedef long long  ll;
const int INF = 0x3f3f3f3f;
const int moder = 10000;
const int maxn = 10005;

int main()
{
    int t;
    scanf("%d",&t);
    while(t--)
    {
        int n,m;
        cin >> n >> m;
        int res = 0;
        while(n)
        {
            res += n/m;
            n = n/m;
        }
        cout << res << endl;
    }
    return 0;
}
```
