字符串格式化符号含义

|符   号|说     明|
|----|----|
|%c|格式化字符及其ASCII码|
|%s|格式化字符串|
|%d|格式化整数|
|%o|格式化无符号八进制数|
|%x|格式化无符号十六进制数|
|%X|格式化无符号十六进制数（大写）|
|%f|格式化定点数，可指定小数点后的精度|
|%e|用科学计数法格式化定点数|
|%E|作用同%e，用科学计数法格式化定点数|
|%g|根据值的大小决定使用%f活%e|
|%G|作用同%g，根据值的大小决定使用%f或者%E|



格式化操作符辅助指令

|符   号|说     明|
|----|----|
|m.n|m是显示的最小总宽度，n是小数点后的位数|
|-|用于左对齐|
|+|在正数前面显示加号（+）|
|#|在八进制数前面显示 '0o'，在十六进制数前面显示 '0x' 或 '0X'|
|0|显示的数字前面填充 '0' 取代空格|

|符   号|说     明|
|----|----|
|\'|单引号|
|\"|双引号|

|\b|退格符|

|\t|横向制表符（TAB）|
|\v|纵向制表符|
|\r|回车符|

|\x|十六进制数代表的字符|

 ####################################################################

1、　　%c（格式化字符及其ASCII码）

print '%c' % 97　　　　#输出a  因为a的ascii码是 ：97

print '%c' % (97,98,99)　　#输出a,b,c

2、　　%s（字符）

print '%s' % 'i love you '　　#输出i love you

3、　　%d（数字）

print '%d+%d=%d' % (4,5,4+5)　　#输出4+5=9

4、　　%o（八进制）

print '%o' % 10　　　　#输出12

5、　　%x（十六进制（小写））

print '%X' % 10　　　　#输出a

6、　　%X（十六进制（大写））

print '%X' % 10　　　　#输出A

7、　　%f（定点数）

print '%f' % 45.564　　#输出45.564000，因为默认精确到小数点后六位

8、　　%e（科学计数法）

print '%e' % 45.564　　#输出4.556400e+01，一样同上，小数点后六位

9、　　%g(自动选择用%e还是%f)

print '%g'% 45.564　　　#输出45.564

####################################################################

格式化操作辅助符号

1、　　m.n（决定这个数的点前点后的数字为几个）

print '%5.1f' % 45.564　　　　　　#输出“  45.6”

print '%.2e' % 45.564　　　　#输出4.56e+01

print '%10d' % 45　　　　　　#输出“　　　　45”，因为他会在前面占用8个空位

2、　　-（左对齐）

print '%-10d' % 45　　　　　　#输出“　　　　45” 

3、　　+（在正数前面加+）

print '%+d' % 55　　　　　　#输出+55

print '%+d' % -55　　　　　　#输出-55　　就不会显示+了

4、　　#（在八进制与十六进制前面显示一个东西）

print '%#o' % 10　　　　#输出012 这是八进制的10

print '%#x' % 10　　　　#输出0xa 这是十六进制的10

print '%#d' % 10　　　　#输出10　这是10

5、　　0（看栗子）

print '%010d' % 5　　　　#输出：'0000000005'

print '%-010d' % 5　　　　#输出：'5               '





举栗子

print "{0} love {1} {2}".format("I","you","forever")     #会输出i love you forever  #这里使用的是位置参数

print "{a} love {b} {c}".format(a="I",b="you",c="forever")   #也会输出i love you forever　#这里使用关键字参数

print "{0} love {b} {c}".format"I",b="you",c="forever")   　　#位置参数与关键字参数混用 

　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　#但位置参数一定要在关键字前

下面有点不一样

print "{{0}}".format("你输出不出来")　　　　　　#大括号被大括号转义了  所以会输出{0} 不会输出那些中文

下面有点难度的

print '{0:.1f}{1}'.format(22.99,'Gb')　　#输出23.0Gb。“：” 这个冒号是格式化符号的开始 ； 

　　　　　　　　　　　　　　　　　       　　#“.1”这个点一的意思，四舍五入，保存一位有效数字

　　　　　　　　　　　　　　　　　　　　　　#　　”f”　　是保存的形式  这里的f是定点数
