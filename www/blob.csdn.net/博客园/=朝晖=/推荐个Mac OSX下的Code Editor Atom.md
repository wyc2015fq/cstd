# [推荐个Mac OSX下的Code Editor:Atom](https://www.cnblogs.com/dhcn/p/7120951.html)
       首先只是当Editor用，不是整成IDE级。
        先说几个大家耳熟能详的：
         1、Sublime，Sublime在Mac下的安装并不完全，CLI启动需要自己ln个链接。还有一些其他原因，比如Packages安装不太方便，有些Package配置起来也很麻烦，我是用惯Notepad++的人，用起来UI交互觉得不太顺手。最后，这个东西其实是收费软件
         2、Textmate，似乎直接就得花钱买。
         3、Vim，我只能玩玩CLI下的vim。
         相对而言Atom的插件安装和配置都很简单，正好适合我这种一般用editor查看代码或者临时编辑的人。
