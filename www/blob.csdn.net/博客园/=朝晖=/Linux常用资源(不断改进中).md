解决 BADSIG 40976EAF437D05B5 in Ubuntu :，执行以下命令（注意：这是一个指令，不是两个指令，宽度的问题，显示两行了）
```
sudo aptitude -o Acquire::http::No-Cache=True -o Acquire::BrokenProxy=true update
```
