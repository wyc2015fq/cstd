# [Visual Studio Community 2013，功能完整，免费使用](https://www.cnblogs.com/dhcn/p/7294636.html)
http://www.infoq.com/cn/news/2014/11/VSC2013
微软刚刚宣布了.NET平台的开源计划，与此同时，它还推出了源自Visual Studio Professional的一个新版本开发平台，名为Visual Studio Community 2013（VSC2013）。这一版本[现在已经可以下载](http://www.visualstudio.com/en-us/products/visual-studio-community-vs)，并且在许多常见的应用场景下，该版本对用户都是免费的。
这里所说的“免费”，是指个人用户、非盈利用户与非企业级组织可以任意使用VSC2013，而不会产生任何附加免费。之所以微软能够迅速推出VSC2013，是因为它本质上是由VS2013 Professional拷贝而来的一个不同名称的工具。这意味着这一工具为用户提供了比之前发布的Visual Studio Express版本更多的功能。使用微软的这一旗舰产品的用户不仅能够开发Windows平台的应用，也包括针对Android、iOS和Unity的应用。
正如Microsoft OneCode团队所说，VSC2013能够[支持](http://blogs.msdn.com/b/onecode/archive/2014/11/12/differences-between-visual-studio-community-edition-and-express-editions.aspx)Visual Studio Gallery中的插件，与不支持扩展能力的Visual Studio Express相比，新版本显然更具实用价值。而对于企业应用来说，仍然[可以继续使用](http://blogs.msdn.com/b/onecode/archive/2014/11/14/differences-between-visual-studio-2013-community-edition-and-visual-studio-2013-pro-edition.aspx)Visual Studio Professional版本。
用户可以通过常见的[web安装包](http://go.microsoft.com/?linkid=9863608)直接安装VSC2013，或是下载完整的[DVD ISO镜像](http://go.microsoft.com/?linkid=9863609)，其中已经包含了Update 4的内容。
