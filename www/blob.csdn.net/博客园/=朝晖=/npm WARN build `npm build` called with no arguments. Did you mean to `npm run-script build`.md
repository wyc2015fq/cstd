# [npm WARN build `npm build` called with no arguments. Did you mean to `npm run-script build`?](https://www.cnblogs.com/dhcn/p/9842116.html)
    跑npm build结果如下：
```
npm WARN build `npm build` called with no arguments. Did you mean to `npm run-script build`?
```
  把指令改成npm run build即可：
```
npm run build
```
