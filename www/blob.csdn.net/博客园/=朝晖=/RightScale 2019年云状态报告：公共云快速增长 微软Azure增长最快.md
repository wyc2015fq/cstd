# [RightScale 2019年云状态报告：公共云快速增长 微软Azure增长最快](https://www.cnblogs.com/dhcn/p/10497267.html)
https://www.rightscale.com/
全球云管理服务厂商RightScale发布了年度云状态报告，今年报告的十大主要内容包括：企业在多云平台上投入巨资、公共云继续快速增长，但是私有云却表现不佳。而微软Azure是增长最快的公共云，虽然亚马逊的AWS仍然占比最大。
每年RightScale都会对企业和中小型企业进行调查，分析云的采用情况。此前，RightScale已经被Flexera收购，今年的报告也是该公司被收购后的第一份报告。
今年，Flexera调查了超过123691名用户，他们在广泛的跨部门组织中采用云计算。其中，只有21％的受访者是RightScale的用户。在786名受访者中，456名来自1000多名员工的大型组织，而330名受访者来自中小型企业。
以下是今年的报告十大主要内容：
1.企业在多云平台上投入巨资
企业用户继续投资多个云平台，包括公共云和私有云。84％的受访者表示，使用了4个以上的云平台，10%表示只用了公共云平台，而3%是私有云，剩下3%表示没有任何计划。
使用4个以上云平台的受访者中，58%是私有和公共云混合使用，17%使用多个公共云平台，而9%使用多个私有云平台。

与2018年相比，混合云的采用率增加了7％，而使用多个公共云平台则增加4%。

2.公共云继续快速增长
23％的受访者每年在公共云上花费至少240万美元，而33％的受访者每年花费至少120万美元。
在大企业中，云支出更高，38％企业每年花费超过240万美元的，一半（50％）表示每年花费超过120万美元。

3.成本优化是企业面临的最大挑战
云用户（包括大企业和中小企业）专注于优化成本和云使用。成本优化（84％）是大企业最关心的问题，其次是监管（84%）、安全（81%）、合规（79%）、缺少资源/专业人才（79%）、多云管理（76%）和云转移（73%）。
而对于中小企业来说，缺少资源/专业人才（77%）是首要关注问题，其他包括成本优化（69%）、监管（73%）、安全（72%）、合规（71%）、多云管理（59%）和云转移（60%）。
大企业和中小企业在云服务方面的挑战

4.大企业的容器工具采用率呈上升趋势
Docker的采用率从2018年的49％增加到57％，这项技术正在成为主流。Kubernetes（K8s）的使用率从27％增加到48％。而亚马逊AWS ECS/EKS的采用率持平，为44%，微软Azure Container Service的采用率从20%上升到28%，而谷歌的Google Container Service (GKE)只上升了1%至15%。

5. Ansible是首选的DevOps工具
Red Hat的Ansible是最受欢迎的DevOps工具。在所有受访者中，Ansible的采用率为41％，其次是Chef和Puppet，采用率均为37％。有趣的是，Terraform自去年以来采用率增幅最大增长了55％，从20％增加到31％。

6.微软Azure是增长最快的公共云
Azure采用率从45％增加到52％，逐渐接近AWS。Azure采用率现已相当于AWS采用率的85％，高于去年的70％。而AWS的使用率有所下降，从2018年的64%小幅下滑至61%。
之后是谷歌云，今年使用率19%（2018年18%），其次是VMware Cloud on AWS（12%）、IBM Cloud（12%）、甲骨文的云服务（11%）以及阿里云（3%）。

7. AWS增长几乎持平，但工作负载数量正在快速增长
AWS超过100个虚拟机的用户比例从2018年的31％增加到2019年的33％。
![](http://5b0988e595225.cdn.sohucs.com/images/20190304/4f48c6d2a76e4ec2b668fb222a8722e1.jpeg)
8.托管数据库服务是最常用的公共云产品
托管数据库服务是最常用的公共云产品，几乎每个应用程序都依赖事务性关系数据库。大多数参与者（53％）将RDBMS评选为最佳公共云服务，其中包括托管数据库服务和在虚拟机中运行的自托管工作负载。
9.无服务器和流处理平台是增长最快的服务
无服务器（36%）是公共云中增长最快的服务之一。而如Azure Event Hub、Amazon Kinesis和Google Cloud Dataflow等流处理服务（30%）在云用户中越来越受欢迎。两者的增幅都是50%。
除了这两项外，其他增长超过30%的服务包括机器学习（18%增至26%）、容器工具即服务（26%增至37%）、IoT（15%增至21%）、数据储存（29%增至40%）和批量处理（26%增至36%）。
![](http://5b0988e595225.cdn.sohucs.com/images/20190304/c9dffe73d3234105b5b78bcf00f2ecc2.jpeg)
10.私有云表现不佳
虽然VMware vSphere继续以50％的使用率领先，但与去年几乎持平。
其次是Open Stack（28%）、VMware vCloud Director（27%）、微软的Microsoft System Center（25%）、Bare-metal Cloud（24%）、Microsoft Azure Stack（22%）、CloudStack（14%）和AWS Outposts（12%）。而预计Azure Stack和AWS Outposts将改变私有云格局。

随着经济的发展，企业越来越重视云服务，AWS本身就占据了一定的份额，但微软从各方面来看似乎增长势头也很足，而谷歌云也在努力追赶，云服务这块大饼，技术巨头在2019年必然都不会轻松放弃，竞争也会越来越激烈
