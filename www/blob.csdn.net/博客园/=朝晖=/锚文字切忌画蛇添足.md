# [锚文字切忌画蛇添足](https://www.cnblogs.com/dhcn/p/7115395.html)
 最近升级bootstrap版本，把pager由 
```
<ul class="pager">   
   <li><a href="#">Previous</a></li>   
   <li><a href="#">Next</a></li>   
</ul>
```
换成 
```
<ul class="pager">   
   <li class="previous"><a href="#">← Older</a></li>   
   <li class="next"><a href="#">Newer →</a></li>   
</ul>
```
   结果偶然发现导致Bing索引量爆减,分析了一下，似乎是因为专给搜索引擎准备的archives页面的换页链接好像被Bing无视了，可能是实体字符导致语义分析混乱。 
    百度影响不大，Google最近看不了。
