# [电脑安装双系统可能出现的问题](https://www.cnblogs.com/xcw0754/p/5135323.html)
** 如果你遇到如下问题，再继续往下看：**
1）双系统引导不了多个系统？
2）ubuntu / win7 的系统引导被破坏，或者覆盖了？
3）只剩下一个启动选项，或者直接就登录win7 / ubuntu了？ 
**先装的win7怎么办？ **
一般情况下，win7是已经在用的，那么再装ubuntu时，只要选择另外一个盘（非c盘）安装即可，启动时会默认调用ubuntu中的grub来选择启动项，一般情况下，win7会出现在最下边的一项。一般默认是第一个项为默认项，长时间未选择时就默认启动该项。
如果想改默认的启动项呢？也是可以的，先用以root打开/boot/grub/grub.cfg，这里是以ubuntu14.04为准的，不过一般这个文件也叫grub.xxx，后缀不清楚。打开后里面有一项需要修改。
```
1 ### BEGIN /etc/grub.d/00_header ###
 2 if [ -s $prefix/grubenv ]; then
 3   set have_grubenv=true
 4   load_env
 5 fi
 6 if [ "${next_entry}" ] ; then
 7    set default="${next_entry}"
 8    set next_entry=
 9    save_env next_entry
10    set boot_once=true
11 else
12 # 这里可以设置默认的启动项，0为ubuntu，一般情况下，4为win7。
13    set default="0"
14 fi
```
如果要修改选择系统时那个倒数时间，也是很容易的，配置文件中总共才只有几个数字，相信你可以找到的。
**先装的ubuntu怎么办？**
很多情况下，我们都会重装win7，这时直接就默认启动win7的引导项了，你会发现，它并不自动引导ubuntu。又或者你直接就ubuntu，再装的win7，也可能出现这样的情况。
解决办法是，装一个easyBCD软件，启动后选择"添加新条目"，也就是下图的“Add New Entry”这一项。接着按照图示来选择就行了，一共4步，完成后重启试试吧! 图来自[这里](http://www.crazyant.net/781.html)。
