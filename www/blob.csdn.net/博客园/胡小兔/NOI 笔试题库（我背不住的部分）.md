为什么C++选手要会编译Pascall啊！为什么Emacs选手要会使用Vim啊！

- Linux 中为文件改名使用的命令是：mv <旧文件名> <新文件名>
- 在Linux 中删除当前目录下的test 目录的命令是：rm -r test
- 使用高级语言编写的程序称之为：源程序
- Linux 系统中杀死名为test 的后台进程的命令是：killall test
- Linux 系统中可以查看隐藏文件的命令是：ls -a
- vim 编辑器中，强制退出不保存修改应当输入：:q!
- vim 编辑器中，强制退出并保存修改可输入以下三种命令之一：:wq、ZZ、:x
- vim 编辑器中，定位到文件中第12 行应当输入：:12
- vim 编辑器中，在文件中查找字符串“12”应当输入：/12
- 使用gcc 编译C 程序时，只编译生成目标文件的命令行选项是：-c
- 使用gcc 编译C 程序时，指定输出文件名的命令行选项是：-o
- 如果C 程序中使用了math.h 中的函数，在编译时需要加入选项：-lm
- 在Linux 的各个虚拟控制台间切换的快捷键是：Ctrl+Alt+F[1-6]
- 在NOI Linux 中，从字符控制台切换回桌面环境使用的快捷键是：Ctrl+Alt+F7
- 在NOI Linux 中默认使用的Shell 是：bash
- 在Linux 中查看当前系统中的进程使用的命令是：ps
- 在Linux 中查看进程的CPU 利用率使用的命令是：ps
- 使用Reset 键强行重新启动计算机可能会对系统造成的后果是：文件系统损坏
- 在Linux 系统中，用于查看文件的大小的命令是：ls -l
- 评测系统中对程序源文件大小的限制是：小于100KB
- 如无另行说明，评测系统中对程序使用内存的限制是：以硬件资源为限
- 终止一个失去响应的进程（$pid 代表进程号）的命令是：kill $pid
- 为程序my.c创建一个备份myc.bak时，使用的命令是：cp my.c myc.bak
- 在Anjuta 中调试程序，继续执行的快捷键是：F4
- 在Lazarus 中开始运行程序的快捷键是：F9
- 在Anjuta 中调试程序，单步运行(Step over)的快捷键是：F6
- 在Lazarus 中调试程序，单步运行(Step over)的快捷键是：F8
- 如果需要在Lazarus 中使用单步调试，则：在Environment->Debugger Options 中配置
- Pascal 中integer 和long integer 类型的长度和编译选项是否有关系：有关系
- NOI 考试对C++语言模板的使用有限制吗？没有
- NOI 考试对PASCAL 语言的使用有限制吗？有
- Lazarus 是可以支持多窗口编辑的IDE 吗？是
- Anjuta 是可以支持多窗口编辑的IDE 吗？是
- 一个完整的计算机系统应包括硬件系统和软件系统。
- 目前微型计算机中采用的逻辑组件是大规模和超大规模集成电路。
- 软件与程序的区别是软件是程序以及开发、使用和维护所需要的所有文档的总称,而程序是

软件的一部分。
- 在微机中，bit 的中文含义是二进制位。
- 计算机所能辨认的最小信息单位是位。
- ASCII 的含义是美国信息交换标准代码。
- 在计算机术语中经常用RAM 表示随机存取存储器。
- 计算机内所有的信息都是以二进制数码形式表示的。
- Linux 中查看当前路径使用的命令是pwd 。
- 在Linux 下建立目录使用的命令是mkdir 。
