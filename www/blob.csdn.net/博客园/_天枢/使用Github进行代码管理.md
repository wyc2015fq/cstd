准备：
Github地址:[https://github.com](https://github.com)
注册Github帐号
**一、新建仓库**
![](https://images2017.cnblogs.com/blog/1149221/201709/1149221-20170925133532167-18242537.png)
 进入如下界面：
![](https://images2017.cnblogs.com/blog/1149221/201709/1149221-20170925133834057-1110748649.png)
![](https://images2017.cnblogs.com/blog/1149221/201709/1149221-20170925134022089-1749518024.png)
**到这里仓库就创建完成了。**
**二、安装Github-window-desktop**
安装github客户端
下载地址:[https://github-for-windows.en.softonic.com/](https://github-for-windows.en.softonic.com/)
安装完成后，在客户端，用github帐号登录
![](https://images2017.cnblogs.com/blog/1149221/201709/1149221-20170925134440542-1337848955.png)
安装完成，点击file，会显示几个菜单项
**New repository 新建仓库**
**Add local repository 本地仓库上传**
**clone repository   github上仓库关联到本地**
我们这里选择 clone repository，将已创建好的仓库关联在本地创建。
![](https://images2017.cnblogs.com/blog/1149221/201709/1149221-20170925135928714-420083298.png)
输入仓库地址和要下载到本地的位置
点Clone会在本地创建一个文件夹，我们可以把自己的python项目创建在这个文件夹下，之后可以更新到git hub
![](https://images2017.cnblogs.com/blog/1149221/201709/1149221-20170925140217964-339406606.png)
摘要和描述要填写，要不然提交是灰色
 点Commit to master提交更新到github
 在github上查看，已经更新了
![](https://images2017.cnblogs.com/blog/1149221/201709/1149221-20170925140726651-2053627413.png)
