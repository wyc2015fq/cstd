可能会遇到如下的错误：

```
Connection: keep-alive
Date: Sat, 24 Nov 2018 00:58:30 GMT
Content-Type: application/json; encoding=utf-8
Content-Length: 71
{
    "errcode": 40016, 
    "errmsg": "invalid button size hint: [dNjLEa01101961]"
}
```
