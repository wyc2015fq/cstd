# [AWS Add Key Pair to Elastic Beanstalk Instance 给EB实例加密钥](https://www.cnblogs.com/grandyang/p/10049160.html)







Go to **Elastic Beanstalk** -> **Configuration** -> **Security** -> **Virtual machine permissions** -> **EC2 key pair**.

Choose your **key pair.** If you do not have one,

Then go to **EC2 Dashboard** -> **NETWORK & SECURITY** -> **Key Pairs** -> **Create Key Pair**.
