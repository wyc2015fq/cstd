# [验证LeetCode Surrounded Regions 包围区域的DFS方法](https://www.cnblogs.com/grandyang/p/4759189.html)







在LeetCode中的[Surrounded Regions 包围区域](http://www.cnblogs.com/grandyang/p/4555831.html)这道题中，我们发现用DFS方法中的最后一个条件必须是j > 1，如下面的红色字体所示，如果写成j > 0的话无法通过OJ，一直百思不得其解其中的原因，直到有网友告诉我说他验证了最后一个大集合在本地机子上可以通过，那么我也来验证看看吧。



```
class Solution {
public:
    void solve(vector<vector<char> >& board) {
        for (int i = 0; i < board.size(); ++i) {
            for (int j = 0; j < board[i].size(); ++j) {
                if ((i == 0 || i == board.size() - 1 || j == 0 || j == board[i].size() - 1) && board[i][j] == 'O')
                    solveDFS(board, i, j);
            }
        }
        for (int i = 0; i < board.size(); ++i) {
            for (int j = 0; j < board[i].size(); ++j) {
                if (board[i][j] == 'O') board[i][j] = 'X';
                if (board[i][j] == '$') board[i][j] = 'O';
            }
        }
    }
    void solveDFS(vector<vector<char> > &board, int i, int j) {
        if (board[i][j] == 'O') {
            board[i][j] = '$';
            if (i > 0 && board[i - 1][j] == 'O') 
                solveDFS(board, i - 1, j);
            if (j < board[i].size() - 1 && board[i][j + 1] == 'O') 
                solveDFS(board, i, j + 1);
            if (i < board.size() - 1 && board[i + 1][j] == 'O') 
                solveDFS(board, i + 1, j);
            if (**j > 1** && board[i][j - 1] == 'O') 
                solveDFS(board, i, j - 1);
        }
    }
};
```



验证的方法无非就是两种方法都算出个结果，然后比较两者是否相同，难点就是在于如何将超大数据集合存入二维vector<vector<char>>中，这里我先将超大集合存入vector<string>中，然后再写了个程序将其转化为vector<vector<char>> 运算，最后再将结果保存出来，验证代码如下所示：

vector<vector<char> > convert(vector<string> v) {
    vector<vector<char> > res;
    for (int i = 0; i < v.size(); ++i) {
        vector<char> tmp;
        for (int j = 0; j < v[i].size(); ++j) {
            tmp.push_back(v[i][j]);
        }
        res.push_back(tmp);
    }
    return res;
}

vector<string> convert2(vector<vector<char> > v) {
    vector<string> res;
    for (int i = 0; i < v.size(); ++i) {
        string tmp(v[i].size(), ' ');
        for (int j = 0; j < v[i].size(); ++j) {
            tmp[j] = v[i][j];
        }
        res.push_back(tmp);
    }
    return res;
}
