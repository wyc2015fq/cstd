# [用神经网络指挥坦克扫雷(Java实现)](https://www.cnblogs.com/zhangchaoyang/articles/2218771.html)





想法来自于《游戏编程中的人工智能》，坦克扫雷的相关介绍[zzwu老兄的博客](http://blog.csdn.net/zzwu/article/details/575108)上有，原版是VC++写的，我重写了个Java版。

初始，在随机的位置上布置地雷和坦克，未扫的地雷是红色，扫过的是绿色。

![](https://pic002.cnblogs.com/images/2011/103496/2011102011520383.png)

单击游戏界面，则游戏暂停。再次单击则游戏继续。

![](https://pic002.cnblogs.com/images/2011/103496/2011102015095127.png)

Game over后报告本次游戏所用的时间。

可以点击这里下载[我的源代码](http://files.cnblogs.com/zhangchaoyang/ANNs.zip)。
