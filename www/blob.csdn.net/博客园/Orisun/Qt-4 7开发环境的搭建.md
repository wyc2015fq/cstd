# [Qt-4.7开发环境的搭建](https://www.cnblogs.com/zhangchaoyang/articles/1984198.html)





依然是在Ubuntu10.04下。另外注意现在已不是Qtopia时代，现在的qt一次编写，到处运行，下图中Qt libraries 4.7.2 for Linux/X11和Qt libraries 4.7.2 for embedded Linux两个包的内容是完全一样的。

![](https://pic002.cnblogs.com/images/2011/103496/2011031421121836.png)

1.上图的这个网址是http://qt.nokia.com/downloads/，我们把Qt SDK for Linux/X11 32-bit**这个包下载下来，使其具有可执行权限，再执行安装：

# chmod u+x qt-sdk-linux-x86-opensource-2010.05.1.bin

# ./qt-sdk-linux-x86-opensource-2010.05.1.bin　　　　　　

以root身份执行安装是因为待会儿我们把它装在/opt下

接下来就是图形化的安装界面了。

2.在.bashrc中添加环境变量

#Qt-4.7.2
export QTDIR=/opt/qtsdk-2010.05/qt
export PATH=$QTDIR/bin:$PATH
export LD_LIBRARY_PATH=$QTDIR/lib:$LD_LIBRARY_PATH

然后source一下

3.安装依赖的工具包

sudo apt-get install libglib2.0-dev libSM-dev libxrender-dev libfontconfig1-dev libxext-dev libglui-dev

4.安装编译工具

sudo apt-get install g++ make cmake pkg-config

5.打开QtCreator

Tools==>Options，手动添加一个Qt Version

![](https://pic002.cnblogs.com/images/2011/103496/2011031421592522.png)

然后switch to project mode

![](https://pic002.cnblogs.com/images/2011/103496/2011031422014377.png)

6.卸载

进入/opt/qtsdk-2010.05/bin，运行uninstall文件
