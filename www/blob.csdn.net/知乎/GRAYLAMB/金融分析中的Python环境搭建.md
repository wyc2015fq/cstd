# 金融分析中的Python环境搭建 - 知乎
# 

首先安装Anaconda，Anaconda下载地址如下：
[Download Anaconda now!](https://link.zhihu.com/?target=https%3A//www.continuum.io/downloads)
我的环境时OS X python2.7环境，直接找到匹配的版本下载链接：
![](https://pic3.zhimg.com/67123ca77c53bb6f5bbcf4d5d6e2c69a_b.png)
按照安装说明，通过command－line执行安装命令（个人比较喜欢命令行方式）

$ bash Anaconda2-2.4.1-MacOSX-x86_64.sh

这时候会提示安装欢迎界面：

Welcome to Anaconda2 2.4.1 (by Continuum Analytics, Inc.)

In order to continue the installation process, please review the license

agreement.

Please, press ENTER to continue

>>> 

敲Enter后，会展示Anaconda的协议内容：

================

Anaconda License

================

Copyright 2015, Continuum Analytics, Inc.

All rights reserved under the 3-clause BSD License:

Redistribution and use in source and binary forms, with or without

modification, are permitted provided that the following conditions are met:

……

按空格键，跳到协议末尾：

Do you approve the license terms? [yes|no]

>>> 

敲击“yes”，表示同意协议内容，继续下面的安装步骤。

Anaconda2 will now be installed into this location:

/Users/graylamb/anaconda2

  - Press ENTER to confirm the location

  - Press CTRL-C to abort the installation

  - Or specify a different location below

按照提示，敲击Enter键，建立Anaconda2目录，并安装相关文件：

……

installing: anaconda-2.4.1-np110py27_0 ...

Python 2.7.11 :: Continuum Analytics, Inc.

creating default environment...

installation finished.

Do you wish the installer to prepend the Anaconda2 install location

to PATH in your /Users/graylamb/.bash_profile ? [yes|no]

安装完毕会提示将profile文件建立在上面的路径中，敲击yes确认。

Prepending PATH=/Users/graylamb/anaconda2/bin to PATH in

newly created /Users/graylamb/.bash_profile

For this change to become active, you have to open a new terminal.

Thank you for installing Anaconda2!

Share your notebooks and packages on Anaconda Cloud!

Sign up for free: [Anaconda Cloud](https://link.zhihu.com/?target=https%3A//anaconda.org)

安装完毕啦～

如果成功安装，那么在开启新的终端窗口，敲击spyder，是可以打开spyder的IDE环境的。
