# GRE单词记忆连载8 - 知乎
# 

101.**demur **- vi.提出异议, 反对, 抗辩, 犹豫n.异议, 反对
mur来自mor，more，更多时间，犹豫，犹豫就会出现异议；

102.**proliferate **-v.增生扩散 

pro + oli + fer + ate[动词后缀]，pro - 向前；oli - 类似oil - 油，有动力；fer - carry，比如ferry；

103.**burgeon **- n.嫩芽v.萌芽

bud[芽] + org[organ]；

104.**wane **- vi.变小, 亏缺, 衰落, 退潮, 消逝, 呈下弦n.月亏, 月亏期, 衰退, 衰退期

wan - 完，完了，没了 - 变小；

105.**inscrutable **- adj.难以了解的, 不能预测的

in - not；scrutiny - n.详细审查；rutiny - routine，有点类似search routine，例行搜查，无法搜查，难以预测；

106.**procurable **- adj.可得到的, 得手的

pro + cure + able，可以向前去关心照顾的，可得手的；

107.**peripheral **- adj.外围的n.外围设备

peri + pher + al；peri表示周围，per second，每一秒，每秒都是；都是，周围都是，到处都是，引申出周围；pher，ph发音同f，pher - fer - ferry；周围携带的，外围设备；

108.**acclimate **- vt.使适应新环境, 使服水土vi.服水土, 适应

ac + climate[n.气候, 风土, 思潮],去适应气候的；

109.**accolade **- n.赞美, 骑士爵位的授予

col表示圆圈，项颈，比如collect，挑选围在一起；collaboration，围在一起工作；

110.**distinct **- adj.清楚的, 明显的, 截然不同的, 独特的

dis + st + ing = distinguish；st = stand；dis - 分开；分开站的；清楚的；
这里ct与g互换；

111.**attune **- vt.(为乐器)调音, 使合调, 使相合

at + tune;to tune

112.**conciliate **- vt.安慰, 赢得, 调和

council + ate；坐在一起讨论协调；安慰；或者concil读音“共私了”，一起私了，调和；

113.**accrete **- vi.共生, 合生, 附着vt.使依附, 使连接adj.共生的

ac + cre + ete ;cre - create;ete就是ate，动词后缀；去生长，一起生长；

114.**accumulate **- v.积聚, 堆积

ac[to] + cumulate[读音近似“枯木垒的”]，堆积；

115.**dissipate **- v.驱散, (使)(云、雾、疑虑等)消散, 浪费(金钱或时间)

dis - 分开；sip读音近似“撕破”；撕开，驱散；
