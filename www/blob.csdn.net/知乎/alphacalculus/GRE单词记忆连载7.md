# GRE单词记忆连载7 - 知乎
# 

86.**ludicrous** - adj.可笑的, 滑稽的, 愚蠢的
lud表示 to play，lud读音类似“录的”，录制的，录的（录制的）电影最初就是为了玩乐，因此滑稽可笑；

87.**preposterous **- adj.荒谬的

pre + post + erous，pre - before；post - 位置，岗位，代表现在，当下，目前，现实，现实之前的，超前的，虚幻的，荒唐可笑的；

88.**abundant **- adj.丰富的, 充裕的, 丰富, 盛产, 富于

ab = to,und表示起波浪，un看起来像凹凸型，类似波浪，水深才有浪，水多，丰富；ant形容词后缀；

89.**ample **- adj.充足的, 丰富的

读音近似“安沛”，又安逸又充沛；

90.**cornucopian **- adj.丰富的, 丰饶的

corn + copy，大量复制谷物, 五谷的；

91.**teeming **- adj.丰富的

与team有关，一组起码有好几个个体，表示多，尤其是两个e表示多；ing形容词后缀；

92.**abuse **- n.滥用, 虐待, 辱骂, 陋习, 弊端v.滥用, 虐待, 辱骂

ab表示away，使用不当，用歪了，滥用，滥用会招人骂，辱骂；

93.**assail **- vt.攻击, 质问

ass - 屁眼，屁股，美国骂人口头禅；sail - 航行；乘风破浪，迎风浪头而上，引申出质问；

94.**castigate **- v.严惩

cast表示“投射，投掷”，castle - 城堡，城堡都比较高，易守难攻，因为可以从上面投郑石头、弓箭；ig表示act，两者读音近似，ct读音类似g；有很多相互替换的单词；石刑就是向...投郑石头；

95.**lambaste **- vt.痛打, 严责

lam + baste[不死的]；

96.**abusive **- adj.辱骂的, 滥用的

abuse + ive[adj.]

97.**abut **- v.邻接, 毗邻

about表示附近, 大约,  左右, 周围，去掉o更邻近；

98.**abysmal **- adj.深不可测的, 无底的

a + bysm + al,a - not;bysm - bottom;

99.**accede **- v.同意, 加入, 同意

100.**acquiesce **- vi.默许, 勉强同意

ac + quies + ce ;quies - quiet，注意st的相似性；ac - to；

变得安静，默认，勉强同意，真正的同意会说出来而不是默认；
