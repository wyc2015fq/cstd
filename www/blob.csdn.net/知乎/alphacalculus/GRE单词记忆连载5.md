# GRE单词记忆连载5 - 知乎
# 

56.**surreptitious** - adj.暗中的, 秘密的, 偷偷摸摸的
sur - under，表示暗中，秘密地；rept - rapid - rape；

57.**abrade** - v.磨损, 摩擦, 折磨

ab - to，rad - radio，射线很细如针，铁棒磨成绣花针.

58.**chafe** - v.(将皮肤等)擦热, 擦破, 使恼火

cha类似“擦”f类似皮肤的肤；来自 calorie + facere "to make, do" (factitious).

59.**excoriate** - v.批判

excoriate = ex + cor + i + ate ，ex = out,cor - core - 核心，ate表示动词后缀，把内部的，隐藏的揭露出来，就是批判.

60.**scuff** - vi.拖足而行, 磨损vt.以足擦地, 践踏, 使磨损n.拖着脚走, 磨损之处

有点类似cutt，切割也是摩擦，与之类似的词是shove - n.推, 挤vt.推挤, 猛推, 强使vi.推，除了口语和航海上，shove都被push代替了；

sh**o**ve，脚f**oo**t擦地，sh**a**ve，刮擦脸f**a**ce

61.**quash** - v.取消

qu - 取，ash - 灰，取消干净，灰尘都不剩.

62.**abscond** - vi.潜逃, 避债

abs表示away，离开；cond读音类似“空的”，逃了，人去楼空；本意：con表示com - 一起；d表示do；cond表示放在一起，abs表示否定.

63.**lam** - 潜逃n.潜逃

lam读音近似“喇嘛”，喇嘛潜逃印度.

64.**arbitrary **- adj.任意的, 武断的, 独裁的, 专断的

arbitr + ary ，arbitr是arbiter的缩写，词根经常将er缩写为r；arbit读音近似“傲逼的”，又骄傲又装逼，很武断；

65.**autocratic **- adj.独裁的, 专制的

auto - 自我；c + rat（老鼠）+ic，自我的，独裁的人就像老鼠，人人喊打.

67.**despotic **- adj.专制的, 暴虐的

despotic = des + pot  +ic，des是domestic - adj.家庭的, 国内的；dome很像home，st - stand；pot - potent - adj.有力的, 有效的；在国内，在家里很有势力的就是专制的, 暴虐的；

68.**dictatorial **- adj.独裁的
dict - speak，比如dictionary；

69.**monocratic** - 独裁的

mono[single] + cratic;mono - more?no! - >single;

70.**tyrannical **- adj.残暴的

tyre - 轮胎，tyran - 轮奸胎儿的人，轮奸孕妇的人，残暴的；
