# 11301 「万和」为什么读作「ばんな」？—— 罕见音便「连声」 - 知乎
# 

　　4 月 1 日，日本公布了「平成」之后的下一个年号 ——「令和」。几天之后，又披露了落选的另外五个元号：英弘、久化、広至、万和、万保，它们的读音如题图所示。不知道你有没有注意到其中有一个元号的读音比较奇怪：「万和」不读作「ばんわ」，而是读成了「ばんな」。没准「读音不规则」这一点，也是它落选的原因之一呢！不过话说回来，这里「和」字的读音，是怎么变成「な」的呢？

　　我们知道，日语里汉字词的读音，常常发生各种音便。比较常见的有：

　　● 连浊，如「誕生」たん + しょう → たんじょう；
　　● 连半浊，如「先輩」せん + はい → せんぱい；
　　● 连促，如「学校」がく + こう → がっこう。

而「万和」读作「ばんな」，则是一种名叫「[连声](https://link.zhihu.com/?target=https%3A//ja.wikipedia.org/wiki/%25E9%2580%25A3%25E5%25A3%25B0)」的音便的结果。

　　「连声」这种音便是十分罕见的，日语中只有十几个词带有这种音便，且相当一部分集中于宗教领域。而且，这种音便只产生于古代（平安时代至室町时代），到了现代已经「不能产」（non-[productive](https://link.zhihu.com/?target=https%3A//en.wikipedia.org/wiki/Productivity_%28linguistics%29)）了：新出现的词汇，即使满足连声的条件，也不会发生音便了。这与连浊、连半浊、连促形成鲜明对比：连半浊和连促依然非常能产，只要满足条件就几乎一定会发生，比如另一个落选元号「万保」（ばんぽう）就是一个例子；连浊在音读词中虽然少了许多，但在训读词中依然能产。

　　说了这么多，这个「连声」到底是怎么回事呢？我们来看一些例子（词义均与汉语相同）：

　　● 銀杏：ぎん + あん → ぎんなん；
　　● 云々：うん + うん → うんぬん；
　　● 天皇：てん + おう → てんのう；
　　● 反応：はん + おう → はんのう；
　　● 観音：かん + おん → かんのん； 
　　● 因縁：いん + えん → いんねん；
　　● 輪廻：りん + え → りんね。 

不难总结出规律：如果前一个字以 ん 结尾，后一个字以 あ 行音开头，那么后一个字的开头会变成相应的 な 行音，或者说是在两个字中间插入了一个 n 音。感觉一下，ん 后面接 な 行音，是不是比接 あ 行音更顺畅一些呢？

　　「万和」的读音从「ばんわ」变成「ばんな」，道理也类似，只是细节不同：「和」字本来的读音在 わ 行而不是 あ 行。不过 わ 也可以算以元音开头，前面插入一个 n 音以求更顺畅也可以理解。插入 n 音之后，「和」字的读音变成了 nwa，这个音在日语中并不存在，所以就干脆把 w 省略，读成 na 了。

　　实际上，如果使用「[历史假名遣](https://link.zhihu.com/?target=https%3A//zh.wikipedia.org/zh-cn/%25E5%258E%2586%25E5%258F%25B2%25E5%2581%2587%25E5%2590%258D%25E9%2581%25A3)」，那么「天皇」的「皇」字应该拼作「わう」，也属于 ん + わ 行的情况。

　　再来看另外两个例子：

　　● 三位：さん + い → さんみ（指基督教的「三位一体」）；
　　● 陰陽：おん + よう → おんみょう。

 咦？这里的后字为什么不是变成 な 行，而是变成 ま 行了呢？原来，在中古汉语中，「三」「陰」两个字是收 -m 韵尾的，而第一组例子中各词的前字都是收 -n 韵尾的。中古汉语中 -n、-m 两种韵尾在日语中都变成了 ん，正如在普通话中也合并成了 -n 一样。但是在发生「连声」时，这个 -m 韵尾就会原形毕露了。

　　同时我们也可以注意到，ん 后面接 や 行音时也可以发生连声，比如「陰陽」中 m + yo 变成了 myo。由于日语中有拗音的存在，这个 y 就得以保留，不像 nwa 中的 w 一样脱落了。

　　连声还有另外一组例子：

　　● 雪隠：せつ + いん → せっちん（「茅厕」的意思）；
　　● 屈惑：くつ + わく → くったく（「烦恼」的意思）。

其中，「屈惑」根据连声后的发音，重新选取了汉字写法，写作「屈託」。

　　这一组例子，体现的是中古汉语中「雪」「屈」两字的「入声」韵尾 -t。这个韵尾在普通话中已经不存在了，但在粤语中还保留着，「雪」「屈」两字分别读作 syut、wat。韵尾 -t 在日语单字读音中一般表现为 つ 或 ち。在后字以元音开头时，这个韵尾就会让后字开头变成 た 行音，而韵尾 -t 本身则表现为促音。

　　总结一下「连声」现象，大概是这样：

　　● 它发生的条件是：
　　　　○ 前字收 -n、-m 韵尾（表现为 ん）或 -t 韵尾（表现为 つ 或 ち）；
　　　　○ 后字以元音（あ、や、わ 行）开头；
　　● 其结果是：
　　　　○ 根据前字的韵尾，后字开头变成 な、ま 或 た 行音；
　　　　○ 若后字本来以 や 行音开头，则会产生拗音；
　　　　○ 若前字收 -t 韵尾，则此韵尾表现为促音。

　　「连声」的本质，是前字的辅音韵尾与后字的元音开头相接时，为了发音顺畅，前字的辅音韵尾发生了延长（[gemination](https://link.zhihu.com/?target=https%3A//zh.wikipedia.org/zh-cn/%25E8%25BC%2594%25E9%259F%25B3%25E5%25BB%25B6%25E9%2595%25B7)）。中古汉语中除了 -n、-m、-t 三种辅音韵尾以外，还有 -ng、-k、-p 三种辅音韵尾，理论上它们也可能导致「连声」，只不过因为「连声」现象本就罕见，没有留下例子罢了。
