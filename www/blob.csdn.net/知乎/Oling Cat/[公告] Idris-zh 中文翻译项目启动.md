# [公告] Idris-zh 中文翻译项目启动 - 知乎
# 

Idris 是个通用的依赖类型纯函数式编程语言，比起 Coq 来说更注重现实世界的编程。说起来咱对这货感兴趣还是受了某莎某 be5 和某智猫的影响= =||

其实 Idris 中文文档翻译的坑很早就挖下了，可惜咱能力有限现在才正式启动。目前只翻译了官方教程的第一章，大家可以在 [Idris-zh 文档主页](https://link.zhihu.com/?target=http%3A//idris-zh.readthedocs.io/) 查看。对应的 [Idris-zh 项目地址](https://link.zhihu.com/?target=https%3A//github.com/Idris-zh/Idris-dev/tree/master/docs) 中可以找到源码。

官方文档的量并不算太多，本打算翻译完官方教程再说，不过咱最近忙于考试，加上人多力量大，思来想去还是在魔法店发个公告比较好。有兴趣的译者可以戳咱留下 Github ID，加入项目组一起翻译。
