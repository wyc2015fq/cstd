# 动力系统与丢番图逼近 - 知乎
# 



这一篇我们简单谈谈动力系统与丢番图逼近的联系。在上一讲中，我们讲了Howe-Moore遍历定理。在这一讲中，我们将用Howe-Moore定理证明一个丢番图逼近领域中的著名结果。

在本文中，我们计算向量的长度都用最大模。

先讲讲丢番图逼近是什么。给定一个无理数 ![\alpha](https://www.zhihu.com/equation?tex=%5Calpha) ，我们想要找到非零整数 ![n](https://www.zhihu.com/equation?tex=n) ，使得 ![n \alpha](https://www.zhihu.com/equation?tex=n+%5Calpha) 与整数的距离尽可能的小。这就是最简单的丢番图逼近，也是所有丢番图逼近研究的起点。对于这个问题，我们有下面这个著名的Dirichlet定理（定理1）：对于任意的 ![N >1](https://www.zhihu.com/equation?tex=N+%3E1) ，存在整数 ![0 < |n| \leq N](https://www.zhihu.com/equation?tex=0+%3C+%7Cn%7C+%5Cleq+N) 以及整数 ![m](https://www.zhihu.com/equation?tex=m) ，使得

![|n \alpha + m| < N^{-1}](https://www.zhihu.com/equation?tex=%7Cn+%5Calpha+%2B+m%7C+%3C+N%5E%7B-1%7D) 。从这个定理很容易推出下面的定理（定理2）：存在无穷多个整数 ![n](https://www.zhihu.com/equation?tex=n) ，使得 ![\langle n \alpha \rangle < |n|^{-1}](https://www.zhihu.com/equation?tex=%5Clangle+n+%5Calpha+%5Crangle+%3C+%7Cn%7C%5E%7B-1%7D) ，这里 ![\langle x \rangle](https://www.zhihu.com/equation?tex=%5Clangle+x+%5Crangle) 表示 ![x](https://www.zhihu.com/equation?tex=x) 与最近的整数的距离。从这个定理可以引出连分数的理论，我们就不展开讨论了。

从上面的两个结果我们如何展开丢番图逼近的研究呢？第一个方向是向高维推广。给定一个 ![n](https://www.zhihu.com/equation?tex=n) 维向量 ![\mathbf{x} = (x_1, \dots , x_n)](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D+%3D+%28x_1%2C+%5Cdots+%2C+x_n%29) 和 ![N >1](https://www.zhihu.com/equation?tex=N+%3E1) ，我们想找非零整数 ![q](https://www.zhihu.com/equation?tex=q) 使得 ![q \mathbf{x}](https://www.zhihu.com/equation?tex=q+%5Cmathbf%7Bx%7D) 离整数向量尽可能的近，即每个 ![\langle q x_i \rangle](https://www.zhihu.com/equation?tex=%5Clangle+q+x_i+%5Crangle) 都很小。在这方面我们也有Dirichlet定理的高维版本：对于任意的 ![N > 1](https://www.zhihu.com/equation?tex=N+%3E+1) ，存在整数 ![0 < |q| \leq N ](https://www.zhihu.com/equation?tex=0+%3C+%7Cq%7C+%5Cleq+N+) ，使得 ![\langle q x_i \rangle \leq N^{-1/n}](https://www.zhihu.com/equation?tex=%5Clangle+q+x_i+%5Crangle+%5Cleq+N%5E%7B-1%2Fn%7D) 对每个 ![i=1,\dots,n](https://www.zhihu.com/equation?tex=i%3D1%2C%5Cdots%2Cn) 都成立。由这个定理很容易推出存在无穷多个整数 ![\{q_i: i = 1, 2, \dots\}](https://www.zhihu.com/equation?tex=%5C%7Bq_i%3A+i+%3D+1%2C+2%2C+%5Cdots%5C%7D) 使得 ![\langle q_i \mathbf{x} \rangle \leq |q_i|^{-1/n}](https://www.zhihu.com/equation?tex=%5Clangle+q_i+%5Cmathbf%7Bx%7D+%5Crangle+%5Cleq+%7Cq_i%7C%5E%7B-1%2Fn%7D) ，这里 ![\langle \mathbf{x} \rangle](https://www.zhihu.com/equation?tex=%5Clangle+%5Cmathbf%7Bx%7D+%5Crangle) 表示向量与最近的整数点之间的距离。由此出发，我们可以定义实数向量不同的丢番图性质。

第一个性质叫very well approximable，以后简称VWA。我们说一个实数向量 ![\mathbf{x} \in \mathbb{R}^n](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D+%5Cin+%5Cmathbb%7BR%7D%5En) 是VWA的，如果存在一个 ![\epsilon >0](https://www.zhihu.com/equation?tex=%5Cepsilon+%3E0) 使得存在无穷多个整数 ![\{q_i: i = 1,2,\dots\}](https://www.zhihu.com/equation?tex=%5C%7Bq_i%3A+i+%3D+1%2C2%2C%5Cdots%5C%7D) 使得 ![\langle q_i \mathbf{x} \rangle \leq |q_i|^{-1/n - \epsilon}](https://www.zhihu.com/equation?tex=%5Clangle+q_i+%5Cmathbf%7Bx%7D+%5Crangle+%5Cleq+%7Cq_i%7C%5E%7B-1%2Fn+-+%5Cepsilon%7D) 。我们知道几乎所有的点都不是VWA。

第二个性质叫Dirichlet定理可改进，以后简称DT可改进。我们说一个实数向量 ![\mathbf{x} \in \mathbb{R}^n](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D+%5Cin+%5Cmathbb%7BR%7D%5En) 是DT可改进的，如果存在一个常数 ![0 < c < 1](https://www.zhihu.com/equation?tex=0+%3C+c+%3C+1) ，使得对于任意充分大的 ![N \gg 1](https://www.zhihu.com/equation?tex=N+%5Cgg+1) ，存在整数 ![0 < |q| \leq c N](https://www.zhihu.com/equation?tex=0+%3C+%7Cq%7C+%5Cleq+c+N) 使得 ![\langle q \mathbf{x} \rangle \leq c N^{-1/n}](https://www.zhihu.com/equation?tex=%5Clangle+q+%5Cmathbf%7Bx%7D+%5Crangle+%5Cleq+c+N%5E%7B-1%2Fn%7D) 。在这种情况下，我们也可以说 ![\mathbf{x}](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D) 是 ![c](https://www.zhihu.com/equation?tex=c) -DT可改进的。我们知道几乎所有的点都不是DT可改进的。

第三个重要的性质叫Badly approximable，我们简称BA。我们说 ![\mathbf{x} \in \mathbb{R}^n](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D+%5Cin+%5Cmathbb%7BR%7D%5En) 是BA的，如果存在一个常数 ![0 < c < 1](https://www.zhihu.com/equation?tex=0+%3C+c+%3C+1) ，使得对于任意的非零整数 ![q](https://www.zhihu.com/equation?tex=q) ， ![\langle q \mathbf{x} \rangle \geq c |q|^{-1/n}](https://www.zhihu.com/equation?tex=%5Clangle+q+%5Cmathbf%7Bx%7D+%5Crangle+%5Cgeq+c+%7Cq%7C%5E%7B-1%2Fn%7D) 。我们知道几乎所有的点都不是BA的，但是BA的点组成的集合的Hausdorff维数为 ![n](https://www.zhihu.com/equation?tex=n) 。

（补充：还有一个重要的性质，叫奇异，singular。我们说 ![\mathbf{x} \in \mathbb{R}^n](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D+%5Cin+%5Cmathbb%7BR%7D%5En) 是奇异的，如果对于任意的 ![\epsilon >0](https://www.zhihu.com/equation?tex=%5Cepsilon+%3E0) ,都有当 ![N>1](https://www.zhihu.com/equation?tex=N%3E1) 充分大时，存在非零整数 ![0< |q| < N](https://www.zhihu.com/equation?tex=0%3C+%7Cq%7C+%3C+N) 使得 ![\langle q \mathbf{x} \rangle < \epsilon T^{-1/n}](https://www.zhihu.com/equation?tex=%5Clangle+q+%5Cmathbf%7Bx%7D+%5Crangle+%3C+%5Cepsilon+T%5E%7B-1%2Fn%7D)。感谢@是也 老师的提醒。）

（补充：我们讲一下在一维的情况，也就是一个数 ![x](https://www.zhihu.com/equation?tex=x) 的丢番图性质。以上这四个性质都可以用 ![x](https://www.zhihu.com/equation?tex=x) 的连分数展开来描述。假设 ![x](https://www.zhihu.com/equation?tex=x) 的连分数表示为 ![[a_1, a_2, a_3, \dots]](https://www.zhihu.com/equation?tex=%5Ba_1%2C+a_2%2C+a_3%2C+%5Cdots%5D) ，又该表示我们可以得到一个连分数逼近的有理序列 ![\left\{\frac{p_n}{q_n}: n=1,2,3,\dots\right\}](https://www.zhihu.com/equation?tex=%5Cleft%5C%7B%5Cfrac%7Bp_n%7D%7Bq_n%7D%3A+n%3D1%2C2%2C3%2C%5Cdots%5Cright%5C%7D) ，具体的对应公式可参阅任何一本书连分数的书。那么VWA对应的性质为存在一个正常数 ![\epsilon](https://www.zhihu.com/equation?tex=%5Cepsilon) ，使得存在无穷多的 ![n](https://www.zhihu.com/equation?tex=n) ，满足 ![a_n \geq q_{n-1}^{\epsilon}](https://www.zhihu.com/equation?tex=a_n+%5Cgeq+q_%7Bn-1%7D%5E%7B%5Cepsilon%7D) 。Dirichlet可改进对应的条件为存在一个常数 ![c>1](https://www.zhihu.com/equation?tex=c%3E1) ，使得对所有充分大的 ![n](https://www.zhihu.com/equation?tex=n) ， ![a_n \geq c](https://www.zhihu.com/equation?tex=a_n+%5Cgeq+c) 。BA所对应的条件为存在一个常数 ![N >1](https://www.zhihu.com/equation?tex=N+%3E1) 使得对所有的 ![n](https://www.zhihu.com/equation?tex=n) ， ![a_n \leq N](https://www.zhihu.com/equation?tex=a_n+%5Cleq+N) 。奇异所对应的条件为 ![a_n \to \infty](https://www.zhihu.com/equation?tex=a_n+%5Cto+%5Cinfty) 。感谢@是也 老师的建议。）

对于VWA，比较重要的问题是对于一个 ![\mathbb{R}^n](https://www.zhihu.com/equation?tex=%5Cmathbb%7BR%7D%5En) 的子流形，是否该子流形上的几乎所有点都不是VWA的。如果这个性质成立，则我们叫这个子流形为Extremal的。对于DT可改进，重要的问题跟上面的类似，对于一个子流形，是否其上的几乎所有点都DT不可改进。对于BA，我们比较关心的是存在性的问题，一个重要的问题是对于一个子流形，该子流形上是否存在BA点，或者更进一步的，该子流形上的BA点的集合的Hausdorff维数是什么。

这三个问题都有了几乎完美的答案。关于VWA的问题，如果子流形不包含在一个线性子流形中，则结论是成立的，如果子流形包含在一个有理线性子空间中，则所有点都是VWA的。这是Kleinbock和Margulis在1998年[Annals of Math] 证明的结果。关于DT可改进的问题，如果子流形不包含在一个线性子流形中，则结论成立，若包含在有理线性子流形中，则所有点DT可改进。这是Shah在2009年[Inventiones Math]证明的结果。关于BA的问题，答案也一样，如果子流形不包含在一个线性子流形中，则结论成立，若包含在有理线性子流形中，则不存在BA点。这是Beresnevich在2015年[Inventiones Math]证明的结果。值得一提的是，前两个问题的解决都依赖于齐性动力系统，最后一个问题在方法上没有用动力系统，但是也依赖于Bernik，Kleinbock和Margulis之前的一个用动力系统证明的定理。

下面我们谈谈这些问题是如何与动力系统联系起来的。这个联系叫做Dani correspondence。

首先，对于 ![\mathbb{R}^{n+1}](https://www.zhihu.com/equation?tex=%5Cmathbb%7BR%7D%5E%7Bn%2B1%7D) 中一个 ![n+1](https://www.zhihu.com/equation?tex=n%2B1) 维的离散子群（格子） ![\Lambda](https://www.zhihu.com/equation?tex=%5CLambda) ，我们可以考虑 ![\mathbb{R}^{n+1}/\Lambda](https://www.zhihu.com/equation?tex=%5Cmathbb%7BR%7D%5E%7Bn%2B1%7D%2F%5CLambda) 的体积，我们称这个体积为个格子 ![\Lambda](https://www.zhihu.com/equation?tex=%5CLambda) 的模。我们记 ![\mathbb{R}^{n+1}](https://www.zhihu.com/equation?tex=%5Cmathbb%7BR%7D%5E%7Bn%2B1%7D) 中所有模为 ![1](https://www.zhihu.com/equation?tex=1) 的格子组成的空间为 ![X](https://www.zhihu.com/equation?tex=X) 。Dani correspondence的关键就是发现 ![X](https://www.zhihu.com/equation?tex=X) 实际上等同于齐性空间 ![\mathrm{SL}(n+1,\mathbb{R})/\mathrm{SL}(n+1,\mathbb{Z})](https://www.zhihu.com/equation?tex=%5Cmathrm%7BSL%7D%28n%2B1%2C%5Cmathbb%7BR%7D%29%2F%5Cmathrm%7BSL%7D%28n%2B1%2C%5Cmathbb%7BZ%7D%29) 。齐性空间中的一个点 ![x = [g]](https://www.zhihu.com/equation?tex=x+%3D+%5Bg%5D) ( ![g \in \mathrm{SL}(n+1, \mathbb{R})](https://www.zhihu.com/equation?tex=g+%5Cin+%5Cmathrm%7BSL%7D%28n%2B1%2C+%5Cmathbb%7BR%7D%29) )对应格子 ![g \mathbb{Z}^{n+1}](https://www.zhihu.com/equation?tex=g+%5Cmathbb%7BZ%7D%5E%7Bn%2B1%7D) 。 ![X](https://www.zhihu.com/equation?tex=X) 不是一个紧致空间，趋近于无穷远的方式是对应的格子存在越来越短的向量。给定一个 ![\epsilon >0](https://www.zhihu.com/equation?tex=%5Cepsilon+%3E0) ，我们记 ![K_{\epsilon} \subset X](https://www.zhihu.com/equation?tex=K_%7B%5Cepsilon%7D+%5Csubset+X) 为不包含长度小于 ![\epsilon](https://www.zhihu.com/equation?tex=%5Cepsilon) 的非零向量的格子的集合，这是一个紧致集合。反之，任何 ![X](https://www.zhihu.com/equation?tex=X) 的紧致子集都包含在某个 ![K_{\epsilon}](https://www.zhihu.com/equation?tex=K_%7B%5Cepsilon%7D) 中。这个判定紧致的法则叫做Mahler判定法则。利用Dani correspondence，我们可以将丢番图逼近的问题转化为齐性空间中的某些动力系统的问题来研究。

**我们最后看看如何用Howe-Moore遍历定理证明几乎所有的向量都不是DT可改进的。**

![Z(A) = \left\{ z(X) : X \in \mathrm{SL}(n, \mathbb{R}) \right\}](https://www.zhihu.com/equation?tex=Z%28A%29+%3D+%5Cleft%5C%7B+z%28X%29+%3A+X+%5Cin+%5Cmathrm%7BSL%7D%28n%2C+%5Cmathbb%7BR%7D%29+%5Cright%5C%7D) 其中

![z(X) = \begin{bmatrix}1 & \\ & X\end{bmatrix}](https://www.zhihu.com/equation?tex=z%28X%29+%3D+%5Cbegin%7Bbmatrix%7D1+%26+%5C%5C+%26+X%5Cend%7Bbmatrix%7D) ，满足 ![a(t) g a(-t) = g](https://www.zhihu.com/equation?tex=a%28t%29+g+a%28-t%29+%3D+g) 。 ![G](https://www.zhihu.com/equation?tex=G) 中所有的元素（除掉一个低维子集意外） ![g](https://www.zhihu.com/equation?tex=g) 都可以写成 ![g = u^- z u^+](https://www.zhihu.com/equation?tex=g+%3D+u%5E-+z+u%5E%2B) ，其中 ![u^- \in U^-(A), z \in Z(A) , u^+ \in U^+(A)](https://www.zhihu.com/equation?tex=u%5E-+%5Cin+U%5E-%28A%29%2C+z+%5Cin+Z%28A%29+%2C+u%5E%2B+%5Cin+U%5E%2B%28A%29) 。

我们看看DT可改进这个条件。它等价于当 ![t](https://www.zhihu.com/equation?tex=t) 充分大时， ![[a(t) u(\mathbf{x})] \not\in K_{c}](https://www.zhihu.com/equation?tex=%5Ba%28t%29+u%28%5Cmathbf%7Bx%7D%29%5D+%5Cnot%5Cin+K_%7Bc%7D) 。很容易知道整点格子 ![\mathbb{Z}^{n+1} \in K_c](https://www.zhihu.com/equation?tex=%5Cmathbb%7BZ%7D%5E%7Bn%2B1%7D+%5Cin+K_c) ，而且它的一个开领域也包含在 ![K_{c}](https://www.zhihu.com/equation?tex=K_%7Bc%7D) 中。这说明，如果 ![\mathbf{x}](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D) 是DT可改进的，则轨道 ![\{[a(t)u(\mathbf{x})]: t >0\}](https://www.zhihu.com/equation?tex=%5C%7B%5Ba%28t%29u%28%5Cmathbf%7Bx%7D%29%5D%3A+t+%3E0%5C%7D) 不是稠密的。根据Howe-Moore遍历定理， ![A](https://www.zhihu.com/equation?tex=A) 在 ![X](https://www.zhihu.com/equation?tex=X) 上的作用为遍历的，特别的，对于几乎所有的 ![[g] \in X](https://www.zhihu.com/equation?tex=%5Bg%5D+%5Cin+X) ，轨道 ![\{[a(t) g] : t >0\}](https://www.zhihu.com/equation?tex=%5C%7B%5Ba%28t%29+g%5D+%3A+t+%3E0%5C%7D) 在 ![X](https://www.zhihu.com/equation?tex=X) 中趋于均匀分布，更特别的，轨道为稠密的。另一方面，几乎所有的 ![g \in G](https://www.zhihu.com/equation?tex=g+%5Cin+G) 都可以写成 ![g = u^- z u(\mathbf{x})](https://www.zhihu.com/equation?tex=g+%3D+u%5E-+z+u%28%5Cmathbf%7Bx%7D%29) ，其中 ![u^- \in U^-(A), z \in Z(A)](https://www.zhihu.com/equation?tex=u%5E-+%5Cin+U%5E-%28A%29%2C+z+%5Cin+Z%28A%29) ，这样

![a(t) g = a(t) u^- a(-t) z a(t) u(\mathbf{x})](https://www.zhihu.com/equation?tex=a%28t%29+g+%3D+a%28t%29+u%5E-+a%28-t%29+z+a%28t%29+u%28%5Cmathbf%7Bx%7D%29) ，注意到 ![a(t) u^- a(-t) \to e](https://www.zhihu.com/equation?tex=a%28t%29+u%5E-+a%28-t%29+%5Cto+e) ， ![z](https://www.zhihu.com/equation?tex=z) 为一个固定的元素，所以 ![[a(t)g]](https://www.zhihu.com/equation?tex=%5Ba%28t%29g%5D) 的极限行为与 ![[a(t) u(\mathbf{x})]](https://www.zhihu.com/equation?tex=%5Ba%28t%29+u%28%5Cmathbf%7Bx%7D%29%5D) 的极限行为是一样的。于是我们就可以得到对于几乎所有的 ![\mathbf{x} \in \mathbb{R}^n](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D+%5Cin+%5Cmathbb%7BR%7D%5En) ，轨道 ![\{[a(t) u(\mathbf{x})]: t >0\}](https://www.zhihu.com/equation?tex=%5C%7B%5Ba%28t%29+u%28%5Cmathbf%7Bx%7D%29%5D%3A+t+%3E0%5C%7D) 稠密，这就证明了几乎所有的 ![\mathbf{x} \in \mathbb{R}^n](https://www.zhihu.com/equation?tex=%5Cmathbf%7Bx%7D+%5Cin+%5Cmathbb%7BR%7D%5En) 都不是DT可改进的。

总结一下，根据Dani correspondence，我们可以把实向量的丢番图性质转化成齐性空间中某些轨道的极限性质，这使得我们可以使用动力系统中的工具来研究丢番图逼近的问题。以后我们还会谈到如何用动力系统来研究VWA性质和BA性质，并简单介绍在丢番图逼近方向上一些最新的研究成果。
