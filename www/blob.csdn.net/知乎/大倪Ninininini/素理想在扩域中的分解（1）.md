# 素理想在扩域中的分解（1） - 知乎
# 

代数数论的一个重要主题是刻画素理想在扩域中的分解情况。这篇笔记我们讨论Galois扩张（或者更强：Abel扩张）的情形。在Galois扩张中，素理想的分解形式非常“整齐”：

**定理1.** 假设 ![K](https://www.zhihu.com/equation?tex=K) 是一个数域，![L](https://www.zhihu.com/equation?tex=L) 是 ![K](https://www.zhihu.com/equation?tex=K) 的（有限）Galois扩张。![A](https://www.zhihu.com/equation?tex=A) 是 ![\mathbb{Z}](https://www.zhihu.com/equation?tex=%5Cmathbb%7BZ%7D) 在 ![K](https://www.zhihu.com/equation?tex=K) 中的整闭包，![B](https://www.zhihu.com/equation?tex=B) 是 ![A](https://www.zhihu.com/equation?tex=A) 在 ![L](https://www.zhihu.com/equation?tex=L) 中的整闭包，![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 是 ![K](https://www.zhihu.com/equation?tex=K) 中的非零素理想。那么 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 在 ![B](https://www.zhihu.com/equation?tex=B) 中分解为： ![\mathfrak{p}B=(\mathfrak{B}_{1}\mathfrak{B}_{2}\cdot\cdot\cdot\mathfrak{B}_{g})^{e}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7DB%3D%28%5Cmathfrak%7BB%7D_%7B1%7D%5Cmathfrak%7BB%7D_%7B2%7D%5Ccdot%5Ccdot%5Ccdot%5Cmathfrak%7BB%7D_%7Bg%7D%29%5E%7Be%7D) ，这里 ![\mathfrak{B}_{i}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7BB%7D_%7Bi%7D) 是 ![B](https://www.zhihu.com/equation?tex=B) 中素理想。

由于在Galois情形下每个 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 上素理想的分歧指数 ![e](https://www.zhihu.com/equation?tex=e) 相同，剩余域扩张次数 ![f](https://www.zhihu.com/equation?tex=f) 也相同，**所以当我们想刻画 ![e](https://www.zhihu.com/equation?tex=e) 和 ![f](https://www.zhihu.com/equation?tex=f) 时，只需要考虑其中任何一个素理想。**描述 ![e,f,g](https://www.zhihu.com/equation?tex=e%2Cf%2Cg) 三个量的方法是把它们和Galois群 ![\text{Gal}(L/K)](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L%2FK%29) 的某些子群联系起来：**分解群和惯性群**描述的就是 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 上某个素理想分裂、分歧的情况。然而**还不能完全确定这些群内部“长什么样子”**（除了特殊情形如二次域和分圆域），因此不严格地，我们称之为对素理想分解“invisible”的描述方式。

后续的笔记我们会说明Frobenius element ![\text{Frob}_{\mathfrak{p}}](https://www.zhihu.com/equation?tex=%5Ctext%7BFrob%7D_%7B%5Cmathfrak%7Bp%7D%7D) 确定了素理想的分解情况，以及idele class group和Ray class group是相对“visible”的对象，因此一旦确定了 ![\text{Frob}_{\mathfrak{p}}](https://www.zhihu.com/equation?tex=%5Ctext%7BFrob%7D_%7B%5Cmathfrak%7Bp%7D%7D) 在idele class group和Ray class group的样子，便可以更清晰地描述素理想的分解。

**（一）Local-global**

一开始我们说：“Galois情形下每个 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 上素理想的分歧指数 ![e](https://www.zhihu.com/equation?tex=e) 相同，剩余域扩张次数 ![f](https://www.zhihu.com/equation?tex=f) 相同，只需要考虑其中任何一个。”这个意思就是，我们可以通过局部的情形来反映整体的情形，这个准则被描述成下面两个引理：

**引理1.**![K](https://www.zhihu.com/equation?tex=K) 是一个数域，![L](https://www.zhihu.com/equation?tex=L) 是 ![K](https://www.zhihu.com/equation?tex=K) 的有限可分扩张，![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 是 ![K](https://www.zhihu.com/equation?tex=K) 中的非零素理想。 ![L](https://www.zhihu.com/equation?tex=L) 中 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 上的非零素理想为 ![\mathfrak{q}_{1},....\mathfrak{q}_{g}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D_%7B1%7D%2C....%5Cmathfrak%7Bq%7D_%7Bg%7D)， ![f=\text{irr}(\alpha,K)](https://www.zhihu.com/equation?tex=f%3D%5Ctext%7Birr%7D%28%5Calpha%2CK%29) , ![L=K(\alpha)](https://www.zhihu.com/equation?tex=L%3DK%28%5Calpha%29) 。那么有同构： ![L\otimes_{K}K_{\mathfrak{p}}=\prod_{i=1}^{g}L_{\mathfrak{q}_{i}}](https://www.zhihu.com/equation?tex=L%5Cotimes_%7BK%7DK_%7B%5Cmathfrak%7Bp%7D%7D%3D%5Cprod_%7Bi%3D1%7D%5E%7Bg%7DL_%7B%5Cmathfrak%7Bq%7D_%7Bi%7D%7D) 。具体地， ![f](https://www.zhihu.com/equation?tex=f) 在 ![K_{\mathfrak{p}}](https://www.zhihu.com/equation?tex=K_%7B%5Cmathfrak%7Bp%7D%7D) 中分解为 ![g](https://www.zhihu.com/equation?tex=g) 个不可约多项式 ![f_{i}](https://www.zhihu.com/equation?tex=f_%7Bi%7D) 的乘积，调整下标后有： ![K_{\mathfrak{p}}[x]/(f_{i}(x))\simeq L_{\mathfrak{q}_{i}}](https://www.zhihu.com/equation?tex=K_%7B%5Cmathfrak%7Bp%7D%7D%5Bx%5D%2F%28f_%7Bi%7D%28x%29%29%5Csimeq+L_%7B%5Cmathfrak%7Bq%7D_%7Bi%7D%7D) ，并且 ![[L:K]=\text{deg}f=\sum_{i=1}^{g}\text{deg}f_{i}=\sum_{i=1}^{g}[L_{\mathfrak{q}_{i}}:K_{\mathfrak{p}}]](https://www.zhihu.com/equation?tex=%5BL%3AK%5D%3D%5Ctext%7Bdeg%7Df%3D%5Csum_%7Bi%3D1%7D%5E%7Bg%7D%5Ctext%7Bdeg%7Df_%7Bi%7D%3D%5Csum_%7Bi%3D1%7D%5E%7Bg%7D%5BL_%7B%5Cmathfrak%7Bq%7D_%7Bi%7D%7D%3AK_%7B%5Cmathfrak%7Bp%7D%7D%5D) 。

**引理2. **![K](https://www.zhihu.com/equation?tex=K) 是一个数域，![L](https://www.zhihu.com/equation?tex=L) 是 ![K](https://www.zhihu.com/equation?tex=K) 的有限可分扩张，![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 是 ![K](https://www.zhihu.com/equation?tex=K) 中的非零素理想。![\mathfrak{q}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D) 是 ![L](https://www.zhihu.com/equation?tex=L) 中 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 上的非零素理想。那么： ![e(\mathfrak{q}B_{\mathfrak{q}}/\mathfrak{p}A_{\mathfrak{p}})=e(\mathfrak{q}/\mathfrak{p}),f(\mathfrak{q}B_{\mathfrak{q}}/\mathfrak{p}A_{\mathfrak{p}})=f(\mathfrak{q}/\mathfrak{p})](https://www.zhihu.com/equation?tex=e%28%5Cmathfrak%7Bq%7DB_%7B%5Cmathfrak%7Bq%7D%7D%2F%5Cmathfrak%7Bp%7DA_%7B%5Cmathfrak%7Bp%7D%7D%29%3De%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bp%7D%29%2Cf%28%5Cmathfrak%7Bq%7DB_%7B%5Cmathfrak%7Bq%7D%7D%2F%5Cmathfrak%7Bp%7DA_%7B%5Cmathfrak%7Bp%7D%7D%29%3Df%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bp%7D%29) 。

第一个引理是说 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 完全分解等价于每个 ![L_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=L_%7B%5Cmathfrak%7Bq%7D%7D) 不带来 ![K_{\mathfrak{p}}](https://www.zhihu.com/equation?tex=K_%7B%5Cmathfrak%7Bp%7D%7D) 的扩张：

![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 完全分解 ![\Leftrightarrow g=[L:K]\Leftrightarrow \text{For every}~\mathfrak{q}_{i}~,~[L_{\mathfrak{q}_{i}}:K_{\mathfrak{p}}]=1](https://www.zhihu.com/equation?tex=%5CLeftrightarrow+g%3D%5BL%3AK%5D%5CLeftrightarrow+%5Ctext%7BFor+every%7D~%5Cmathfrak%7Bq%7D_%7Bi%7D~%2C~%5BL_%7B%5Cmathfrak%7Bq%7D_%7Bi%7D%7D%3AK_%7B%5Cmathfrak%7Bp%7D%7D%5D%3D1)

第二个引理是说，整体域的剩余域扩张、分歧的情形和局部域上的情况一样。

我们关心的是Galois扩张的情形，这时每个 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 上素理想的分歧指数 ![e](https://www.zhihu.com/equation?tex=e) 相同，剩余域扩张次数 ![f](https://www.zhihu.com/equation?tex=f) 相同，再结合引理2，我们发现每个 ![[L_{\mathfrak{q}_{i}}:K_{\mathfrak{p}}]](https://www.zhihu.com/equation?tex=%5BL_%7B%5Cmathfrak%7Bq%7D_%7Bi%7D%7D%3AK_%7B%5Cmathfrak%7Bp%7D%7D%5D) 也相同。那么引理1最后那个等式就可以写成： ![[L:K]=g[L_{\mathfrak{q}}:K_{\mathfrak{p}}]=gef=ge(\mathfrak{q}B_{\mathfrak{q}}/\mathfrak{p}A_{\mathfrak{p}})f(\mathfrak{q}B_{\mathfrak{q}}/\mathfrak{p}A_{\mathfrak{p}})](https://www.zhihu.com/equation?tex=%5BL%3AK%5D%3Dg%5BL_%7B%5Cmathfrak%7Bq%7D%7D%3AK_%7B%5Cmathfrak%7Bp%7D%7D%5D%3Dgef%3Dge%28%5Cmathfrak%7Bq%7DB_%7B%5Cmathfrak%7Bq%7D%7D%2F%5Cmathfrak%7Bp%7DA_%7B%5Cmathfrak%7Bp%7D%7D%29f%28%5Cmathfrak%7Bq%7DB_%7B%5Cmathfrak%7Bq%7D%7D%2F%5Cmathfrak%7Bp%7DA_%7B%5Cmathfrak%7Bp%7D%7D%29) 。

**Ex1.**![L/K](https://www.zhihu.com/equation?tex=L%2FK) Galois时， ![L_{\mathfrak{q}}/K_{\mathfrak{p}},l/k](https://www.zhihu.com/equation?tex=L_%7B%5Cmathfrak%7Bq%7D%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D%2Cl%2Fk) 也是Galois扩张。（ ![l,k](https://www.zhihu.com/equation?tex=l%2Ck) 分别是 ![B,A](https://www.zhihu.com/equation?tex=B%2CA) 的剩余域）

**（二）分解群（为了描述**![g](https://www.zhihu.com/equation?tex=g)**）**

首先我们注意到下面的性质：

**Ex2.**![\text{Gal}(L/K)\times\{\text{nonzero prime ideals over}~\mathfrak{p}\}\rightarrow \{\text{nonzero prime ideals over}~\mathfrak{p}\}](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L%2FK%29%5Ctimes%5C%7B%5Ctext%7Bnonzero+prime+ideals+over%7D~%5Cmathfrak%7Bp%7D%5C%7D%5Crightarrow+%5C%7B%5Ctext%7Bnonzero+prime+ideals+over%7D~%5Cmathfrak%7Bp%7D%5C%7D)是一个可迁作用（transitive action）

而我们知道**对于可迁作用： ![G\times X\rightarrow X](https://www.zhihu.com/equation?tex=G%5Ctimes+X%5Crightarrow+X)，有双射： **![G/\text{Stab}(x)\leftrightarrow X](https://www.zhihu.com/equation?tex=G%2F%5Ctext%7BStab%7D%28x%29%5Cleftrightarrow+X) 。因此在上面的作用里， ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 上素理想的个数 ![g=\#\{\text{nonzero prime ideals over}~\mathfrak{p}\}=\#\text{Gal}(L/K)/\text{Stab}({\mathfrak{q}})](https://www.zhihu.com/equation?tex=g%3D%5C%23%5C%7B%5Ctext%7Bnonzero+prime+ideals+over%7D~%5Cmathfrak%7Bp%7D%5C%7D%3D%5C%23%5Ctext%7BGal%7D%28L%2FK%29%2F%5Ctext%7BStab%7D%28%7B%5Cmathfrak%7Bq%7D%7D%29)，其中 ![\mathfrak{q}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D) 是 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 上任一个非零素理想。这里 ![\text{Gal}(L/K)](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L%2FK%29) 的子群 ![\text{Stab}({\mathfrak{q}})](https://www.zhihu.com/equation?tex=%5Ctext%7BStab%7D%28%7B%5Cmathfrak%7Bq%7D%7D%29) 称为 ![{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%7B%5Cmathfrak%7Bq%7D%7D) 的分解群，记为 ![D_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=D_%7B%5Cmathfrak%7Bq%7D%7D) 。于是立即得到：

![{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%7B%5Cmathfrak%7Bq%7D%7D) 在 ![L](https://www.zhihu.com/equation?tex=L) 中完全分解 ![\Leftrightarrow](https://www.zhihu.com/equation?tex=%5CLeftrightarrow)![g=\#\text{Gal}(L/K)/\text{Stab}({\mathfrak{q}})=\#\text{Gal}(L/K)](https://www.zhihu.com/equation?tex=g%3D%5C%23%5Ctext%7BGal%7D%28L%2FK%29%2F%5Ctext%7BStab%7D%28%7B%5Cmathfrak%7Bq%7D%7D%29%3D%5C%23%5Ctext%7BGal%7D%28L%2FK%29)![\Leftrightarrow](https://www.zhihu.com/equation?tex=%5CLeftrightarrow)![D_{\mathfrak{q}}=\{1\}](https://www.zhihu.com/equation?tex=D_%7B%5Cmathfrak%7Bq%7D%7D%3D%5C%7B1%5C%7D)

由此可知，分解群描述的是 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 在 ![L](https://www.zhihu.com/equation?tex=L) 中分裂情况：**![D_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=D_%7B%5Cmathfrak%7Bq%7D%7D) 越大， ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 分裂出的素理想越少。**

**Ex3.**![\text{Gal}(L_{\mathfrak{q}}/K_{\mathfrak{p}})\simeq D_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L_%7B%5Cmathfrak%7Bq%7D%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D%29%5Csimeq+D_%7B%5Cmathfrak%7Bq%7D%7D)

这里我们还要更细致地描述 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 是怎么分裂的。下面的讨论都**假设 ![L/K](https://www.zhihu.com/equation?tex=L%2FK) 是Abel扩张**。假设 ![D_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=D_%7B%5Cmathfrak%7Bq%7D%7D) 在Galois对应下的域为 ![Z_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=Z_%7B%5Cmathfrak%7Bq%7D%7D) ，考虑extension tower： ![K\subset Z_{\mathfrak{q}}\subset L](https://www.zhihu.com/equation?tex=K%5Csubset+Z_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+L) ；假设 ![\mathfrak{q} \cap Z_{\mathfrak{q}}=\mathfrak{q}_{Z}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D+%5Ccap+Z_%7B%5Cmathfrak%7Bq%7D%7D%3D%5Cmathfrak%7Bq%7D_%7BZ%7D) 。

**由Galois基本定理立即可知： ![D_{\mathfrak{q}}=\text{Gal}(L/Z_{\mathfrak{q}})](https://www.zhihu.com/equation?tex=D_%7B%5Cmathfrak%7Bq%7D%7D%3D%5Ctext%7BGal%7D%28L%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29) ， ![\text{Gal}(Z_{\mathfrak{q}}/K)=\text{Gal}(L/K)/\text{Gal}(L/Z_{\mathfrak{q}})](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28Z_%7B%5Cmathfrak%7Bq%7D%7D%2FK%29%3D%5Ctext%7BGal%7D%28L%2FK%29%2F%5Ctext%7BGal%7D%28L%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29) ； 由分解群的定义：![\#\text{Gal}(Z_{\mathfrak{q}}/K)=\#\text{Gal}(L/K)/\text{Gal}(L/Z_{\mathfrak{q}})=g\Rightarrow\#D_{\mathfrak{q}}=\#\text{Gal}(L/Z_{\mathfrak{q}})=ef ](https://www.zhihu.com/equation?tex=%5C%23%5Ctext%7BGal%7D%28Z_%7B%5Cmathfrak%7Bq%7D%7D%2FK%29%3D%5C%23%5Ctext%7BGal%7D%28L%2FK%29%2F%5Ctext%7BGal%7D%28L%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29%3Dg%5CRightarrow%5C%23D_%7B%5Cmathfrak%7Bq%7D%7D%3D%5C%23%5Ctext%7BGal%7D%28L%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29%3Def+) 。**

**Claim1.**![\mathfrak{q}_{Z}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D_%7BZ%7D) 在 ![L](https://www.zhihu.com/equation?tex=L) 中不分裂，并且 ![e(\mathfrak{q}/\mathfrak{q}_{Z})=e,f(\mathfrak{q}/\mathfrak{q}_{Z})=f ](https://www.zhihu.com/equation?tex=e%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29%3De%2Cf%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29%3Df+) ; ![e(\mathfrak{q}_{Z}/\mathfrak{q})=1,f(\mathfrak{q}_{Z}/\mathfrak{q})=1](https://www.zhihu.com/equation?tex=e%28%5Cmathfrak%7Bq%7D_%7BZ%7D%2F%5Cmathfrak%7Bq%7D%29%3D1%2Cf%28%5Cmathfrak%7Bq%7D_%7BZ%7D%2F%5Cmathfrak%7Bq%7D%29%3D1) 。

**Proof.** 对于第一个断言，我们只需要考虑在Galois扩张 ![Z_{\mathfrak{q}}\subset L](https://www.zhihu.com/equation?tex=Z_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+L) 中， ![\mathfrak{q}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D) 的分解群 ![D^{\prime}_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=D%5E%7B%5Cprime%7D_%7B%5Cmathfrak%7Bq%7D%7D) 。事实上，**根据定义可知 ![\text{Gal}(L/Z_{\mathfrak{q}})](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29) 中任一个元素都固定 ![\mathfrak{q}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D)，那么 ![D^{\prime}_{\mathfrak{q}}=\text{Gal}(L/Z_{\mathfrak{q}})=D_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=D%5E%7B%5Cprime%7D_%7B%5Cmathfrak%7Bq%7D%7D%3D%5Ctext%7BGal%7D%28L%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29%3DD_%7B%5Cmathfrak%7Bq%7D%7D)，**![\mathfrak{q}_{Z}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D_%7BZ%7D) 上素理想个数为 ![\#\text{Gal}(L/Z_{\mathfrak{q}})/D^{\prime}_{\mathfrak{q}}=1](https://www.zhihu.com/equation?tex=%5C%23%5Ctext%7BGal%7D%28L%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29%2FD%5E%7B%5Cprime%7D_%7B%5Cmathfrak%7Bq%7D%7D%3D1) 。其次，注意到：

(1) ![e(\mathfrak{q}/\mathfrak{q}_{Z})f(\mathfrak{q}/\mathfrak{q}_{Z})\cdot1=[L:Z_{\mathfrak{q}}]=\#D_{\mathfrak{q}}=ef](https://www.zhihu.com/equation?tex=e%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29f%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29%5Ccdot1%3D%5BL%3AZ_%7B%5Cmathfrak%7Bq%7D%7D%5D%3D%5C%23D_%7B%5Cmathfrak%7Bq%7D%7D%3Def) ;

(2) ![ e(\mathfrak{q}/\mathfrak{q}_{Z})e(\mathfrak{q}_{Z}/\mathfrak{q})=e,f(\mathfrak{q}/\mathfrak{q}_{Z})f(\mathfrak{q}_{Z}/\mathfrak{q})=f](https://www.zhihu.com/equation?tex=+e%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29e%28%5Cmathfrak%7Bq%7D_%7BZ%7D%2F%5Cmathfrak%7Bq%7D%29%3De%2Cf%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29f%28%5Cmathfrak%7Bq%7D_%7BZ%7D%2F%5Cmathfrak%7Bq%7D%29%3Df) 。

因此这个过程可看作：**![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 先在 ![Z_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=Z_%7B%5Cmathfrak%7Bq%7D%7D) 中分裂成 ![g](https://www.zhihu.com/equation?tex=g) 个不同素理想，但剩余域不发生扩张，也不发生分歧。然后这 ![g](https://www.zhihu.com/equation?tex=g) 个不同素理想在 ![L](https://www.zhihu.com/equation?tex=L) 分歧（分歧指数为 ![e](https://www.zhihu.com/equation?tex=e)），剩余域发生 ![f](https://www.zhihu.com/equation?tex=f) 次扩张，不再分裂出新的素理想。**

然而我们还不清楚这 ![g](https://www.zhihu.com/equation?tex=g)个不同素理想到底是在 ![Z_{\mathfrak{q}}\subset L](https://www.zhihu.com/equation?tex=Z_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+L) 的哪个中间域发生的剩余域的扩张和分歧，为此我们需要考虑惯性群。

**（三）惯性群（为了描述 ![e](https://www.zhihu.com/equation?tex=e) ）**

**第一种定义方式：**

定义 ![{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%7B%5Cmathfrak%7Bq%7D%7D) 的惯性群 ![I_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=I_%7B%5Cmathfrak%7Bq%7D%7D) 为 ![\text{Ker}(D_{\mathfrak{q}}\rightarrow\text{Gal}(l/k))](https://www.zhihu.com/equation?tex=%5Ctext%7BKer%7D%28D_%7B%5Cmathfrak%7Bq%7D%7D%5Crightarrow%5Ctext%7BGal%7D%28l%2Fk%29%29) 。（Ex1和 ![D_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=D_%7B%5Cmathfrak%7Bq%7D%7D) 的定义使得该映射是良定义的）

那么我们立即发现 ![\#I_{\mathfrak{q}}=e](https://www.zhihu.com/equation?tex=%5C%23I_%7B%5Cmathfrak%7Bq%7D%7D%3De) 。因此 ![{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%7B%5Cmathfrak%7Bq%7D%7D) 不分歧 ![\Leftrightarrow](https://www.zhihu.com/equation?tex=%5CLeftrightarrow)![I_{\mathfrak{q}}=\{1\}](https://www.zhihu.com/equation?tex=I_%7B%5Cmathfrak%7Bq%7D%7D%3D%5C%7B1%5C%7D) 。

**Ex4. **![D_{\mathfrak{q}}\rightarrow\text{Gal}(l/k)](https://www.zhihu.com/equation?tex=D_%7B%5Cmathfrak%7Bq%7D%7D%5Crightarrow%5Ctext%7BGal%7D%28l%2Fk%29) 是满射

**第二种定义方式：**

首先我们回忆局部域非分歧扩张的性质（注意我们说“非分歧”指的是分歧指数为 ![1](https://www.zhihu.com/equation?tex=1)** 并且剩余域扩张是可分的**）：

**引理3. **设 ![K](https://www.zhihu.com/equation?tex=K) 是一个局部域， ![k](https://www.zhihu.com/equation?tex=k) 是其剩余域， ![L/K ](https://www.zhihu.com/equation?tex=L%2FK+) 是有限可分扩张。那么我们有一一对应：

{包含在 ![L](https://www.zhihu.com/equation?tex=L) 中的 ![K](https://www.zhihu.com/equation?tex=K) 的非分歧扩张 ![K^{\prime}](https://www.zhihu.com/equation?tex=K%5E%7B%5Cprime%7D) } ![\Leftrightarrow](https://www.zhihu.com/equation?tex=%5CLeftrightarrow) {包含在 ![l](https://www.zhihu.com/equation?tex=l) 中的 ![k](https://www.zhihu.com/equation?tex=k) 的有限可分扩张 ![l^{\prime}](https://www.zhihu.com/equation?tex=l%5E%7B%5Cprime%7D) }

并且成立： ![[K^{\prime}:K]=[k^{\prime}:k]](https://www.zhihu.com/equation?tex=%5BK%5E%7B%5Cprime%7D%3AK%5D%3D%5Bk%5E%7B%5Cprime%7D%3Ak%5D) ; ![K^{\prime}/K](https://www.zhihu.com/equation?tex=K%5E%7B%5Cprime%7D%2FK) Galois ![\Leftrightarrow](https://www.zhihu.com/equation?tex=%5CLeftrightarrow)![k^{\prime}/k](https://www.zhihu.com/equation?tex=k%5E%7B%5Cprime%7D%2Fk) Galois，![\text{Gal}(K^{\prime}/K)\simeq\text{Gal}(k^{\prime}/k)](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28K%5E%7B%5Cprime%7D%2FK%29%5Csimeq%5Ctext%7BGal%7D%28k%5E%7B%5Cprime%7D%2Fk%29) 。

对于Abel扩张 ![L/K ](https://www.zhihu.com/equation?tex=L%2FK+)，假设 ![K_{0}](https://www.zhihu.com/equation?tex=K_%7B0%7D) 包含于 ![L_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=L_%7B%5Cmathfrak%7Bq%7D%7D) 的 ![K_{\mathfrak{p}}](https://www.zhihu.com/equation?tex=K_%7B%5Cmathfrak%7Bp%7D%7D) 的最大非分歧扩张，剩余域分别为 ![l_{\mathfrak{q}},k_{\mathfrak{p}}](https://www.zhihu.com/equation?tex=l_%7B%5Cmathfrak%7Bq%7D%7D%2Ck_%7B%5Cmathfrak%7Bp%7D%7D) 。**注意到我们考虑的是数域的Abel扩张，所以 ![l_{\mathfrak{q}}/k_{\mathfrak{p}}](https://www.zhihu.com/equation?tex=l_%7B%5Cmathfrak%7Bq%7D%7D%2Fk_%7B%5Cmathfrak%7Bp%7D%7D) 是可分的（它们都是有限域的有限扩张）**。那么由引理3可知 ![\text{Gal}(K_{0}/K_{\mathfrak{p}})\simeq\text{Gal}(l_{\mathfrak{q}}/k_{\mathfrak{p}})](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28K_%7B0%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D%29%5Csimeq%5Ctext%7BGal%7D%28l_%7B%5Cmathfrak%7Bq%7D%7D%2Fk_%7B%5Cmathfrak%7Bp%7D%7D%29) 。

这里定义惯性群 ![I_{\mathfrak{q}}=\{\sigma\in\text{Gal}(L_{\mathfrak{q}}/K_{\mathfrak{p}})|~\sigma x\equiv x~\text{mod}~\mathfrak{q},\text{for all}~x\in B_{\mathfrak{q}}\} ](https://www.zhihu.com/equation?tex=I_%7B%5Cmathfrak%7Bq%7D%7D%3D%5C%7B%5Csigma%5Cin%5Ctext%7BGal%7D%28L_%7B%5Cmathfrak%7Bq%7D%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D%29%7C~%5Csigma+x%5Cequiv+x~%5Ctext%7Bmod%7D~%5Cmathfrak%7Bq%7D%2C%5Ctext%7Bfor+all%7D~x%5Cin+B_%7B%5Cmathfrak%7Bq%7D%7D%5C%7D+) ，注意到这和第一种定义是一样的。其次在这里可以把 ![I_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=I_%7B%5Cmathfrak%7Bq%7D%7D) 看作 ![\text{Ker}(\text{Gal}(L_{\mathfrak{q}}/K_{\mathfrak{p}})\rightarrow\text{Gal}(l_{\mathfrak{q}}/k_{\mathfrak{p}}))](https://www.zhihu.com/equation?tex=%5Ctext%7BKer%7D%28%5Ctext%7BGal%7D%28L_%7B%5Cmathfrak%7Bq%7D%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D%29%5Crightarrow%5Ctext%7BGal%7D%28l_%7B%5Cmathfrak%7Bq%7D%7D%2Fk_%7B%5Cmathfrak%7Bp%7D%7D%29%29) ，即可得同构：

![\text{Gal}(L_{\mathfrak{q}}/K_{\mathfrak{p}})/I_{\mathfrak{q}}=\text{Gal}(K_{0}/K_{\mathfrak{p}})\simeq\text{Gal}(l_{\mathfrak{q}}/k_{\mathfrak{p}})](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L_%7B%5Cmathfrak%7Bq%7D%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D%29%2FI_%7B%5Cmathfrak%7Bq%7D%7D%3D%5Ctext%7BGal%7D%28K_%7B0%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D%29%5Csimeq%5Ctext%7BGal%7D%28l_%7B%5Cmathfrak%7Bq%7D%7D%2Fk_%7B%5Cmathfrak%7Bp%7D%7D%29) 。

以及exact sequence： ![1\rightarrow I_{\mathfrak{q}}\rightarrow D_{\mathfrak{q}}\rightarrow \text{Gal}(l_{\mathfrak{q}}/k_{\mathfrak{p}})=\text{Gal}(l/k)\rightarrow1](https://www.zhihu.com/equation?tex=1%5Crightarrow+I_%7B%5Cmathfrak%7Bq%7D%7D%5Crightarrow+D_%7B%5Cmathfrak%7Bq%7D%7D%5Crightarrow+%5Ctext%7BGal%7D%28l_%7B%5Cmathfrak%7Bq%7D%7D%2Fk_%7B%5Cmathfrak%7Bp%7D%7D%29%3D%5Ctext%7BGal%7D%28l%2Fk%29%5Crightarrow1)

对于extension tower： ![K\subset Z_{\mathfrak{q}}\subset T_{\mathfrak{q}}\subset L](https://www.zhihu.com/equation?tex=K%5Csubset+Z_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+T_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+L) ，设 ![I_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=I_%7B%5Cmathfrak%7Bq%7D%7D) 在Galois对应下的域为 ![T_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=T_%7B%5Cmathfrak%7Bq%7D%7D) ，**由Galois基本定理立即得到：**

**![\text{Gal}(L/T_{\mathfrak{q}})=I_{\mathfrak{q}},\text{Gal}(T_{\mathfrak{q}}/Z_{\mathfrak{q}})=Z_{\mathfrak{q}}/I_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L%2FT_%7B%5Cmathfrak%7Bq%7D%7D%29%3DI_%7B%5Cmathfrak%7Bq%7D%7D%2C%5Ctext%7BGal%7D%28T_%7B%5Cmathfrak%7Bq%7D%7D%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29%3DZ_%7B%5Cmathfrak%7Bq%7D%7D%2FI_%7B%5Cmathfrak%7Bq%7D%7D) ；**

![\#\text{Gal}(T_{\mathfrak{q}}/K)=fg,\#\text{Gal}(T_{\mathfrak{q}}/Z_{\mathfrak{q}})=f](https://www.zhihu.com/equation?tex=%5C%23%5Ctext%7BGal%7D%28T_%7B%5Cmathfrak%7Bq%7D%7D%2FK%29%3Dfg%2C%5C%23%5Ctext%7BGal%7D%28T_%7B%5Cmathfrak%7Bq%7D%7D%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29%3Df) 。 

**Ex5. **只要求 ![L_{\mathfrak{q}}/K_{\mathfrak{p}}](https://www.zhihu.com/equation?tex=L_%7B%5Cmathfrak%7Bq%7D%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D) 是Galois扩张（不必Abel），也可得到 ![I_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=I_%7B%5Cmathfrak%7Bq%7D%7D) 是 ![\text{Gal}(L_{\mathfrak{q}}/K_{\mathfrak{p}})](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L_%7B%5Cmathfrak%7Bq%7D%7D%2FK_%7B%5Cmathfrak%7Bp%7D%7D%29) 的正规子群。

与讨论分解群的时候一样，我们考察 ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 在 ![K\subset Z_{\mathfrak{q}}\subset T_{\mathfrak{q}}\subset L](https://www.zhihu.com/equation?tex=K%5Csubset+Z_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+T_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+L) 中的分解过程，设 ![\mathfrak{q} \cap T_{\mathfrak{q}}=\mathfrak{q}_{T}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D+%5Ccap+T_%7B%5Cmathfrak%7Bq%7D%7D%3D%5Cmathfrak%7Bq%7D_%7BT%7D)。

**Claim2. **![e(\mathfrak{q}/\mathfrak{q}_{T})=e,f(\mathfrak{q}/\mathfrak{q}_{T})=1;e(\mathfrak{q}_{T}/\mathfrak{q}_{Z})=1,f(\mathfrak{q}_{T}/\mathfrak{q}_{Z})=f](https://www.zhihu.com/equation?tex=e%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BT%7D%29%3De%2Cf%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BT%7D%29%3D1%3Be%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29%3D1%2Cf%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29%3Df) 。

**Proof. **首先我们考察 ![ Z_{\mathfrak{q}}\subset T_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=+Z_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+T_%7B%5Cmathfrak%7Bq%7D%7D) 中 ![\mathfrak{q}_{T}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D_%7BT%7D) 的分解群 ![D^{\prime}_{\mathfrak{q}_{T}}](https://www.zhihu.com/equation?tex=D%5E%7B%5Cprime%7D_%7B%5Cmathfrak%7Bq%7D_%7BT%7D%7D)，由定义可知其分解群即 ![\text{Gal}(T_{\mathfrak{q}}/Z_{\mathfrak{q}})](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28T_%7B%5Cmathfrak%7Bq%7D%7D%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29)。即**![\mathfrak{q}_{Z}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D_%7BZ%7D) 在 ![T_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=T_%7B%5Cmathfrak%7Bq%7D%7D) 中不分裂(分裂已在 ![K\subset Z_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=K%5Csubset+Z_%7B%5Cmathfrak%7Bq%7D%7D) 完成），于是：**

![e(\mathfrak{q}_{T}/\mathfrak{q}_{Z})f(\mathfrak{q}_{T}/\mathfrak{q}_{Z})\cdot1=[T_{\mathfrak{q}}:Z_{\mathfrak{q}}]=f](https://www.zhihu.com/equation?tex=e%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29f%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29%5Ccdot1%3D%5BT_%7B%5Cmathfrak%7Bq%7D%7D%3AZ_%7B%5Cmathfrak%7Bq%7D%7D%5D%3Df) 。

其次，考虑 ![T_{\mathfrak{q}}\subset L](https://www.zhihu.com/equation?tex=T_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+L) 中 ![\mathfrak{q}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D) 的惯性群 ![I^{\prime}_{\mathfrak{q}}=\text{Ker}(D^{\prime}_{\mathfrak{q}}\rightarrow\text{Gal}(l/l({\mathfrak{q}_{T}})))](https://www.zhihu.com/equation?tex=I%5E%7B%5Cprime%7D_%7B%5Cmathfrak%7Bq%7D%7D%3D%5Ctext%7BKer%7D%28D%5E%7B%5Cprime%7D_%7B%5Cmathfrak%7Bq%7D%7D%5Crightarrow%5Ctext%7BGal%7D%28l%2Fl%28%7B%5Cmathfrak%7Bq%7D_%7BT%7D%7D%29%29%29) ，这里 ![l({\mathfrak{q}_{T}})](https://www.zhihu.com/equation?tex=l%28%7B%5Cmathfrak%7Bq%7D_%7BT%7D%7D%29) 是 ![T_{\mathfrak{q}}/K](https://www.zhihu.com/equation?tex=T_%7B%5Cmathfrak%7Bq%7D%7D%2FK) 的剩余域；以及 ![K\subset L](https://www.zhihu.com/equation?tex=K%5Csubset+L) 中 ![{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%7B%5Cmathfrak%7Bq%7D%7D) 的惯性群 ![I_{\mathfrak{q}}=\text{Ker}(D_{\mathfrak{q}}\rightarrow\text{Gal}(l/k))](https://www.zhihu.com/equation?tex=I_%7B%5Cmathfrak%7Bq%7D%7D%3D%5Ctext%7BKer%7D%28D_%7B%5Cmathfrak%7Bq%7D%7D%5Crightarrow%5Ctext%7BGal%7D%28l%2Fk%29%29) ，**根据定义可知二者相等：**

![T_{\mathfrak{q}}\subset L](https://www.zhihu.com/equation?tex=T_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+L) 中 ![{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%7B%5Cmathfrak%7Bq%7D%7D) 的分解群 ![D^{\prime}_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=D%5E%7B%5Cprime%7D_%7B%5Cmathfrak%7Bq%7D%7D) 是 ![\text{Gal}(L/T_{\mathfrak{q}})\subset\text{Gal}(L/Z_{\mathfrak{q}})=D_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L%2FT_%7B%5Cmathfrak%7Bq%7D%7D%29%5Csubset%5Ctext%7BGal%7D%28L%2FZ_%7B%5Cmathfrak%7Bq%7D%7D%29%3DD_%7B%5Cmathfrak%7Bq%7D%7D) 中固定 ![\mathfrak{q}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bq%7D) 的元素，即 ![\text{Gal}(L/T_{\mathfrak{q}})](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28L%2FT_%7B%5Cmathfrak%7Bq%7D%7D%29) ，于是 ![I^{\prime}_{\mathfrak{q}}=\text{Ker}(I_{\mathfrak{q}}\rightarrow\text{Gal}(l/l({\mathfrak{q}_{T}})))=I_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=I%5E%7B%5Cprime%7D_%7B%5Cmathfrak%7Bq%7D%7D%3D%5Ctext%7BKer%7D%28I_%7B%5Cmathfrak%7Bq%7D%7D%5Crightarrow%5Ctext%7BGal%7D%28l%2Fl%28%7B%5Cmathfrak%7Bq%7D_%7BT%7D%7D%29%29%29%3DI_%7B%5Cmathfrak%7Bq%7D%7D) 。而**Ex4**告诉我们 ![I_{\mathfrak{q}}\rightarrow\text{Gal}(l/l({\mathfrak{q}_{T}}))](https://www.zhihu.com/equation?tex=I_%7B%5Cmathfrak%7Bq%7D%7D%5Crightarrow%5Ctext%7BGal%7D%28l%2Fl%28%7B%5Cmathfrak%7Bq%7D_%7BT%7D%7D%29%29) 是满射，因此 ![\text{Gal}(l/l({\mathfrak{q}_{T}}))=\{1\}](https://www.zhihu.com/equation?tex=%5Ctext%7BGal%7D%28l%2Fl%28%7B%5Cmathfrak%7Bq%7D_%7BT%7D%7D%29%29%3D%5C%7B1%5C%7D) 。

于是 ![T_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=T_%7B%5Cmathfrak%7Bq%7D%7D) 和 ![L](https://www.zhihu.com/equation?tex=L) 的剩余域相同： ![l=l({\mathfrak{q}_{T}})](https://www.zhihu.com/equation?tex=l%3Dl%28%7B%5Cmathfrak%7Bq%7D_%7BT%7D%7D%29) ，![f(\mathfrak{q}_{T}/\mathfrak{p})=f(\mathfrak{q}/\mathfrak{p})=f](https://www.zhihu.com/equation?tex=f%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bp%7D%29%3Df%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bp%7D%29%3Df) 。那么：

![f(\mathfrak{q}_{T}/\mathfrak{p})=f(\mathfrak{q}_{T}/\mathfrak{q}_{Z})f(\mathfrak{q}_{Z}/\mathfrak{p})=f(\mathfrak{q}_{T}/\mathfrak{q}_{Z})=f](https://www.zhihu.com/equation?tex=f%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bp%7D%29%3Df%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29f%28%5Cmathfrak%7Bq%7D_%7BZ%7D%2F%5Cmathfrak%7Bp%7D%29%3Df%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29%3Df) ；

![e(\mathfrak{q}_{T}/\mathfrak{q}_{Z})=1](https://www.zhihu.com/equation?tex=e%28%5Cmathfrak%7Bq%7D_%7BT%7D%2F%5Cmathfrak%7Bq%7D_%7BZ%7D%29%3D1)，![f(\mathfrak{q}/\mathfrak{q}_{T})=1](https://www.zhihu.com/equation?tex=f%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BT%7D%29%3D1) ， ![e(\mathfrak{q}/\mathfrak{q}_{T})=e](https://www.zhihu.com/equation?tex=e%28%5Cmathfrak%7Bq%7D%2F%5Cmathfrak%7Bq%7D_%7BT%7D%29%3De) 。

总结下来就是， ![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D) 在 ![K\subset Z_{\mathfrak{q}}\subset T_{\mathfrak{q}}\subset L](https://www.zhihu.com/equation?tex=K%5Csubset+Z_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+T_%7B%5Cmathfrak%7Bq%7D%7D%5Csubset+L) 中的分解过程为：

![\mathfrak{p}](https://www.zhihu.com/equation?tex=%5Cmathfrak%7Bp%7D)**先在 **![Z_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=Z_%7B%5Cmathfrak%7Bq%7D%7D)** 中分裂成 **![g](https://www.zhihu.com/equation?tex=g)** 个不同素理想，但剩余域不发生扩张，也不发生分歧；**

**这**![g](https://www.zhihu.com/equation?tex=g)** 个不同素理想在**![T_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=T_%7B%5Cmathfrak%7Bq%7D%7D)**中发生次数为 ![f](https://www.zhihu.com/equation?tex=f) 的剩余域扩张，但仍然不发生分歧；**

![T_{\mathfrak{q}}](https://www.zhihu.com/equation?tex=T_%7B%5Cmathfrak%7Bq%7D%7D)**中**![g](https://www.zhihu.com/equation?tex=g)** 个不同素理想在 ![L](https://www.zhihu.com/equation?tex=L) 中发生分歧。**

**即分裂、剩余域扩张、分歧分别只在第一次、第二次、第三次扩张中发生。**
