好了好了，有了这个神奇的网站，今天我们来解决第一大问题，让我们和基于汉语拼音的变量名系统说再见吧。
![](https://pic4.zhimg.com/v2-d97268e07c4e89005bd40964ae5dfabb_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='258'></svg>)- 网站地址：[https://unbug.github.io/codelf/](https://link.zhihu.com/?target=https%3A//unbug.github.io/codelf/)
- GitHub 链接：[https://github.com/unbug/codelf#codelf-for-vs-code](https://link.zhihu.com/?target=https%3A//github.com/unbug/codelf%23codelf-for-vs-code)

我们做了一些尝试，比如在搜索框中键入中文「残差」，搜索框下方会出现 Suggestions，列出了多个英文翻译。Suggestions 下方那片「五彩斑斓」默认为是包含第一个建议英文翻译的所有变量名和函数名。
![](https://pic3.zhimg.com/v2-4d81100c6a510897648485f7a4ea310a_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='331'></svg>)
点击其他建议翻译，下方「五彩斑斓」也会随之改变。比如点击「error」，搜索结果的变化如下所示。
![](https://pic3.zhimg.com/v2-e032f71f03fb53d7efbdc3df8b4f8a72_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='383'></svg>)
你也可以选择自己喜欢的编程语言，比如 Python。
![](https://pic4.zhimg.com/v2-d38c8abd4029a2df4ff5ba9a4a32c48f_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='352'></svg>)
看看搜索结果的变化。
![](https://pic3.zhimg.com/v2-a154a050b572a5a6844d12f3f2a01ed6_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='252'></svg>)
将鼠标指针放置在某个彩色框上，会出现「Search」、「Repo」、「Copy」、「Codes」四种选项。用户可以点击这些选项，使用对应功能。
![](https://pic4.zhimg.com/v2-206c2047b2b97cb3a6c7137089f4074f_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='247'></svg>)
点击「Search」就是基于当前命名搜索其它相关的命名。比如，在「residuals」上点击「Search」后结果如下所示。
![](https://pic4.zhimg.com/v2-92b96bad52a41529c6b933acf006a99f_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='147'></svg>)
点击「Repo」就是链接到使用该命名的代码所在的资源库。
![](https://pic2.zhimg.com/v2-52f1147193f7d952aae223c9618e1df1_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='518'></svg>)
点击「Copy」是复制该命名。

点击「Codes」可以查看使用命名的示例代码。
![](https://pic1.zhimg.com/v2-ca00530834a25c41bd61b0cd0e38dfdc_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='946' height='715'></svg>)
可以根据需要选择不同语言的示例代码。

据作者介绍，Codelf 的内部原理是：先到有道翻译 API 取中文的英文翻译，然后搜索 GitHub 取变量。该网站支持直接搜索中文，在搜索框键入中文后，Codelf 会提供查好的单词及其近义词，然后通过搜索 Github、Bitbucket、Google Code、Codeplex、Sourceforge、Fedora Project 上的开源项目的源码匹配出与这些词汇相关的变量名和函数名。当然，直接输入英文也是可以的，Codelf 会提供相关的命名。
![](https://pic4.zhimg.com/v2-f7d49f55dbb6db9e4faea3f446c902e7_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='580' height='442'></svg>)
此外，Codelf 网站还可以管理你的 GitHub 项目，点击网站右上方左侧图标，登录 GitHub 账户，管理此用户名下的 Star 和 Repo。
![](https://pic3.zhimg.com/v2-eb06f663f3ef64d7493ea0df6155ae86_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='633'></svg>)
