# 谷歌新手机 Pixel3 终于发布! 但「相框」智能音箱有点出乎意料 - 知乎
# 



> 谷歌做硬件没有优势，但他擅长将人的注意力从硬件上挪开。

**撰文 | 宇多田**

对于很多开发者来说，每年春季举行谷歌 I/O 开发者大会或许更富盛名，但自 2016 年开始，每年于秋季举行的硬件发布会，才是让谷歌「走下神坛，深入群众」的最佳舞台。

被评价为「平庸之作」，且销量成迷的 Pixel2；一直在跟随亚马逊 Echo 脚步的智能音箱 Google Home……这些产品让一些人觉得谷歌的硬件水平太过寻常，甚至连像大众乐于给予苹果手机的「批判声」都少有。

但是，只有拿到手才能发现，在谷歌发布的每一件产品中，你都窥探到这家公司对技术细节的想象力。

譬如，谷歌手机 Pixel2 上的「拍照自动去前景」这个功能就有点意思： 
![](https://pic3.zhimg.com/v2-a49bc799ed41042b4f5400c358592dfe_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='722'></svg>)左边为原图，右边为去掉玻璃遮挡后的效果
再譬如，Google 输入法会帮你「马赛克」掉所有打出来的脏话：
![](https://pic2.zhimg.com/v2-436ae035989d366d82b3be100c33bccd_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='958'></svg>)
不过，除了想象力，谷歌也略带一些工程师文化中不可或缺的「傻气」（谁知道是不是故意泄露的）——这次还没开大会之前，关于新一代手机 Pixel3 的消息就几乎被泄露完了。

除了手机，一些配件的信息也陆续被曝光。

譬如新一代数字电视棒 Chromecast，据说已经被百思买网站代替旧版卖给了消费者，还被收到货的网友把图片晒在了 Reddit 的论坛里……

而外媒 Verge 对此的评价是：「谷歌应该是历史上新品信息被泄露的最全的公司。」

所以，今晚的硬件大会，更像是对着答案复查一下自己的试卷，看看有没有爆料者错过，而我们又觉得有些惊喜的漏网之鱼。

而事实证明，谷歌又一次跟我们开了一次玩笑——「大件」（手机+智能音箱+平板电脑）全中，而小配件……除了无线充电器，其它一件都没有发布。
![](https://pic1.zhimg.com/v2-0e06ded661a7934259f5a7492665d2f8_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='518'></svg>)
这次发布会依然秉承了前两届朴素简洁的作风——展示空间很像一个大学里的普通报告厅，大概仅能容纳 200~300 人（感觉是经费不足）。

而除了第一届谷歌 CEO 劈柴兄曾到场支持做串场嘉宾外，此后都是由谷歌的硬件团队负责硬件新品的全程讲解与演示。这次的主持兼串场人，仍然是 Rick Osterloh，谷歌硬件高级副总裁。

这位已经在谷歌呆了将近 30 个月的前摩托罗拉总裁，一上任就被赋予了艰巨的任务——向外界展示公司对硬件的态度是严肃而认真的。

因此，在他的监督下，谷歌产品品类开始变丰富的同时，也正在积极拓展亚洲市场——在印度陆续设立的线下门店也许可以证明这一点。
![](https://pic1.zhimg.com/v2-e6221e631e5ee4fd40baa71a95feb374_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='804'></svg>)Rick Osterloh
尽管开场要多煽情、讲讲公司做产品秉承的精神与信念，过去取得的成绩，几乎是每一家硬件厂商的惯例，但 Osterloh 仍然把更多时间都花在了对软件的「歌颂」上：

Google 翻译、Google Assistant（语音助手）、Google 搜索、Google 地图以及 Google Lens ……明明都是一些让人不能再眼熟的软件。

但他认为这些软件能够凸显谷歌硬件家族的与众不同，也能顺便引出本次硬件大会的主题——

谷歌硬件究竟是什么？AI+软件+硬件。

这不是一个新主题，至少 Osterloh 已经在公开场合讲了一年多。为了提升谷歌的硬件销售成绩与生产规模，这位出身传统硬件行业的工程师一直都在试图如何将谷歌的核心优势应用到硬件产品上。

没错，就像文章开头所举例的，谷歌的优势不是设计昂贵好看的外观，而是将技术巧妙而又平滑地融入到产品中。

当然，你可能会说Pixel3 也不能免俗地被换上 OLED 显示屏，但有媒体爆料，这个要求其实也是根据用户的强烈要求进行改进的，因为去年发布 Pixel2 时就有不少人抱怨屏幕使用感太差。

「我们现在在硬件上优势吗？答案是否定的。」Osterloh 在一次采访中承认，「但我们擅长让人把注意力从硬件上转移开。」
![](https://pic4.zhimg.com/v2-5b14319c003ab7a51b06251844d362cb_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='480'></svg>)
## **Pixel3**

这部手机其实是发布会上的最后一个产品，显然被谷歌拿来做「压轴」。

实际上，如果仅从外观来看，亮点不仅不多，而且都是很多手机厂商用过的「老梗」。

譬如采用了 OLED 全面屏，譬如增加了一种被称为「砂粉」的颜色，再譬如前置摄像头变成了两个（一个标准，一个广角，都是 800 万像素）。
![](https://pic2.zhimg.com/v2-a6ba347bda9d5fb37ff48d3e0beb33f5_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='633'></svg>)
由于有两个前置摄像头，因此不可避免会有「刘海」

而在壳子里面，Pixel 3 和 Pixel 3 XL 都配备了高通公司的 10nm Snapdragon 845 处理器，以及最新的安全芯片 Titan M；

这两款手机也是第一次支持无线充电，而 Pixel Stand 就是谷歌刚刚推出的无线充电器。
![](https://pic2.zhimg.com/v2-f3aa87c69bfa9a861732a3c657c18f95_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='609'></svg>)Pixel Stand
除此以外，这部手机的硬件配置就跟 Pixel2 没有太大区别了——仍然有两个尺寸（分别是 5.5 英寸与 6.3 英寸，比 Pixel 稍大一些），仍然只是后置单摄（1220 万像素）。
![](https://pic3.zhimg.com/v2-d374975bd85090210f7d8ecb978109be_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='408'></svg>)
不过 Osterloh 在发布会开场时就强调，Pixel3 的拍摄效果较上一代又有了之显著提升。因为其更多是在依靠算法与谷歌图片搜索提供的大量数据，而不是通过硬件来进行图像校准与分析工作。

「Pixel 的摄像头会『学习』图像被捕捉的角度、地理位置等多种信息，我们可以根据这些信息反向追踪到谷歌图片中的大量数据。通过大量训练，摄像头的『专业性』会达到新的高度。」
![](https://pic3.zhimg.com/v2-ad8cb4cb2b93f05ab4e6c186582995e6_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='739'></svg>)
因此，利用算法及配置双向升级后的摄像头，工程师在现场介绍并演示了四种新增的摄像功能：
- Top Shot（最佳瞬间）即便你按下快门的时间不是很理想，摄像头也会自动选择拍照的最佳时刻。
- Super Res Zoom（超级变焦）即便你拉近镜头，也可以把照片拍的非常清楚。据谷歌称这一功能使用了与 NASA 拍摄火星表面相同的技术。
- Night Sight（夜拍）可以在低光环境中拍出清晰的照片，这个功能小米等手机也已经实现。
- Wide Selfies（广域自拍）在有了两个前置镜头后，你与多个朋友以及背后的景色可以全部装进狭窄的屏幕中，这在一些手机品牌中也已实现。

不过值得高兴的是，Google Lens 这一让机器能够「看图说话」的智能应用，也终于集成到了 Pixel3 的摄像头里。

举个例子，如果现在你把 Pixel 3的摄像头对准一张拍卖海报，那么你就会立刻拿到电话号码与地址这类文本信息。

而另外一个非常有意思的功能叫做 Call Screen feature。

这个功能会接听自动屏蔽的电话，帮你询问对方是谁，为什么要打来。而这段对话会被实时转录到手机屏幕上。
![](https://pic4.zhimg.com/v2-9be335de52f5ecf0fa330764fc7e60bb_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='751'></svg>)
## **Google Home Hub**

给智能音箱装上屏幕，这也不是一个新鲜的概念。

至少亚马逊的带屏音箱已经出了两款，而国内也有包括百度在内的多家公司推出了带屏产品。

但谷歌的这款音箱，从内而外几乎就是一个会说话的「相框」。
![](https://pic2.zhimg.com/v2-a53417a9208d1d2e2ec39fff6f41b2c1_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='782' height='445'></svg>)
价格149美元，内置两个远场麦克风与一个扬声器驱动程序

「这个外形是设计团队受到了相册的启发，」谷歌的产品管理主管 Micah Collins 表示，无论是白色边框轻柔的弧度，还是顶部内置的 RGB 色彩传感器，都是为了更好地展示照片而设计的。

「传感器可以自动调整屏幕的亮度和温度，让整个设备更加自然地融合到所在的环境中。」

为了能够让照片更加投用户所好，工程师将谷歌照片的 app 与 Hub 进行了打通，并针对 Hub 推出了两个新的照片功能——

Live Albums（动态相册）与 Recent Highlights（近期精彩瞬间）。
![](https://pic1.zhimg.com/v2-20775d644cfd51c2016d79cfd74a1218_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='675'></svg>)
其中，Live Albums 也应用了AI技术。

首先，你需要选择出最想让其出现在屏幕相册中的人。假如你选择了父母，那么谷歌就会从你的照片中筛选出所有与你父母相关的照片，并分享到你的音箱主屏幕上。

**而 Recent Hightlights，则能够帮你展示出最近在谷歌图片账户中刚刚更新的照片。**

不过，即便再像相框，它的灵魂也还是一台智能音箱。

谷歌工程师把他比作是一个附身于相框的 Google Assistant（语音助手），不仅能与谷歌所有常用的应用程序相连接，也扮演着「中枢控制中心」的角色。

「在 Hub 的屏幕上，你可以直接点击谷歌搜索、图片、Youtube、Duo（语音聊天）、谷歌日历以及地图等应用。

当然，除了可以连接 Google Home，你可以用它来控制所有 Nest 的智能家居产品。」谷歌家居产品的另一个产品经理 Ashton Udall 表示，

「Hub 的本质是谷歌助手的延伸，但加强了视觉层面的互动。实际上，它不仅仅是设备的控制中心，也是包括 Youtube 等视频软件的重要入口。」
![](https://pic3.zhimg.com/v2-02ae5ed513edd1c1eab01f96c51fb3fa_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='620'></svg>)
## **Google Slate**

早在会前爆料中，Google Slate 就被誉为是让此次发布会中最有意思的一项产品。
![](https://pic4.zhimg.com/v2-51b71a78812fc9254ec4822479752fc7_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='770'></svg>)12.3 英寸，3000 x 2000 分辨率的液晶显示屏
因为自谷歌发布基于安卓系统的旗舰级平板电脑 Pixel C 已经过去了三年，而该产品在去年年底就被下架。

因此，很多人都在猜测这个产品线将会以怎样的形式被重新启动。

从发布会的演示图来看，它外表就是一个普通的平板电脑，既有可拆卸键盘，又有一个前盖，这样的设计让我们感到似曾相识。

譬如有同事就认为这款平板电脑更像是介于苹果 iPad Pro 与微软平板电脑 Surface Pro之间的一个产品。
![](https://pic4.zhimg.com/v2-4d0a1dbd0a97c71538ebc938f93f718f_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='800' height='450'></svg>)左为Surface Pro，右为iPad Pro
这个 12.3 寸平板电脑四周配有两个 USB-C 端口，一个自定义键盘连接器与两个麦克风。此外，在电源按钮上也有一个指纹识别，方便登录电脑。

不过，它没有 SD 卡槽，也没有耳机孔，这对于很多人来说可能是坏消息。

但更值得注意的是，Slate 运行的是 Chrome OS 系统而非安卓系统，基本上具有计算机的内核，包括两种英特尔处理器以及自研的安全芯片 Titan C。
![](https://pic3.zhimg.com/v2-ee585990bc628dedbf7faf9021f691fe_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='664'></svg>)
基本型号为 599 美元，采用第八代赛扬处理器，4 GB 内存和 32 GB 的存储空间；升级型号为 999 美元，采用 Core i5 处理器，8GB 内存与 128G 存储空间

显然，尽管安卓在手机上取得了成功，但在平板电脑上的竞争力还是很有限的。

因此，谷歌或许将 Slate 作为一个全新的开始，也是大力推动 Chrome 操作系统的开始。

这意味着它将不仅仅是发烧友们聚集的平台；也标志着谷歌终于开始认真对待 Windows 和 iPad。
![](https://pic3.zhimg.com/v2-68def3bfa3bb16171da628b95be772fe_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='720'></svg>)
近年来，科技巨头们一直在寻找笔记本电脑这条产品线的后续发力点。

譬如，苹果公司一方面在为 iPad 持续增加多任务功能，同时也在试图恢复 Mac 的活力；

而微软则一直致力于使 Windows 系列硬件更加轻便，并在积极为软件寻找一些新形式。

现在，谷歌正以自己对计算机的想象力，跳入这场漩涡的中心。

在过去四年里，比较意外的是，随着 Chrome OS 在校园中的普及，谷歌将有机会突破单一的市场份额。

这可能意味着，我们即将看到一个发生在 Windows，iOS，macOS 和 ChromeOS 之间的竞争，也就是所谓的技术平台之争。
![](https://pic4.zhimg.com/v2-5a750d67153b619421468dac992ef1ff_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='675'></svg>)
从这场发布会整体来看，硬件的配置并不出乎意料，也没有什么亮眼之处，但产品的部分软件设计的确能让人眼前一亮。

那么你对于这场发布会的产品会做出何种评价？
