*本文作者在 Github 上建立了一个代码速查表，对机器学习初学者来说是不可多得的一个资源。*



对于初学者来讲，入门机器学习和深度学习非常困难；同时深度学习库也难以理解。通过收集多方资源，我在 Github 上创建了一个速查表库，希望能对你有所帮助。欢迎访问这个库，并完善它（如果你也有速查表）。

项目地址：[kailashahirwar/cheatsheets-ai](https://link.zhihu.com/?target=https%3A//github.com/kailashahirwar/cheatsheets-ai)


**1.Keras**


Keras 是一个非常强大且容易上手的深度学习库；当 Keras 接入 Theano 和 TensorFlow 时，后两者可提供高水平的神经网络 API 以开发和评估深度学习模型。该速查表可用于 Python 数据科学和机器学习。

**2. Numpy**


Numpy 库是 Python 中科学性计算的核心库，它提供高性能、多维度的数组对象，以及对这些数组进行运算的工具。该速查表可用于 Python 数据科学和机器学习。

![](https://pic3.zhimg.com/v2-a3f08effc3e5a62d80737b3e1ace4842_b.png)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1753' height='1240'></svg>)
**3. Pandas**


这张 Pandas 速查表主要用于数据整理（data wrangling）。

![](https://pic3.zhimg.com/v2-6214dbcfc6fc4af8b6db0355beace77a_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='3056' height='2362'></svg>)![](https://pic2.zhimg.com/v2-dfe9931ccdde8e8ac9c0b9e3b88a5535_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='3056' height='2362'></svg>)
Pandas 库构建在 NumPy 上，并为 Python 编程语言提供易于上手的数据结构和数据分析工具。该速查表可用于 Python 数据科学和机器学习。

![](https://pic4.zhimg.com/v2-d9126924c722f32b146eb14e0b33ca03_b.png)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1753' height='1240'></svg>)
**4. SciPy**


SciPy 库是科学性计算的核心包之一，科学性计算可提供数学算法和构建在 Python 的 NumPy 扩展上的便捷函数 (Convenience Functions) ；该表中还包括线性代数的部分。该速查表可用于 Python 数据科学和机器学习。

![](https://pic1.zhimg.com/v2-f601737ba7e4f441dc9b56e026c9a014_b.png)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1753' height='1240'></svg>)
**5. Matplotlib**


Matplotlib 是一个 Python 2D 绘图库，它在平台上以多种硬拷贝格式和交互环境生成发表品质的图。该速查表用于 Python 数据科学。

![](https://pic1.zhimg.com/v2-ca4c69ecea600576d1447823cd291e08_b.png)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1753' height='1240'></svg>)
**6. Scikit-learn**


Scikit-learn 是一个开源 Python 库，通过统一接口实现一系列的机器学习、预处理、交叉验证和视觉化算法。该速查表可用于 Python 数据科学和机器学习。

**7. Neural Networks Zoo（神经网络大全）**


该速查表几乎涵盖了所有类型的神经网络。

![](https://pic2.zhimg.com/v2-81c72e30357c7d78dcbf3d514a4e2481_b.png)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='2000' height='3000'></svg>)
**8. ggplot2**


ggplot2 基于图形语法，其思想是你可以利用相同的几个组件构建所有的图形：一个数据集、一个几何集（表征数据点的视觉化标记）和一个协作系统。该速查表用于数据可视化。
