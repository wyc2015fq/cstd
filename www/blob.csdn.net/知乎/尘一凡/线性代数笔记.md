# 线性代数笔记 - 知乎
# 

**定义（不变子空间）**  设 ![\mathcal{A}:V\rightarrow V](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%3AV%5Crightarrow+V) 是线性变换， ![U](https://www.zhihu.com/equation?tex=U) 是 ![V](https://www.zhihu.com/equation?tex=V) 的子空间。如果对 ![\forall \alpha \in U](https://www.zhihu.com/equation?tex=%5Cforall+%5Calpha+%5Cin+U) 都有 ![\mathcal{A} (\alpha) \in U](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D+%28%5Calpha%29+%5Cin+U) ，即 ![\mathcal{A} ( U ) \subseteq U](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D+%28+U+%29+%5Csubseteq+U) ，那么 ![U ](https://www.zhihu.com/equation?tex=U+) 叫做 ![\mathcal{A}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D) 的*不变子空间。*
- ![\mathcal{A}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D) 有两个平凡的不变子空间。由定义， ![\mathcal{A}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D) 是 ![V](https://www.zhihu.com/equation?tex=V) 到 ![V](https://www.zhihu.com/equation?tex=V) 的线性变换，故![\mathcal{A}(V)\subseteq V](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%28V%29%5Csubseteq+V)， 故![V](https://www.zhihu.com/equation?tex=V)是 ![\mathcal{A}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D) 是不变子空间。另外对 ![V](https://www.zhihu.com/equation?tex=V) 的零子空间 ![\{ \bf{0} \}](https://www.zhihu.com/equation?tex=%5C%7B+%5Cbf%7B0%7D+%5C%7D) ， ![\mathcal{A}(\bf{0})=\bf{0}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%28%5Cbf%7B0%7D%29%3D%5Cbf%7B0%7D) ，故零子空间 ![\{ \bf{0} \}](https://www.zhihu.com/equation?tex=%5C%7B+%5Cbf%7B0%7D+%5C%7D) 也是 ![\mathcal{A}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D) 的不变子空间。
- ![\text{Im}(\mathcal{A}) ](https://www.zhihu.com/equation?tex=%5Ctext%7BIm%7D%28%5Cmathcal%7BA%7D%29+) ( =![\mathcal{A}(V)](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%28V%29) )和  ![\text{Ker}(\mathcal{A})](https://www.zhihu.com/equation?tex=%5Ctext%7BKer%7D%28%5Cmathcal%7BA%7D%29) 也是 ![\mathcal{A}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D) 的不变子空间。由于 ![\mathcal{A}(V)\subseteq V](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%28V%29%5Csubseteq+V) ,![\mathcal{A}(\mathcal{A}(V))\subseteq \mathcal{A}(V)](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%28%5Cmathcal%7BA%7D%28V%29%29%5Csubseteq+%5Cmathcal%7BA%7D%28V%29) ，因此 ![\text{Im}(\mathcal{A})](https://www.zhihu.com/equation?tex=%5Ctext%7BIm%7D%28%5Cmathcal%7BA%7D%29) 是 ![\mathcal{A}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D) 的不变子空间（解释：A作用于V得到A的像，而A的像仍是V一部分，故A作用于A的像仍落在A的像空间中）。设 ![\alpha \in \text{Ker}(\mathcal{A})](https://www.zhihu.com/equation?tex=%5Calpha+%5Cin+%5Ctext%7BKer%7D%28%5Cmathcal%7BA%7D%29) ，即 ![\mathcal{A}(\alpha)=0](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%28%5Calpha%29%3D0) ，故 ![\mathcal{A}(\mathcal{A}(\alpha))=0](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%28%5Cmathcal%7BA%7D%28%5Calpha%29%29%3D0) ，即 ![\mathcal{A}(\alpha) \in \text{Ker}(\mathcal{A})](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D%28%5Calpha%29+%5Cin+%5Ctext%7BKer%7D%28%5Cmathcal%7BA%7D%29) ，因此 ![\text{Ker}(\mathcal{A})](https://www.zhihu.com/equation?tex=%5Ctext%7BKer%7D%28%5Cmathcal%7BA%7D%29) 是 ![\mathcal{A}](https://www.zhihu.com/equation?tex=%5Cmathcal%7BA%7D) 的不变子空间。
