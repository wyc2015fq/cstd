# 1.1 Ring和Algebra - 知乎
# 

本节有两个主题，一是介绍集合列（sequence of sets）的极限，二是介绍代数结构——环（ring）和代数（algebra）。为什么介绍集合列的极限？因为在实变函数的证明中，构造集合列及其极限是重要的技巧。为什么介绍环和代数？因为可测集构成一个 ![\sigma](https://www.zhihu.com/equation?tex=%5Csigma) 代数。

**术语说明：**

![X](https://www.zhihu.com/equation?tex=X) :表示全集，更形象地称为“空间”， ![X](https://www.zhihu.com/equation?tex=X) 中的元素叫做点。

![E](https://www.zhihu.com/equation?tex=E) :表示 ![X](https://www.zhihu.com/equation?tex=X) 的子集，即 ![E\subseteq X](https://www.zhihu.com/equation?tex=E%5Csubseteq+X) 。

集族：集合的集合，即以集合作为元素构成的集合，也即 ![E_\alpha \in](https://www.zhihu.com/equation?tex=E_%5Calpha+%5Cin) 集族（其中![E_\alpha \subseteq X](https://www.zhihu.com/equation?tex=E_%5Calpha+%5Csubseteq+X) ）。

幂集 ![\mathscr{P}(X)](https://www.zhihu.com/equation?tex=%5Cmathscr%7BP%7D%28X%29) : ![\mathscr{P}(X)=\{](https://www.zhihu.com/equation?tex=%5Cmathscr%7BP%7D%28X%29%3D%5C%7B)![X](https://www.zhihu.com/equation?tex=X) 中所有子集 ![\}](https://www.zhihu.com/equation?tex=%5C%7D) ，即 ![X](https://www.zhihu.com/equation?tex=X) 所有集合作为元素构成幂集。

**Section 0 数列的极限**

为方便理解集合列的极限，先来看一下数列的上下极限。设有数列 ![\{ x_n\} ](https://www.zhihu.com/equation?tex=%5C%7B+x_n%5C%7D+) ,

数列的上极限定义为 ![\[\mathop {\lim \sup x_n}\ (= \overline {\mathop {\lim }\limits_{n \to \infty } } {\rm{ }}{x_n} )= \mathop {\inf }\limits_{n \ge 1} {\rm{ }}\mathop {{\rm{sup}}}\limits_{m \ge n} {\rm{ }}{x_m}\]](https://www.zhihu.com/equation?tex=%5C%5B%5Cmathop+%7B%5Clim+%5Csup+x_n%7D%5C+%28%3D+%5Coverline+%7B%5Cmathop+%7B%5Clim+%7D%5Climits_%7Bn+%5Cto+%5Cinfty+%7D+%7D+%7B%5Crm%7B+%7D%7D%7Bx_n%7D+%29%3D+%5Cmathop+%7B%5Cinf+%7D%5Climits_%7Bn+%5Cge+1%7D+%7B%5Crm%7B+%7D%7D%5Cmathop+%7B%7B%5Crm%7Bsup%7D%7D%7D%5Climits_%7Bm+%5Cge+n%7D+%7B%5Crm%7B+%7D%7D%7Bx_m%7D%5C%5D) , 

数列的下极限定义为 ![\[\mathop {\lim \inf x_n}\ (=\mathop {\underline {\lim } }\limits_{n \to \infty } {\rm{ }}{x_n} )= \mathop {\sup }\limits_{n \ge 1} {\rm{ }}\mathop {\inf }\limits_{m \ge n} {\rm{ }}{x_m}\]](https://www.zhihu.com/equation?tex=%5C%5B%5Cmathop+%7B%5Clim+%5Cinf+x_n%7D%5C+%28%3D%5Cmathop+%7B%5Cunderline+%7B%5Clim+%7D+%7D%5Climits_%7Bn+%5Cto+%5Cinfty+%7D+%7B%5Crm%7B+%7D%7D%7Bx_n%7D+%29%3D+%5Cmathop+%7B%5Csup+%7D%5Climits_%7Bn+%5Cge+1%7D+%7B%5Crm%7B+%7D%7D%5Cmathop+%7B%5Cinf+%7D%5Climits_%7Bm+%5Cge+n%7D+%7B%5Crm%7B+%7D%7D%7Bx_m%7D%5C%5D) 。

**例子1**  如图，蓝色点为数列 ![\{ x_n\}](https://www.zhihu.com/equation?tex=%5C%7B+x_n%5C%7D) 。数列的上极限是怎么得到的呢？先**固定**一个 ![n](https://www.zhihu.com/equation?tex=n) ，然后求数列中所有 ![m\geq n](https://www.zhihu.com/equation?tex=m%5Cgeq+n) 项的上确界，即得到一个数 ![\[\mathop {{\rm{sup}}}\limits_{m \ge n} {\rm{ }}{x_m}\]](https://www.zhihu.com/equation?tex=%5C%5B%5Cmathop+%7B%7B%5Crm%7Bsup%7D%7D%7D%5Climits_%7Bm+%5Cge+n%7D+%7B%5Crm%7B+%7D%7D%7Bx_m%7D%5C%5D) ，然后 ![n](https://www.zhihu.com/equation?tex=n) 从1取到 ![\infty](https://www.zhihu.com/equation?tex=%5Cinfty)

，我们就得到一个数列{ ![\[\mathop {{\rm{sup}}}\limits_{k \ge n} {\rm{ }}{x_k}\]](https://www.zhihu.com/equation?tex=%5C%5B%5Cmathop+%7B%7B%5Crm%7Bsup%7D%7D%7D%5Climits_%7Bk+%5Cge+n%7D+%7B%5Crm%7B+%7D%7D%7Bx_k%7D%5C%5D) }，如图最上边红色线所示。易知，数列{ ![\[\mathop {{\rm{sup}}}\limits_{k \ge n} {\rm{ }}{x_k}\]](https://www.zhihu.com/equation?tex=%5C%5B%5Cmathop+%7B%7B%5Crm%7Bsup%7D%7D%7D%5Climits_%7Bk+%5Cge+n%7D+%7B%5Crm%7B+%7D%7D%7Bx_k%7D%5C%5D) }是递减的，我们再对它取下确界，即 ![\mathop {\inf }\limits_{n \ge 1} {\rm{ }}\mathop {{\rm{sup}}}\limits_{m \ge n} {\rm{ }}{x_m}](https://www.zhihu.com/equation?tex=%5Cmathop+%7B%5Cinf+%7D%5Climits_%7Bn+%5Cge+1%7D+%7B%5Crm%7B+%7D%7D%5Cmathop+%7B%7B%5Crm%7Bsup%7D%7D%7D%5Climits_%7Bm+%5Cge+n%7D+%7B%5Crm%7B+%7D%7D%7Bx_m%7D) ，就得到了数列 ![\{ x_n \}](https://www.zhihu.com/equation?tex=%5C%7B+x_n+%5C%7D) 的上极限。仿照上述步骤可得到数列 ![\{ x_n \}](https://www.zhihu.com/equation?tex=%5C%7B+x_n+%5C%7D) 下极限的求法。
![](https://pic2.zhimg.com/v2-e08f2b017717c170452c1a847cafe91d_b.jpg)
**注释：**

1. 若不理解定义，建议按照例子动手画一画，写一写。

2. 直观上理解，数列的上极限就是先取最大，然后取最小；数列的下极限就是先取最小，然后取最大。

2. 数列的极限不一定存在，但是数列的上极限和下极限一定存在。而当上极限=下极限时，数列的极限存在。

**Section 1 集合列的极限**

设有集合列 ![\{{ E_n\}}](https://www.zhihu.com/equation?tex=%5C%7B%7B+E_n%5C%7D%7D) ，

**定义1（集合列的上极限与下极限） **

集合列的上极限符号记作 ![\[\mathop {{\rm{lim \ sup}}}\limits_{n \to \infty } {\rm{ }}{E_n} = \overline {\mathop {\lim }\limits_{n \to \infty } } {\rm{ }}{E_n} = \overline {\lim } {\rm{ }}{E_n}\]](https://www.zhihu.com/equation?tex=%5C%5B%5Cmathop+%7B%7B%5Crm%7Blim+%5C+sup%7D%7D%7D%5Climits_%7Bn+%5Cto+%5Cinfty+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D+%3D+%5Coverline+%7B%5Cmathop+%7B%5Clim+%7D%5Climits_%7Bn+%5Cto+%5Cinfty+%7D+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D+%3D+%5Coverline+%7B%5Clim+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D%5C%5D) ，其定义为 ![\overline {\mathop {\lim }\limits_{n \to \infty } } {\rm{ }}{E_n} = \bigcap_{n=1}^{\infty}\bigcup_{k=n}^{\infty}{E_k}](https://www.zhihu.com/equation?tex=%5Coverline+%7B%5Cmathop+%7B%5Clim+%7D%5Climits_%7Bn+%5Cto+%5Cinfty+%7D+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D+%3D+%5Cbigcap_%7Bn%3D1%7D%5E%7B%5Cinfty%7D%5Cbigcup_%7Bk%3Dn%7D%5E%7B%5Cinfty%7D%7BE_k%7D) 。集合列的下极限符号记作 ![\[\mathop {{\rm{lim \ inf}}}\limits_{n \to \infty } {\rm{ }}{E_n} = \mathop {\underline {\lim } }\limits_{n \to \infty } {\rm{ }}{E_n} = \underline {\lim } {E_n}\]](https://www.zhihu.com/equation?tex=%5C%5B%5Cmathop+%7B%7B%5Crm%7Blim+%5C+inf%7D%7D%7D%5Climits_%7Bn+%5Cto+%5Cinfty+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D+%3D+%5Cmathop+%7B%5Cunderline+%7B%5Clim+%7D+%7D%5Climits_%7Bn+%5Cto+%5Cinfty+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D+%3D+%5Cunderline+%7B%5Clim+%7D+%7BE_n%7D%5C%5D) ，其定义为 ![ \mathop {\underline {\lim } }\limits_{n \to \infty } {\rm{ }}{E_n}= \bigcup_{n=1}^{\infty}\bigcap_{k=n}^{\infty}{E_k}](https://www.zhihu.com/equation?tex=+%5Cmathop+%7B%5Cunderline+%7B%5Clim+%7D+%7D%5Climits_%7Bn+%5Cto+%5Cinfty+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D%3D+%5Cbigcup_%7Bn%3D1%7D%5E%7B%5Cinfty%7D%5Cbigcap_%7Bk%3Dn%7D%5E%7B%5Cinfty%7D%7BE_k%7D) 。

**定义2 （集合列的极限）**

如果集合列的上极限等于下极限，就说集合列的极限存在，此时 ![ {\lim } {\rm{ }}{E_n}=\overline {\lim } {\rm{ }}{E_n} =\underline {\lim } {\rm{ }}{E_n}](https://www.zhihu.com/equation?tex=+%7B%5Clim+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D%3D%5Coverline+%7B%5Clim+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D+%3D%5Cunderline+%7B%5Clim+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D) 。

任取正整数 ![n](https://www.zhihu.com/equation?tex=n) ，由上面式子可知，![x\in\bigcap_{n=1}^{\infty}\bigcup_{k=n}^{\infty}{E_k}\Rightarrow x\in{E_n} \cup {E_{n+1}} \cup \cdots ](https://www.zhihu.com/equation?tex=x%5Cin%5Cbigcap_%7Bn%3D1%7D%5E%7B%5Cinfty%7D%5Cbigcup_%7Bk%3Dn%7D%5E%7B%5Cinfty%7D%7BE_k%7D%5CRightarrow+x%5Cin%7BE_n%7D+%5Ccup+%7BE_%7Bn%2B1%7D%7D+%5Ccup+%5Ccdots+) ，再由并集的定义得到 ![x](https://www.zhihu.com/equation?tex=x) 属于某个 ![E_k(k\geq n)](https://www.zhihu.com/equation?tex=E_k%28k%5Cgeq+n%29)。**因此**![x\in\bigcap_{n=1}^{\infty}\bigcup_{k=n}^{\infty}{E_k}](https://www.zhihu.com/equation?tex=x%5Cin%5Cbigcap_%7Bn%3D1%7D%5E%7B%5Cinfty%7D%5Cbigcup_%7Bk%3Dn%7D%5E%7B%5Cinfty%7D%7BE_k%7D)** 写成逻辑语言就是，对任意正整数 **![n](https://www.zhihu.com/equation?tex=n)** ，存在某个 **![k\geq n](https://www.zhihu.com/equation?tex=k%5Cgeq+n)** ，使得 **![x\in E_k](https://www.zhihu.com/equation?tex=x%5Cin+E_k)** 。**余下另一个的等式仿照上面的步骤可以推出（其实是懒，占坑以后有空来填）。
首先，我们证明"\Rightarrow "方向 \\
因为存在某个正整数n，使得对于任意的k \geq n，都有x \in E_k(可以粗略地看成是 x \in $${\lim_{n to \infty}}$$(E_n \cup E_{n+1} \cup \cdots) )所以除去前面的有限项外，x属于所有的E_n，这就证明了充分性，下面来看必要性。 \\
若不包括元素x的所有集合的指标组成的集合为M，因为|M|<\infty ，所以supM=m<\infty ，下面我们令n=m+1，则 \forall x \in 下面的集合， x \in \varliminf E_n(因为x属于E_m后的所有集合项)，这就完成了必要性的证明。

3. 集合列的极限不一定存在，但是**集合列的上极限和下极限一定存在**。

4. （我的一些叨叨絮絮的话，可以不看）

> 为什么集合列的上下极限是这样定义的？集合列 ![\{E_n \}](https://www.zhihu.com/equation?tex=%5C%7BE_n+%5C%7D) 按照 ![E_1,E_2,...,E_k...](https://www.zhihu.com/equation?tex=E_1%2CE_2%2C...%2CE_k...) 变化，每个 ![E_k](https://www.zhihu.com/equation?tex=E_k) 的里面元素也在变化。一个直觉是，有没有一些元素比较“稳定地出现”呢？ 如果有，那么这些元素就是该集合列核心的元素。 **下极限 **![ \underline {\lim } {\rm{ }}{E_n}](https://www.zhihu.com/equation?tex=+%5Cunderline+%7B%5Clim+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D)** 中的元素就是最核心的，因为除去前面有限个 **![E_N](https://www.zhihu.com/equation?tex=E_N)** 外，里面的元素属于所有的 **![E_k](https://www.zhihu.com/equation?tex=E_k)** ；而上极限 **![ \overline {\lim } {\rm{ }}{E_n}](https://www.zhihu.com/equation?tex=+%5Coverline+%7B%5Clim+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D)** 中的元素是次核心的，因为里面的元素只要求属于无限个 **![E_k](https://www.zhihu.com/equation?tex=E_k)**。从“元素出现”这一点理解，就显然有**![ \underline {\lim } {\rm{ }}{E_n}\subseteq \overline {\lim } {\rm{ }}{E_n}](https://www.zhihu.com/equation?tex=+%5Cunderline+%7B%5Clim+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D%5Csubseteq+%5Coverline+%7B%5Clim+%7D+%7B%5Crm%7B+%7D%7D%7BE_n%7D) 。
集合列的上极限就是先取并集再取交集（先取大后取小），集合列的下极限就是先取交集再取并集（先取小后取大）。

**定义3 (集合的特征函数)**

设有 ![E \subseteq X](https://www.zhihu.com/equation?tex=E+%5Csubseteq+X) ，定义 ![E](https://www.zhihu.com/equation?tex=E) 的特征函数 ![\chi_E : X\rightarrow \{ 0,1\}](https://www.zhihu.com/equation?tex=%5Cchi_E+%3A+X%5Crightarrow+%5C%7B+0%2C1%5C%7D)为： ![\[{\chi _E} (x)= \left\{ {\begin{array}{*{20}{c}} {\rm{1}}, if \ x \in E\\ {\rm{0}} ,if \ x \notin E \end{array}} \right.\]](https://www.zhihu.com/equation?tex=%5C%5B%7B%5Cchi+_E%7D+%28x%29%3D+%5Cleft%5C%7B+%7B%5Cbegin%7Barray%7D%7B%2A%7B20%7D%7Bc%7D%7D+%7B%5Crm%7B1%7D%7D%2C+if+%5C+x+%5Cin+E%5C%5C+%7B%5Crm%7B0%7D%7D+%2Cif+%5C+x+%5Cnotin+E+%5Cend%7Barray%7D%7D+%5Cright.%5C%5D)

注意：（1）![\chi_E](https://www.zhihu.com/equation?tex=%5Cchi_E) 是定义在整个空间 ![X](https://www.zhihu.com/equation?tex=X) 上的。（2） ![\chi_E](https://www.zhihu.com/equation?tex=%5Cchi_E) 的定义是逐点的（pointwise）。

**性质：**

（1）若 ![A\subseteq B](https://www.zhihu.com/equation?tex=A%5Csubseteq+B) ，则 ![\chi_A(x)\leq \chi_B(x)](https://www.zhihu.com/equation?tex=%5Cchi_A%28x%29%5Cleq+%5Cchi_B%28x%29)

> ![( B \backslash A )=\{x: x \in B \ and \ x \notin A\}](https://www.zhihu.com/equation?tex=%28+B+%5Cbackslash+A+%29%3D%5C%7Bx%3A+x+%5Cin+B+%5C+and+%5C+x+%5Cnotin+A%5C%7D)
因为![A\subseteq B](https://www.zhihu.com/equation?tex=A%5Csubseteq+B) ，故 ![B=A \cup ( B \backslash A )](https://www.zhihu.com/equation?tex=B%3DA+%5Ccup+%28+B+%5Cbackslash+A+%29) (且 ![A \cap ( B \backslash A )=\emptyset ](https://www.zhihu.com/equation?tex=A+%5Ccap+%28+B+%5Cbackslash+A+%29%3D%5Cemptyset+) )。接下来将整个空间 ![X](https://www.zhihu.com/equation?tex=X) 分区域讨论。
当 ![x](https://www.zhihu.com/equation?tex=x) 属于 ![A](https://www.zhihu.com/equation?tex=A) 时， ![x](https://www.zhihu.com/equation?tex=x) 也属于 ![B](https://www.zhihu.com/equation?tex=B) ，故此时 ![\chi_A(x)= \chi_B(x)=1](https://www.zhihu.com/equation?tex=%5Cchi_A%28x%29%3D+%5Cchi_B%28x%29%3D1)。当 ![x](https://www.zhihu.com/equation?tex=x) 属于 ![ ( B \backslash A )](https://www.zhihu.com/equation?tex=+%28+B+%5Cbackslash+A+%29) 时， ![ \chi_B(x)=1 > \chi_A(x)=0](https://www.zhihu.com/equation?tex=+%5Cchi_B%28x%29%3D1+%3E+%5Cchi_A%28x%29%3D0) 。当 ![x](https://www.zhihu.com/equation?tex=x) 属于 ![X\backslash B](https://www.zhihu.com/equation?tex=X%5Cbackslash+B) 时， ![\chi_A(x)= \chi_B(x)=0](https://www.zhihu.com/equation?tex=%5Cchi_A%28x%29%3D+%5Cchi_B%28x%29%3D0) 。综上就证明了 ![\chi_A(x)\leq \chi_B(x)](https://www.zhihu.com/equation?tex=%5Cchi_A%28x%29%5Cleq+%5Cchi_B%28x%29) 。

（2） ![\overline{ \text{lim}} \ \chi_{E_{n}}(x)=\chi_{\overline{ \text{lim} }\ E_n}(x)](https://www.zhihu.com/equation?tex=%5Coverline%7B+%5Ctext%7Blim%7D%7D+%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29%3D%5Cchi_%7B%5Coverline%7B+%5Ctext%7Blim%7D+%7D%5C+E_n%7D%28x%29) ; ![\underline{ \text{lim}} \ \chi_{E_{n}}(x)=\chi_{\underline{ \text{lim} }\ E_n}(x)](https://www.zhihu.com/equation?tex=%5Cunderline%7B+%5Ctext%7Blim%7D%7D+%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29%3D%5Cchi_%7B%5Cunderline%7B+%5Ctext%7Blim%7D+%7D%5C+E_n%7D%28x%29)

（即对集合列取上下极限与取特征函数可以交换顺序)

> （a）先明确 ![\overline{ \text{lim}} \ \chi_{E_{n}}(x)](https://www.zhihu.com/equation?tex=%5Coverline%7B+%5Ctext%7Blim%7D%7D+%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29) 和 ![\chi_{\overline{ \text{lim} }\ E_n}(x)](https://www.zhihu.com/equation?tex=%5Cchi_%7B%5Coverline%7B+%5Ctext%7Blim%7D+%7D%5C+E_n%7D%28x%29) 的含义。由于集合的特征函数是逐点定义的，因此固定一个 ![x](https://www.zhihu.com/equation?tex=x) ,当 ![n](https://www.zhihu.com/equation?tex=n) 从1到 ![\infty](https://www.zhihu.com/equation?tex=%5Cinfty) 变化时， ![\ \chi_{E_{n}}(x)](https://www.zhihu.com/equation?tex=%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29) 取变化的值，因此得到一个数列 ![\{\ \chi_{E_{n}}(x)\}](https://www.zhihu.com/equation?tex=%5C%7B%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29%5C%7D) ，那么 ![\overline{ \text{lim}} \ \chi_{E_{n}}(x)](https://www.zhihu.com/equation?tex=%5Coverline%7B+%5Ctext%7Blim%7D%7D+%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29) 就是该数列的上极限，即 ![\overline{ \text{lim}} \ \chi_{E_{n}}(x)= \mathop {\inf }\limits_{n \ge 1} {\rm{ }}\mathop {{\rm{sup}}}\limits_{k \ge n} {\rm{ }}{\chi_{E_{n}}(x)}](https://www.zhihu.com/equation?tex=%5Coverline%7B+%5Ctext%7Blim%7D%7D+%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29%3D+%5Cmathop+%7B%5Cinf+%7D%5Climits_%7Bn+%5Cge+1%7D+%7B%5Crm%7B+%7D%7D%5Cmathop+%7B%7B%5Crm%7Bsup%7D%7D%7D%5Climits_%7Bk+%5Cge+n%7D+%7B%5Crm%7B+%7D%7D%7B%5Cchi_%7BE_%7Bn%7D%7D%28x%29%7D) 。 ![\chi_{\overline{ \text{lim} }\ E_n}(x)](https://www.zhihu.com/equation?tex=%5Cchi_%7B%5Coverline%7B+%5Ctext%7Blim%7D+%7D%5C+E_n%7D%28x%29) 表示集合 ![\overline{ \text{lim}} \ E_n](https://www.zhihu.com/equation?tex=%5Coverline%7B+%5Ctext%7Blim%7D%7D+%5C+E_n) 的特征函数。
（b）如果我们能证明对任意的 ![x](https://www.zhihu.com/equation?tex=x), ![\overline{ \text{lim}} \ \chi_{E_{n}}(x)](https://www.zhihu.com/equation?tex=%5Coverline%7B+%5Ctext%7Blim%7D%7D+%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29) 和 ![\chi_{\overline{ \text{lim} }\ E_n}(x)](https://www.zhihu.com/equation?tex=%5Cchi_%7B%5Coverline%7B+%5Ctext%7Blim%7D+%7D%5C+E_n%7D%28x%29) 取相同的值，也就完成了等式的证明。由于 ![\chi_{\overline{ \text{lim} }\ E_n}(x)](https://www.zhihu.com/equation?tex=%5Cchi_%7B%5Coverline%7B+%5Ctext%7Blim%7D+%7D%5C+E_n%7D%28x%29) 只能取0或者1两个值，故不妨令 ![\overline{ \text{lim}} \ \chi_{E_{n}}(x)](https://www.zhihu.com/equation?tex=%5Coverline%7B+%5Ctext%7Blim%7D%7D+%5C+%5Cchi_%7BE_%7Bn%7D%7D%28x%29) =1，然后证明在这情况下也有 ![\chi_{\overline{ \text{lim} }\ E_n}(x)=1](https://www.zhihu.com/equation?tex=%5Cchi_%7B%5Coverline%7B+%5Ctext%7Blim%7D+%7D%5C+E_n%7D%28x%29%3D1) 。

**Section 3 环和代数**

**定义4 （环）**

定义**集族**![R\subseteq \mathscr{P}(X)](https://www.zhihu.com/equation?tex=R%5Csubseteq+%5Cmathscr%7BP%7D%28X%29) 为环(ring)，如果满足：

![(1)\ \varnothing\in R ](https://www.zhihu.com/equation?tex=%281%29%5C+%5Cvarnothing%5Cin+R+)

![(2) \ A,B\in R\Rightarrow A \backslash B \in R](https://www.zhihu.com/equation?tex=%282%29+%5C+A%2CB%5Cin+R%5CRightarrow+A+%5Cbackslash+B+%5Cin+R)

![(3) A, B \in R \Rightarrow A \cup B \in R](https://www.zhihu.com/equation?tex=%283%29+A%2C+B+%5Cin+R+%5CRightarrow+A+%5Ccup+B+%5Cin+R)

性质：

（1） ![A_1,A_2,...A_n\in R\Rightarrow \bigcup_{i=1}^{n}A_i\in R](https://www.zhihu.com/equation?tex=A_1%2CA_2%2C...A_n%5Cin+R%5CRightarrow+%5Cbigcup_%7Bi%3D1%7D%5E%7Bn%7DA_i%5Cin+R) (有限个 ![A_i](https://www.zhihu.com/equation?tex=A_i) )

> 由定义中的（3）可证

（2） ![A,B \in R \Rightarrow A \cap B \in R](https://www.zhihu.com/equation?tex=A%2CB+%5Cin+R+%5CRightarrow+A+%5Ccap+B+%5Cin+R)

> 由定义中的（3）有 ![A \cap B=A\backslash( A\backslash B)\in R](https://www.zhihu.com/equation?tex=A+%5Ccap+B%3DA%5Cbackslash%28+A%5Cbackslash+B%29%5Cin+R) （集合的交可用差集表示，用韦恩图容易看出表示方法）

（3） ![A_1,A_2,...A_n\in R\Rightarrow \bigcap_{i=1}^{n}A_i\in R](https://www.zhihu.com/equation?tex=A_1%2CA_2%2C...A_n%5Cin+R%5CRightarrow+%5Cbigcap_%7Bi%3D1%7D%5E%7Bn%7DA_i%5Cin+R) (有限个 ![A_i](https://www.zhihu.com/equation?tex=A_i) )

> 由上面性质的（2）可证

注：为什么满足上面定义的集族 ![R](https://www.zhihu.com/equation?tex=R) 称为“环”呢？因为若在 ![R](https://www.zhihu.com/equation?tex=R) 上定义“ ![+](https://www.zhihu.com/equation?tex=%2B) ”和“ ![·](https://www.zhihu.com/equation?tex=%C2%B7) ”运算为 ![A+B=A\Delta B](https://www.zhihu.com/equation?tex=A%2BB%3DA%5CDelta+B) （对称差）, ![A·B=A \cap B](https://www.zhihu.com/equation?tex=A%C2%B7B%3DA+%5Ccap+B) ，那么 ![（R,+,·）](https://www.zhihu.com/equation?tex=%EF%BC%88R%2C%2B%2C%C2%B7%EF%BC%89) 就是一个代数环

**定义5 （代数）**

定义集族 ![R](https://www.zhihu.com/equation?tex=R) 为一个代数（algebra），如果 ![R](https://www.zhihu.com/equation?tex=R) 是一个环，并且有 ![X\in R](https://www.zhihu.com/equation?tex=X%5Cin+R)

**定义6 （ ![\sigma](https://www.zhihu.com/equation?tex=%5Csigma) 环）**(读作西格玛环)

(a)若 ![(\chi_A\cdot\chi_B)(x)=\chi_A(x)\cdot\chi_B(x)=1](https://www.zhihu.com/equation?tex=%28%5Cchi_A%5Ccdot%5Cchi_B%29%28x%29%3D%5Cchi_A%28x%29%5Ccdot%5Cchi_B%28x%29%3D1) ，这等价于 ![\chi_A(x)=\chi_B(x)=1](https://www.zhihu.com/equation?tex=%5Cchi_A%28x%29%3D%5Cchi_B%28x%29%3D1) ，等价于 ![x\in A \text{ and } x\in B](https://www.zhihu.com/equation?tex=x%5Cin+A+%5Ctext%7B+and+%7D+x%5Cin+B) ,等价于 ![x\in A \cap B](https://www.zhihu.com/equation?tex=x%5Cin+A+%5Ccap+B) ，等价于 ![\chi_{A\cap B}(x)=1](https://www.zhihu.com/equation?tex=%5Cchi_%7BA%5Ccap+B%7D%28x%29%3D1) 。
(b)若 ![(\chi_A\cdot\chi_B)(x)=\chi_A(x)\cdot\chi_B(x)=0](https://www.zhihu.com/equation?tex=%28%5Cchi_A%5Ccdot%5Cchi_B%29%28x%29%3D%5Cchi_A%28x%29%5Ccdot%5Cchi_B%28x%29%3D0) ，这等价于 ![\chi_A(x)=0 \text{ or } \chi_B(x)=0](https://www.zhihu.com/equation?tex=%5Cchi_A%28x%29%3D0+%5Ctext%7B+or+%7D+%5Cchi_B%28x%29%3D0) ，等价于 ![x\notin A \text{ or } x\notin B](https://www.zhihu.com/equation?tex=x%5Cnotin+A+%5Ctext%7B+or+%7D+x%5Cnotin+B) ,等价于![x\in A^c\cup B^c=(A\cap B)^c](https://www.zhihu.com/equation?tex=x%5Cin+A%5Ec%5Ccup+B%5Ec%3D%28A%5Ccap+B%29%5Ec) ，等价于 ![x \notin A\cap B](https://www.zhihu.com/equation?tex=x+%5Cnotin+A%5Ccap+B) ,故 ![\chi_{A\cup B}(x)=0](https://www.zhihu.com/equation?tex=%5Cchi_%7BA%5Ccup+B%7D%28x%29%3D0)
综合(a)(b)知对任意 ![x](https://www.zhihu.com/equation?tex=x) , ![\chi_A\cdot \chi_B(x)=\chi_{A\cap B}(x)](https://www.zhihu.com/equation?tex=%5Cchi_A%5Ccdot+%5Cchi_B%28x%29%3D%5Cchi_%7BA%5Ccap+B%7D%28x%29)
(2）仿照（1）对每一点 ![x](https://www.zhihu.com/equation?tex=x) 讨论 ![\chi_A\cdot \chi_B(x)](https://www.zhihu.com/equation?tex=%5Cchi_A%5Ccdot+%5Cchi_B%28x%29) 和 ![\chi_{A\cap B}(x)](https://www.zhihu.com/equation?tex=%5Cchi_%7BA%5Ccap+B%7D%28x%29) 的值。
