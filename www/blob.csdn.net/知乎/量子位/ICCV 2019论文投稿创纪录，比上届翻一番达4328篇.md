# ICCV 2019论文投稿创纪录，比上届翻一番达4328篇 - 知乎
# 



> 允中 发自 麦蒿寺
量子位 出品 | 公众号 QbitAI

历史新高。

计算机视觉领域的顶级会议ICCV 2019，刚刚公布了论文投稿的情况：一共收到了4328篇论文。

这个数量是上一届，也就是ICCV 2017的两倍。
![](https://pic2.zhimg.com/v2-073914bc162a4a494d99c1377e1afd71_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1080' height='425'></svg>)
可怕啊可怕。

上一届ICCV 2017共收到2143篇论文投稿，其中621篇被选为大会论文（入围比例29%），其中56篇亮点报告（Spotlight），45篇口头报告（Oral）。

看起来，今年比拼肯定是要更激烈了。
![](https://pic1.zhimg.com/v2-82b55fdd198c9fe1638a0882ce0a31e4_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='720' height='540'></svg>)
上次ICCV 2017大会，参会人数再创新高，达到3107名。在30年ICCV大会历史上，已经被称为火爆出奇的一届。

ICCV 2017最佳论文（Marr Prize），颁给了何恺明为一作的Mask R-CNN。另外，何恺明参与的另一篇论文：Focal Loss for Dense Object Detection，也被大会评为最佳学生论文。

ICCV 2019将于今年10月27日在韩国首尔召开。
