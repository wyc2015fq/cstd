# 吃瓜社第二期报名 | 旷视研究院解读COCO2017人体姿态估计竞赛冠军论文 - 知乎
# 



昨天，量子位·吃瓜社联合Face++推出了线上论文分享系列第一期，旷视研究院研究员、论文一作彭超解读了COCO 2017物体检测夺冠算法论文《MegDet: A Large Mini-Batch Object Detector》，并分享了夺冠历程和经验。


除物体检测竞赛之外，Face++在COCO 2017人体姿态估计竞赛中也摘得桂冠，并在近期完成了相关论文**《Cascaded Pyramid Network for Multi-Person Pose Estimation》**。

基于这篇论文所提出的算法，Megvii(Face++)队在COCO2017人体姿态估计竞赛上获得了历史最好成绩，相对 2016年人体姿态估计的最好成绩提高了19%，毫无疑问是当前人体姿态估计的最高水平。

12月13日（下周三）晚19:30，**量子位·吃瓜社第二期**就邀请到了**竞赛冠军队owner、论文共同一作王志成**，他将深度解析COCO 2017人体姿态估计夺冠算法论文《Cascaded Pyramid Network for Multi-Person Pose Estimation》，聊一聊人体姿态估计课题下提出的新的网络结构，以及COCO竞赛中的经验和收获。
