# 我心中好的数学学习辅导材料的样子（顺便谈谈做题） - 知乎
# 


谨以本文献给网剧《你好，旧时光》。

可能因为我是东北人，又是过来人，比很多95后观众更理解这部剧的背景。个人认为导演还原的很好。尤其让我感动的是剧中校服蓝白相间，跟我当年的高中校服设计样式颜色相近，倍感亲切。

不过我所在高中比振华中学差远了，但是大家的学习材料是一样的。我们高中当年是北京四中网校用户，看过他们的课堂录像，说实话讲的真一般。其实，大学也差不多，会讲课的老师很少。后来偶然接触了育才中学相关人士，得知育才中学很多好学生靠课后报补习班提升姿势水平。

读大学时我意识到小地方吸引不了优秀教师，真正拉高学生平均水平主要靠辅导材料了（在我看来）。很多人觉得对同一道真题，哪个老师照着答案讲都一样。这种观点是错误的。以争议最大的科目语文为例，考试中心一位马甲叫余闻（类似任仲平）的老师基本上每年都会写一篇语文全国卷真题分析。不怕不识货，就怕货比货，你看过文科高手的文章后自然有所体会，但这种体会很难用言语描述。

在我心中，不论是高中或者是大学，优秀的辅导材料应该是如下构成：
试题+标答+评注+引申。

对于老师们来说，评注或讲解是体现自身水平的地方。但在我看过的高考学习材料中，包括五三，很少见到令我满意的评注。至于引申，那就更为少见了。

什么叫引申?不好定义。但可以举栗子：我在2017年为小说《遇见》写了一个数学读本《明月随笔数学篇》（见博士数学论坛），其中一篇科普文章叫《从AGM到FLT》，这就是引申的经典栗子。我后来又看到Q赛题，其中一道与Fourier级数有关的试题A6令自己印象深刻。

评注：这道题最关键的地方是利用Fejer和做差，算是一个数学技巧吧。A6的原型是《微积分学教程》第三卷的一个结论。

![](https://pic3.zhimg.com/v2-dc9b5f99f2ad97020f0e90b07a4417d6_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='800' height='1280'></svg>)
利用Fejer和做差技巧也出现在Rudin数学分析原理课后题中。
![](https://pic4.zhimg.com/v2-606ae8caf0a7894d4b7e86f6e2edaacb_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1536' height='882'></svg>)

引申：A6实质问的是一致有界，而考虑一直收敛是很自然的。为此需要加强假设f是连续周期函数。

![](https://pic4.zhimg.com/v2-fd1abdf1f5a97d15cc61600c265db237_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='800' height='1280'></svg>)
从证明进一步发现其并不很依赖于Fourier级数，可以推广到一般函数级数（读者可以思考一下）。汪林在《实分析中的反例》这本书，收集了两个几乎处处无界发散的Fourier反例，第一个出自柯尔莫哥洛夫，第二个出自哈代。后者变换（3）中也用到了Fejer和做差技巧。
![](https://pic2.zhimg.com/v2-e2bbc86eda78b1e1ee05ab38235a0f61_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1536' height='2048'></svg>)

柯尔莫哥洛夫还构造了一个处处无界发散的Fourier级数反例，细看之下发现哈代那个反例其实是效仿柯尔莫哥洛夫的例子。

![](https://pic1.zhimg.com/v2-1973d77c3c3c052d5dab05e6b035fcec_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='800' height='1280'></svg>)

![](https://pic4.zhimg.com/v2-7e3b4ce2623179c8bedbe67583b67b13_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='800' height='1280'></svg>)
知乎上很多数学大仙儿bs做题。别的科目不了解，但至少分析这科，我可以断定本质在于他们不会用科研眼光做题。做完就忘，也不理解自己做的题，只是习惯了而已。眼高手低，基本上几乎处处收敛于名词党。

题外话。《你好，旧时光》原著里，一些学霸防着同班别的学霸，生怕XXX。这种事情是事实，育才中学的学生都不能例外。但我高中时不担心别人偷学，因为我的学习方法（包括抄书）在当时很多同学看来非笨即累，浪费时间。在我印象中，同学们也许私下说我大道理，低EQ，但应该没人说我学习自私（那是时有人请教必详细解答）。

但有一点，不自私不代表慷慨倒贴。什么叫倒贴，就是网剧《你好，旧时光》里林杨同学给余周周同学搜集资料的事情。倒贴的动力往往不是来自于理性。
