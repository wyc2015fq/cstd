# 时代记忆：魔兽游戏，Beyond the Game，ACM-ICPC和林杨（你好旧时光） - 知乎
# 


随着网剧《你好，旧时光》的热播，勾起了这一代人的回忆。所以借机把林杨写进了小说《遇见番外篇》，在逻辑接口上我用的是网剧的逻辑接口，而非八月长安的原著。就我的记忆，林杨那个时代的热门专业认知（东北标准）是21世纪某专业和计算机专业。再考虑到网剧《最强男神》，于是钦点林杨去了T大计算机专业。

林杨那个年代最流行的网游是魔兽，这个游戏是一代人的记忆。但对我个人来说，我不会打游戏，也没玩过魔兽游戏，我大概把打游戏的时间都用到自学数学上了吧。耳濡目染，即便不会打魔兽也能感受到这个游戏的火爆，连我都听过人皇sky大名。我在番外篇里设定了一个ACM算法编程界的顶尖高手skywalker，原型是大名鼎鼎的tourist。我原来准备的番外篇里的名字是rambler，但后来看了李晓峰的自传，有点感慨，加上我是星战迷，于是最后将ITMO那位算法编程高手绰号钦定为skywalker，也是为了向人皇致敬！

那个年代比sky印象更深的是WCG主题曲《beyond the game》。当我计划动笔写点ACM故事时，第一想到的歌曲就是这首歌。搜集资料时，我看到知乎上有些ACM选手似乎内心bs打电竞的选手，我觉得哪一行只要能做到顶尖就值得尊敬。好比假如很多人搞大热门抽象代数几何，结果成绩一般，有人搞解析数论，结果用意想不到的传统方法证明了RH，这也值得尊敬。

最初给林杨设定《番外篇》角色只考虑了高考方面，算是对网剧你好旧时光里一个情节的扩写。后来不知怎么的，产生一个想法，何不让林杨打ACM finals！

查看了小说，林杨似乎参与了高中数学物理竞赛，并没有参加noi，其实现在noi也是南强北弱。在搜集ACM资料时发现一个事实：中国的ACM世界总冠军队员中几乎没有ioi金银牌选手，有的甚至没拿过NOI金牌。所以，我设定林杨可以参加finals。但是，小说里设置是T大，没拿过noi奖牌想参加finals，基本是奇迹了，不知道存在与否。于是出于逻辑上考虑，借鉴了楼天城复出的故事，设定一个逻辑：原决赛选手顾凡意外受伤，让他的中学学弟顶替他参加finals，又设定林杨和肖明琮认识，如此主程序手陈城也得遵从民主原则，这样林杨加入校队参加finals。

![](https://pic2.zhimg.com/v2-a5a52130f100d091e9b2abbe47de5e11_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='1000' height='2089'></svg>)
因为小说是虚构的，所以可以遐想，设定比较随意。比如入场时，可以播放音乐beyond the game，甚至极乐净土之类。我觉得beyond这首歌不论是电竞，还是ACM都很适合，大家都是电脑操作，都有手速和脑速要求，更重要的是比赛精神是一样的。

![](https://pic4.zhimg.com/v2-b5aa61d1b0eab42a855b59d3ab6641ef_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='800' height='1165'></svg>)
比赛不是踏青，不是老朋友聊天。场外可以聊天，场内就要全身心投入。我看过楼天城的长篇回忆录，网上很容易找到，楼教主第一次参加总决赛时不是十分专注，尤其轻视主程序手自我状态的调整。他的文字里多少流露出反思的痕迹。第二次复出时，楼教主近乎巅峰状态，战斗力也惊人。但是，他原计划中没复出打算，是一个意外导致他被临时拉入队中顶替某人。因为是临时入队，磨合时间短，总决赛时没有太充分发挥团体力量（感觉团体配合不如第一次总决赛时）。最后是平题输罚时，再次获得总决赛亚军。

![](https://pic4.zhimg.com/v2-fc477df3774d0bf142392350fc8c1143_b.jpg)![](data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='522' height='473'></svg>)

在搜集ACM资料时，我发现了一个问题：T大的队员单兵战斗力并不差，也不是没怎么做题，但总感觉团体配合和战术组织上有点问题（也可能是我这个外行人的错觉）。交大的制霸固然是刻苦训练，但从搜集的资料来看，公平地说，感觉交大的配合水平胜于T大。楼教主两次亚军之后，很多人（包括我）预计T大在十年内拿ACM世界总冠军应该问题不大，毕竟一代更比一代强。现在回看，如今ACM比赛日渐成熟，高中时代的OI优势将日渐趋少。系统训练和团队战术配合的作用可能慢慢超过了天赋的作用。

ACM怪像如今也蔓延到丘成桐大学生数学竞赛，这个比赛主要是数学系在PK，近几年的情况是IMO选手可能被虐，国家集训队选手优势渐少。

最后附上人皇sky的WCG总决赛回忆，相信很多人感触很深吧。
