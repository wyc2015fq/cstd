【伯乐在线导读】：2015年04月15日消息，知名开发者工具厂商 JetBrains 于今日在[官博](http://blog.jetbrains.com/clion/2015/04/clion-1-0-has-finally-arrived/)正式发布了一个跨平台的 C/C++ 集成开发环境 —— CLion （1.0 版）。
两年前（2013年）的愚人节，JetBrains 说要做这款 IDE。开发一年后发布了一个预览版，后来收到了很多令人激动的反馈，数百份详细的需求，还有很多鼓励他们团队的留言。
![](http://ww4.sinaimg.cn/mw690/7cc829d3gw1er62qrqjcvj20xc0hg408.jpg)
### 关于 JetBrains
JetBrains 是一家捷克的软件开发公司，总部位于捷克的布拉格，并在俄罗斯圣彼得堡及美国波士顿都设有办公室。旗下的 IDE 产品有：
- IntelliJ IDEA：一套智慧型的Java整合开发工具，特别专注与强调程式师的开发撰写效率提升
- PHPStorm：PHP 集成开发工具
- PyCharm：智能Python集成开发工具
- RubyMine ：一个为Ruby 和Rails开发者准备的IDE，其带有所有开发者必须的功能，并将之紧密集成于便捷的开发环境中。
- WebStorm：HTML/CSS/J S开发工具
- AppCode：ObjC的IDE，是一个XCode的替代品
- CLion： C/C++ 集成开发环境
### 关于 CLion
JetBrains 官网的宣传口号是：A Power Tool for APower Languages。
**跨平台。**CLion 1.0 可以用于 64 位的 Linux、OS X、64 位的 Windows。在 Linux 和 OSX，可选择 GCC 或 Clang 作为编译器。在 Windows 平台，可选 MinGW 32/64 或 Cygwin。
**支持 C++ 11 标准；**
**构建系统：CMake**
![](http://ww4.sinaimg.cn/mw690/7cc829d3gw1er62qpf3k6j20yg0le79z.jpg)
**强大的编辑器 和 one-click 导航**
- 有可以匹配左值类型的智能自动完成的建议。
- 多个游标同时处理多个编辑任务。
- 根据预设的编码风格，代码自动格式化。
- 支持选择、重新排列、并迅速注释代码的键盘快捷键。
- 还有更多 （比如：不知道函数传了哪个参数？Ctrl + P 启用 Parameter Info 特性。
**代码分析、快速修复 和 重构**
![](http://ww3.sinaimg.cn/mw690/7cc829d3gw1er62qqxdvzj20w80crjxk.jpg)
**集成的调试器**
![](http://ww2.sinaimg.cn/mw690/7cc829d3gw1er62qq7fodj20sa0a2acx.jpg)
**集成了主流的版本控制系统**
Subversion、Git、GitHub、Mercurial、 CVS、Perforce (via plugin)和 TFS
