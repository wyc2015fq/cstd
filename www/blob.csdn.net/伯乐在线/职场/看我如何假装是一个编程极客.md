【伯乐在线导读】：有网友在 Quora 上提问：「如果我不是程序员，那我改如何假装是一个编程极客呢？」
众多程序员给题主出主意了，来看看他们是如何「教唆」非程序员伪装的。
### Kamran Khan 的回答，7900+ 顶
- 尽可能用一些时髦的术语。例如让“云”、“区块链”、“全栈”等术语遍布在你的谈话中。
- 要一直保持愤世嫉俗的态度。当面对一个任务，要装作不爱理睬：“这个之前已经完成了”或者断言这个任务会有无法遇见的技术挑战——或者说这个任务不够有趣。
- 寻求别人帮助时要伪装成指导他们。做到这一条是非常棘手的，但这可能是在这个列表中最具决定性的建议。
- 对于其他程序员，要像PITA一样跟他们一起工作。
- 当和管理层打交道时，让他们感觉你很聪明。
- 搞一个Macbook。
- 开特斯拉，并将它与你的iPhone连接。
- 搬到旧金山，多使用像“湾区”和“硅谷”这样的词语，即使是在阿肯色州的小岩城与人交谈。“在硅谷，每个人都是非常有野心的”是一个好的话头。
- 在你的书架上多放一些编程书籍。不要忘了《[计算机程序设计艺术](https://www.amazon.cn/gp/product/B019ZQR9GW/ref=as_li_qf_sp_asin_il_tl?ie=UTF8&camp=536&creative=3200&creativeASIN=B019ZQR9GW&linkCode=as2&tag=vastwork-23)》。（不要声称你已经在终端上看过这些书了。这会让你暴露的比吃了类固醇的博尔特跑的还快）
- 无论何时何地都要抨击微软。
- 要喝很多种咖啡，并且多讨论咖啡。每个人都知道程序员是把咖啡转换成代码的机器。
- 去参加编程马拉松，然后获取 T恤衫。
- 讲“Hello world”的笑话。
- 说愚蠢的和毫无根据的话，例如“软件工程是一场程序员生产软件与世界生产傻瓜之间的比赛，到目前为止世界赢了”。
- 多讨论 TED/TEDX 的演讲。
- 抓住每次机会讨论人工智能的威胁。
- 讨论科技公司的股票，然后利用你的小聪明让人听起来你的知识很渊博：“雅虎注定会失败，看看他们都做了什么……”
- “今天我在Hacker News上看到……”，以这样的句子开始对话。
- 多引用 xkcd 和 The Oatmeal 漫画网站上的漫画
- 在看起来挺幽默但并不怎么好笑的上下文中使用愚蠢、无意义的代码。例如：


```
void smartassCodeCommentOnAnswer(Answer *a)
{
    if (answer->isRelatedToProgramming) {
        smartassCodeCommentOnAnswer(a);    
    }
    else {
        regularCommentOnAnswer(a);
    }
}
```

### Unnit Metaliya 的回答，6700+ 顶
是的，你可以做到的。让我告诉你怎么弄。
第1步：不要穿得太正式。那不是极客们的着装风格。搞一些T恤衫什么的。还有像这样的眼镜。
![](https://qph.ec.quoracdn.net/main-qimg-2733a839b53a5c458923f29d61a90929-c?convert_to_webp=true)
第2步: 然后，去一些咖啡馆或者其他任何公共场所，在那里别人能清楚地看到你的屏幕。
第3步：打开你的笔记本，然后把亮度调到最大。
第4步：打开 [Hacker Typer](http://hackertyper.com/)（hackertyper.com）
第5步：进入全屏模式然后，开始随便敲点什么。网站会自己调整代码风格。
让我给你演示一下。
![](https://qph.ec.quoracdn.net/main-qimg-0a1699efd149df35ce010a703f0f3d95?convert_to_webp=true)
– 按“ALT”键3次弹出“ACCESS GRANTED”消息。
![](https://qph.ec.quoracdn.net/main-qimg-c3348c813c5eb3719c3758f4be5c7413?convert_to_webp=true)
-然后，充满激情地说：“耶！我成功了！让我再破解下其它的东西”。然后按“ESC”键让信息消失。
– 再次开始随机地打字然后按“Caps Lock”键3次弹出“ACCESS GRANTED”消息。
![](https://qph.ec.quoracdn.net/main-qimg-08e45e5e34bc604f26e673722dee38b4?convert_to_webp=true)
– 说，”我的天啊。哪里出错了？？！！让我再检查一下…”然后按”ESC”键。
– 开始随便打字并再次弹出“ACCESS GRANTED”消息。
– 然后，假装给你的同伴打电话，低声的抱怨并让周围的人听到，“连接了很多的节点以后，出现了许多困难，最后成功地进入了数据库，然后下一步怎么做？”
好了。一个浮夸的超级编程极客诞生了。
### Pankaj Parashar 的回答
这很容易，只要你有一点现金（发音为cache）并且跟随以下步骤：
- 搞一个高端的Macbook Pro ，然后在上面贴上一些极客贴纸。
- 买一个全新闪亮的Dr. Dre的Beats耳机，然后缠在你的脖子上。
- Wear a geeky t-shirt accompanied by a thin sweater and make sure you’re hair is unkempt.
- 穿上极客T恤并搭一件薄毛衣，然后确保你是蓬头垢面的。
- 无论有没有wifi，走进最近的咖啡馆。 在角落找一个最隐蔽的地方。
- 把你手机的背景换成一个呆板的壁纸。保持解锁状态然后把它放到一个别人很容易注意到的地方。
- 在任意浏览器打开[这个连接](http://fediafedia.com/neo/umbrella/)，切换到满屏幕模式然后开始疯狂的快速敲击你的键盘。
- 避免眼神交流。如果某人想要小声讨论，不要看他的脸而是看着他的脚。尽你所能的发誓，没有理由，不要停。
- 当参与谈话时，让他们重复两次，然后你要假装正忙着为下一个Facebook搞定算法。
- 如果某人跟你打招呼，你就回复：“这100万行代码怎么编译这么久！”
- 学习一些程序员的笑话。如果你希望给一个女孩留下深刻的印象，那么它就派上用场了。 如果那个女孩也用Quora，那么你就告诉她这些笑话都是你原创的。
### Sushant Somkuwar 的回答
- 弄一个笔记本电脑，连上网络连接；
- 在你的浏览器打开 [Hackertyper](http://hackertyper.com/) 网站；
- 按 Fn+F11 或者任何什么按钮，把窗口扩大到全屏。
- 开始随便敲键盘，一些随机的代码就会出现。
抿一口咖啡并立即装作思考状。好了，你就是编程极客！！！ 愿力量与你同在。
