Github 上最具影响力的组织/机构有哪些？**gh-impact** 可以对此进行评估。
## 简介
gh-impact 可以测量开源项目的影响。gh-impact 基于开源项目在 Github 上获得的 star 数量：如果一个账户有 n 个 star 的 n 个 project，那么此账户的 gh-impact 得分为 n。高 gh-impact 分数意味着账户下有很多好用的 project。
下面的表格列出了 Github 上 **gh-impact** 排名前 10 的组织。这 10 个组织混合了私人营利公司和非营利基金会。每个名称最终是如何纳入列表有一个完整的解释，但是我想要简单地强调两个公司：Facebook 和 Microsoft。
Facebook 值得注意，因为它尽管在 Github 组织中的成员相对较少，但是 gh-impact 分数很高。Facebook 的 user-id 低，也就是说他们很早就尝试 Github了。社交网络中遵循“富人更富”效应，所以这个早期的行动可能有利于 Facebook 的 **gh-impact** 评估。
另一方面，Microsoft 的高 user-id 值得注意；Microsoft 加入 Github 前，GitHub 就已经有超过 3 百万账户了。Microsoft 有一个著名的传统，就是 Open Source 的反对者。前 Microsoft CEO Ballmer 也讽刺过：“Linux 就像癌症一样，把所及一切都和自身的知识产权理念扯上关系。这就是许可证的工作方式。”从 **gh-impact** 组织排名可以明显看出，Microsoft 在 Nadella 的监管下已经成为了一家全然不同的公司。Microsoft 在这么晚加入 Github 后，排名已经上升至第 4 位，这个现象就直接反映着，Microsoft 转向开源的背后有着某些重要打算。

这个列表本身有一些特点，因为不同组织创建账户的方式。比如，名为“google”和“googlesamples”的账户出现在前 10 中。它们多半情况都只是 Google。我有理由怀疑编程语言的混杂、组织大小和一些其它因素会影响 **gh-impact**，未来工作会研究这些问题。
我很惊讶有一些公司**没有**进入前 10，但是顶尖的名额也就这些。个人账户的排名也计算出来了。[Leader Board](http://www.gh-impact.com/leaderboard/) 可以查看完整名单。
