# 适用于开发者的最佳火狐插件 - 文章 - 伯乐在线
原文出处： [OSC-两味真火](https://my.oschina.net/editorial-story/blog/881207)
嗨…你还记得火狐么？ 谷歌的 Chrome 浏览器现在是浏览器群英中的王者，不过火狐依然独树一帜并且依旧是网页应用开发者最好的浏览器。我知道它稍微有点慢，并且略占内存，不过其各种定制化功能绝对是无懈可击的。我们列举出了一些我们收藏的火狐插件，让我们作为开发者的生活变得更加简单，并且希望它能帮助节省你一点点时间。
![firefox add ons for developers](https://static.oschina.net/uploads/img/201704/17180707_dLMf.jpg)
### **1、[Web Developer](https://addons.mozilla.org/en-US/firefox/addon/web-developer)**
一个绝对能称之为顶尖的浏览器网页开发工具，Web Developer 是一款不仅仅在Firefox上可用的扩展插件，其还风靡 Chrome，Opera，并且其可以在任何支持上述浏览器的操作系统上运行自如，包括 Windows,OS X, 以及 Linux。
![best firefox add-ons for developers](https://static.oschina.net/uploads/img/201704/17180709_707P.png)
### **2、[JavaScript Deobfuscator](https://addons.mozilla.org/en-US/firefox/addon/javascript-deobfuscator/)**
查看并调试 JavaScript，如 JavaScript 本身在有 JavaScript Deobuscator 的页面上编译执行一样。 JavaScript Deobuscator 会为你显示页面上的所有 JavaScript —— 甚至那些在fly上生成并混淆的代码。即使你工作于没有使用混淆的网站，它也是一个很有价值的工具。
![best firefox add-ons for developers](https://static.oschina.net/uploads/img/201704/17180709_a0Tz.png)
### **3、[HackBar](https://addons.mozilla.org/en-us/firefox/addon/hackbar/)**
HackBar 是一个简单的安全审核和渗透测试工具，它是所有希望确保其工作安全的开发人员的必备工具。 它并不是一个用来执行标准漏洞的工具，它不会教你如何破解网站，但它可以帮你测试SQL注入，XSS漏洞和站点安全性。 您可以在 [GitHub](https://github.com/johan-adriaans/hackbar) 查看源代码。
![HackBar](https://static.oschina.net/uploads/img/201704/17180710_wFF3.png)
### **4、[SQLite Manager](https://addons.mozilla.org/en-US/firefox/addon/sqlite-manager/)**
SQLite Manager 能直接从您的浏览器管理 SQLite 数据库，它提供了大量的功能：包括直观地显示数据库对象的结构树，用于管理表，索引，视图和触发器的有用对话框，具有语法帮助功能的下拉菜单能帮助你更轻松地写 SQL ，还能通过菜单，工具栏，按钮和上下文菜单等直观查看功能和常用操作。
![best firefox add-ons for developers](https://static.oschina.net/uploads/img/201704/17180711_TNhC.png)
### **5、[FirePHP](https://addons.mozilla.org/en-US/firefox/addon/firephp)**
FirePHP 让你能够在不干扰实时内容的情况下快速轻松地调试代码。它允许你使用简单的 PHP 方法调用登录到你的 Firebug Console，对于 AJAX 开发人员来说简直是个完美的 Firefox 附加组件。
![best firefox add-ons for developers](https://static.oschina.net/uploads/img/201704/17180719_YGzs.png)
### **6、[JSONView](https://addons.mozilla.org/en-us/firefox/addon/jsonview/)**
有了JSONViewAAAA，你可以很容易地查看在浏览器右边查看 JSON 文档。不像火狐提示你下载文件，文档在浏览器中显示如 XML 文档一样。文档以及被格式化并且高亮可收缩的显示了数组和对象，并且即使这个JSON 文本包含错误，JSONView 依旧可以显示其每行文本。
![JSONView](https://static.oschina.net/uploads/img/201704/17180720_DXUY.png)
### **7、[Ghostery](https://addons.mozilla.org/en-us/firefox/addon/ghostery/)**
如果你访问网站时想要隐身或者大幅提升浏览速度的话，你可以添加 Ghostery  插件。它会让你明智地选择与所访问的网站分享的数据，在你工作时能为你提供一个更干净、更安全、更快速的浏览体验。
![Ghostery](https://static.oschina.net/uploads/img/201704/17180720_Cc70.png)
### **8、[All-in-One Sidebar](https://addons.mozilla.org/en-us/firefox/addon/all-in-one-sidebar/)**
All-in-One Sidebar 能让你同时打开几个窗口作为侧边栏面板，并能轻松切换它们，包括书签、历史记录，以及对话框、下载、扩展工具、页面信息、错误控制台等对话。它们不会妨碍你的浏览，或者说：你很容易就可以通过开关显示或隐藏它们。而且，你还可以在浏览器启动或者全屏模式时配置这个扩展组件的行为。[点此](http://firefox.exxile.net/aios/features.php)获取更多该组件的特性。
![All-in-One Sidebar](https://static.oschina.net/uploads/img/201704/17180721_N5DF.png)
### **9、[Resurrect Pages](https://addons.mozilla.org/en-us/firefox/addon/resurrect-pages/)**
页面崩溃对用户和开发者来说都是令人沮丧的。幸亏有 Resurrect Pages，它能够让死了很久的页面重新复活并继续浏览。这个组件会搜索包括 Google 缓存在内的四个主页缓存/镜像（但并非全部，因为并非所有页面都被缓存）。你可以在 [GitHub](https://github.com/arantius/resurrect-pages)上查看源代码。
![Resurrect Pages](https://static.oschina.net/uploads/img/201704/17180721_TxGh.png)
### **10、[FireShot Pro](https://addons.mozilla.org/en-US/firefox/addon/fireshot/?src=cb-dl-users)**
快速抓取整个网页的屏幕截图，并使用FireShot Pro进行注释，这个插件十分有用，你可以快速捕获并保存屏幕截图（为图片或PDF），而且你还可以将屏幕截图发送到剪贴板，打印，添加注释，上传 ，以及用一个简单的工具发送。 请注意，某些功能（添加注释，上传和发送）仅适用于Windows操作系统。
![best firefox add-ons for developers](https://static.oschina.net/uploads/img/201704/17180722_HUxX.png)
