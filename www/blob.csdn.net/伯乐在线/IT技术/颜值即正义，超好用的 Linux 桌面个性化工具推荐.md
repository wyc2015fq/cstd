# 颜值即正义，超好用的 Linux 桌面个性化工具推荐 - 文章 - 伯乐在线
原文出处： [开源中国编辑部](https://my.oschina.net/editorial-story/blog/1527138)
对于系统桌面，有人追求美观，有人追求简洁，有人追求实用，也有人就是喜欢默认的设置效果。接下来，推荐几款适用于 Linux 的开源桌面个性化工具，代码敲累了的时候，换换主题，说不定也能让你更放松。
### [1、Arc](https://github.com/horst3180/arc-theme)
一款主打透明元素的扁平化主题，支持基于 GTK 3 和 GTK 2 的桌面环境，如 GNOME、Unity、Budgie、Pantheon、Xfce、MATE 等，内含 3 种不同的风格可供选择。

### [2、**Flatabulous**](http://github.com/anmoljagetia/Flatabulous)
一款扁平化主题，适用于 Ubuntu 以及其它基于 Gnome 的 Linux 系统。
![](http://jbcdn2.b0.upaiyun.com/2017/09/6522206a9c1be1c6ef3506fbe28c85cc.png)
### [3、Adapta](https://github.com/adapta-project/adapta-gtk-theme)
一款自适应的 Gtk+ 主题，遵循当前比较流行的 Material Design 设计风格。支持 GNOME、Budgie、Cinnamon、Unity、XFce4、Mate、LXDE 等桌面环境。
![](http://jbcdn2.b0.upaiyun.com/2017/09/1874fddce2d1634263d80adbfa498f14.png)
### [4、Numix](https://github.com/numixproject/numix-gtk-theme)
一款结合了明暗元素的现代化平面主题，支持 Gnome、Unity、XFCE 和 Openbox 。
![](http://jbcdn2.b0.upaiyun.com/2017/09/9ec4cee40a68985bdd7163f64eefcfe7.png)
### [5、La Capitaine](https://krourke.org/projects/art/la-capitaine-icon-theme)
这是一个图标主题，目标是适配大部分桌面环境。里面的图标受 macOS 和 Material Design 启发，主要是使用视觉上容易吸引眼球的渐变、阴影和简单的几何形式元素。
![](http://jbcdn2.b0.upaiyun.com/2017/09/4700d547b2265628ab762d53a2d5a658.png)
### [6、Papirus](https://github.com/PapirusDevelopmentTeam/papirus-icon-theme)
基于非开源的 [Paper](https://snwh.org/paper) 主题，并有一些额外的特性（如硬编码托盘支持，kde 颜色方案支持，libreoffice 图标主题，filezilla 主题，smplayer 主题…）以及其它的修改的 SVG 图标主题。有 Papirus 、Papirus Dark、Papirus Light、ePapirus 四种风格可供选择。
![](http://jbcdn2.b0.upaiyun.com/2017/09/6bd12b2013111e5a71af592a691fba2c.png)
### [7、Hardcode Tray](https://github.com/bil-elmoussaoui/Hardcode-Tray)
这是一个托盘脚本，通过在 Linux 中自动探测你的默认主题，正确的图标尺寸，硬编码内嵌图标应用，以及正确的状态图标，来修正硬编码的托盘图标。它还支持上面提到的 Numix 和 Papirus 。
![](http://jbcdn2.b0.upaiyun.com/2017/09/c5ddb413c4c4f2777d2a8a12475a112a.png)
