# 2015 年最受 Linux 爱好者欢迎的软硬件大盘点 - 文章 - 伯乐在线
原文出处： [Michael Larabel](http://www.phoronix.com/scan.php?page=news_item&px=LB-2015-Stats&utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+Phoronix+%28Phoronix%29)   译文出处：[LinuxStory-GUOLONG LIU](http://www.linuxstory.org/linux-hardwaresoftware-enthusiast-trends-2015/)
Linux 爱好者都喜欢用哪些硬件，哪些发行版呢？近日 OpenBenchmarking.org 做了一个 2015 年度数据的统计和梳理，LinuxStory特意整理了一下，分享给大家。
OpenBenchmarking.org 启用新的 PTS 6.2-Gamvik 代码库进行重构，同时采用更强大的服务器——这些才能保障我们统计的乐趣。我们一直坚持以不同的方式统计显示访问 OpenBenchmarking.org 的计算机的不同软硬件组成，所以敬请期待未来几周里的相关更新，同时也欢迎你分享任何有趣的想法。OpenBenchmarking.org 有超过1,400 万的测试下载和数百万的软硬件组合。
其中第一个统计特征是即将到来的新版 OpenBenchmarking.org 重新启用的通过 PTS/OpenBenchmarking.org 按月追踪到的不同组件信息，比如显卡驱动、CPU、文件系统等等。下面将展示过去12个月的一些趋势，注意那些运行 Phoronix Test Suite 的并不是典型用户，而更可能是软硬件爱好者、企业服务管理员和其他专业人士。
![image](http://jbcdn2.b0.upaiyun.com/2016/03/bf8ad2e583c81edfd7f97d579c12ba12.jpg)
EXT4目前仍然是基准测试时使用的最流行的文件系统。
![image (1)](http://jbcdn2.b0.upaiyun.com/2016/03/b4f940866d21cb64a74c7da1b77a2a71.jpg)
Ubuntu 仍然是基准测试时最流行的 Linux 发行版，但是 CentOS、Arch、Debian 和其他的发行版也表现不错。其他发行版的峰值可能是当公司使用新上千台板载服务器进行基准测试产生的，而不一定是有独立用户堆叠的。
![image (2)](http://jbcdn2.b0.upaiyun.com/2016/03/21ccfb6ee480418091e1268711f47215.jpg)
Radeon 和 Intel 的显卡驱动依然是最流行、最普遍的，它们竞争相当严重而 Nvidia 偶有突起。鉴于使用 Phoronix Test Suite 的有很多企业用户，所以 VMware 和 Matrox 很常见。
![image (3)](http://jbcdn2.b0.upaiyun.com/2016/03/0f7772e233518b8fc5a00175b91ce1cc.jpg)
Intel 处理器毫无疑问称霸市场，80%左右的结果是来自搭载 Intel 处理器的系统，其次是 AMD 和 ARM（当然也有一些小份额的其他处理器，我尽力去更好的分类和统计这些硬件，但不能就此认为 PTS 运行在不同的架构上同时经常有预生产的硬件）。
![image (4)](http://jbcdn2.b0.upaiyun.com/2016/03/14def9cb2f4b7cf161cefb85c3a56368.jpg)
ASUS 和 Gigabyte 主板最常见。
![image (5)](http://jbcdn2.b0.upaiyun.com/2016/03/53772eac3061413fe3f56593253de104.jpg)
GCC 仍然是占主导地位的编译器。
敬请关注新版 OpenBenchmarking.org 挖掘到更多的 Linux 软硬件数据和信息，测试版到下一代 OpenBenchmarking.org  应该在未来数天实现，接着在二月份会发布 Phoronix Test Suite 6.2。
