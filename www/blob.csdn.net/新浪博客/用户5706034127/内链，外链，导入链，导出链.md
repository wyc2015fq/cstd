# 内链，外链，导入链，导出链_用户5706034127_新浪博客
||分类：[自然语言处理](http://blog.sina.com.cn/s/articlelist_5706034127_3_1.html)|
inlinks是导入链 就是别人网站给自己网站应该链接 不是内链
内链是自己网站内部的链接
outlinks是导出链 就是你的网站给别的网站添加链接 不是外链
外链是别人的网站给你的网站加入链接
# What is Inlink and Outlink
1.
To Web page W, an**inlink**is a URL of
another Web page which contains a link pointing to W. To Web page
W, an**outlink**is a link (URL)
appearing in W which points to another Web page.[Learn
more in: Web Mining to Identify People of Similar
Background](http://www.igi-global.com/chapter/web-mining-identify-people-similar/21736)
这么一说，inlinks就是外链了
