conda并不比pip厉害，，比如安装keras包，conda2.7就安装不了。但pip可以轻松安装。
所以安装包的时候，要两个都试一下。
更坑爹的是，，pip install -r requirements.txt
并不会安装包，还是得一个一个自己安装。
还有tensorflow 和
theano的底层是由python或c++实现了，也就是说是有tensorflow的python包的。
先下载，在用pip安装
