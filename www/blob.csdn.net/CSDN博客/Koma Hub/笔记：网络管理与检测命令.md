##### 1.查询网络配置信息命令
###### 1）ipconfig（linux对应ifconfig）
ipconfig用于显示基本的网络配置信息，包括适配器配置的IP地址、子网掩码（网络掩码）、默认网关地址等，*是隐藏的信息。为简洁省去了实际显示内容。
```
D:\>ipconfig
Windows IP 配置
以太网适配器 vEthernet (默认开关):
以太网适配器 以太网 2:
无线局域网适配器 WLAN:
以太网适配器 以太网:
以太网适配器 VMware Network Adapter VMnet1:
以太网适配器 VMware Network Adapter VMnet8:
```
###### 2）ipconfig/all
显示详细的网络配置信息，除了ipconfig的信息外，还有MAC地址（物理地址）、本地域名服务器地址等。为简洁省去了实际显示内容。
```
D:\>ipconfig/all
Windows IP 配置
以太网适配器 vEthernet (默认开关):
以太网适配器 以太网 2:
无线局域网适配器 WLAN:
以太网适配器 以太网:
以太网适配器 VMware Network Adapter VMnet1:
以太网适配器 VMware Network Adapter VMnet8:
```
###### 3）ipconfig/renew
该指令使得所有选择自动获得IP地址方式的适配器重新向DHCP服务器申请网络配置信息。
```
D:\>ipconfig/renew
Windows IP 配置
```
###### 4）ipconfig/release
该指令使得所有选择自动获取IP地址的适配器释放从DHCP服务器申请到的网络配置信息。
##### 2.管理ARP缓冲器命令
###### 1）arp -a
该命令用于显示所有IP地址与MAC地址之间的绑定关系。为简洁省去了实际显示内容。
```
D:\>arp -a
接口: 180.201.178.4 --- 0xd
  Internet 地址         物理地址              类型
  180.201.128.1         d8-49-0b-b8-49-95     动态
  180.201.191.255       ff-ff-ff-ff-ff-ff     静态
.....
接口: 192.168.244.129 --- 0xe
接口: 192.168.179.1 --- 0x10
接口: 192.168.198.1 --- 0x12
```
###### 2)arp -d
该命令用于删除所有动态的IP地址与MAC地址之间的绑定关系。为了不搞坏自己的电脑网络，不搞了。

###### 2）ping 目的终端IP地址
该命令用于检测与该目的终端之间的传输通路是否正常，如：ping 192.168.1.1用于检测终端与无线路由器之间的传出路径是否正常，IP地址192.168.1.1是无线路由器LAN端口的IP地址。由于电脑没用无线。。
```
D:\>ping 192.168.1.1
正在 Ping 192.168.1.1 具有 32 字节的数据:
请求超时。
请求超时。
请求超时。
192.168.1.1 的 Ping 统计信息:
    数据包: 已发送 = 3，已接收 = 0，丢失 = 3 (100% 丢失)，
```
