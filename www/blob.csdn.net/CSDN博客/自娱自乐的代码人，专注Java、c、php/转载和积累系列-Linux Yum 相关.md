- 搜索磁盘阵列相关软件
`yum search raid`- 查找某个软件的相关信息
`yum info mdadm`- 列出服务器上所提供的软件名称
`yum list`- 查看服务器上哪些软件可以更新
`yum list updates`- 安装软件
`yum install gcc`- 删除软件
`yum remove gcc`- yum升级
`yum -y update`
