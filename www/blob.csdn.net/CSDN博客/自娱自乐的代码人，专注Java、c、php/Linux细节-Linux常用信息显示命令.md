**Linux常用信息显示命令**
|**命令**|**说明**|**使用**|
|----|----|----|
|uname|显示系统信息|uname     显示操作系统uname –a   显示全部信息|
|hostname|显示主机信息|hostname   显示主机信息hostname –a 显示主机全部信息|
|dmsg|开机显示信息|dmsg|
|uptime|开机到现在多少时间|uptime|
|file|显示文件类型|file  filename  显示文件类型|
|du|统计目录磁盘用量|du|
|df|磁盘使用量|df –h 显示磁盘使用量|
|top|显示进程状态|top     实时显示进程状态top  -s  安全模式显示top –u root 执行用户显示|
|free|查看内存使用状况|free     查看内存free –m  用M的方式显示|
|w|显示登录用户||
|date|显示时间||
|cal|显示日历||
|witch|在环境变量 PATH 设置的目录下查找指定文件的位置|which find|
|whereis|在特定目录中查找符合条件的文件（包括二进制文件、手册页文件、源码文件）|whereis ls|
