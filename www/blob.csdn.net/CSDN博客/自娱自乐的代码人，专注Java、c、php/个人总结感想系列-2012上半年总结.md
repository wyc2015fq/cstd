一年一年其实过的挺快的。从08年开始学html，09年开始在卡塔科技写的php代码，10年进的阿云到现在，已经是第四个年头了，中间感触很多，感谢每一个曾经帮助过和鼓励过我的人。

2012年上半年总结来说，两个字：**变化**。从pw变到无线，从迷茫中走来，慢慢走向自信。

从以下几个方面总结**过去的工作**吧：

**工作上**：
- 协助开发红包6期和机票的开发工作。主要是在3月-5月之间，中间有过空的阶段，也有忙的时候，主要还是做了这两部分工作。其实现在回头去看下红包6期和机票的项目，review一下自己的代码，总体来说是可以接受的，良好的代码风格和注释，以及相对质量比较高的代码，这部分对自己的评价是肯定的。项目开发中也遇到一些难题，但是也克服过来了。
- 负责彩票应用和5+项目的后端开发工作。这个过程中，是痛苦的，但是走过来了，回头去看看，是美好的。做彩票的时候，其实遇到一个很困难的事情，淘宝的API并没有提供非常完善，导致我们这边一个需求无法实现。最后迫不得已只能发邮件提这个功能的需求，他们也很积极配合。5+项目遇到了进度危机，但是在困难的跟进中也算顺利上限了。感悟非常多，有时候跨出勇敢的第一步比闷头写代码要好多了，因为接手的项目都是需要去不断沟通，不断和各种团队交流的项目。

**学习上：**
- 非常遗憾的是一直想写点C，但是看了很多次，都看到c有一种想睡觉的感觉，也许真的不太适合吧。最近看到lua的语法很兴奋，看来我比较适合看一些不是最底层的代码。
- Liunx主要把鸟哥的私房菜看了大部分，现在对linux的操作相对比较熟练。但是linux系统非常庞大，需要不断在实践中继续探索。
- 服务器方面，现在能很熟练的配置Nginx，mysql，php，memcache等，然后做一些基本的配置优化等。有了自己的安装文档和安装包。
- 设计模式方面，基本看的差不多了，也是边看边写边理解。总体消化良好
- 熟练了一下 memcache,redis,Mongodb等工具
- InitPHP框架最终3.2稳定版本


**个人能力：**
- 沟通能力增强了。经常和各种团队成员沟通一些细节需求。
- 胆子和信心变大了。跨出去了，其实就不怕了。
- 恒心一直都是我心中的痛，不过开始有所好转。
- 更稳重了些。
- 决断力还不够。
- 自信心还需要有待提高。
