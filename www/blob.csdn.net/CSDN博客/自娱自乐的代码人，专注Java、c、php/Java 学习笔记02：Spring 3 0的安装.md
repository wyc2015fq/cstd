- Spring 3.0下载

去官方网站下载Spring框架的版本，例如我下载的是：spring-3.2.0.M2-dist.zip包



- 创建项目

创建一个SpringTest的java项目，点击下一步，然后点击完成，就创建完毕。




![](https://img-my.csdn.net/uploads/201211/15/1352948538_8189.jpg)

- 导入Spring3.0Jar文件

打开窗口 - 打开首选项 - 找到 Java - 打开构建路劲 - 找到用户库 - 在里面创建一个新的名为Spring 3.0的用户库文件夹




![](https://img-my.csdn.net/uploads/201211/15/1352948729_3290.jpg)




添加JAR文件，点击 Add Externel JAR，找到下载的Spring包中的libs文件夹下的JAR包，然后全选导入


![](https://img-my.csdn.net/uploads/201211/15/1352948857_7319.jpg)


> 
![](https://img-my.csdn.net/uploads/201211/15/1352948943_1791.jpg)

- 在项目中导入Spring库

> 
点击SpringTest项目右键，找到构建路劲，选择点击配置构建路劲。可以看到“库”这个Tab栏目

![](https://img-my.csdn.net/uploads/201211/15/1352949176_3372.jpg)



> 



选择添加库，点击用户库，点击下一步，选择Spring

![](https://img-my.csdn.net/uploads/201211/15/1352949302_9651.jpg)


![](https://img-my.csdn.net/uploads/201211/15/1352949346_8287.jpg)



> 
点击确认，然后在项目中查看是否已经有了Spring库文件，有了说明安装成功

![](https://img-my.csdn.net/uploads/201211/15/1352949430_7832.jpg)
