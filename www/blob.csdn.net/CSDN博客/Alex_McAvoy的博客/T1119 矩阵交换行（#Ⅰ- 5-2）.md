### 【题目描述】

给定一个5×5的矩阵(数学上，一个r×c的矩阵是一个由r行c列元素排列成的矩形阵列)，将第n行和第m行交换，输出交换后的结果。

### 【输入】

输入共6行，前5行为矩阵的每一行元素,元素与元素之间以一个空格分开。
    第6行包含两个整数m、n，以一个空格分开（1 ≤ m,n ≤ 5）。


### 【输出】

输出交换之后的矩阵，矩阵的每一行元素占一行，元素之间以一个空格分开。


### 【输入样例】

1 2 2 1 2
5 6 7 8 3
9 3 0 5 3
7 2 1 4 6
3 0 8 2 4
1 5

### 【输出样例】

3 0 8 2 4
5 6 7 8 3
9 3 0 5 3
7 2 1 4 6
1 2 2 1 2


### 【源程序】

```cpp
#include<iostream>
#include<algorithm>
using namespace std; 
int main() 
{
    int n,m;
    int	a[6][6];
    int i,j;
    
    for(i=1;i<=5;i++)//输入矩阵元素
        for(j=1;j<=5;j++)
            cin>>a[i][j];
    cin>>n>>m;//输入要交换的两行
    
    for(i=1;i<=5;i++)//交换两行元素
    	swap(a[m][i],a[n][i]);
    
    for(i=1;i<=5;i++)//输出矩阵
    {
        for(j=1;j<=5;j++)
    		cout<<a[i][j]<<" ";
    	cout<<endl;
    }
    return 0;
}
```
