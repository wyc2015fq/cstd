### 【题目描述】

小云和朋友们去爬香山，为美丽的景色所陶醉，想合影留念。如果他们站成一排，男生全部在左（从拍照者的角度），并按照从矮到高的顺序从左到右排，女生全部在右，并按照从高到矮的顺序从左到右排，请问他们合影的效果是什么样的（所有人的身高都不同）？

### 【输入】

第一行是人数n（2 ≤ n ≤ 40，且至少有1个男生和1个女生）。后面紧跟n行，每行输入一个人的性别（男male或女female）和身高（浮点数，单位米），两个数据之间以空格分隔。

### 【输出】

n个浮点数，模拟站好队后，拍照者眼中从左到右每个人的身高。每个浮点数需保留到小数点后2位，相邻两个数之间用单个空格隔开。

### 【输入样例】

```
6
male 1.72
male 1.78
female 1.61
male 1.65
female 1.70
female 1.56
```

### 【输出样例】
`1.65 1.72 1.78 1.70 1.61 1.56`
### 【源程序】

```cpp
#include<iostream>
#include<cstdio>
#include<cstring>
using namespace std;
int main()
{
    int n;
    double d;
    char s[20];
    double a[50],b[50];
    int p=0,q=0;
    double temp;
    int i,j;
    
    
    cin>>n;
    for(i=0;i<n;i++)
    {
        cin>>s>>d;
        if(strcmp(s,"male")==0)
            a[p++]=d;
        else 
            b[q++]=d;
    }
    
    for(i=0;i<p;i++)
        for(j=i+1;j<p;j++)
            if(a[i]>a[j])
            {
                temp=a[i];
                a[i]=a[j];
                a[j]=temp;
            }
            
    for(i=0;i<q;i++)
        for(j=i+1;j<q;j++)
            if(b[i]<b[j])
            {
                temp=b[i];
                b[i]=b[j];
                b[j]=temp;
            }
            
    for(i=0;i<p;i++)
        printf("%.2lf ",a[i]);
    for(i=0;i<q;i++)
        printf("%.2lf ",b[i]);
    return 0;
}
```
