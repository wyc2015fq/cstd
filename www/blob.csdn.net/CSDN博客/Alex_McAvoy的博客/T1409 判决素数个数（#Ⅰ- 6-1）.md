### 【题目描述】

输入两个整数X和Y，输出两者之间的素数个数（包括X和Y）。


### 【输入】

两个整数X和Y（1 ≤ X,Y ≤ 105）。


### 【输出】

输出一个整数，表示X，Y之间的素数个数（包括X和Y）。


### 【输入样例】

1 100

### 【输出样例】

25

### 【源程序】

```cpp
#include<iostream>
#include<cmath>
using namespace std;
bool prime(int x);

int main()
{
	int x,y;
	int ans=0;
	int i;
	
	cin>>x>>y;
	for(i=x;i<=y;i++)//从x枚举到y
		if( prime(i) )//分别判断i是否是素数
			ans++;
	cout<<ans<<endl;
	return 0;
}

bool prime(int x)//判断素数
{
	int i=2;
	if(x==1||x==0)	return false;
	while( i<=floor(sqrt(x)) && (x%i!=0) )
		i++;
	if(i>floor(sqrt(x)))
		return true;
	return false;
}
```
