### 【题目描述】

输入b，p，k的值，求bp mod k的值。其中b，p，k×k为长整型数。

### 【输入】

输入b，p，k的值。



### 【输出】

求bp mod k的值。

### 【输入样例】

```
2 10 9
```

### 【输出样例】
`2^10 mod 9=7`
### 【源程序】

```cpp
#include<iostream>
#include<cstdio>
#include<cstdlib>
#include<cstring>
#include<algorithm>
#include<string>
#define INF 999999999
#define N 1001
#define MOD 1000000007
using namespace std;
int b,p,k;
int calculate(int p)
{
    int temp;

    if(p==0)
        return 1;
    temp=calculate(p/2)%k;
    temp=(temp*temp)%k;
    if(p%2==1)
        temp=(temp*b)%k;
    return temp;
}
int main()
{
    cin>>b>>p>>k;
    int temp=b;
    b%=k;
    cout<<temp<<"^"<<p<<" mod "<<k<<"="<<calculate(p)<<endl;
    return 0;
}
```
