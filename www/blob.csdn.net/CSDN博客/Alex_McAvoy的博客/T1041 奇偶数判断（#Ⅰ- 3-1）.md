### 【题目描述】






给定一个整数，判断该数是奇数还是偶数。如果n是奇数，输出odd；如果n是偶数，输出even。

### 【输入】






输入仅一行，一个大于零的正整数n。 

### 【输出】








输出仅一行，如果n是奇数，输出odd;如果n是偶数，输出even。

### 【输入样例】

5


### 【输出样例】

odd

### 【源程序】


```cpp

```

```cpp
#include <iostream>
using namespace std;
int main()
{
	int N;
	cin>>N;
 	if(N%2) cout<<"odd"<<endl;
 	else cout<<"even"<<endl;
        return 0;
}
```
