### 【题目描述】

利用公式e=1+1/1!+1/2!+1/3!+...+1/n! ，求e的值，要求保留小数点后10位。

### 【输入】

输入只有一行，该行包含一个整数n（2≤n≤15），表示计算e时累加到1/n!



### 【输出】



输出只有一行，该行包含计算出来的e的值，要求打印小数点后10位。

### 【输入样例】

10

### 【输出样例】

2.7182818011

#include<iostream>
#include<cstdio>
using namespace std;
int main()
{
    int num,i;
    double e=1;
    long long temp=1;

    cin>>num;
    for(i=1;i<=num;i++)
    {
        temp=temp*i;//计算阶乘
        e=e+1.0/temp;//累加
    }
    printf("%.10lf\n",e);
    
    return 0;
}
