### 【题目描述】

给定一个只包含小写字母的字符串，请你找到第一个仅出现一次的字符。如果没有，输出no。


### 【输入】

一个字符串，长度小于100000。


### 【输出】

输出第一个仅出现一次的字符，若没有则输出no。


### 【输入样例】

abcabd

### 【输出样例】

c

### 【源程序】

```cpp
#include<iostream>
#include<cstdio>
#include<cstring>
using namespace std;
char ch[100001];
int main()
{
    int letters[26]={0};
    int len;
    int sum=0;
    int i;
    
    gets(ch);//获取一行字符串
    len=strlen(ch);//求字符串长度
    
    for(i=0;i<len;i++)//桶排的思想统计26个小写字母出现频次
        letters[ch[i]-'a']++;
    
    for(i=0;i<len;i++)//对字符串进行遍历
	{
		if(letters[ch[i]-'a']==1)//如果出现过一次
		{
			cout<<ch[i]<<endl;//输出字母
			return 0;//结束
		}
	}
	cout<<"no"<<endl;
    return 0;
}
```
