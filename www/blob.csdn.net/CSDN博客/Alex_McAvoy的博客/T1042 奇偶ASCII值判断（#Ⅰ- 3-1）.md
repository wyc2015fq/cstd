### 【题目描述】






任意输入一个字符，判断其ASCII是否是奇数，若是，输出YES，否则，输出NO。

### 【输入】





输入一个字符。


### 【输出】






如果其ASCII值为奇数，则输出YES，否则，输出NO。


### 【输入样例】

A


### 【输出样例】

YES

### 【源程序】


```cpp

```

```cpp
#include <iostream> 
#include <cstdio>
using namespace std;  
int main()  
{  
    char ch;  
    scanf("%c",&ch);
    if (ch%2 == 0) cout << "NO" << endl;  
    else cout << "YES" << endl;  
    return 0;  
}
```
