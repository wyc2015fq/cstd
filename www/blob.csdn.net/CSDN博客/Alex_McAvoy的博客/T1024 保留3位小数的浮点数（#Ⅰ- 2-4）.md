# 【题目描述】

读入一个单精度浮点数，保留3位小数输出这个浮点数。

# 【输入】

只有一行，一个单精度浮点数。

# 【输出】

也只有一行，读入的单精度浮点数。

# 【输入样例】
`12.34521`
# 【输出样例】
`12.345`
# 【源程序】

```cpp
#include<iostream> 
#include<iomanip> 
using namespace std; 
int main() 
{ 
    float n; 
    cin>>n; 
    cout<<setiosflags(ios::fixed)<<setprecision(3); 
    cout<<n<<endl; 
    return 0; 
}
```
