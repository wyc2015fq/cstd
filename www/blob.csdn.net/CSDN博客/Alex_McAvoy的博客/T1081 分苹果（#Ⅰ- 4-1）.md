### 【题目描述】



把一堆苹果分给n个小朋友，要使每个人都能拿到苹果，而且每个人拿到的苹果数都不同的话，这堆苹果至少应该有多少个？

### 【输入】



一个不大于1000的正整数n，代表小朋友人数。

### 【输出】



 一个整数，表示满足条件的最少苹果个数。

### 【输入样例】

8

### 【输出样例】

36

#include<iostream>
using namespace std;
int main()
{
	int n,sum=0;
	int i;

	cin>>n;//人数
	for(i=1;i<=n;i++)//第一个从1开始，其后每人比前一人多一个
		sum+=i;//累计
	cout<<sum<<endl;
	return 0;
}
