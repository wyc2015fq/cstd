### 【题目描述】


利用公式C = 5*(F-32)/9(其中C表示摄氏温度，F表示华氏温度)进行计算转化,输入华氏温度f，输出摄氏温度c，要求精确到小数点后5位。


### 【输入】


输入一行，包含一个实数f，表示华氏温度。（f >= -459.67）




### 【输出】


输出一行，包含一个实数，表示对应的摄氏温度，要求精确到小数点后5位。


### 【输入样例】

41

### 【输出样例】

5.00000

#include<iostream>
#include<iomanip>
using namespace std;
int main()
{ 
	double c,f;
	cin>>f;
	c=5.0*(f-32)/9;
	cout<<setiosflags(ios::fixed)<<setprecision(5);
        cout<<c<<endl;
  	return 0;
}
