## 题目描述

设有n个正整数（n≤20），将它们联接成一排，组成一个最大的多位整数。

例如：n=3时，3个整数13，312，343联接成的最大整数为：34331213

又如：n=4时，4个整数7，13，4，246联接成的最大整数为：7424613

## 输入输出格式

**输入格式：**

第一行，一个正整数n。

第二行，n个正整数。

**输出格式：**

一个正整数，表示最大的整数

## 输入输出样例

**输入样例#1：**

```
3
13 312 343
```

**输出样例#1：**

```
34331213
```

———————————————————————————————————————————————

思路：水题一枚，string类输入，用sort按拼接后降序排序，输出即可

## 源代码

```cpp
#include<iostream>
#include<algorithm>
using namespace std;
bool cmp(string a,string b)
{
    return a+b>b+a;
}
int main()
{
    int n;
    string str[20];
    int i;

    cin>>n;
    for(i=0;i<n;i++)
        cin>>str[i];

    sort(str,str+n,cmp);//按拼接后最大排序

    for(i=0;i<n;i++)
        cout<<str[i];
    return 0;
}
```
