# 【安装】

## 1.下载 Eclipse 安装包

进入官网，根据操作系统的位数来选择所需的安装包

官网：[点击这里](https://www.eclipse.org/downloads/)

## 2.安装

下载完成后，选择好路径安装即可

![](https://img-blog.csdn.net/20180910103747225?watermark/2/text/aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3UwMTE4MTU0MDQ=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70)

# 【汉化】

## 1.确定版本

找到 Eclisp 的安装位置,打开 readme 文件夹，可以看见一个 HTML 文件

![](https://img-blog.csdn.net/20180910103908415?watermark/2/text/aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3UwMTE4MTU0MDQ=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70)

用浏览器打开，然后就可以看到版本信息了，如下图的版本是 Neon 的 4.6.0

![](https://img-blog.csdn.net/20180910103958864?watermark/2/text/aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3UwMTE4MTU0MDQ=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70)

## 2.下载相关汉化包

进入官网，选择相应版本的多国语言包

官网：[点击这里](http://www.eclipse.org/babel/downloads.php)

![](https://img-blog.csdn.net/20180910104143912?watermark/2/text/aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3UwMTE4MTU0MDQ=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70)

进去后，选择所需的汉化包下载即可

![](https://img-blog.csdn.net/20180910104331182?watermark/2/text/aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L3UwMTE4MTU0MDQ=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70)

## 3.安装汉化版

下载完成后，解压得到两个文件夹，将文件夹拖入安装目录的 dropins 中，然后重启 Eclipse 即可
