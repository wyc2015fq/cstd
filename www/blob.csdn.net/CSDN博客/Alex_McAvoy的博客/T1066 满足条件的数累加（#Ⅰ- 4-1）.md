### 【题目描述】

将正整数m和n之间(包括m和n)能被17整除的数累加，其中0<m<n<1000。


### 【输入】



一行，包含两个整数m和n，其间，以一个空格间隔。

### 【输出】

输出一行，包行一个整数，表示累加的结果。


### 【输入样例】

50 85

### 【输出样例】

204

#include<iostream>
using namespace std;
int main()
{
	int m,n,sum=0;
	int i;

	cin>>m>>n;//输入m、n
	for(i=m;i<=n;i++)//在区间内循环
		if(i%17==0)//满足条件
			sum+=i;//累加

	cout<<sum<<endl;
        return 0;
}
