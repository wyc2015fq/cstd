### 【题目描述】



给定一个长度为n的非负整数序列，请计算序列的最大跨度值(最大跨度值 = 最大值减去最小值)

### 【输入】



一共2行，第一行为序列的个数n（1 ≤ n ≤ 1000)，第二行为序列的n个不超过1000的非负整数，整数之间以一个空格分隔。

### 【输出】



 输出一行，表示序列的最大跨度值。


### 【输入样例】

6
3 0 8 7 5 9

### 【输出样例】

9

#include<iostream>
using namespace std;
int main()
{
        int n,a[1001];
	int min=9999,max=0;//初始化
	int i;
    
	cin>>n;
	for(i=1;i<=n;i++)
	{
		cin>>a[i];
		if(a[i]>max)//最大值
			max=a[i];
		if(a[i]<min)//最小值
			min=a[i];
	}

	cout<<max-min<<endl;//跨度值
        return 0;
}
