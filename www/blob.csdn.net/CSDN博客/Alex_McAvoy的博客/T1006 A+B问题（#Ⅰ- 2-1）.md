### 【题目描述】


求两个整数的和。

### 【输入】


只有一行，两个用空格隔开的整数。



### 【输出】


两个整数的和。

### 【输入样例】

1 2

### 【输出样例】

3

### 【源程序】


```cpp

```

```cpp
#include<iostream>
using namespace std;
int main()
{
	int a,b;
	int sum;
	cin>>a>>b;
	sum=a+b;
	cout<<sum<<endl;
	return 0;
}
```
