### 【题目描述】




输入一个浮点数，输出这个浮点数的绝对值，保留到小数点后两位。


### 【输入】








输入一个浮点数，其绝对值不超过10000。

### 【输出】








输出这个浮点数的绝对值，保留到小数点后两位。

### 【输入样例】

-3.14

### 【输出样例】

3.14

### 【源程序】


```cpp

```

```cpp
#include <iostream>
#include <iomanip>
using namespace std;
int main()
{
	double N;
	cin>>N;
 	cout<<setiosflags(ios::fixed)<<setprecision(2);
 	if(N>=0) cout<<N<<endl;
 	else cout<<-N<<endl;
 	return 0;
}
```
