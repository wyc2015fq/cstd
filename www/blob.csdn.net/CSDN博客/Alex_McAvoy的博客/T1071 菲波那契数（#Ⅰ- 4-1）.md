### 【题目描述】



菲波那契数列是指这样的数列: 数列的第一个和第二个数都为1，接下来每个数都等于前面2个数之和。给出一个正整数k，要求菲波那契数列中第k个数是多少。

### 【输入】





输入一行，包含一个正整数k。（1 ≤ k ≤ 46）

### 【输出】



输出一行，包含一个正整数，表示菲波那契数列中第k个数的大小。

### 【输入样例】

19

### 【输出样例】

4181

#include<iostream>
using namespace std;
int main()
{
	int k;
	int a=1,b=1,temp;//a、b为第1、2个数，temp用于存储第i个数
	int i;

	cin>>k;
	
	for(i=3;i<=k;i++)
	{
		temp=a+b;//每个数都为前两数之和
		a=b;
		b=temp;
	}
	cout<<b<<endl;
        return 0;
}
