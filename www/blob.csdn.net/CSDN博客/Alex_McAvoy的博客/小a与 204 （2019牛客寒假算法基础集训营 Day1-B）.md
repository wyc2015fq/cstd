# 【题目描述】

小a非常喜欢204这个数字，因为′a′+′k′=204。

现在他有一个长度为n的序列，其中只含有2,0,4这三种数字

设ai为序列中第i个数，你需要重新排列这个数列，使得 ![\sum _{i=1}^n(a_i-a_{i-1})^2](https://private.codecogs.com/gif.latex?%5Csum%20_%7Bi%3D1%7D%5En%28a_i-a_%7Bi-1%7D%29%5E2) 最大

(公式的含义是：每个数与前一个数差的平方的和)

注意：我们默认a0=0

# 【输入描述】

第一行一个整数n

接下来一行n个整数，第i个数表示ai

# 【输出描述】

输出一个整数，表示 ![\sum _{i=1}^n(a_i-a_{i-1})^2](https://private.codecogs.com/gif.latex?%5Csum%20_%7Bi%3D1%7D%5En%28a_i-a_%7Bi-1%7D%29%5E2) 的最大值

# 【样例】

示例1

输入

2

2 4

输出

20

说明

按(4,2)排列是最优的，此时sum=(4−0)^2+(2−4)^2=20

示例2

输入

3

2 0 4

输出

36

说明

按(4,0,2)排列是最优的，此时sum=(4−0)^2+(0−4)^2+(2−0)^2=36


思路：贪心+构造

由于不需要输出方案，因此只需要按照题意进行构造即可
