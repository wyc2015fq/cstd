### 【题目描述】




给出一个等差数列的前两项a1，a2，求第n项是多少。。

### 【输入】






一行，包含三个整数a1，a2，na1，a2，n。−100≤a1,a2≤100，0<n≤1000。

### 【输出】






一个整数，即第n项的值。。

### 【输入样例】

1 4 100

### 【输出样例】

298

### 【源程序】


```cpp

```

```cpp

```

```cpp
#include<iostream>
using namespace std;
int main()
{
	int a_1,a_2,n;
	int a_n,d;	
	cin>>a_1>>a_2>>n;
	d=a_2-a_1;//求公差
	a_n=a_1+(n-1)*d;//等差数列通项公式
	cout<<a_n<<endl;	
	return 0;
}
```
