### 【题目描述】

给定一棵树，输出树的根root，孩子最多的结点max以及他的孩子。

### 【输入】



第一行：n（结点个数≤100），m（边数≤200）。

以下m行：每行两个结点x和y，表示y是x的孩子(x,y≤1000)。

### 【输出】

第一行：树根：root；

第二行：孩子最多的结点max；

第三行：max的孩子（按编号由小到输出）。

### 【输入样例】

```
8 7
4 1
4 2
1 3
1 5
2 6
2 7
2 8
```

### 【输出样例】

```
4
2 
6 7 8
```

### 【源程序】

```cpp
#include<iostream>
#include<cstdio>
#include<cstring>
#include<cmath>
#include<algorithm>
#include<string>
#include<cstdlib>
#include<queue>
#include<vector>
#define INF 0x3f3f3f3f
#define PI acos(-1.0)
#define N 101
#define MOD 123
#define E 1e-6
int tree[101];
using namespace std;
int main()
{
    int n,m;
    int x,y;

    cin>>n>>m;
    for(int i=1;i<=m;i++)
    {
        cin>>x>>y;
        tree[y]=x;
    }

    int root;
    for(int i=1;i<=n;i++)
        if(tree[i]==0)
        {
            root=i;
            break;

        }
    int maxx=-INF;
    int maxroot;
    for(int i=1;i<=n;i++)
    {
        int sum=0;
        for(int j=1;j<=n;j++)
            if(tree[j]==i)
                sum++;
        if(maxx<sum)
        {
            maxx=sum;
            maxroot=i;
        }
    }

    cout<<root<<endl;
    cout<<maxroot<<endl;
    for(int i=1;i<=n;i++)
        if(tree[i]==maxroot)
            cout<<i<<" ";
    cout<<endl;

    return 0;
}
```
