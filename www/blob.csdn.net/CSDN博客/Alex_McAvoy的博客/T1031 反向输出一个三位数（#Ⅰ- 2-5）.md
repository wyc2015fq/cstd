### 【题目描述】






将一个三位数反向输出，例如输入358，反向输出853。

### 【输入】



一个三位数n。

### 【输出】




反向输出n。

### 【输入样例】

100

### 【输出样例】

001

### 【源程序】


```cpp

```

```cpp

```

```cpp
#include<iostream>
using namespace std;
int main()
{
	int n;
	int a,b,c;
	cin>>n;
	a=n/100;
	b=n%100/10;
	c=n%10;
	cout<<c<<b<<a<<endl;
	return 0;
}
```
