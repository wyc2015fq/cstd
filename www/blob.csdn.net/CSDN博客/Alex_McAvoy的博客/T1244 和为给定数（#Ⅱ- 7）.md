### 【题目描述】

给出若干个整数，询问其中是否有一对数的和等于给定的数。

### 【输入】

第一行是整数n(0 < n ≤ 100,000)，表示有n个整数。

第二行是n个整数。整数的范围是在0到108之间。

第三行是一个整数m（0≤m≤230)，表示需要得到的和。

### 【输出】

若存在和为m的数对，输出两个整数，小的在前，大的在后，中间用单个空格隔开。若有多个数对满足条件，选择数对中较小的数更小的。若找不到符合要求的数对，输出一行No。

### 【输入样例】

```
4
2 5 1 4
6
```

### 【输出样例】
`1 5`
### 【源程序】

```cpp
#include<iostream>
#include<cstdio>
#include<cstdlib>
#include<cstring>
#include<cmath>
#include<algorithm>
#include<string>
#define INF 999999999
#define N 1000001
#define MOD 1000000007
#define E 1e-3
using namespace std;
int a[N];
int main()
{
    long long n,m,i,j;
    int tot=0;
    cin>>n;
    for(int i=1;i<=n;i++)
        cin>>a[i];
    cin>>m;
    sort(a+1,a+1+n);
    i=1;j=n;
    while(i<=j)
    {
        if(a[i]+a[j]==m)
        {
            cout<<a[i]<<" "<<a[j]<<endl;
            exit(0);
        }
        else if(a[i]+a[j]<m)
            i++;
        else
            j--;
    }
    cout<<"No"<<endl;
    return 0;
}
```
