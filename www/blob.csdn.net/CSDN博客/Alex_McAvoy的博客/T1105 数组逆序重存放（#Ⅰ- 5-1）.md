### 【题目描述】

将一个数组中的值按逆序重新存放。例如，原来的顺序为8,6,5,4,1。要求改为1,4,5,6,8。


### 【输入】

输入为两行：第一行数组中元素的个数n（1<n<100)，第二行是n个整数，每两个整数之间用空格分隔。


### 【输出】

输出为一行：输出逆序后数组的整数，每两个整数之间用空格分隔。

### 【输入样例】

5
8 6 5 4 1


### 【输出样例】

1 4 5 6 8

### 【源程序】

```cpp
#include<iostream>
using namespace std;
int main()
{
    int n;
    int num[101];
    int i;
    
    cin>>n;//元素个数n
    for(i=1;i<=n;i++)
            cin>>num[i];//n个整数
    
    for(i=n;i>=1;i--)//倒序输出
    	cout<<num[i]<<" ";
    
    cout<<endl;
    return 0;
}
```
