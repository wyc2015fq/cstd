### 【题目描述】

如果一个数从左边读和从右边读都是同一个数，就称为回文数。例如6886就是一个回文数，求出所有的既是回文数又是素数的三位数。


### 【输入】

（无）


### 【输出】

所有的既是回文数又是素数的三位数。一个数一行。


### 【输入样例】

（无）

### 【输出样例】

（无）

### 【源程序】

```cpp
#include<iostream>
#include<cmath>
using namespace std;
bool judge(int x);

int main()
{
	int i;
	for(i=100;i<=999;i++)//从枚举所有三位数
		if( judge(i) && (i/100==i%10))//该数是素数且百位与个位相同
			cout<<i<<endl;//输出该数
	return 0;
}

bool judge(int x)//判断素数
{
	int i=2;
	while( i<=floor(sqrt(x)) && (x%i!=0) )
		i++;
	if(i>floor(sqrt(x)))
		return true;
	return false;
}
```
