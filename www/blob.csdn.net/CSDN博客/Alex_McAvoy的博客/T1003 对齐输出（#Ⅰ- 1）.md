### 【题目描述】




读入三个整数，按每个整数占8个字符的宽度，右对齐输出它们，按照格式要求依次输出三个整数，之间以一个空格分开。

### 【输入】


只有一行，按照格式要求依次输出三个整数，之间以一个空格分开。



### 【输出】


只有一行，一个整数，即输入的第二个整数。

### 【输入样例】

123456789 0 -1

### 【输出样例】

123456789       0      -1

### 【源程序】


```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int main()
{
	int a,b,c;
	cin>>a>>b>>c;
	printf("%8d %8d %8d",a,b,c);
	return 0;
}
```
