# 【程序】

1.程序：一组计算机能识别和执行的指令。

2.程序设计语言：计算机工作的工具，是由字、词和语法规则构成的指令系统。

3.程序设计：根据特定的问题，使用某种程序设计语言，设计出计算机执行的指令序列。

4.程序设计要完成的工作

1）数据描述：将被处理信息转换为计算机可接受的数据形式。

2）数据处理：对数据进行输入、输出、整理、计算等工作。

# 【程序设计语言的发展】

1.机器语言：机器语言指令系统，由 0、1 构成的指令码组。

2.汇编语言：用助记符号描述的指令系统。

3.高级语言：接近于人的自然语言和数学语言，简单易懂，基本上不依赖于具体的机型。

注：① 机器语言、汇编语言统称低级语言，其是面向机器的。

       ② 高级语言可分为面向过程与面向对象。

# 【C 语言与 C++】

C 语言，其语言简洁、紧凑、灵活，运算符和数据类型丰富，程序设计可实现结构化、模块化，允许直接访问物理地址，可直接对硬件操作，生成目标代码质量高、可移植性好，语法限制不太严格，程序设计自由度大。

其是一门面向过程的中级语言，而 C++ 是在 C 语言基础上发展起来的一门兼容 C 语言的面向对象的高级语言。

# 【C++ 程序结构特点】

## 1.函数是基本组成单位

1）有且仅有一个主函数（ main() ）

2）运行时，从主函数开始，到主函数结束

3）主函数无需放在整个程序最前面

4）其他函数均是被主函数直接或间接调用来执行

5）其他函数为自定义函数，也可为库函数

## 2.程序的书写自由

1）一行可有多个语句（以分号结尾为一语句）

2）提倡一行一个语句，每个语句后均要有分号

3）常用锯齿形书写格式

4）大小写敏感

5）习惯是小写，个别大写（均为半角状态）

6）可以空行

**注：**

**① 半角：一个字符占一个标准字符位置（1字节）**

**② 全角：一个字符占两个标准字符位置（2字节）**

## 3.程序中应习惯包含注释

1）行注释：以 " // " 开始的文本置于行末

2）块注释：以 " /*...*/ " 相括的文本可置于程序任何位置

## 4.一个程序可由多个源程序组成

源程序：高级语言编写的程序，计算机无法直接识别，需要通过 “编译程序” 将其编译成机器语言才可执行。

## 5.源程序的编译执行

1）编辑：使用编辑器录入源程序，最后生成文本文件（*.cpp）

2）编译（*.cpp→*.exe）

① 预处理：执行编译前的准备，包括执行包含指令、宏替换命令等

② 语法检查：检查是否有语法错误，无误后生成目标代码文件（*.obj）

③ 连接处理：将目标代码文件与库文件连接成为可执行文件（*.exe）

3）运行：可执行文件可直接于 Dos、Windows 等环境下执行

## 6.源程序编写时的错误种类

1）编译错误：编译时的语法错误，如缺少分号

2）连接错误：连接时的错误，如找不到连接库文件

3）运行错误：运行时的错误，如求 sqrt(x) 时，x 值为负

4）逻辑错误：编译、运行均无法发现的错误，如执行 2/4 时，期望值是 0.5，但由于 C++ 作整除运算，实际值为 0

# 【库文件与头文件】

1.库文件：将一系列源文件编译打包后形成的二进制文件包，其中封存函数借口，可由其他源文件调用，是编译生成的，仅于连接时使用。

2.头文件：以 #include 形式包含的文件，对于 C 语言，一般以 .h 为扩展名，对于 C++，一般无扩展名，属于文本文件，是手动编写的，于编译时使用。

3.库文件与头文件的联系

1）当调用库函数时需要引用接口函数所在的头文件

2）一个库文件均有一组头文件与其对应

3）头文件中有函数的声明，库文件中有函数的定义。

# 【字符集】

所有的语言系统都是由字符集和规则集组成的，“字符”是语言的不可分解的最基本语法单位，由字符可以组成“词”，由词组成“表达式”、“句子”，又由各种句子构成“函数”、“程序”。

C++ 的字符集是 ASCII码的子集，包括：26个大小写字母、10个数字、基本符号等。

# 【词汇】

词汇是语言中有意义的最小语法单位。根据形成规则，一个单词由一个或多个字符组成。

## 1.关键字

系统预定义的语义符，不允许重复定义。

```
auto    break    case    char    class    const    continue    default    delete    else
enum    explicit    extern    float    for    friend    goto    if    inline    int    long 
new    operator   private    protected    public    register    return    short    signed 
sizeof    static    struct    switch    this    typedef    union    unsigned    virtual   void    while
```

## 2.标识符

1）定义：用来标识变量、函数、数组等的字符序列。

2）语法：由字母、数字、下划线组成，且第一个字母必须是字母或下划线，大小写敏感，不能使用关键字。

3）长度：随系统而异，但至少前8个字符有效，建议标识符的长度最好不要超过8个字符。

4）命名原则：见名知意，不易混淆

## 3.运算符

对数据进行操作的简洁表达，以单词的形式调用系统预定义函数。

```
34种运算符：
算术运算符：+  -  *  /  %  ++  --
关系运算符：<  <=   ==   >   >=   !=
逻辑运算符：！  &&  ||
位运算符  ：<<   >>   ~  |  ^  &
赋值运算符：= 及其扩展
条件运算符：?:
逗号运算符：,
指针运算符：*  &
求字节数   ：sizeof
强制类型转换：(类型）
分量运算符：.  ->
下标运算符：[]
其它     ：( )  -
```

## 4.分隔符

用于分隔不同的语法单位

1）空格：多用于语句各单词之间，作间隔符。

2）逗号：逗号主要用于类型说明和函数参数表中，分隔各个变量。

3）分号：用于语句结束。
