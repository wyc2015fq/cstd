### 【题目描述】

给定两个正整数，求它们的最大公约数。

### 【输入】

输入一行，包含两个正整数(<1,000,000,000)。

### 【输出】

输出一个正整数，即这两个正整数的最大公约数。

### 【输入样例】
`6 9`
### 【输出样例】
`3`
### 【源程序】

```cpp
#include<iostream>
#include<cstdio>
#include<cstdlib>
#include<cstring>
#define N 1000010
using namespace std;
int calculate(int a,int b)
{
    if(b==0)
        return a;
    return calculate(b,a%b);
}
int main()
{
    int a,b;
    cin>>a>>b;
    if(a>b)
        cout<<calculate(a,b)<<endl;
    else
        cout<<calculate(b,a)<<endl;
    return 0;
}
```
