### 【题目描述】



非负整数n，求2^n，即2的n次方。。

### 【输入】





一个整数n。0 ≤ n < 31。

### 【输出】






一个实数，即线段AB的长度，保留到小数点后3位。

### 【输入样例】

2

### 【输出样例】

8

### 【源程序】


```cpp

```

```cpp

```

```cpp
#include<iostream>
#include<cmath>//pow()函数需调用<cmath>
using namespace std;
int main()
{
	int n,result;
	cin>>n;
	result=pow(2,n);//power(x,y)，求x^y,其中x为底数，y为指数
	cout<<result<<endl;
	return 0;
}
```
