### 【题目描述】

在程序中定义一函数digit(n,k)，它能分离出整数n从右边数第k个数字。


### 【输入】

正整数n和k。


### 【输出】

一个数字。

### 【输入样例】

31859 3

### 【输出样例】

8

### 【源程序】

```cpp
#include<iostream>
using namespace std;
int calculate(int n,int k);

int main()
{
	int n,k;
	cin>>n>>k;
	cout<<calculate(n,k)<<endl;
	return 0;
}

int calculate(int n,int k)
{
	k--;
	if(k==0)//判断是否到达递归边界k=0
		return n%10;
	else//否则继续递归
		return calculate(n/10,k);
}
```
