## 题目描述

有 N 级的台阶，你一开始在底部，每次可以向上迈最多 K 级台阶（最少 1 级），问到达第 N 级台阶有多少种不同方式。

## 输入输出格式

**输入格式：**

两个正整数 N，K。

**输出格式：**

一个正整数，为不同方式数，由于答案可能很大，你需要输出 ans mod100003 后的结果。

## 输入输出样例

**输入样例#1：**
`5 2`
**输出样例#1：**
`8`
## 源代码

```cpp
#include<iostream>
using namespace std;
int main()
{
    
    int n,k;
    int a[100003]={0};
    int i,j;
    
    cin>>n>>k;//输入n、k
    a[0]=1;//初始条件
    for(i=1;i<=n;i++)//从头到尾进行递推
    {	
        for(j=1; j<=k && i-j>=0 ;j++)//当前阶数最少1到最大k阶依次累加
            a[i]+=a[i-j];
        a[i]%=100003;//按要求进行求模，防止数太大导致溢出
    }	
    cout<<a[n]<<endl;
    return 0;
}
```
