用递归函数求解。

### 【输入】

第一数是x的值，第二个数是n的值。


### 【输出】

函数值，保留两位小数。


### 【输入样例】

1 2

### 【输出样例】

0.40

### 【源程序】

```cpp
#include<iostream>
#include<cstdio>
#include<cmath>
using namespace std;
double calculate(double x,double n);

int main()
{
	double n,x;
	double result;
	cin>>x>>n;
	result=calculate(x,n);
	printf("%.2lf\n",result);
	return 0;
}

double calculate(double x,double n)
{
	if(n==1)//判断是否到达递归边界n=1
		return x/(1+x);
	else//否则继续递归
		return x/(n+calculate(x,n-1));
}
```
