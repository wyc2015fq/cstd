# 【cin】






- 作用：把输入的一行作为一个字符串传送到字符数组中
- 特点：以空格、回车为结束标志，自动加'\0'

```cpp
char str[10];
cin>>str;
```







# 【cin.get()】


- 作用：接收从键盘输入的单个字符数据（输入多个字符时，只接收第一个）
- 特点：可以接收回车
- 字符数组调用时：要用到循环，不会自动加'\0'

```cpp
char str[10];
for(i=0;i<10;i++)
    str[i]=cin.get();
```

# 【gets()】


- 作用：把输入的一行作为一个字符串传送到字符数组中
- 特点：以回车结束，自动加'\0'

```cpp
char str[10];
gets(str);
```



# 【cin.getline()】


- 作用：把输入的一行作为一个字符串传送到字符数组中
- 格式：cin.getline(数组名,输入字符串的最大长度,结束标识符（默认为'\n'）)
- 特点：以回车结束，自动加'\0'

```cpp
char str[10];
cin.getline(str,10);
```
