# 【逐个比较】

```cpp
bool isPalindrome(string str){
    for(int i=0;i<str.size()/2;i++)
        if(str[i]!=str[str.size()-1-i])
            return false;
    return true;
}
```

# 【反转比较】

```cpp
bool isPalindrome(string str){
    string temp=str;
    reverse(temp.begin(),temp.end());//反转字符串
    return str==temp;//比较与原字符串是否相同
}
```
