一、LIMIT子句
LIMIT子句用于规定要返回的记录的条数。对于拥有成千上万条记录的大型数据表来说，LIMIT就显得非常有用了。
当要求返回前几行或者中间某几行记录的时候，我们就可以运用LIMIT子句了。
二、LIMIT 语法
SELECT * from 表名 LIMIT 开始位置，行数
其中，开始位置默认是0。LIMIT语句可以接受一个或两个参数，参数必须是一个整数常量。（开始位置和行数，必须是整数常量）第一个参数指定第一个返回记录行的偏移量，也就是返回记录的开始行位置；第二个参数指定返回的记录行数。
注意：开始位置是0二不是1。
三、例子
SELECT * FROM table LIMIT 5,10
查询结果：返回第六行开始的10行记录（6-15行）
四、特殊
为了检索从某一行开始的所有记录行，可以指定第二个参数为-1
例如：SELECT * FROM table LIMIT 95,-1
查询结果：96行开始的所有记录
如果只给定一个参数，它表示返回最大的记录行数目
例如：SELECT * FROM table LIMIT 5
查询结果：返回前5行记录，等价于SELECT * FROM table LIMIT 0,5
五、查找入职员工时间排名倒数第三的员工的所有信息
原始表：
```
   CREATE TABLE `employees` (
`emp_no` int(11) NOT NULL,
`birth_date` date NOT NULL,
`first_name` varchar(14) NOT NULL,
`last_name` varchar(16) NOT NULL,
`gender` char(1) NOT NULL,
`hire_date` date NOT NULL,
PRIMARY KEY (`emp_no`)); 
```
入职时间倒数第三，先按入职时间由大到小排列
SELECT * FROM employees order by hire_date DESC 
然后从第三条记录开始，选取一条记录，也就是倒数第三条。
SELECT * from employees order by hire_date DESC LIMIT 2,1
