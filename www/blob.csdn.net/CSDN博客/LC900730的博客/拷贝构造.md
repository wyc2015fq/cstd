## 拷贝构造函数
如果一个构造函数的第一个参数是自身类类型的引用，且任何额外的参数都有默认值，则此构造函数是拷贝构造函数。
```
class Foo{
    public :
        Foo();
        //大多数下是const，但不是必须的
        Foo(const Foo&); 
}
```
