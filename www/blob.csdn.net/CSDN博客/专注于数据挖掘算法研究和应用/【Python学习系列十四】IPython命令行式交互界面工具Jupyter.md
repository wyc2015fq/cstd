好的IDE对提高编码质量很有帮助，Jupyter Notebook的即编即视效果很有利于调试。

IPython 是 Python 的原生交互式 shell 的增强版，可以完成许多不同寻常的任务，比如帮助实现并行化计算；主要使用它提供的交互性帮助，比如代码着色、改进了的命令行回调、制表符完成、宏功能以及改进了的交互式帮助。
