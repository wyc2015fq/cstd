1.DB2数据库V8.2版本的安装，获取到的版本集合服务端和客户端，选择典型安装，默认即可，服务端可以默认创建一个实例和数据库，创建用户名和密码！

2.客户端和服务端都安装好后，打开服务端的控制中心，创建一个数据库testdb；

3.客户端，菜单：IBM DB2->命令行工具->命令窗口，执行如下命令：

->db2 catalog tcpip node ABC remote ServerIP server 50000


 ->db2 catalog db testdb at node ABC 

 ->db2 connect to testdb user Uid using Pwd

 说明：其中ABC是节点名，任意取的；ServerIP是远程连接服务器的IP地址；Server后面是服务器的实例端口，DB2不同实例端口递增；db后面的testdb是远程服务器要连接的数据库名；Uid是远程服务器实例的用户名；Pwd是远程服务器实例的密码；


4.执行后，打开客户端控制中心查看，发现多了ServerIP 的节点，可以在该节点上操作远程数据库；

5.常用命令：

        1)查看本地节点目录

　　命令窗口中输入：db2 list node directory

　　2)编目一个TCP/IP节点

　　命令窗口：db2 catalog tcpip node <node_name> remote <hostname|ip_address> server <svcname|port_number> ostype <OS2|AIX|WIN95|NT|HPUX|SUN|MVS|OS400|VM|VSE|SCO|SGI|LINUX|DYNIX>

　　3)取消节点编目

　　db2 uncatalog node <node_name>

　　4)查看系统数据库目录

　　db2 list database directory

　　5) 查看本地数据库目录

　　db2 list database directory on <盘符>

　　在本地数据库目录中有而系统数据库目录中没有的数据库不能访问，可以在控制中心中选中<数据库>右键单击选择添加，然后输入需要添加的数据库名称或者点击刷新按钮选择数据库，加入数据库后即可以访问。

　　6) 编目数据库

　　db2 catalog database <db_name> as <db_alias> at node <node_name>

　　7) 取消数据库编目

　　db2 uncatalog database <db_name>
