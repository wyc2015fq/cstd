# 为什么思考？
我们常常听说“思考力决定竞争力”，“思考就是力量”，“思考致富”，“思考方式改变人生道路”，“思考改变命运”，“思考改变生活”。
# 思考是什么？
思考是指对某一个或多个对象进行进行分析﹑综合﹑推理﹑判断等思维活动。
# 思考的内涵：
如何思考，怎样思考，思考什么，如何提高思考效率，显得尤为重要。
思考什么，体现是内容，是目标，是方向；
如何思考，体现的是方法和工具；
而如何改进思考能力，则体现是效率，是加速度。
独立思考，冷静思考，指的是思考的态度；
水平思考，六顶思考帽法，换位思考，指的是思考的方法；
像经济学家一样的思考，指的是思考的方式和思维的模式，他具有固定的程式，流程；
# 持续改进思考能力，才是核心竞争力
我们每天也思考，可谓绞尽脑汁，为什么我们依然囊中羞涩，为什么人生道路依然步履艰难。
难道我们思考的不够多，思考的方向不够对？
其实不尽然。
现在，这个社会，比的是速度，拼的是加速度，持续加速的原动力。
试问君，2008年的29届奥林匹克运动会，100米以及200米短跑的冠军是牙买加“闪电”博尔特，不是其他选手没有跑，也不是其他选手跑得不够快，他们可都是全球各国的顶级选手，而是博尔特的加速度太快了。
所以，**持续改进思考能力才是思考的核心竞争力**。持续改进，说明能够不断自我完善，
长远发展，自我纠正，可以真正实现从优秀走向卓越。
# 如何持续改进思考能力？
倾听大师、业界领袖、传媒、政府、专家学者的声音，分析问题，思考问题的角度，看问题的着眼点，剖析思考的历史根源，多对比，多总结。
# 展望

让我们共同努力吧！持续改进我们的思考能力！
