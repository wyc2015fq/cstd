● Many of  the activities performed during  the preliminary  investigation are  still being conducted in  （71） , but in much greater depth than before. During this phase, the analyst must  become  fully  aware  of  the  （72）and must  develop  enough  knowledge  about  the   （73） and  the  existing  systems  to  enable  an  effective  solution  to  be  proposed  and implemented. Besides the （74）for process and data of current system, the deliverable from this phase also includes the （75）for the proposed system.
（71）
A. analysis phase                 B.  design phase
    C.  implementation phase          D. maintenance phase
（72）
A. main symptom                 B.  root problem
    C.  final blueprint                 D.  data specification
（73）
A. hardware environment          B. testing environment
    C. software environment           D. business environment
（74）
A. logical models                 B. physical models
    C. design models                 D. implementation models
（75）
A. hardware and software specification  B. system performance specification
    C. formal requirements definition       D. general problem statement

(71) A
(72) B
(73) D
(74) A
(75) C

译文：在系统分析时，初步的调研正在被执行的同时，许多活动也要进行开展，但是要比以前更有深度。在这个阶段，分析师必须充分认识到问题的根源，对商业环境和现有的系统，必须拥有足够的知识，以便寻找有效的解决方案，以便建议和实施。从这一阶段成果，除了关于当前系统的过程和数据的逻辑模型，还包括给建议系统的正式的需求定义。
（71）
初步的调查正在被执行，是在哪个阶段
系统分析---系统设计-系统实现—系统维护
初步的调查，只能是系统分析阶段。
所以，答案选择：A
（72）
A. main symptom  主要症状               B.  root problem 问题根源
C.  final blueprint  最后蓝图               D.  data specification数据规范
系统分析，就是需求分析，主要是找出问题，也就是找出问题的根源
C D都不合适
A main symptom主要症状也不合适
the analyst must  become  fully  aware  of  the  （72）
（73）
A. hardware environment   硬件环境      B. testing environment 测试环境
C. software environment   软件环境        D. business environment 商业环境
选C更符合语境
系统分析阶段，关注的是商业环境
系统设计、系统实现阶段关注的是硬件环境、软件环境以及测试环境。
（74）
A. logical models   逻辑模型              B. physical models物理模型
C. design models  设计模型               D. implementation models实现模型
这个模型是为“process and data of current system 当前系统的过程和数据”服务的，
过程和数据，选A
（75）
A. hardware and software specification 硬件和软件规范
B. system performance specification 系统性能指标
C. formal requirements definition  正式需求定义D. general problem statement 一般问题的说明
在这个阶段，需要提交的交付物，应该是正式的需求规格说明书。因为这个阶段，是系统分析阶段。
