# 什么是BREW 分发系统？ - 我相信...... - CSDN博客





2009年10月16日 19:18:00[半吊子全栈工匠](https://me.csdn.net/wireless_com)阅读数：1062








BREW 分发系统(BDS)是专为开发商及运营商设计的商业解决方案, 主要解决如何向用户分发应用软件，以及向开发商就应用软件付款等问题，它在BREW端到端解决方案中占有极其重要的地位，是商业模式的集中体现。

 BDS 提供全面的目录管理功能，可支持运营商维护其产品目录，使运营商可以通过目录定制向用户提供的移动增值业务。通过BDS的管理，在用户的手机设备上安全可靠地空中下载、执行和删除应用。BDS可与运营商预付费、后付费及用户授权业务相集成，实现全面的服务。

 BDS 支持多种财务服务，其中包括：通过包含用于计费的综合使用记录的运营商提取；基于 XML 方式来记录和传播用户的 BREW交易；集成的开发商计费服务，支持根据综合使用记录开具发票、进行调整、收费和多方结算；SMNP、HTTP、Java 和 XML 数据接口可以支持运营商计费集成和其它服务；提供基于企业外联网报告与接口，提供用于用户支持与计费的协调报告。

 同时，BDS 系统非常灵活，支持众多嵌入式操作系统（包括用于 Pocket PC 的 Microsoft Windows 移动软件、Windows 智能电话或 Palm 操作系统）上运行的 BREW 移动软件商店应用程序，向智能电话和 PDA 等其它设备发布应用程序。使用 BDS 还有一个更大的优点，即运营商在外购数据解决方案的同时仍能掌握控制权。拥有这些功能后，运营商可通过安全的应用程序下载增加 ARPU值，增强其移动增值业务的销售服务，从而获得竞争优势。

 BREW 分发系统具有模块化的可扩展架构，运营商可以自行决定对系统的管理程度，与现有系统快速集成并根据需要进行扩展（例如集成DeliveryOne的技术平台）。BREW不仅灵活性高，而且还可向网络运营商提供网络托管的部署选项，例如位于美国明尼苏达州的Midwest Wireless就是使用BREW网络托管服务的一个运营商。同时，运营商还可以选择 BREW 管理的服务选项，这一解决方案能够为运营商提供无需专业数据服务人员托管型共享目录。
