**简评：**GitHub 是一个面向开源及私有软件项目的托管平台，同时它也为程序员提供了生产力工具，旨在帮助开发者更高效地构建软件。本文给大家推荐七种高效生产力工具。
**1. Oh My Zsh**
**![](https://img-blog.csdn.net/20170804142347984?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzA1MTM0ODM=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)**获得超过 43000 颗 star，[Oh My Zsh](https://github.com/robbyrussell/oh-my-zsh) 可以说是 Github 上最受欢迎的生产力工具了。它是一款社区驱动的命令行工具，基于 zsh 命令行，提供了主题配置，插件机制，以及内置的便捷操作。
Github：[robbyrussell/oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)
**2. The Silver Searcher**
**![](https://img-blog.csdn.net/20170804142353072?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzA1MTM0ODM=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)**
比 ack 速度更快的代码搜索工具，而且还可以跟 vim，textmate，emacs 集成。
Github：[ggreer/the_silver_searcher](https://github.com/ggreer/the_silver_searcher)
**3. Quick Look plugins**
**![](https://img-blog.csdn.net/20170804142358034?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzA1MTM0ODM=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)**
Mac OS X 下快速查看工具，要用于补充 MacOS 当中用于实现 Finder 文件管理及 UI shell 的 Quick Look 预览功能。当用户选中一个文件，只需按下空格键即可查看其内容，再按一下空格键则退出 QuickLok，不需要启动关闭任何软件。
Github：[sindresorhus/quick-look-plugins](https://github.com/sindresorhus/quick-look-plugins)
**4. ShareX**
**![](https://img-blog.csdn.net/20170804142402993?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzA1MTM0ODM=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)**
ShareX 是开源的高级截图工具和屏幕记录器。使用 ShareX，只需要一个快捷键就可以保存截图到你的粘贴板，硬盘或者上传到 80 个不同的文件存储服务上。ShareX 的插件还可以上传图片，文本文件和其他各种文件类型。
Github：[ShareX/ShareX](https://github.com/ShareX/ShareX)
**5. Homebrew**
**![](https://img-blog.csdn.net/20170804142407721?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzA1MTM0ODM=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)**
Homebrew 是 Mac OS X 上的软件包管理工具，能在 Mac 中方便地安装或者卸载软件，可以说它就是 Mac 下的 apt-get、yum 神器。
Github：[Homebrew](https://github.com/Homebrew)
**6. dotjs**
**![](https://img-blog.csdn.net/20170804142412965?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzA1MTM0ODM=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)**
dot.js 是一个短小精悍的 js 模板引擎，压缩版仅有4K大小。
Github：[defunkt/dotjs](https://github.com/defunkt/dotjs)
**7. GitHub Changelog Generator**
**![](https://img-blog.csdn.net/20170804142417948?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzA1MTM0ODM=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)**
它是一款命令行工具，能够自动通过 GitHub 上的标签、问题以及 pull 请求生成变更日志。该项目旨在帮助用户与贡献者更为轻松地查看项目新版本中的重要变更。
GitHub：[skywinder/github-changelog-generator](https://github.com/skywinder/github-changelog-generator)
