# 方正大搞内幕交易，北大为何捂盖子？
# 简而言之，捂盖子就是“护短”、遮丑。
# 堂堂世界知名学府北京大学，养育出方正这个劣子，可悲也。
# 什么叫“内幕交易”？怎么搞法？近日，中央有关部门发文，要求整顿金融市场秩序。北大领导不知有何想法？
# 2014年11月2日，北大医药股价暴跌7.57%，应声停牌。这一切的始作俑者和实际操控者就是北大方正CEO李友!依托着北大方正集团这个国企平台，方正集团CEO李友把魔爪伸向金融股市，通过内幕交易获取暴利，使众多小股民惨遭血洗。可恨否？
# 北大方正党委书记兼董事长魏新高度评价李友的能力。但是，李友是如何通过操纵股票市场获取暴利的，至今还缺乏深入分析、批判。此事，我们还要从2013年说起。2013年6月份，北大医药发出公告，称根据相关信息披露规定，拟转让的本公司股份合计不超过7000万股，占总股本的11.75%以下。该公司控股股东西南合成医药集团转让总数量为3000万股，由北大教育基金作为受让方，转让价格为9.2元/股，占总股本的比例为5.03%。与此同时，北大国际医院集团转让的股份总数量为4000万股，受让方为政泉控股，转让价格也为9.2元/股，占总股本的比例为6.71%。
# 这是方正集团CEO李友运作的开始，在受让股份之前，李友指使其掌控的深圳康隆科技发展有限公司向政泉控股汇款1.104亿元，并告知该款是股权收购款的一部分，随后方正集团与政泉控股签订股票“代持协议”（所谓“代持门”事件）。9月2日，方正李友再次指使北大资源向政泉控股支付了剩余股权收购款共计人民币2.576亿元。9月25日，李友又指使政泉控股将该4000万股股票质押给中信证券股份有限公司为其融资2亿元，
# 然后，在9月26日将融资的2亿元款项支付至北大资源。
# 2014年7月4日，李友指示政泉控股解除部分股票的质押，当日政泉控股收到李友掌控的成都市华鼎文化发展有限公司转来1.05亿元，将该款全部用于偿还中信证券的融资欠款，将2000万股解押。接下来，根据李友指示，2014年8月以其抛售所得归还中信证券融资的剩余本金9500万元，其余款项扣除股票交易手续费后均支付至北大资源指定账户中（注意：这是私人账户）。政泉控股在2014年8月27日至29日共向北大资源交付股票转让款2560万元，随后该款又全部被李友转移到了其掌控的河南元和园林绿化工程有限公司。9月9日，李友在幕后的全局操弄，政泉控股出手2000万股，股价在18.87元（注意：每股股价增加近10元人民币！）。
# 李友这一系列的幕后操作行为共获利3.55亿余元，是再明显不过的通过内幕交易获取暴利，破坏国家金融秩序的行为。
