# UbuntuKylin 13.04发行版究竟应该由谁来发布？ - 袁萌专栏 - CSDN博客
2013年04月26日 10:43:37[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：1541
     这个问题是小题大做吗？显然不是。为什么？
4月25日，在Ubuntu官方网站的“发布注意”（ReleaseNotes）栏目的引言中，第一句话就是：”The
 Ubuntu developers are moving quickly to bring you the absolute latest and greatest software the Open Source community has to offer. The Ubuntu 13.04 Release is the next version of Ubuntu“，核心意思是，开源社区提供了最新且伟大的软件，......由此可见，软件发布的主体（版权人）是Ubuntu开发者(团体)。
4月25日，Ubuntu
 13.04在UbuntuGeek的发布文字里面，是这样说的:”The 
 Ubuntu team is very pleased to announce the release of Ubuntu 13.04 for Desktop，......”这就是说，Ubuntu团队是发布的版权人主体。
4月25日，在北京举行的Ubuntu麒麟13.04发布会上却完全变了味儿，由挂靠工信部CSIP的CCN联合实验室来发布，而根本不提Ubuntu麒麟的开发团队。这是很不对的。实际上，GNU/Linux发行版是有版权人的，只不过遵守GNU
 GPL版权规则而已。
