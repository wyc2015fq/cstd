# 微软为何推出便携式Win 8？ - 袁萌专栏 - CSDN博客
2013年04月09日 03:33:14[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：1634
    2012年10月26日，微软正式发布视窗操作系统Windows
 8，但是，对我们国内不再销售盒装Win 8，盗版Win
 8更加困难了。人们急切地等待盗版Win 8从天而降，我看，没戏。Win
 8盗版绝非一件易事。
在Win 8专业版或企业版里面有个U盘安装工具，叫Win
 8 To Go，利用这个安装工具，Win
 8操作系统就可以植入自启动U盘直接启动计算机了。对微软而言，这算是一种历史性的进步。ZDNet记者称：Windows
 To Go "One of the more interesting features of Windows 8",并且指出，"Even though
 we were using a USB 2.0 port performance was good, with nonoticeable lag"，最后评价说：这是一种非常有用的方法（"a
 very useful way of running of Windows 8）。为什么会是这样呢？连微软也搞起了便携式软件？
现在，世界上主要的Linux发行版几乎都有Live
 USB版本（共有72种），比前几年上升近一倍。为什么会出现便携式软件（主要是操作系统）的这股“旋风”？最后，连微软也扛不住了？微软参与便携式软件业务（顺应时代潮流）是一种信号：便携式软件将会成为发展的潮流，不再是旁门左道，不再是雕虫小技。
其实，便携式软件的主要优点就是计算机隐私信息的保护。比如，不论你在谁的计算机上运行便携式Win8，当你拔出U盘走人之后，在60秒钟之后，计算机就会自动关机。你的计算机隐私信息（所谓使用计算机的“痕迹”）就会烟消云散，一切化为乌有。家人也找不到你在外面有“外遇”的蛛丝马迹。
大约在2007年的下半年，我就开始鼓动人们搞“电子书包”租赁业务，其实这就是出租自启动U盘的业务，比如，企业与中小学的校方联手搞这项U盘出租业务（利益共享）。按照教学大纲，分年级发放U盘，学生家长需要为此预付押金。学生升级之后，退回押金，重新租用相应的U盘（各年级使用不同的U盘）。学生在学校的电脑教室上课，插入各自的U盘即可。回家之后，把U盘插入计算机就可以继续完成白天的学习内容了。要让孩子有自己的“小世界”，尽早懂得正确使用电子信箱，保护自己的计算机隐私，做一个现代文明的孩子。
政府文职人员也应当学会使用便携式U盘，比如，LibreOffice
 Portable4.0.1，该版本与微软Office办公套件百分之百兼容。但是，制作自启动U盘却不是一件易事。这就是一种商业机会，市场前景可以预期。。
