       当前，有一帮技术高强的Linux死硬分子，他们与我国有关电脑生产厂商沟通、洽谈两年之久，决定在我国境内大批量生产售价只有100元人民币（约15美元）的Linux计算机，性能也相当适用，你相信吗？
在这个世界上，什么怪人（包括企业）都有，Rhombus(菱形)公司便是一例。首先，我们要搞清楚这个所谓的”菱形公司“是一种”CIC组织“，就是”Community
 Interest Company“（性质很怪），意思是为某种社区利益而整日奔忙的（非赢利）企业。在我们看来，他们无疑就是一帮现代乌托邦分子。
他们超低价Linux计算机的硬件配置如下：
- 
1.5ghzCortex A8 ARM Core
- 
MALI 400MP  OpenGL ES 2.0 GPU
- 
2160p Hardware-accelerated Video playback
- 
Up to 1gbof DDR3 (800mhz) RAM
- 
A NAND Flash Controller that is capable of 8-way concurrent DMA (8 NANDICs) as well as supporting up to 500mhz DDR2 RAM
- 
4 SDIO interfaces (SD 3.0, UHI class)
- 
USB 2.0 Host as well as a 2nd USB-OTG Interface (USB-OTG can be reconfigured as USB 2.0 Host, automatically)
- 
24-pinRGB/TTL as well as simultaneous HDMI out
- 
SATA-II3gb/sec
- 
10/100Ethernet (MII compatible)
- 
A 2nd 24-pin RGB/TTL interface that is multiplexed (shared) on the samepins for a standard IDE (PATA) interface.
- 
GPIO, I2C,PWM, Keyboard Matrix (8x8), built-in Resistive Touch screen Controller；
从以上硬件配置来看，1.5GHz的ARM处理器，1GB内存，还有图形加速卡，等等，确实不错。他们想直接到超市销售，避开北京中关村的黑心老板。此举，我双手赞成。什么时间才能生产上市呢？答案是：“The
 development is going on rapidly and sooner or later they will deliver it”，意思是说，事情进展很快，或早或晚，“They will deliver it“。

 A10“，中文意思是”全胜者A10“。如今，计算机比手机还要便宜，是不是人们都发疯了？请见“Now
 $15  Linux Computer?”及文中所附的“FAQ“（精彩之极！）。
