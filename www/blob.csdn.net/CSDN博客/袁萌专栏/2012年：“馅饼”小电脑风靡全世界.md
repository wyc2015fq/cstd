    本文中的所谓“馅饼“小电脑是指源自英国的“山莓馅饼”（“RaspberryPi”）小电脑，零售价25美元一个“单元”（暂无包装盒，裸体机）。这真是天上掉”馅饼“（英国人叫“Pi“，中国人也叫”派“），形成抢购风。何故也？
这种馅饼电脑是纸糊的吗？非也。其所用的“片上系统”是：The
 SoC is a Broadcom BCM2835. This contains an ARM1176JZFS, with floating point, running at 700Mhz, and a Videocore 4 GPU. The GPU is capable of BluRay quality playback, using H.264 at 40MBits/s. It hasa fast 3D core accessed using the supplied OpenGL ES2.0 and
OpenVG libraries“，本领可谓“高强”，结构图如下：

[在此，实体图省略。该小电脑的体积不足香烟盒大小，重量只有50克，电耗不足5瓦。详细情况，请见www.raspberrypi.org网站的Q&As栏目，内容十分详尽。该小电脑自2月29日开始征订以来，订单从全球四面八方飞向“山莓馅饼”基金会，简直应付不了。在3～4月订货（下订单），5月就可以收到该小电脑产品，不论客户在何方。](http://www.raspberrypi.org/wp-content/uploads/2011/07/raspi_blue_white.png)
