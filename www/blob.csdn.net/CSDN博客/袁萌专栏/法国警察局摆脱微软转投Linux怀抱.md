# 法国警察局摆脱微软转投Linux怀抱 - 袁萌专栏 - CSDN博客
2010年11月20日 06:06:00[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：1506
         11月18日消息，法国警察局近日已经（has upgraded)把8万5千台PC计算机升级到Ubuntu桌面版，完全摆脱了对微软操作系统的依赖性（原文是（“removing its reliance on the Microsoft operating system completely”）。我们应该怎样看待此事？
        
       回顾往事，法国巴黎市的女警察，全身武装，英姿焕发，是一道亮丽的巴黎风景线。让我没有想到的是，到了今天，他（她）们和我一样都在使用Ubuntu桌面了。法国人是傻子吗？不，法国人办事精明透顶，决不干吃亏的事儿。实际上，法国警察局采用Ubuntu系统，每年就可节省200万欧元的软件使用许可费，而且，他们把其中的4500台Ubuntu系统（高档PC计算机）改成了服务器使用，进一步减低了硬件的投入成本。
    法国人算了一笔帐。在以往，“an IT administrator needed to be traveling all year to physically install new versions of anti-virus applications on the desktops”，意思是说，需要安排一名IT系统管理员全国到处跑，到各地警察分局去安装新版杀毒软件。他们还发现，Ubuntu相比Windows更加易于管理和维护，在全国范围内部署什么新的应用软件，不必和以前那样需要指派专人前往安装系统，现在，可以通过“远程部署”的方式，2周以内就可全国“搞定”。
       回到我们国内，现在，全国政府机关都要实现“正版化”，其中仅是办公套件就至少需要100万套。我们来算个帐。假定一套办公套件（至少）需要200元人民币，这就需要2个亿。在某些人看来，两个亿不算什么，是个“小意思”。但是，我不同意这种观点。对于我们国家而言，能省下20块钱都是好的。平日里，我外出时身边只带着20～30块钱，已经感觉不少了。为什么我们不能学习法国警察局改用Ubuntu系统？我使用Ubuntu桌面已经有5～6年，从来没有死过机，不知病毒、木马为何物，而且，Ubuntu桌面越来越好用了。凭什么说部分政府机关人员不能用Ubuntu桌面？谁说不行？你站出来说话啊！
      实际上，还有一则消息报道说：“The German police also is massively switching to Linux. And some other German institutions already are using various Linux distribution, not only because of saving lots of money, also because of security”，意思是说，德国警察早已大量使用Linux，而且某些其他的（德国）社会机构已经开始使用各种Linux发行版，不仅是为了省钱，而且也是考虑到应用系统的安全性（不能吊死在某个特定的Linux版本上）。
    
      说明：我不曾看到有哪位政府官员说自己要使用Linux桌面。我不知道，这是怎么搞的（是什么原因）。这不是“反政府”言论。“反政府”，这种事儿我不敢（做）。
