     在将来，个人电脑是什么样子？有何根据？
3月25日，日本Sony公司推出一款新型“云电脑”，外观既像传统的笔记本，又像所谓的“云终端”，介乎两者之间，起名“ChromeBook“。“云电脑”其实就是我们以前所说的”浏览器电脑“，开机只需几秒钟，直接登录某个”云中心“，各种联网工具都齐备，浏览网页，玩游戏，听音乐，看电影，似乎不缺什么。这就是未来人们普遍使用的个人“云电脑”。
实际上，“云电脑”不同于平板电脑，前者是”永远在线“，不管是借助Wi-Fi局域网，还是3G广域网，而后者随便。“云电脑”的配置一般较高，具有内存2GB，主频1.6GHz，采用Intel的Atom处理器，不像平板电脑那么“小巧玲珑”，节电“省油”（少花钱）。
目前，”云电脑“在中国境内还推广不开，因为，中国境内没有云办公套件可用，我们只能“望洋兴叹”。所以，我还是鼓吹小电脑，超低价，低功耗，简单皮实抗折腾。在我的心目中，这种小电脑也是采用Unity风格，“启动栏”忽隐忽现，孩子很喜欢。说穿了，云电脑与小电脑都是Linux计算机，绝对与XP无缘。一提Linux计算机，国内有人就摇头，转角180度。我就是不明白，Linux有啥不好？中国有句俗话，免费没有好东西。这种思想意识，根深蒂固，不易除去。昨天，我已经发出“信号”，准备制作一批Ubuntu
 12.04自启动U盘，准备送人（类似Linux大棚车计划）。
实际上，Unity看似简单，实则不然。在Unity的背后，凝聚着无数编程高手的心血。至今为止，没有任何一款别的Linux发行版敢于套用Unity。Unity的不断完善，对于某些人而言，不是一个好兆头。个人计算机处于屏幕左侧的最佳位置（”制左点“），现在已经被Unity的“启动栏”（即Launcher）牢牢占据，苹果、微软只能“干蹬眼”。
说明：根据官方定义，“Chromebooks（又叫“云电脑”）are
 shipped with the Google Chrome OS operating system, which uses the[Linux kernel](http://en.wikipedia.org/wiki/Linux_kernel),and the
[Google Chrome](http://en.wikipedia.org/wiki/Google_Chrome) web browser with an integrated
[media player](http://en.wikipedia.org/wiki/Media_player_%28application_software%29)”，由此可见，云电脑是一款新型的Linux计算机，与小电脑一样，将来必然要流行起来。
