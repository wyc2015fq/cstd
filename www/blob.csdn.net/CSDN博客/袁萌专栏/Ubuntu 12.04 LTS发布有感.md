         4月26日（北京时间）19点，Ubuntu12.04终于发布了，其截图如下：
![](http://i1-news.softpedia-static.com/images/extra/LINUX/small/ubuntu1204lts-small_005.png)
[](http://i1-news.softpedia-static.com/images/extra/LINUX/large/ubuntu1204lts-large_005.jpg)
         从上图可见，该版本有4个工作区，位于屏幕最左侧的启动栏“安然无恙”，具体模块如下：
·Unity 3D interface 5.10;
· Unity 2D interface 5.10;
· GNOME3.4.1;
· Linux kernel 3.2.12;
· X.Org 7.6;
· Xorg Server1.11.4;
· Upstart 1.5;
· GCC 4.6.3 toolchain;
· Python2.7.3 and Python 3.2.3;
· Compiz Fusion 0.9.7.6.
         为什么Python编程语言赫然在目？足见其重要性。昨日上午，我去新华书店寻找有关Python语言的书籍，结果失望而归。由此可见，在我们国内普及Linux的旅途仍然十分遥远。让我们大家共同努力吧！
