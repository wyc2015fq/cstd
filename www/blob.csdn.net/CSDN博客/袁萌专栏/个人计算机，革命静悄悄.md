    大家似乎都认为，个人台式机是老计算设备（如同手摇计算机），没有什么话要说的了？我看，未必。
    老实说，我每天写作使用的就是一台老式的台式计算机，处理器耗电100多瓦，机内散热风扇不停地在嗡嗡响，（计算机）浑身热乎乎的，像个小电热炉似的，家人都说我是一只“电老虎”。到了这个年头（已经进入互联网时代），为了写点东西（博客短文）还这么费劲，这能算正常吗？实际上，这主要是因为Intel的CISC处理器太耗电儿造成的，只得不停地散热（即耗电）才行。那么，该怎么办呢？
近日，英伟达（NVIDIA）推出一款基于RISC（精简指令集）的处理器（也叫ARM处理器），这款处理器可以用于台式机、服务器以及高性能计算机。这款新型处理器耗电功率只有几瓦（甚至1～2瓦），那么，很显然的后果是，今后计算机的散热风扇全不用了，体积也大大变小了（机箱甚至也没有了），计算机仿佛变成了一台显示器，我的“电老虎”帽子也就摘掉了！我想，为了此项发明，全世界的PC用户都要内心发狂，拥到大街上去跳舞，我们的生活将会充满（RISC）阳光，完全变了个模样儿！据此，新一轮科幻小说可以动手写了。
     今年1月5日，微软在CES大展上宣布下一代Windows(NT)和办公套件Office将支持ARM处理器（跑到精简指令集上面了），与Intel说再见了，（Intel不好意思）。这是一件大好事。从此，我国的盗版XP（包括盗版Office）也就算彻底退出历史舞台（在新款处理器上不能跑）。或者很有可能，微软新一代Windows
NT使用硬件单元发布，叫你盗版者变成真正的窃贼（偷窃物理器件者），软件盗版成为历史的记忆。
    在新一代台式机到来之前，Windows与Linux都站在同一起跑线上，枪声一响，大家一起奋力向前跑去，胜负的机会完全一样。1月18日，Mark
Shuttleworth宣布把Qt引入Ubuntu就是为了这一天的到来。我国大牌计算机生产厂商直到今天才想起来要预装Linux，还妞妞捏捏（舍不得给钱，几毛钱一套！），显得不好意思，真是少见多怪。Linux是什么？不就是一堆程序代码，你可以自由使用，但是，不可据为己有，如此而已。搞Linux，为什么要国家花钱？这件事情我就是想不明白，为此得罪了不少人。
    英伟达CEO
Jen-Hsun Huang（黄）在今年CES上说："This
marks the beginning of the Internet Everywhere era, where every
device provides instant access to the Internet, using advanced CPU
cores and rich operating
systems“，意思是说，节能型的ARM处理器（台式机通用）以及丰满操作系统的推出标志着互联网无处不在时代的到来。我们把话说白了，计算机、电视机、电冰箱、甚至洗衣机统统连接到互联网上已经不再是乌托邦空想。如果没有30年前关于精简指令集计算机（RISC，Reduced
Instruction Set
Computer）的梦想，怎么会有今日英伟达（处理器）的发明？毫无疑问，Intel与AMD做RISC计算机可谓“轻而易举”，但是，RISC计算机的相关专利已经不在自己手中了，很是无奈也。微软公司是识时务者为俊杰，革命不分先后。当前，世界范围的计算机革命正在静悄悄地进行中，我们还在为争夺“国产”桂冠（向政府要钱）而战斗。
