# Ubuntu云战略浮出水面 - 袁萌专栏 - CSDN博客
2011年10月22日 07:36:53[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：1373
    今年10月20日是Ubuntu的7周岁生日，当前，一场深思熟虑的“云战斗”即将打响。是什么事情呢？
2005年1月18日，从《Ubuntu实验室》全体员工在福州“金百合”（“LillyHotel“）一层餐厅聚会的那一天开始，我算是与Ubuntu绑在一起了。现在，我在CSDN上的博客密码仍然是“UbuntuNow“，这就是一个证明。密码”UbuntuNow“的意思是，现在就Ubuntu吧！
在我看来，Ubuntu11.10的发布代表着Ubuntu发展已经达到了一个全新的高度：云计算。可以这么说，离开了云计算，你就不能准确理解站在Ubuntu后面的人们此刻在想什么，他们在干什么，因而也就不能抓住Ubuntu的灵魂。那么。全球的这股Ubuntu社会势力究竟是想干什么呢？光是想自己发财吗？非也。为何？
10月5日，Ubuntu”小毛头“竟然在美国波斯顿召开的“OpenStack云技术讨论会“的一楼展厅里面，展示他们用手（byhand）组装的一台基于ARM处理器的服务器，以便表达他们的急切心情（“demonstrateda
 server running on an Arm processor”，“Theserver was assembled
 by Canonical byhand”），因为，至今为止，在全球范围内还很少有生产厂商能够出品“ARM服务器”。
10月20日，Ubuntu领头人MarkShuttleworth书来说话了，在其博文“PrecisionPlanning;
 Prepping for 12.04 LTS”（“精密规划，为12.04LTS做准备”）一文里面，他点出了本月31日在美国佛罗里达州召开的Ubuntu开发者峰会（UDS）的“基调”，即普及、推动云计算，特别是ARM云计算。他说：“12.04will
 be the first LTS to support the ARM architecture on selected ARMSoC parts”，还说：“theentry
 of ARM to the server market is a very interestingshift”，以及“Ubuntuhas
 established a very strong competence in ARM and I think the 12.04LTS release will power a new generation of power-focused hardware forthe data centre”，总之一句话：Ubuntu将在ARM服务器架构上有一“搏”。
10月21日，Canonical对外正式宣布：12.04LTS版本对其桌面用户的技术支持延长至5年，与服务器版本保持“一致”，由此，摆开阵势专攻企业级应用。有人似乎认为，Ubuntu桌面只是“小儿科”，不值一提。非也。比如：跨国公司高通（Qualcomm）与德国慕尼黑市（Munich）已经大规模地部署了Ubuntu计算机桌面，近日，印度法院系统已经决定将全面部署UbuntuLTS版本。
本文的中心意思是说，在未来5年里面，Ubuntu全球团队将围绕Ubuntu11.10LTS版本（所达到的“高度”）展开“云保卫战”，特别是ARM云架构的实际应用。坦率地说，站在Ubuntu后面的人们使我们看清了未来世界IT产业发展的大趋势：ARM云的普及与应用，意义特别重大，谁敢与其“争风吃醋”呢？大家知道，微软的Win8虽然支持ARM处理器，但是，就是不敢说：Win8支持ARM服务器！在我们国内呢？有些人只会跟在谷歌Android4.0屁股后面“瞎吆喝”，简直是在胡闹！
说明：为什么站在Ubuntu背后的人们有如此长远的眼光？是神灵的奥秘启示，还是人的社会实践？当然是人的社会实践决定了人们的意识。那么，我们呢？？
