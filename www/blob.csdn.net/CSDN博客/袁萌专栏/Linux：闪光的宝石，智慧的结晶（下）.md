2005年4月7日，Linus
 Torvalds发布了一款新型通用工具软件包，叫做”Git”(the Git source code management system）。“Git”（源代码管理系统），一般人不用，但是，系统开发人员必备，尤其是大系统开发。
Git有什么能耐（特性）呢？对于大系统而言，Git运行速度极快，特别是对于软件系统中的模块“说明文档”的管理非常有效。也就是说，借助Git的能耐，任何大软件系统的电子文档说明可以“一目了然”，不再糊糊涂涂，模棱两可。

严格地说，Linux是闪光的宝贝，是智慧的结晶，指的就是说：Linux是一本透明、开放的系统软件教科书，不像”国产软件“躲躲闪闪，言不由衷，是骗人害己的东西。
