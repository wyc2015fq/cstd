# Debian 7.0正式发布 - 袁萌专栏 - CSDN博客
2013年05月06日 04:02:45[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：1790
   5月5日，全球最大、最老的GNU/Linux发行版Debian
 7.0正式发布，3张DVD光盘，2万多个自由软件包。
2009年2月15日，Debian
 5.0发布。记得，Debian 5.0发布之后，“Debian亚洲峰会”（兼Release
 Party）是由江苏南京新华科技公司组织的，在北京西郊友谊宾馆举办。这次活动规模很大，Debian领导层换届。我被邀请参加。
2011年2月6日，Debian
 6.0发布。很有意思的是，在那个时期，LibreOffice已经从OpenOffice独立出来，可是，OpenOffice仍然被Debian收录进去了。但是，今年发布的7.0版本，却只收录LibreOffice，而OpenOffice不见了。与此同时，国内与OpenOffice社区深度合作的红旗中文2000也烟消云散，人走楼空，不见了踪影。
2007年4月8日，Debian
 4.0发布；2002年10月20日，Debian
 3.0发布，1998年7月24日，Debian
 2.0发布，那是上世纪的事情了。有趣的是，Debian 1.0版本从来没有发布过。对Linux小毛头而言，最早的Debian
 0.01版本发布在1993年8月，至今已经快二十年了。
大约在十年之前，在2004年年初，面对Debian发布的缓慢节拍，Mark
 Shuttleworth决定发起所谓的“Ubuntu运动”，继续高举GNU/Linux旗帜，同年，在Debian基础上，发布了Ubuntu
 4.10版本，至此，GNU/Linux延续至今。
2013年3月11日，UbuntuKylin成了Ubuntu的新分支（flavour），4月25日，UbuntuKylin
 13.04版本发布。这是我们国内第一款GNU/Linux发行版。5月11日在北京举行该发行版的“Release
 Party”，会上请来商业软件公司金山做“主  题报告”，显得很不和谐，冷落了新贵LibreOffice。
中国人口多，要圆“中国梦”，需要GNU/Linux。如今，中国的GNU/Linux，你在哪里？坦率地说，GNU软件并不复杂，可是令人没有想到的是，我们的伟大竟会对GNU理念如此抵触，想方设法把它驱逐出去。5月4日，北大校庆，习主席给北大学生回复函件，鼓励北大学子奋发图强，圆“中国梦”。对此，我想了一天，停下笔来，不再写短文。今日，在Debian
 7.0发布之际，发出了上述的感叹。
人老了就无用了，只能说点历史故事。只有在历史的长河中，在历史的背景中，才能看清楚事物的本质。近日，在OpenStack峰会上，Ubuntu奠基人Mark
 Shuttleworth发出呼吁：”the world wants a Google Android alternative“，而且表示：“Canonical'ssecret
 sauce is a single Ubuntu code base that scales across smartphones, tablets, PCs, servers and cloud systems and TVs”，意思是，世界需要一个谷歌安卓的替代品，Canonical的秘密武器（调味剂）是”asingle
 Ubuntu codebase“，将智能手机、平板电脑、PC计算机、服务器与云系统以及智能电视统一起来，实现”大一统“。这就是2014年我国”核高基“重大软件专项的核心内容。呜呼！
