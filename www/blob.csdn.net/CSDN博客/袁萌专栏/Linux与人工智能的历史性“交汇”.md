# Linux与人工智能的历史性“交汇” - 袁萌专栏 - CSDN博客
2012年02月02日 04:47:01[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：1009
    老实说，灵巧计算（Smart Computing）应当属于人工智能的研究领域，但是，让人们没有想到的是，它来得（或出现）竟然是这么快，很是突然，完全出乎人的意料。
1月24日，业界资深分析评论家Steven
 J. Vaughan-Nichols发表文章，题为“Beyond the desktop:
 Ubuntu Linux's new Head-Up Display“，文中有一段话，画龙点睛。他说：”HUD
 will use a “vocabulary UI”, or VUI. In this you’ll start to type or say a command and in 12.04 LTS, the HUD starts a smart look-ahead search through the app and system (indicator) menus. This uses fuzzy matching, combined with a learning function so HUD will
 prioritize the actions you use do“。
从以上文字，我们可以看出，文章作者认为：HUD技术已经超出了传统计算机桌面的研究领域，进入到（或涉及到）所谓的“Fuzzy
 matching“（模糊匹配）与“E-learning”（机器学习）领域，后两个研究领域明显就是典型的人工智能研究领域，从原理上讲，这就是Linux与人工智能研究领域的“交汇”。
今天（2月2日），Ubuntu
 12.04(LTS)的第二个测试版就要发布了。在这个版本中，Ubuntu开发者预先做了一定的“技术处理”，在必要时才让HUD“框框”出现，让用户慢慢适应，逐渐减少对系统Menu（菜单）的依赖性。
从计算机系统的“菜单”入手搞“革命”（或”改革“），这确是Ubuntu开发人员的一大“发明”，真是“奇招儿”也。此举令人“茅塞顿开”，眼前“豁然开朗，一片阳光”。原来，计算机系统的“菜单”（Menu）就是那个“茅塞”。如今，“茅塞”已经除去，棋盘就活了，余下的事情也就好办了。
当今，在世界软件界，Ubuntu是个“捣蛋鬼”，四处惹祸，得罪了不少人（包括自己大用户）。大家知道，要领跑技术创新，就不要怕得罪人。你即便把Ubuntu掐死（不准它喘气），它死得也光荣。......中国的现行教育体制，不断地为微软培养大批人才和潜在的用户（后备军），对此，我们（这一代人）只好认“倒霉”算了。现在，让我们的孩子们玩玩Ubuntu，使他们及早开阔眼界，鄙视盗版XP瘾君子，健康地成长，成为我们的“希望”（接班人）。
至今，令我感到非常遗憾的是，在200年年底，《Ubuntu实验室》在北京的《赛迪大厦》15层被人突然“掐死”了。如果让它继续存活到现在，如今也有8岁了。据说，深圳乌邦托公司准备“复活”《Ubuntu实验室》，如果真是这样，我就打个电话，叫他们高举“HUD“这面旗帜，率先进行深入研究，紧紧咬住。决不掉队！现在，我人已老，又有轻度”青光眼“，快不中用了，但是，我的眼光还算行，看东西（把握事物的发展大方向）不模糊。
说明：我我们国内，有人以拒绝”新鲜事物“为己任，脸不红，心不跳，如同死人一般（浑身冰凉）。。
