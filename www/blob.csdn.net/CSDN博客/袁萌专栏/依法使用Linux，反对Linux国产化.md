# 10月28日，中国操作系统应用推广大会在京举行，包括中央办公厅、公安部、工信部在内的多个中央国家机关部委出席该会。此次推广大会的召开或标志着操作系统国产化进程的进一步推进。
会上，中国工程院院士、著名计算机专家倪光南发表主题演讲。倪光南表示，目前已经成立中国智能终端操作系统产业联盟来营造国产生态系统，做好应用服务支持，才能赶上微软等国外操作系统厂商。简而言之，倪院士主张我们要追赶微软。但是，追赶什么呢？怎么追赶？
现在，世界已经进入互联网“云时代”，微软视窗系统与Linux不再是对立的双方，而是共同打拼市场的同盟军。为什么？
2013年7月25日，甲骨文（*Oracle**）宣布：“*a
 partnership that will let business customers access some of its key products on cloud services provided by its longtime rival*Microsoft”**，大意是，在云计算方面，甲骨文与多年的对头”言和”。*
*2014年10月22日，**IBM**与微软签订新协议：*“*...*that
 will allow each company's software to run in the other's* cloud*.”此举意味着视窗系统与Linux“握手言和”。
由此可见，院士确实年已经老矣，方向不明。当前，国内软件业的问题不是继续纠缠什么“国产操作系统”，而是把握正确的发展方向，抓住良机，不要越赶超，距离世界软件发展前沿越远。关键的问题是，要依法（GPL
 3.0）使用Linux，反对一切Linux“国产化”的错误言行。
