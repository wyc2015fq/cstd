8月7日，“Ask
 a Cloud Expert”网站发表署名文章，题为”How To Tell If Windows 10 Is Succeeding“。我认为，该文作者很有见解。
文章说，目前，微软转向互联网服务，奋力追赶谷歌，其策略是：It’s adding Bing search directly to the
 desktop task bar through its Cortana virtual assistant. Any Cortana query that requires a web search sends users to Bing, giving Microsoft a chance to indirectly steal market share from Google.“微软推出柯塔娜原来是为了从谷歌那里“偷”市场！数亿个柯塔娜登场“大合唱”，力量不可小视。
微软说：”It’s
[expecting a 10% to 15% increase](http://searchengineland.com/with-windows-10-bing-ads-anticipates-10-to-15-percent-more-search-volume-by-september-226432) in Bing queries from Windows 10’s various search engine hooks as early as September.”也就是说，微软的战略转型，在今年9月初即可初见成效。
