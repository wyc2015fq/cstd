# CES 2012：Ubuntu为何抢了Android的风头？ - 袁萌专栏 - CSDN博客
2012年01月12日 02:46:23[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：1569
    一般而言，所谓“风头”是指外在形势发展的大方向，是一种比喻性质的词汇。如果“风头”被人抢去，其主人心中一定会生气（指心里面生闷气）。
至今，CES2012大展已经进行了两天，从国外媒体报道来看，毫无疑问，Ubuntu的智能TV处在各大媒体关注的“焦点”，明显地抢占了谷歌Android平板电脑的风头（相比去年CES大展）。这是为什么呢？
大家知道，对于谷歌Android的“半开放”（或“半开源”），媒体历来批评不断，而且，谷歌对国际Linux标准LSB（Linux
 Standard Base）完全不当一回事（根本不在乎），从来都是“天马行空”、“我行我素”。什么是自由软件？是不是大的应用软件总得“卖钱”（如同商业软件）？否则，就是傻子行为，就是“搅局者”。当今，60英寸大型数字彩电已经成为”主流产品”，在国内，光纤（宽带）入户也非“稀罕”之事。那么，人们能否从“看电视”，不用花钱，就转变为“玩电视”呢？电视怎么“玩”法？就不用我去说了。说句实在话，智能电视（Smart
 TV），就像过去的卡拉OK，才是家庭的“娱乐中心”，又跳又唱，多么痛快！
元月10日，英国典范（Canonical）公司特别针对美国、中国市场，在今年的CES大展上，推出一款大型自由软件，叫”Ubuntu
 Smart TV”，不日即可自由下载。安装Ubuntu应用TV软件包之后，你家的大型彩电在转眼之间就变成“智能电视”了，可以高高兴兴“玩”一把。请见：[How
 To Install **Ubuntu TV** From A**PPA**](http://www.google.com/url?q=http://www.linuxtoday.com/infrastructure/2012011000441OSUB&sa=U&ei=LckNT6CpBOfcmAWosYybBg&ved=0CAgQFjAC&client=internal-uds-cse&usg=AFQjCNFIKTd1AMJIHsEgG4GHtECs2EC7Mw)（**1**月**10**日发表）。
实际上，国内市场上智能电视不是没有，而是（一般人）玩不起，因为价钱太贵。在我们国内，不知是什么原因，富人很多，整天吃吃喝喝，满嘴流油，用“信用卡”大把花钱，仿佛他的“信用卡”有后门，有人不断“打钱”进去。这种人，一旦被抓住，吃点苦头（包括跳楼），活该！活该！Ubuntu电视可不是给他们玩的。据报道，Ubuntu智能电视（产品）今年年底将要上市。我想，其价格大约会在1千美元上下（这种“定价”，比较合适）。Ubuntu智能电视，与苹果、谷歌智能电视相比，价格相差“悬殊”，难道没有市场竞争力？
严格说来，UbuntuTV(软件包)是一款大型自由应用软件，也可以看作是Ubuntu的智能TV发行版（当然完全免费），而且，绝对“开源”，不像谷歌的Android系统，又开又不开，真要把人急死了。难怪Ubuntu智能电视一出来，外电媒体纷纷抢先报道。为此事件，全球业界无不喜气洋洋、欢天喜地（我心里面也出了一口气）。至此，Ubuntu的Unity”人机接口“又得了一分！
有人预计，由于苹果不信任CES大展（早已退出），微软明年也要退出，因此，今后每年一度的世界CES大展将是Linux的天下，几乎成为Linux电子产品的世界博览会。所以，我也想去看看。你呢？
说明：智能电视里面具有许多“高科技”，Ubuntu智能电视捅破了这层窗户纸，为全体人类做了一件大好事（可以”玩电视“），完全符合Ubuntu的宗旨。
