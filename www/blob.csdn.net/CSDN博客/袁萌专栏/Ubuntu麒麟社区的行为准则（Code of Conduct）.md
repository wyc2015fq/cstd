# Ubuntu麒麟社区的行为准则（Code of Conduct） - 袁萌专栏 - CSDN博客
2013年04月14日 02:34:01[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：1530
     4月14日，我进入UbuntuKylin中文社区（[www.ubuntukylin.com](http://www.ubuntukylin.com/)）的技术论坛，有感。
UbuntuKylin社区的官方口号是：“做更有中国味的操作系统”。什么是“中国味”？一般业界人士认为：所谓”中国味“就是不提GNU，尽量回避GNU，表现在行为没有准则，随随便便。为什么这么说呢？
我登录技术论坛，竟然不用承诺遵守社区的行为准则（Code
 of Conduct）就登录成功了，这就是所谓的“中国味”吗？看看UbuntuKylin社区的领导人（Lead）JackYu的个人信息：“Signed
 Ubuntu Code of Conduct: Yes”。这就是说，作为Ubuntu国际社区的子社区Ubuntu麒麟社区也是要遵守社区的行为准则的。我记得，Ubuntu社区的行为准则是：要善待他人，待人有礼貌，做事要替别人着想。这三条行为准则符合GNU的基本精神。Ubuntu人的哲学理念是：软件应该是自由的（Freedom）。离开了这一基本点，Ubuntu就不是Ubuntu了。
说句实在话，现在的UbuntuKylin的主导权还不在我们手中，不是“自主可控”的社区。但是，这个阶段是必须的。因为，我们虚心要学习国际GNU社区的运作规则，主动适应这些规则，努力改掉我们的“中国味儿”（松散、随意、不团结的坏毛病）。
在此，我说句不该说的话，现在90后的孩子们都生活在蜜罐中，他们不知道1962年经济困难时期人们吃不饱饭的滋味儿。我不是作家莫言，不敢发牢骚。在南京大学读书期间，吃不饱饭的经历还是有的。对此说法，请不要说我是“反党、反社会主义”。
我寄希望于Ubuntu麒麟社区的小毛头们，希望你们为了国家的强盛，努力学习，认真生活，服从真理（特别是GNU），做一个对社会有用的人才。德才兼备，德是第一位的。做人，不讲“徳”，就没有意思了。
我建议，Ubuntu麒麟社区的口号是：要做安全稳定的操作系统。在GNU的阳光普照之下，联合起来，团结一致，我们一定能够做出更安全、更稳定的操作系统。......数学在呼唤着我，我不能每天都陪伴着你们这些麒麟小毛头，但是，我会随时关注你们所取得的成就，为你们祝福，为你们赞美。
