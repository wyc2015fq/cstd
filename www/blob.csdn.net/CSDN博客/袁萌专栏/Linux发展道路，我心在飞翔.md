    近日，在博客短文的留言中，我似乎感到一丝温暖，看到一线希望。何故也？
    在我国，Linux究竟应该怎么发展？国家投入了那么多钱，钱都到哪里去了？为什么这个问题不能摆在桌面上讲个明白？谁是“败类”？谁才是蛀虫？我们是一个现代文明国家，不允许再继续肆意扭曲Linux的本来面目，滥用国家钱财。
    有一个Linux发行版叫Mint，世界发行版排名第二，仅次于Ubuntu发行版，值得引起我们的关注。现引用Linux发行版世界排行榜官方网站（Distrowatch)的一组统计数据，位于前五名的Linux发行版如下：
1）Ubuntu   
     得分2,179
2)
 Mint            得分1,796
3)
 Fedora        得分1,457
4）Debian    
   得分1,383
5）openSUSE 
  得分1,316
    以上“得分”的自动评分公式是公开的，透明的，稳定的，持续的，在此我们不必细说。在Mint官方网站上，有这么一句话：“Started
in 2006, Linux Mint is now the 4th most widely used home operating
system behind Microsoft Windows, Apple Mac OS and Canonical's
Ubuntu”。这一结论，你是不能反驳的。我们要问，取得如此成就，Mint使用了多少资源呢？实际上，在Mint社区的背后，并没有国家或公司的财力支持，靠的是全球Mint用户和社会公众的捐助维持其日常运行。捐助的金额，从一美元到数十美元不等，捐助人的国籍与姓名，每个月都公布在Mint的官方网站上，一清二楚，明明白白。从Mint项目开始启动，每月的捐助总额只有数百美元，直到今年，每月接受捐助的总额达到5.000美元的水平。5年以来，捐助总计不足20万美元（约100万人民币）。当然，还有一些微薄的广告收入，不在此列。
为什么Mint会取得如此骄人的业绩？总共有五条经验（做了一点精简）：
1）It
is extremely easy to use. 
2）It's
both free of cost and open source. 
3）It's
community-driven. Users are encouraged to send feedback to the
project so that their ideas can be used to improve Linux Mint. 
4）Based
on Debian and Ubuntu, it provides about 30,000 packages and one of
the best software managers. 
5）It's
safe and reliable. …... Linux Mint requires very little maintenance
(no regressions（退化现象）,
no antivirus, no anti-spyware...etc).
   这五条经验，我几乎一字不改，原样摆在这里（做了必要的精简）。其中经验2），经验3）和经验4），特别值得我们学习与反思。什么叫社区驱动？相信群众，依靠群众，走群众路线，by
the people, for the people，这就是发展Linux的真谛。
回到我们国内，有人整天想方设法游说国家政府官员，说什么发展“国产”Linux，国家需要投入，投入，投入，还是投入。这是什么用心？......西北农村的小学生们坐在什么凳子上课，你知道吗？现在，我们的国家虽然已经是大国，但是，有的地方还是很穷。......我就不信，中国人治不了这个“穷”字！我在这个伟大的国度已经生活了60多年，在离开人世之前，我希望看见GNU/Linux的鲜花布满祖国的大地！
