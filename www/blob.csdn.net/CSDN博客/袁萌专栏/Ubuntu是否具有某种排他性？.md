# Ubuntu是否具有某种排他性？ - 袁萌专栏 - CSDN博客
2012年01月26日 09:05:15[yuanmeng001](https://me.csdn.net/yuanmeng001)阅读数：884
    近年来，Ubuntu为何从一个具有普及性、用户友好的Linux发行版而逐渐转变为一个具有某种排他性、“独一无二”的Linux发行版？当前，这是一个值得我们深入思考的实际问题。
1月23日，业界知名评论家Christopher
 Tozzi发表署名文章，题为“[Why
 Don’t Other Linux Distros Use Unity? A Few Thoughts](http://www.thevarguy.com/2012/01/23/why-dont-other-linux-distros-use-unity-a-few-thoughts/)”，明确地提出了这一问题。也就是说，为什么其他Linux发行版都不采用Unity桌面？究竟是什么原因？问题出在哪里？有何高见？
文章作者认为，无非有这么几种潜在的可能性（或可能的解释）：
1、the
 interface is simply unpopular；
2、Unity
 is harder to work with from a technical standpoint；
3、no
 one has spun off Unity because Canonical has kept the interface soclose to home。
作者最赞成的解释是第3种，也就是说，没有人能够从Unity演化（spun
 off）出其他的Ubuntu变体，因为：“Canonical
 has been able to exert much more rigid control over it”，意思是说，典范（Canonical）公司能够对其（指Unity）施加(exert)许多“硬性”的控制，别人根本插不上手。
大家知道，苹果公司搞的是软、硬件“一体化”，使其最优，利润滚滚而来。如今，典范搞系统软件与”人机接口“（Unity）的“一体化”，制作软件精品，抢占市场，比如：Ubuntu的智能电视。如此以来，谁也不想跟它玩了，比如，国内的深度Linux（拒不采用Unity桌面）。同时，文章作者指出：“Unity
 has not been distributed outside of Ubuntu’s own channel”，也就是说，目前，Unity还局限在Ubuntu自己的“小圈子”里面，不授权他人使用。
那么，有什么办法能够挽救Ubuntu的孤独“忧郁症”呢？1月13日，Ubuntu国际社区发布Unity5.0，为Unity的普及应用打开了一扇“大门”（实际上，是个“门缝儿”），Ubuntu在正确的方向上迈出了可喜的一步。
说明：如果Ubuntu不是如此这般“天马行空”（具有一定的排他性），勇往直前，谈何与苹果和谷歌的“市场竞争”？Ubuntu也有自己的难处啊！
