1.先安装pyqt4

开终端,依次执行如下命令:


sudo apt-get install libxext6 libxext-dev libqt4-dev libqt4-gui libqt4-sql 


sudo apt-get install qt4-dev-tools qt4-doc qt4-qtconfig qt4-demos qt4-designer 


sudo apt-get install python-qt4 


sudo apt-get install python-qt4-* 


sudo apt-get install python-qscintilla2

然后就可以使用标签制作工具了,我们来看第二步


2.使用标签制作工具制作标签

使用工具labelImg-master


下载地址:链接: https://pan.baidu.com/s/1cNWzhG 密码: eynd

使用技巧:先打开图像文件夹-->更改默认xml文件存放位置-->按W键开始标记-->标记完成后,设置分类类别-->按D键来切换到下一张照片;最后是重复上述操作
