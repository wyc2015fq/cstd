# 如何将ftp服务器映射为本机磁盘 - 纯洁的微笑 - CSDN博客
2012年11月26日 11:31:11[微笑很纯洁](https://me.csdn.net/ityouknow)阅读数：4171
                
例如远程ftp服务器的地址是  202.101.98.54
你的ftp帐号是    user
你的密码是       password
在“我的电脑” 右键  选择菜单 "映射网络驱动器”
路径 填 ftp://user:password@202.101.98.54
按确定就可以了。
需要注意的是。如果是匿名的服务器，一般的用户名和密码都是 guest 。或者干脆直接在路径上填ip或者域名就可以了。
