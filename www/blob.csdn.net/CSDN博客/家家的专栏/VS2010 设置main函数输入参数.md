# main函数的argc、argv参数。



菜单Project -> Properties -> Configuration Properties -> Debugging







在Command Arguments里填上即可。

中文版：
菜单[项目] -> 属性页 -> 配置属性 -> 调试

在[命令行参数]里填上即可。

不同参数之前用空格隔开。
