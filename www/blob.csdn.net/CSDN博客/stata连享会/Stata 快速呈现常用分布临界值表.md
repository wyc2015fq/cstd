### 1. 简介

在计量经济学教学和学习过程中，我们经常需要对常用分布在不同显著性水平下的临界值进行查表，而本项目就给大家介绍 `ttable` (**t 分布**), `ztable` (**正态分布**),  `chitable` (**卡方分布**) 和 `ftable` (**F 分布**) 四个命令，可以在 Stata 结果窗口中快速显示常用分布函数和常用显著水平 α 的临界值表。命令如下：

```
. ttable       // t 分布
. ztable       // 正态分布
. chitable     // 卡方分布
. ftable       // F 分布
```

### 2. 下载方法

在 Stata 命令窗口中输入如下命令即可一次性下载上述文件：
- 第一种： (Note：网络不畅时需要多试几次)

```
. net install probtabl, replace
```
- 第二种：(Note： 亲测可顺畅安装)

```
. net install probtabl, replace ///
   from(https://stats.idre.ucla.edu/stat/stata/ado/teach)
```

或者输入 `findit probtabl` 命令，或 `search probtabl`  命令，在弹出页面中按指引点击链接下载即可。

### 3. 使用方法

要使用这些程序，请在 Stata 命令窗口内键入 `ttable`,  `ztable`, `chitable` 或 `ftable` 之一命令，还可在 `chitable` 或 `ttable` 命令后设置自由度参数，选项 `alpha(#)` 可在 `ftable` 命令中设置，`ztable` 无可设定选项。

### 4. Stata 范例

此四个程序的目的在于提供卡方分布、F 分布、t 分布和 z 分布（正态分布）临界值表的窗口显示。它们提供了一些常用 α 水平的临界值信息。
