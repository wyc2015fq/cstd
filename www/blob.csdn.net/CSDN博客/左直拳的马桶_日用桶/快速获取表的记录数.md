# 快速获取表的记录数 - 左直拳的马桶_日用桶 - CSDN博客
2013年06月24日 22:42:36[左直拳](https://me.csdn.net/leftfist)阅读数：1742
可以这样写：
```
use [mydb]
go
select rows from sysindexes where id = object_id('表名') and indid in (0,1);
```
|**indid**|**smallint**|索引 ID：0 = 堆1 = 聚集索引 > 1 = 非聚集索引|
这样就不必写出 select count(*) from 表 这样耗费资源的语句了。
