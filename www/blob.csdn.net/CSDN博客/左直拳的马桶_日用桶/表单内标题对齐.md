# 表单内标题对齐 - 左直拳的马桶_日用桶 - CSDN博客

做网页表单，标题尽量对齐，看起来就舒服些。
如果每个标题字数都一样，自然最好，但有时做不到，但也可以对齐的：
![](http://hi.csdn.net/attachment/201202/10/0_132886667048TR.gif)
代码如下：
```
<tr>
		        <td class="tdHead">传<span style="color:#fff;">传真</span>真</td><td><asp:Label ID="labFax" runat="server"></asp:Label></td>
	        </tr>
	        <tr>
		        <td class="tdHead" style="letter-spacing:0.5em;">联系人</td><td><asp:Label ID="labService" runat="server"></asp:Label></td>
	        </tr>
```
传真 这个是采取中间隐藏2个字的方式。隐藏的方式是将字体的颜色设置成背景的颜色，看不出。也可以采用visibility=hidden。visibility=hidden不同于display=none。前者不可见，但占地方；后者不可见，且不占地方。
联系人 这个是设置 letter-spacing，字符间距。
