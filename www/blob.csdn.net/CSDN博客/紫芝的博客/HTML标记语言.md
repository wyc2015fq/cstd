## HTML语法

### 1. HTML标记语言

**表示网页信息的符号标记语言**

**特点**
- 可以设置文本的格式，比如标题、字号、文本颜色、段落等
- 可以创建链表
- 可以插入图像和媒体
- 可以创建表格
- 通过鼠标点击超链接实现页面跳转

### 2.HTML语言的标记和属性

**HTML文档的保存格式**
- *.html
- *.htm
- *.xhtml
- 标记的属性，就是用来控制内容图像和文本如何显示的
- 格式<*标记 属性1=属性值 属性2=属性值 属性3=属性值…*> **内容** </*标记*>

### 3.语法不区分大小写

都是相同的标记，但是编写网页时尽量使用小写

### 4.文档注释
- “<!–” 注释开始标志
- “–>” 注释结束标志

### 5.代码格式
- 空格键和回车键在网页中都不会起到任何作用
- 以“Tab”键进行缩进

### 6.字符实体

> 
如果想在网页上显示一个"<“符号，但是”<"是标记的开始符号，需要用实体名称代替


> 
常见字符实体

- 显示结果：`空格` 描述：`空格` 实体名称：`&nbsp` 实体编号：`&#160`
- 显示结果：`<` 描述：`小于` 实体名称：`&lt` 实体编号：`&#60`
- 显示结果：`>` 描述：`大于` 实体名称：`&gt` 实体编号：`&#61`
- 显示结果：`&` 描述：`和号` 实体名称：`&amp` 实体编号：`&#38`
- 显示结果：`"` 描述：`引号` 实体名称：`&quot` 实体编号：`&#34`
- 显示结果：`￥` 描述：`人民币` 实体名称：`&yen` 实体编号：`&#165`

## HTML基本结构

### 1. html标签表示网页的开始和结束

### 2. head标签表示网页的标题和关键字

### 3. title标签表示网页的标题，不能包含其他标记

### 4. body标签表示网页的主体部分

```
body标签常见属性
    - bycolor 设置背景颜色
    - text 设置设置文本颜色
    - link 设置连接颜色
    - vlink 已经访问的连接的颜色
    - alink 正在被点击的连接的颜色
```

### 5. meta标签表示页面的元信息，比如针对搜索引擎和更新频度的描述，必须放在head元素里面

**必须属性：content，定义name属性相关的元信息**

> 
UTF-8:Unicode Transformation Format 8it


```
允许含BOM，但通常不包含BOM，是用以解决国际上字符的一种多字节编码。
对于英文使用8位编码(一个字节)，中文使用24位编码(三个字节)
UTF-8包含全世界所有国家需要用到的字符，属于国际编码，通用性强
UTF-8编码的文字可以在各国支持UTF8字符集的浏览器上显示
无需下载中文语言支持包即可在英文IE浏览器上显示中文
```

> 
GBK是国家标准GB2312基础上扩容后兼容GB2312的标准


```
GBK使用双字节来编码文字，不论中、英文字符均采用双字节表示;中文的最高位为1
GBK包含全部中文字符，通用性交叉，数据库较小
```

## 文档设置标记

**(多个回车或空格等效于一个空格)**

### 1.格式标记

```
- <br> 换行标记，单标记
- <p> 换段落标记
- <center> 居中对齐标志，让段落或文字相对于父级标记居中显示
- <pre> 预格式化标记，保留预先编排好的格式
- <li> 列表项目标记
- <ul> 无序列表
- <ol> 有序列表
- <dl><dt><dd>定义型列表
- <hr>水平分割线标记
- <div>分区显示标记(层标记)
```

### 2.文本标记

```
- hn 标题标记，n的范围是1~6
- font 字体设置标志，size(字体大小),color(颜色),face(字体)
- b 粗字体标记
- i 斜字体标记
- sub 文字下标字体标记
- sup 文字上标字体标记
- tt 打印机字体标记
- cite 引用方式的字体，通常是斜体
- em 表示强调，通常显示为斜体
- strong 表示强调，通常显示为粗体
- small 小型字体标记
- big 大型字体标记
- u 下划线字体标记
```

## 图像标记

```
<img src="路径/文件名.文件类型" width="" height="" border="" alt="">
```
- border 指定图标的边框宽度，单位px、em、cm、mm、n%
- alt 图片的描述

## 超链接的使用

```
<a herf="链接地址" target="打开方式" name="链接锚点名称"> 文字或者图片 </a>
<!--页面内跳转-->
<a name="top"></a>
...
...
...
<a href="#top">turn to top!</a>
```

### target属性
- _blank **在新窗口中打开链接**
- _self 在当前窗口中打开链接 (默认)
- _top 在顶层窗口中打开 (框架中使用较多)
- _parent 在父窗口中打开链接 (框架中使用较多)

## 表格的基本结构

```
<table 属性1="" 属性2="" ...>表格内容</table>
```

### 1. width属性

表示表格的宽度，值可以是像素(px)或者父级元素的百分比(%)

### 2. height属性

表示表格的高度，值可以是像素(px)或者父级元素的百分比(%)

### 3. border属性

表示表格边框的宽度

### 4. align属性

```
表格的显示位置
- left 居左显示
- center 居中显示
- right 居右显示
```

### 5. cellspacing属性

单元格之间的间距，默认是2PX，单位像素

### 6. cellpadding属性

单元格内容与单元格边框的显示距离，单位像素

### 7.frame属性

#### (1).作用

**控制表格边框最外层的四条边框**

#### (2).属性值
- void：表示无边框（默认）
- above：表示仅顶部有边框
- below：表示仅底部有边框
- hsides：表示仅底部和顶部有边框
- lhs：表示仅有左侧边框
- rhs：表示仅有右侧边框
- vsides：表示仅有左右边框
- box：包含全部的四个边框
- border：包含全部的四个边框

### 8.rules属性

**控制如何显示单元格之间的分割线**

#### (1)none：(默认)表示无分割线

#### (2)all：表示所有的分割线

#### (3)rows：表示仅有行分割线

#### (4)clos：表示仅有列分割线

#### (5)groups：表示仅在行组和列祖之间有分割线

```
<caption>表格的标题</caption>
```
- top:标题放在表格的上
- left:标题放在表格的左
- right:标题放在表格的右
- bottom:标题放在表格的下

```
<tr>定义表格的一行，可以嵌套多个<td>、<th>标记</tr>
<th>表头的标记，必须嵌套在tr标记内</th>
<td>数据标记，必须嵌套在tr标记内</td>
```
- bgcolor：设置单元格的背景颜色
- align：设置单元格的对齐方式
- valign：设置单元格垂直对齐方式
- width：设置单元格的宽度
- rowspan：设置单元格所占行数
- height：设置单元格的高度
- colspan：设置单元格所占列数

## HTML框架

**框架将浏览器划分为不同的部分，每一部分加载不同的网页，实现在同一浏览器内加载多个页面的效果**

```
<!--语法格式-->
<frameset>....</frameset>
```

### 1.cols属性

使用像素数和百分比左右分割窗口

### 2.rows属性

使用像素数和百分比上下分割窗口

### 3.frameborder属性

指定是否显示边框：0不显示，1显示

### 4.border属性

设置边框大小，默认值5像素

### 5.frame子窗口属性

**frameset中设置几个子窗口，对应使用几个frame框架，使用src属性指定网页**

#### (1)src:加载网页的URL地址

#### (2)name:框架名称(链接标记的target属性所需参数)

#### (3)noresize:设置框架大小不可调整(默认可以调整大小)

#### (4)scrolling:设置滚动条
- auto:根据需要自动
- yes:有
- no:无

#### (5)frameborder:设置边框
- 1：显示边框
- 0：不显示边框

## 表单标记

### from标记

```
<from action="服务器地址(接受表单内容的地址)" name="表单名称" method="post/get">...</from>
```

#### 1.method属性

**传送数据的方式，分为post和get两种**

##### (1) get方式

get方式提交时，会将表单内容附加在URL地址的后面，限制提交内容的长度，不超过8192个字符，不具备保密性

##### (2) post方式

post方式提交时，将表单数据包含在表单主体中，一起传送到服务器，没有数据大小的限制，具有保密性

#### 2.action属性

**表单数据处理程序的URL地址，如果为空则使用当前文档的URL地址**

#### 3.enctype属性

**设置表单的资料的编码方式**

#### 4.target属性

**用来指定目标窗口**

### input标记

```
<input name="" type="" value="" size="" maxlength="">
```

#### 1.type属性

**输入**

##### (1)text 文本输入

##### (2)password 密码输入

**按钮**

##### (3)submit 提交按钮

##### (4)reset 重置按钮

##### (5)button 普通按钮

**选择框，通过checked设置默认选择项**

##### (6)radio 单选框

##### (7)checkbox 复选框

##### (8)hidden 隐藏域

#### 2.value属性

**初始化，打开页面时，文本框中的内容**

#### 3.size属性

**设置控件的大小**

#### 4.maxlength属性

**输入框最大输入的字符数**

### textarea标记

```
<textarea name="" cols="" rows="" value="">(多行文本域)</textarea>
```

### select标记
