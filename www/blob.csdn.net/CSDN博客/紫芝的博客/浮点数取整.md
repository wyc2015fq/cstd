                ceil函数：对小数向上取整，返回值为double类型 

floor函数：对小数向下取整，返回值为double类型 

```cpp
#include<iostream>
#include<cmath>
using namespace std; 
int main(){
	double n;
	while(cin>>n){
		cout<<"向上取整："<<ceil(n)<<endl;
		cout<<"向下取整："<<floor(n)<<endl; 
	}
	return 0;
}
```
