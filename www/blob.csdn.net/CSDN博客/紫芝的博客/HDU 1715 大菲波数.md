# 大菲波数
**Time Limit: 1000/1000 MS (Java/Others)    Memory Limit: 32768/32768 K (Java/Others)Total Submission(s): 22756    Accepted Submission(s): 8183**


Problem Description


Fibonacci数列，定义如下：
f(1)=f(2)=1
f(n)=f(n-1)+f(n-2)  n>=3。
计算第n项Fibonacci数值。













Input

输入第一行为一个整数N，接下来N行为整数Pi（1<=Pi<=1000）。





Output

输出为N行，每行为对应的f(Pi)。





Sample Input


5
1
2
3
4
5





Sample Output


1
1
2
3
5

#include<bits/stdc++.h>
using namespace std;
const int maxn=1009,bit=220;
char f[maxn][bit+5];//第1000个斐波那契数有210位
int main()
{
	ios::sync_with_stdio(false);
	memset(f,0,sizeof(f));
    f[1][0]=1;f[2][0]=1;
    for(int i=3;i<maxn;i++){
        for(int j=0;j<bit;j++)
            f[i][j]=f[i-1][j]+f[i-2][j];
        for(int j=0;j<bit;j++)
        if(f[i][j]>9){
            f[i][j+1]++;
            f[i][j]%=10;
        } }

      int n,pi;
      cin>>n;
      while(n--){
        cin>>pi;
        int pos=bit;
        while(f[pi][pos]==0)
            pos--;
        for(int i=pos;i>=0;i--)
            printf("%d",f[pi][i]);
        printf("\n");
      }

	return 0;
 }
