# [编程题] [树的子结构](https://www.nowcoder.com/questionTerminal/6e196c44c7004d15b1610b9afca8bd88)

**热度指数：`349541` 时间限制：`1秒` 空间限制：`32768K`**

输入两棵二叉树A，B，判断B是不是A的子结构。（ps：我们约定空树不是任意一个树的子结构）

```java
class TreeNode {
    int val = 0;
    TreeNode left = null;
    TreeNode right = null;

    public TreeNode(int val) {
        this.val = val;

    }

}

public class Solution {
    public boolean HasSubtree(TreeNode root1,TreeNode root2) {
        //如果是空树，直接返回false
        if(root2==null||root1==null)
        return false;

        return IsSubtree(root1, root2)//以root1为起点，判断是否包含Tree2
        ||HasSubtree(root1.left, root2)//以Tree1的左儿子为起点，判断是否包含Tree2
        ||HasSubtree(root1.right, root2);//以Tree1的右儿子为起点，判断是否包含Tree2
    }
    public boolean IsSubtree(TreeNode root1,TreeNode root2){
        if(root2==null)
        return true;//如果Tree2已经遍历完了，并且都可以对应，返回true
        if(root1==null)
        return false;//如果Tree2还没遍历完，Tree1已经遍历完了，返回false
        if(root1.val==root2.val){//如果根节点可以对应上，继续匹配子节点
            return IsSubtree(root1.left, root2.left)&&IsSubtree(root1.right, root2.right);
        }else{//根节点无法对应，直接返回false
            return false;
        }
    }
}
```
