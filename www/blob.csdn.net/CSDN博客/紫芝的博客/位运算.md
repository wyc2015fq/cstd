位运算：用于对整数类型(int,char,long等)变量中的某一位(bit)，或者若干位进行操作。

1.判断某一位是否为 1

2.只改变其中某一位，而保持其他位不变
3.位运算符

①&按位与（双目）：两个操作数对应的两个二进位均为1时，结果的对应二进位为1，否则为0 

②|  按位或（双目）：两个操作数对应的两个二进位均为0时，结果的对应二进位为0，否则为1 

③^按位异或（双目）：两个操作数对应的两个二进位不相同时，结果的对应二进位为1，否则为0 

④~按位非（取反，单目）:将操作数中的二进制位0变成1，1变成0 
⑤a<<b左移（双目）：将a各二进制位全部左移b位后得到的值
高位丢弃，低位补0，a的值不因位运算而改变

左移n位，等于乘以2^n次方，比成乘法操作快 
⑥a>>b 右移（双目）： 将a各二进制位全部右移b位后得到的值
低位丢弃，高位补0，a的值不因位运算而改变
对于有符号的数，右移动时，符号位一起移动，如果符号位为1
右移时最高位补1，原符号位为0，最高位补0； 

左移n位，等于除以2^n次方，将结果往小里取整 

利用异或运算符交换两个数

```cpp
#include<iostream>
using namespace std; 
int main(){
	int a,b;
	cin>>a>>b;
	a=a^b;
	b=b^a;
	a=a^b;
	cout<<"a="<<a<<endl;
	cout<<"b="<<b<<endl;
	return 0;
}
```

```cpp
#include <stdio.h>
int main(){
	int n1=15;
	short n2=-15;
	unsigned short n3=0xffe0;
	char c=15;
	n1=n1>>2;
	n2>>=3;
	n3>>=4;
	c>>=3;
	printf("n1=%d,n2=%x,n3=%x,c=%x",n1,n2,n3,c);
	
	return 0;
}
```
