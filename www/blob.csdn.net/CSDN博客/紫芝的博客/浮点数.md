# 浮点数

## 在机内用指数型式表示，分解为：数符，尾数，指数符，指数四部分。
1..数符占1位二进制，表示数的正负。
2.指数符占1位二进制，表示指数的正负。
3.尾数表示浮点数有效数字，0.xxxxxxx,但不存开头的0和点
4.指数存指数的有效数字。

指数占多少位，尾数占多少位，由计算机系统决定。
可能是数符加尾数占24位，指数符加指数占8位 -- float.
数符加尾数占48位，指数符加指数占16位 -- double.


## float: 
单精度浮点数在机内占4个字节，用32位二进制描述。

## double:
双精度浮点数在机内占8个字节，用64位二进制描述。

## long double:
长双精度浮点数在机内占16个字节，用128位二进制描述 
但是编译器不一样也有差别,比如vc中它们都是64位的。
总之long double >= double =64位是不会错的。 

double 和 float 的区别是double精度高，有效数字16位，float精度7位。
但double消耗内存是float的两倍，double的运算速度比float慢得多，
C语言中数学函数名称double 和 float不同，不要写错，

能用单精度时不要用双精度（以省内存，加快运算速度）。

```cpp
#include<stdio.h>
#include<iostream>
using namespace std;
int main() {

	long double a;

	scanf("%Lf", &a);
	printf("%Lf\n", a);
	system("pause");
	return 0;
}
```
