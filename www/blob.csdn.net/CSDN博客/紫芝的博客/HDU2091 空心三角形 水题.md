# 空心三角形
**Time Limit: 1000/1000 MS (Java/Others)    Memory Limit: 32768/32768 K (Java/Others)Total Submission(s): 54709    Accepted Submission(s): 14696**


Problem Description

把一个字符三角形掏空，就能节省材料成本，减轻重量，但关键是为了追求另一种视觉效果。在设计的过程中，需要给出各种花纹的材料和大小尺寸的三角形样板，通过电脑临时做出来，以便看看效果。





Input

每行包含一个字符和一个整数n(0<n<41)，不同的字符表示不同的花纹，整数n表示等腰三角形的高。显然其底边长为2n-1。如果遇到@字符，则表示所做出来的样板三角形已经够了。





Output

每个样板三角形之间应空上一行，三角形的中间为空。显然行末没有多余的空格。





Sample Input


X 2
A 7
@





Sample Output


 X
XXX
　
      A
     A A
    A   A
   A     A
  A       A
 A         A
AAAAAAAAAAAAA

## 题解：

特定格式输出题，画一次就知道了，注意每个样板三角形之间应空上一行，WA了多次终于AC的水题

![](https://img-blog.csdn.net/20180505162430162)


```cpp
#include<bits/stdc++.h>
using namespace std;

int main()
{
	ios::sync_with_stdio(0);
	int n,Case=0;
	char s[2];
	while(~scanf("%s",s)){
		if(s[0]=='@')
		break;
		
		scanf("%d",&n);
		
		Case++;
		if(Case!=1)
		printf("\n");
		
		if(n>1){
		for(int i=0;i<n-1;i++)
		printf(" ");
		printf("%c\n",s[0]);
		
				int cnt=1;
		for(int i=2;i<n;i++){
			for(int j=0;j<n-i;j++)
			printf(" ");
			printf("%c",s[0]);
			for(int k=0;k<cnt;k++)
			printf(" ");
			printf("%c\n",s[0]);
			cnt+=2;
		}
		
		for(int i=0;i<2*n-1;i++)
		printf("%c",s[0]);
		printf("\n");
		}else{
			printf("%c\n",s[0]);
		}	
	}
	return 0;
}
```
