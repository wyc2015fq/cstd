# 合成特征（synthetic feature）
　　不在输入特征中，而是从一个或多个输入特征中派生出的特征。 
　　合成特征的类型包括：
```
1. 特征与自己或其他特征相乘（叫作特征交叉）。
2. 两个特征相除。
3. 将连续的特征放进 range bin 中。
```
　　注意，由归一化或缩放单独创建的特征不是合成特征。
