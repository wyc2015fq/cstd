1) 插入文本；
2）修改文本；
3）删除文本；
vi & vim 有编译模式与命令模式，下述光标移动操作均为命令模式下操作；
下文中以 ^ 符表示 Ctrl 键被按下，如 ^F 标示 Ctrl 与 F 键被同时按下；
下文中以 + 号表示按下某键后再按另外按键，如 z + Enter 表示先按下 z 键，再按下 Enter 键；
1， 插入文本：
> ![](http://blog.chinaunix.net/attachment/201306/7/20671208_13705752301axe.jpg)
（命令模式下）
> 1.1 命令模式下按下 i 键，进入vi & vim 的插入编辑模式进行文本插入：
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370575313YS09.jpg)
1.2 命令模式下按下 a 键，进入 vi & vim 的追加插入编辑模式进行文本插入：
![](http://blog.chinaunix.net/attachment/201306/7/20671208_13705754037qR6.jpg)
note： 插入与追加插入命令都会使 vi & vim 进入文本插入编辑模式，
唯一区别就是插入命令会保持光标位置不变，追加命令会使光标移到原光标位置的下一个字母的位置进行文本插入。
2， 修改文本：
> 2.1 修改文本字母：
> 
> 2.1.1 命令 r + 字母 ，修改光标所在位置的字母为指定的字母，r 命令只修改一个字母，操作完成后依旧停留在命令模式，不需要按 Esc 键返回命令模式。
![](http://blog.chinaunix.net/attachment/201306/7/20671208_137057662049U4.jpg)
2.1.2 命令 s + 字母，替换光标所在位置的字母为任意输入文本，
此命令会删除光标所在的字母，使 vi & vim 进入插入编辑模式，此时可以输入任意你想输入的文本，直到你按下 Esc 键退出插入编辑模式。
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370577256Oz5s.jpg)
2.1.3 命令 数字 + s + 字母，替换光标所在位置后面数字指定数量的文本为任意输入文本，与前面一个命令的区别就是替换指定数量的字母。
例如命令：3s + eee
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370577549g20I.jpg)
2.1.4 命令 R + 任意输入，进入替换模式，后续字母输入将逐一替换原有文本，直至按下 Esc 键回退到命令模式。
例如命令： R + www
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370578218pdNO.jpg)
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370578228S1w1.jpg)
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370579162BLWb.jpg)
> 2.2 修改文本块：
> 
> ![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370579956CpcN.jpg)
> 
> 2.2.1 命令 c + w ，往后修改一个文本单词，此命令会从光标位置往后删除一个单词，并使 vi 进入插入编辑模式进行文本插入。
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370580018e948.jpg)
2.2.2 命令 c + 数字 + w，往后修改数字指定数量的文本单词，并使 vi 进入插入编辑模式进行文本插入。
例如命令：c2w
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370581635cMRE.jpg)
2.2.3 命令 c + b ，往前修改一个文本单词，此命令会从光标位置往前删除一个单词，并使 vi 进入插入编辑模式进行文本插入。
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370581981UoUe.jpg)
2.2.4 命令 c + 数字 + b，往前修改数字指定数量的文本单词，并使 vi 进入插入编辑模式进行文本插入。
例如命令：c2b
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370582055Jj06.jpg)
2.2.5 命令 c + 0 （数字零），删除光标位置到行首的所有文本，并使 vi 进入插入编辑模式进行文本插入。
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370582292teY9.jpg)
2.2.6 命令 c + $ ，删除光标位置到行末的所有文本，并使 vi 进入插入编辑模式进行文本插入。
![](http://blog.chinaunix.net/attachment/201306/7/20671208_1370582499wCOB.jpg)
note：此命令等效于命令： C
2.2.7 命令 cc ，删除光标所在的一整行文本，并使vi进入插入编辑模式进行文本插入。
note：此命令等效于命令： S
3，删除文本：
> 3.1
 删除文本字母
> 
> 3.1.1
 命令 x ，删除光标所在位置的一个字母，并保持在命令模式。
> 3.2
 删除文本块
> 
> 3.2.1
 命令 dw ,删除光标所在位置往后的一个单词，并保持在命令模式。
3.2.2 命令 d + 数字 + w ，删除光标所在位置往后的由数字指定数量的单词，并保持在命令模式。
3.2.3 命令 dd ，删除光标所在的一整行，并保持在命令模式。
3.2.4 命令 d + 0 ，（数字零），删除光标所在位置直到行首的所有文本，并保持在命令行模式。
3.2.5 命令 d + $ ,删除光标所在位置直到行末的所有文本，并保持在命令行模式。此命令等效于命令： D
