在学习Visual Studio 2017的过程中，总会遇到各种各样的难题，这时候你就会求助书或者是网上大佬们的解释，但是在看视频的过程中，我发现了MSDN这个“好东西”，就立马应用于实践，下面把步骤分享给大家。
说明：MSDN可以是你在装Visual Studio 2017时安装，也可在安装完Visual Studio 2017后添加安装。我的是在Visual Studio 2017安装完后添加的，如果是前期安装的话，步骤大同小异，怎么舒服怎么来吧~
第一步：“Window键”单击“Visual Studio Installer”；（我的在打开的过程中更新了一下，时间有点漫长，你可以选择等待）
第二步：这时你会看到如下界面，单击“更多”下拉列表，在其弹出列表中单击“修改”；（也会有一小段等待时间，请耐心等候。）![在这里插入图片描述](https://img-blog.csdnimg.cn/20181105162935531.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L0VsbGVuNTIwMw==,size_16,color_FFFFFF,t_70)
第三步：在弹出的窗口中，切换到“单个组件”，找到“代码工具”，选中“Help Viewer”，点击“修改按钮”确认安装组件变更；
第四步：当安装成功后，关闭Visual Studio Installer，打开Visual Studio 2017，点击“帮助”——添加和删除帮助内容；
![在这里插入图片描述](https://img-blog.csdnimg.cn/20181105164150475.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L0VsbGVuNTIwMw==,size_16,color_FFFFFF,t_70)
第五步：在弹出的界面中，单击“管理内容”，找到Windows，将其内帮助文档全部“添加”到下载列表中，然后点击右下角“更新”即可；（等待时间更加漫长）
![在这里插入图片描述](https://img-blog.csdnimg.cn/20181105164230253.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L0VsbGVuNTIwMw==,size_16,color_FFFFFF,t_70)
第六步：切换到“索引页”，在搜索框中输入“CreateWindows”。如果结果列表中出现“CreateWindow function [Windows and Messages]”则表示Windows开发所需要的帮助文档安装成功。Very Good！
