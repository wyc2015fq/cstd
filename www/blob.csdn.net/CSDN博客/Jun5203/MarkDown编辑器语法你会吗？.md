## 标题
● 在文字前书写不等数量的#可以完成不同的标题，如下：
```
# 一级标题 
## 二级标题 
### 三级标题 
#### 四级标题 
##### 五级标题 
###### 六级标题
```
● #的个数即是标题的等级。 
● 等号及减号也可以进行标题的书写，不过等号书写一级标题，而减号书写二级标题，并且需要写在文字的下面。减号及等号的数量不会影响标题的基数，如下：
```
二级标题 
=======
二级标题 
--------
```
## 列表
● 列表分为有序和无序两种。 
● 无序列表用*或者-或者+表示。
- 列表一
- 列表二
- 列表三
● 有序列表用阿拉伯数字加点表示。
- 列表一
- 列表二
- 列表三
★★★ 符号和文字之间需要用空格分开（至少一个）。 
● 嵌套列表 
第一级：随意 
第二级：空一格 
第三极：空四格 
之后：空四格
- 列表一 
- 列表二 
- 列表三
- 列表四

## 粗体、斜体、加粗斜体、分割线、删除线
```
粗体：**文字**
斜体：*文字*或_文字_
加粗斜体：***文字***
分割线：***或者___（粗）；---（细）
删除线：~~文字~~
```
