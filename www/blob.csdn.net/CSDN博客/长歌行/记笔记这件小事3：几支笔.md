工欲善其事必先利其器，记笔记，没有笔可不行。爷爷喜欢用软笔写打油诗，爸爸用毛笔写字，我呢，也从小耳濡目染，喜欢摆弄这些笔墨，虽然字写得不好，但是就是爱这一口。

大致介绍下，现在我用着哪些笔吧。

# 百利金M400白色

俗称白乌龟 
![img](http://yuji.bjdn.openstorage.cn/public/image/600AA55C-8ABE-4155-9369-2207A071418C.jpeg)

有一段时间就特别想用钢笔，然后就在那个，淘宝上面就，也买那个m4，400成为百五归，然后来借钢笔的毒，反正当这是我解读的话，要么就是，啊，你们2292失败，然后我觉得你每一桩事，嗯他的是按键，400，明天特别好看，那就冲着颜值买的，要用钱的话也特别好些，不过比我稍微帅的一次，那就，修到能把比赛办成之后写还是挺好写的，但就现在就有点不敢随便的使用它了，但主要是放着供着，是不是把玩一下，张也是几个月都不用的话，也能够一笔出水，也挺好的。
# 凌美狩猎者苹果绿

![img](http://yuji.bjdn.openstorage.cn/public/image/2350FD0B-EFD0-40E7-A039-D94DB15620C4.jpeg)

这是在唯品会买的，大概80块钱。

虽然说凌美狩猎者，凌美恒星是烂大街的笔，本来不想购入，但是经不住大家一遍遍的安利，近期就买了一支，作为日用或者收藏吧。大家说它的优点是外观好，工业设计感。因为我其他的钢笔，要么如78G太细，要么如M400太招摇，不方便用来搬砖写文字写代码，所以想以此作为日用。

# 永生612a

![img](http://yuji.bjdn.openstorage.cn/public/image/0FAC9EEF-1A85-4F61-8EFD-E30B7E56EC50.jpeg)

这支笔购于淘宝，主要也是被别人安利的，说是经典款老笔。我从来都没有使用过永生的。正好，这些老笔，价格都比较便宜，所以入手一个，来试试。笔尖太硬了，写着真不舒服，就像拿着烤串的铁签子书写一样。速写的话还可以。整体素质也行。没有漏墨等现象。
# 翎墨

国产钢笔，仿照写乐幽灵。主要特点是旋转上磨和透明示范。其实写感不好。经常飞白。大概是30多元入手，价格还可以，值得一提的是，它的盒子可以当文具盒用，省去了一个MUJI的文具盒了。

# Bamboo 触控笔

![img](http://yuji.bjdn.openstorage.cn/public/image/B2017FC0-21F9-41B6-ABCB-68409149DAE5.jpeg)

是布头的。京东300多购入了。下完单，我当时觉得自己简直是太冲动了。我基本上不用pad或者手机记笔记，为啥还买这个啊。不过手感确实好。因为在屏幕上写字，一般是要写大字的，笔需要比平时轻一点，后来我为了好玩，也买了几款国产的触控笔。那些笔虽然份量十足，但是写字时间长了会很累很累。这笔的缺点是，外表是铝制的，容易磨花。另外，无压力感应等。

# 蜻蜓 graph

![img](http://yuji.bjdn.openstorage.cn/public/image/6D18426D-742F-46EA-925C-4AF8E73E0CC1.jpeg)

买的0.3的。这笔的优点是，晃动上铅，配重合适，尾端的大橡皮很好用，可以旋转出芯。现在有点后悔没有买0.5的了。因为0.3的HB太淡，2B太脆。这支笔配色也十分好看。不好的地方是，笔前端的护芯管不能收进去，让人没有安全感。（好像晨光也出了这一款的仿版）

# 英雄美工笔

![img](http://yuji.bjdn.openstorage.cn/public/image/12E84508-FB7E-487D-B89A-2B96C14FFAA2.jpeg)

英雄的美工笔，用他写的字，我觉得是挺好看的，不需要练字就能够把横竖撇捺折的各个细节给勾画出来，然后我主要是用它来写一些标题，是一只耐用也耐看的笔。

# 三菱旋转二代

![img](http://yuji.bjdn.openstorage.cn/public/image/FE95978F-63AF-4DC8-AE6E-C010BA3489EB.jpeg)

这个是三菱的旋转笔第二代，然后也是我现在的主力自动铅笔，它能边写边旋转，所以写出来的字笔画很一致，但我一般是用2b的铅芯，在国誉的纸上面写，效果很好。也许有些人会觉得写字的时候会有点晃，但是我觉得还好。

# 比克四色笔

![img](http://yuji.bjdn.openstorage.cn/public/image/7ECCD5C2-9173-4284-97B7-188345EB5446.jpeg)

这个是比克的四色笔，法国产，这里面的弹簧特别有劲儿，0.7的吧，感觉就是粗了一些。颜色倒挺准的。

# 晨光某自动铅笔

![img](http://yuji.bjdn.openstorage.cn/public/image/640976FD-728C-4BD4-9601-F4302C9C8E58.jpeg)

晨光的自动铅笔，12块钱。我觉得也挺好用的，配重啊什么的都挺好的，也是我买的最早的一支自动铅笔，后来想换别的使用一下所有就闲置了起来。
# 文冠某宝珠笔

![img](http://yuji.bjdn.openstorage.cn/public/image/F5E36B81-2E6C-42AC-9338-55310277FCD9.jpeg)

这是文冠的宝珠笔，用了20多年吧，主要是纪念价值，看起来像是女士的，从初中用到现在，目前外观还挺好的。

# 三菱 freeStyle

![img](http://yuji.bjdn.openstorage.cn/public/image/8F16DB12-6636-4895-A820-8466DDB4AC69.jpeg)

我是在淘宝上面买的三菱的三色模块笔，因为我一直很喜欢模块笔。最大缺点，性价比低，笔芯几天就用完了，还贵。所以买自动铅笔模块和圆珠笔芯会比较好。中性笔芯0.38的我觉得不是很好，圆珠笔的话效果很好。

# 笔刀小黑

![img](http://yuji.bjdn.openstorage.cn/public/image/7E65BC6E-5264-4759-8AC4-9F2E83F4FD02.jpeg)

这个就是用于橡皮雕刻上的小黑，反正平时雕刻雕刻橡皮章啊，然后拆快递盒也挺有用的，业余娱乐一下吧。
# MUJI凝胶

![img](http://yuji.bjdn.openstorage.cn/public/image/3F4AAFEC-019A-429B-B86E-D0B6188425FD.jpeg)

无印良品的凝胶墨水笔，八块钱一只，觉得是很好用的，有一只还有点漏墨，但是我目前用的黑色的话都没有问题，我觉得真的是特别好写，八块钱稍微有点贵。他对于办公室工作的人来说，写字量不是很大的话，倒觉得无所谓。
# 某赠品正姿笔

![img](http://yuji.bjdn.openstorage.cn/public/image/816F55FF-61FB-4B3F-AE01-D35BC79F6C47.jpeg)

好像是买别的笔的时候送的，然后真的是非常好用。我会给小孩子买这种笔，非常好写。
# 派克威雅

![img](http://yuji.bjdn.openstorage.cn/public/image/68BD9823-E46C-436D-8978-A456E0963640.jpeg)

我爸送我的，水性签字笔，用来做笔记的话，用得太快不划算，笔芯好像是三十多一支，一两天可以用完。
# 英雄一百全钢

![img](http://yuji.bjdn.openstorage.cn/public/image/76263A7C-4B8B-4F0A-9965-71771B322B51.jpeg)

英雄一百全钢，情怀向，觉得确实不太好用，笔尖打磨有点奇怪，写出来字都是方方正正的，但其他部分用料很实在，做工也挺好。在非常非常滑溜的纸上写字较好。

# 英雄58

![](http:<a href=)http://yuji.bjdn.openstorage.cn/public/image/C346F600-34CC-46D8-A624-65FF651BE3C9.jpeg” alt=”img” title=”” />

价格15吧，我小时候用的比较多，所以以后就买了一个，也是有各种各样的缺点吧，笔经常堵。

# 斑马防断芯

![](http:<a href=)http://yuji.bjdn.openstorage.cn/public/image/0129B55B-2BDC-4B8B-ABD6-30606F332120.jpeg” alt=”img” title=”” /> 

已经给娃用了，确实不会断，样子也不错。但是作为成人使用，一般的自动笔芯也不会断，优势不大。
# MUJI毛笔

![img](http://yuji.bjdn.openstorage.cn/public/image/13903983-2181-42CB-91F9-3537F44698CA.jpeg)

无印良品的那个，好用，但是，用的时间长了，前端有些分叉，分叉后写的字都是毛毛糙糙的。

# 百乐78G

![img](http://yuji.bjdn.openstorage.cn/public/image/8D0F2C56-63B9-49E6-82F7-2B8E7F6EC51B.jpeg)

经典日系笔，我买的是EF尖，真的是超级超级细，大概49元，非常好写，下水流畅，非常好写，不过现在涨价了。

# 英雄216

![img](http://yuji.bjdn.openstorage.cn/public/image/AF5E85E9-29BF-40A2-AD34-54999927DAAB.jpeg)

英雄216也是我初中的时候一直用到现在，外观也非常的大气，下水的话就可能有点多，墨囊已经被腐蚀扭曲了。
