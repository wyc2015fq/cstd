题目描述：给定一个整数数组，找到和为零的子数组。你的代码应该返回满足要求的子数组的起始位置和结束位置

样例：给出 [-3, 1, 2, -3, 4]，返回[0, 2] 或者 [1, 3].

求出和为0的子数组，穷举当然能解决。但是效率未免太低了。所以，我在这里用了一种以空间换取时间的办法：哈希表（所谓哈希表，在这里可简单理解成一个由键值对组成的列表，也就是python里面的字典）

什么意思呢？假设现在我们依次求取数组的和（前1位的和，前2位的和，前3位的和。。。），并且将每次求出的和以及所对应的是前几位对应起来，存储在一个哈希表里（键为和，值为对应前几位）。如果求到前n位的和时，发现与前面的某个和（比如前m位的和）相等，那也就意味着从数组的m + 1位开始到第n位这部分所有元素的和为0。这样一来，就实现了通过一次遍历解决问题。

代码如下：



```python
class Solution:
    """
    @param nums: A list of integers
    @return: A list of integers includes the index of the first number 
             and the index of the last number
    """
    def subarraySum(self, nums):
        sum = 0
        # 给hash表增加一个前置位：键 —— 0；值 —— -1
        hash_table = {0:-1}
        n = len(nums)
        for i in range(n):
            sum += nums[i]
            if sum in hash_table:
                return [hash_table[sum] + 1, i]
            hash_table[sum] = i

```
