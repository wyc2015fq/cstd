题目描述：给定一个只包含字母的字符串，按照先小写字母后大写字母的顺序进行排序。

样例：给出"abAcD"，一个可能的答案为"acbAD"

跟排颜色的题（详见：[点击打开链接](http://blog.csdn.net/guoziqing506/article/details/51227695)）是一样的。（额。。。我也不知道为什么同样的问题，他会出这么多），所以我不讲了。就是通过交换，将小写全部放到数组前面。代码在下面：



```python
class Solution:
    """
    @param chars: The letters array you should sort.
    """
    def sortLetters(self, chars):
        left = -1
        n = len(chars)
        for i in range(n):
            if chars[i].islower():
                left += 1
                chars[i], chars[left] = chars[left], chars[i]
        return chars

```
