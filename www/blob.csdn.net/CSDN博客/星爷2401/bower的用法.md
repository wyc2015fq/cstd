bower是什么？
Bower是一个客户端技术的软件包管理器，它可用于搜索、安装和卸载如JavaScript、HTML、CSS之类的网络资源。其他一些建立在Bower基础之上的开发工具。
说白了bower就是负责下载各种前端的框架，比如jquery，bootstrap，而不用自己手动下载。
1、bower的安装。（先安装好npm）
npm install bower在当前路径下安装
npm install -g bower在全局路径下安装
