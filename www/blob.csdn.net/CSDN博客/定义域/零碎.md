# jCenter
jCenter 是bintray.com维护的依赖仓库，和 Maven Center 类似。作为各种依赖管理工具的依赖源，比如 Maven、Ivy、Gradle。
# AS 构建缓慢如何处理
AS 的构建系统 gradle 每次编译都要解析依赖（连接 jCenter，但国内访问不稳定）。可以 Preference   - Gradle - Offline work，再需要新的依赖时，再让 Gradle 联网工作。减少没有依赖变化时，构建所需的时间。最好的方式是能够稳定的翻墙代理。
# 配置 gradle 使用哪个依赖仓库
```
# 使用 jcenter 
allprojects {
    repositories {
       jcenter()
    }
}
# 使用 maven
allprojects {
    repositories {
        mavenCentral()
    }
}
```
# 配置 gradle 使用自定义依赖仓库
```
# 添加自定义的依赖源
repositories {
    maven { url 'https://maven.fabric.io/public'   }
}
```
