- Git或者说VCS，除了可以用于团队协作的开发，也可用于个人对自己代码的VC。比如个人练习或小项目的优化过程都需要VC来回到之前的版本。
- Git for Windows 和 GitHub Desktop虽都是客户端，但前者多用命令行，后这多GUI。后者因为是对 Github 使用经过优化，可能入手较快且更加适合用于 GitHub。但前者在根据错误信息定位以及更多功能的使用上比前者更好。
- 可同时安装 Git for Windows 和 GitHub Desktop，二者互不影响。GitHub Desktop 使用自带的 Git for Windows 组件。
- Git 默认是针对 Linux 开发的，在 Windows 上使用 Git 有上面两种方式。Git for Windows 内部使用 MSYS2 来提供 Git 运行所需的 POSIX 环境。MSYS2 是从 Cygwin 精简移植过来的，其提供了所需环境。
