- 获取NDK安装包位置 
- 设置代理，打开 AS - Config - SDK Manager，看到 NDK uninstalled，使用 AS 下载安装；阻塞，复制链接，用迅雷下载。
- 移动到 SDK 安装目录下，解压到当前目录，重命名为 ndk-bundle
参考：[http://tools.android.com/tech-docs/android-ndk-preview](http://tools.android.com/tech-docs/android-ndk-preview)
“In this release we’ve tried to unify this; the NDK should now be installed under the SDK home in “ndk-bundle”.”
