使用[地址栏](http://support.google.com/chrome/bin/answer.py?answer=95440)可搜索网页、书签和浏览历史记录。您只需在地址栏中输入搜索字词，然后按 **Enter** 键，即可查看通过[默认搜索引擎](http://support.google.com/chrome/bin/answer.py?answer=95653)搜到的结果。
以下是 Google Chrome 浏览器中进行搜索的其他所有方法。以下提示适用于 Windows、Mac、Linux 和 Chrome 操作系统中的 Google Chrome 浏览器。
- **搜索具体的网站（使用 Tab 键进行搜索）**
首先在地址栏中键入要使用的搜索网站的网址。如果 Chrome 浏览器已记录该网站的搜索引擎，就会自动提供用于在该网站中进行搜索的选项。如果您记得该[搜索引擎的关键字](http://support.google.com/chrome/bin/answer.py?answer=95653)，也可以在地址栏中键入该关键字。按 **Tab**键选择搜索引擎，键入搜索字词，然后按 **Enter** 键。
- **搜索网页**
使用[查找栏](http://support.google.com/chrome/bin/answer.py?answer=95635)可以查找当前查看的网页上的特定字词。按键盘快捷键** Ctrl+F**（适用于 Windows 和 Linux）或 **⌘-F**（适用于 Mac）可在当前网页上打开查找栏。
- **搜索浏览历史记录、下载内容和书签**
您也可以使用“历史记录”页、[“下载内容”页](http://support.google.com/chrome/bin/answer.py?answer=95758)和[书签管理器](http://support.google.com/chrome/bin/answer.py?answer=95714)顶部的搜索框，分别搜索浏览历史记录、下载历史记录和书签。这些资源都可以通过
 Chrome 浏览器菜单使用![Chrome menu](http://storage.googleapis.com/support-kms-prod/SNP_2696434_en_v1)（位于浏览器工具栏上）。
- **右键点击搜索**
用光标突出显示网页上的文字。点击右键（或在 Mac 上按住 Control 键的同时点击鼠标），然后选择相应的选项，即可搜索突出显示的文字。
- **粘贴并搜索**
用光标突出显示网页上的任意文字，再对其进行复制。当您准备开始搜索时，右键点击（或在 Mac 上按住 Control 键的同时点击鼠标）地址栏，然后选择**粘贴并搜索**。
- **突出显示并拖动搜索字词**（仅限 Windows）
使用光标突出显示网页上的任意文本，然后将其拖到地址栏中。Google Chrome 浏览器会自动显示使用默认搜索引擎搜索突出显示的文本所得到的结果。
