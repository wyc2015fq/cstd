1.[Visual studio 2005 恢復默認設置的方法](http://www.cnblogs.com/rainmanstu/archive/2007/09/22/902525.html)----[http://www.cnblogs.com/rainmanstu/archive/2007/09/22/902525.html](http://www.cnblogs.com/rainmanstu/archive/2007/09/22/902525.html)
遇到無法從代碼窗口切換到設計窗口等問題
首先關閉Visual studio 2005
在msdos窗口下進入安裝路徑 
E:\Program Files\Microsoft Visual Studio 8\Common7\IDE
在msdos窗口下可選命令有：
/ResetSettings  還原 IDE 的預設值，選擇性的重設為指定的 VSSettings 檔。
/ResetSkipPkgs  清除所有加入至 VSPackages 的 SkipLoading 標記。
 運行如下命令：
E:\Program Files\Microsoft Visual Studio 8\Common7\IDE> **devenv  /resetskippkgsOK了！**
2.VS2005的Solution Explorer在右边----[http://blog.sina.com.cn/s/blog_4d25c9870100cwqq.html](http://blog.sina.com.cn/s/blog_4d25c9870100cwqq.html)
那天重装了一下XP，再安装VS2005,发现Solution Explorer在右边，所有的.c和.h都在右边。感觉不好用。就调到左边了。
解决方法：
一，英文版的VS2005下面：
    第一步：在VS2005界面下，Tools--->Import and Export Settings Wizard--->Reset all settings --->Next--->NO,just reset settings,overwriting my current settings--->àNext--->Visual C++ Development Settings--->Finish
    第二步：這樣就會得到默認的形式，下面就是要設置好你的字體，Tools--->Options--àFonts設置好，保存。同樣Tools--->Import and Export Settings Wizard--->Import selected environment settings --->Next--->Yes,save current settings(有一個settings
 filenames.記一下一會有用) --->Next--->Visual C++ Development Settings--->Next--->Finish
    第三步：Tools--->Import and Export Settings Wizard--->Import selected environment settings --->Next--->Yes,save current settings(有一個settings filenames.不要記了) --->Next--->上一次讓你記住的那個settings
 filenames，不是這次啊 ---->Next---->Finish
二，中文版的VS2005下面
      VS2005 界面下，选择“工具”-->“导入和导出设置”-->“重置所有设置”-->“下一步”-->任选“是/否”-->“下一步”-->选择“Visual C++ 开发设置”-->“完成”。
之后你再设置你自己所喜欢的字体就行了。并保存。之後再保存你的設置。和英文的設置一樣進行，這樣最後你的VS2005，再在Window---àReset windows layout就不會得到第一步的效果了（你所設置的字體不管用了），而且是第三步的效果。
3.vs2008的solution explorer如何放到左边----[http://zhidao.baidu.com/question/128383323](http://zhidao.baidu.com/question/128383323)
用鼠标左键点击solution explorer顶部的边框，并按住不放并拖动。拖动时，会出现透明的方块，将鼠标放进对应位置的方块后松开即可
4.出现在VS2005可以打开的文件但在SOURCEINSIGHT里面打不开，一般是文件的存储格式出了问题，可以在VS2005里面进行转换----在FILE菜单/高级保存选项里面可以选择保存的编码格式
5.[为VS2005工程添加新的SDK](http://blog.csdn.net/norains/article/details/5600904)----[http://blog.csdn.net/norains/article/details/5600904](http://blog.csdn.net/norains/article/details/5600904)
6.修改VS2005 IDE里面的字体设置----在工具/选项/环境/字体和颜色菜单下可以进行设置，我用的字体设置如下
    font type::Fixedsys++++font size::12------------个人觉得视觉效果不错的一个设置
 7.在写APP的时候在对话框上加按钮切到工具箱没看到IDE系统自带的控件，没显示出来，进行一次reset  toolbox就可以了。
