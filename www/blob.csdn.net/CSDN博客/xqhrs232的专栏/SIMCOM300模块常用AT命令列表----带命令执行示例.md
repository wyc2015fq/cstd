1、AT----测试模块有没正常工作
   //1111----正常工作返回OK
2、ATI----读取模块的信息
   ati
  SIMCOM_Ltd
  SIMCOM_SIM300
  Revision:1008B15SIM300M32_SST34HF3284
  OK
3、ATD----拨打电话-----我的动感地带的卡在SIM300模块上打不了电话----必须用神州行的卡才可以，这也就是为什么我买了个神州行卡的原因
    //1111----atd075526785510;
    //2222----ATd075526785510;-----我的动感地带的卡
      ERROR
4、ATA----接电话
5、ATH----挂电话
6、拨号上网-----APN设置
   AT+CGDCONT=1,"ip","3gnet"------联通卡
   AT+CGDCONT=1,"ip","cmnet"------移动卡
//  at+cgdcont=1,"ip","cmnet"
7、AT+CSQ----信号的质量
   AT+csq
   +CSQ: 23,0
   OK
8、AT+CREG----网络注册情况
   AT+creg=?
   +CREG: (0-2)
   OK
9、AT+CSCS----查询支持的网络
at+cscs=?
+CSCS: ("GSM","HEX","IRA","PCCP","PCDN","UCS2","8859-1")
OK
10、查询SIM卡是联通还是移动的-----我的动感地带的卡读不到，返回的值为0，也打不了电话
   AT+COPS?
   +COPS: 0,0,"CHINA MOBILE"
   OK
11、AT+CGATT=1命令激活GPRS功能，使GPRS模块附着在网络上。如果返回OK，则GPRS 连接成功；如果返回ERROR，则意味着GPRS失败
   //1111----AT+CGATT=?------Test command 
   +CGATT: (0-1)
   OK
    //2222----AT+CGATT=?------read command 
   +CGATT: 1
   OK
   //3333----AT+CGATT=1------write command 
  OK
