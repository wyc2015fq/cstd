什么叫被中断的系统调用，举个例子：经典的服务器模型
当服务器卡在accept的时候（可以理解为服务器进程现在是accept状态），也就是服务器阻塞在accept这个系统调用的时候，如果之前有signal的信号处理函数，假如这时候突然来了个子进程的信号：SIGCHLD， 这时候服务器进程就会把accept中断了转而先处理signal函数，同时置errno为EINTR，如果服务器对这个返回的错误进行了处理：
if((connfd = accept(listenfd, (SA*)&cliaddr, &clilen)) < 0)
{
if(errno == EINTR) continue; // 不管这个错误，继续下一个while循环，相当于重启accept这个系统调用
else err_sys("accept error");
}
