我们看到过很多直线分割平面的题目，今天的这个题目稍微有些变化，我们要求的是n条折线分割平面的最大数目。比如，一条折线可以将平面分成两部分，两条折线最多可以将平面分成7部分，具体如下所示。  
![这里写图片描述](https://odzkskevi.qnssl.com/f38a5cb2aa4e9b9d447d934a7c1b24e7?v=1535073496)

  Input 

  输入数据的第一行是一个整数C,表示测试实例的个数，然后是C 行数据，每行包含一个整数n`(0<n<=10000)`,表示折线的数量。  

  Output 

  对于每个测试实例，请输出平面的最大分割数，每个实例的输出占一行。  

  Sample Input 

  2 

  1 

  2 

  Sample Output 

  2 

  7
一看就是递推  

首先以直线为例子，加上一条直线的话与前面的n-1条直线最多有n-1个交点，会增加n-1+1也就是n个部分

如果是折线的话，加上一条折线与前面的图形最多有2*2*(n-1个交点，会增加4*(n-1)+1个部分

代码：

```cpp
#include <cstdio>
using namespace std;
int main(){
    int n,i,k,j;
    long long a[10010];
    scanf("%d",&n);
    for(i=1;i<=n;i++){
        scanf("%d",&k);
        a[1]=2;
        for(j=2;j<=k;j++)
            a[j]=a[j-1]+4*(j-1)+1;
        printf("%lld\n",a[k]);
    }
    return 0;
}
```
