题目描述  

      现在有一个大小n*1的收纳盒，我们手里有无数个大小为1*1和2*1的小方块，我们需要用这些方块填满收纳盒，请问我们有多少种不同的方法填满这个收纳盒


> 
输入描述: 

  第一行是样例数T 

  第2到2+T-1行每行有一个整数n（n<=80），描述每个样例中的n。
> 
输出描述: 

  对于每个样例输出对应的方法数


> 
示例1 

  输入 

  3 

  1 

  2 

  4 

  输出 

  1 

  2 

  5 

  说明 

  n=4，有五种方法 

  1:1 1 1 1 

  2:2 1 1 

  3:1 2 1 

  4:1 1 2 

  5:2 2
一道简单到不能再简单的dp了，先打表，注意要用long long 

代码：

```cpp
#include <cstdio>
#include <algorithm>
using namespace std;
int main(void){
    long long dp[81];
    dp[0]=1;
    dp[1]=2;
    for(int i=2;i<81;i++){
        dp[i]=dp[i-1]+dp[i-2];
    }
    int t;
    scanf("%d",&t);
    while(t--){
        int n;
        scanf("%d",&n);
        printf("%lld\n",dp[n-1]);
    }
    return 0;
}
