### 普通内部类

```
package com.zxc.innerclass;

public class primaryOuterClass {
    private String name;
    private int age;

    public void setName(String name) {
        this.name = name;
    }

    public void setAge(int age) {
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }

    public void show(){
        System.out.println(name+" "+age+"outer");
    }
    public void showIn(){
        new innerClass().show();
    }

    public class innerClass{
        private String name;
        private int age;

        public void setName(String name) {
            this.name = name;
        }

        public void setAge(int age) {
            this.age = age;
        }

        public String getName() {
            return name;
        }

        public int getAge() {
            return age;
        }

        public void show(){
            System.out.println(name+" "+age+"inner");
        }
        public void showOut(){
            System.out.println(primaryOuterClass.this.name+" "+primaryOuterClass.this.age+"inner");
        }
        public void showOut2(){
            primaryOuterClass.this.show();
        }
        public void showOut3(){
            showIn();
        }
    }

    public static void main(String[] args){
        primaryOuterClass out=new primaryOuterClass();
        out.setAge(12);
        out.setName("hello");

        primaryOuterClass.innerClass in=out.new innerClass();
        out.showIn();
        in.showOut();
        in.showOut2();
        in.showOut3();
        in.setAge(21);
        in.setName("world");
        in.show();
    }
}
```
- 直接在外部类里面定义一个普通的内部类
- 内部类是一个完全独立的类，除了创建实例时需要用到外部类 ，编译会产生两个独立的.class文件，`外部类.class`和`外部类$内部类.class`
- 内部类可以直接访问外部类的属性，方法（无需创建实例），而外部类访问内部类需要创建内部类实例
- 内部类属性（方法）名与外部类属性（方法）名不同时，可以直接使用属性（方法）名，如果相同，内部类访问外部类属性（方法）名时要加上`外部类名.this.属性（方法）名`

### 静态内部类
- 静态内部类可以直接访问外部类的**静态属性（方法）**
- 静态内部类的实例化可以直接`new 外部类.内部类()`，**注意是内部类后面加括号，因为创建的是内部类实例，实际上外部类并没有被实例化**](https://so.csdn.net/so/search/s.do?q=内部类&t=blog)](https://so.csdn.net/so/search/s.do?q=java&t=blog)
