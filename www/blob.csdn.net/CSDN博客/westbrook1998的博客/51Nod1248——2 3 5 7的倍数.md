> 
给出一个数N，求1至N中，有多少个数不是2 3 5 7的倍数。 例如N = 10，只有1不是2 3 5 7的倍数。 

  Input 

  输入1个数N(1 <= N <= 10^18)。 

  Output 

  输出不是2 3 5 7的倍数的数共有多少。 

  Sample Input 

  10 

  Sample Output 

  1
看似简单的题目也做不出来，直接循环肯定超时了，百度了发现是容斥定理，离散数学的

代码：

```cpp
#include <cstdio>
#include <algorithm>
using namespace std;
long long n;
int main(void){
    scanf("%lld",&n);
    long long ans=0;
    ans+=(n/2+n/3+n/5+n/7);
    ans-=(n/6+n/10+n/14+n/15+n/21+n/35);
    ans+=(n/30+n/42+n/70+n/105);
    ans-=(n/210);
    printf("%lld\n",n-ans);
    return 0;
}
```
