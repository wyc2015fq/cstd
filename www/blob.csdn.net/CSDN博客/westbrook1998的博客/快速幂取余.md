### 需要用到的公式

> - **(a^b)%c = ((a%c)*(b%c))%c**
- **a^b=(a^(b/2))或者=(a^(b/2) *a)**
- **直接用例子表示 5^7 = 5^111(2) = 5^(4+2+1) = 5^4*5^2*5^1 = 5^2*5^2*5^2*5^1**


### 代码

```
#include <stdio.h>
int fun(int a,int b,int c){
    int t=1;
    a=a%c;
    while(b>0){   //(b&1)
        if(b%2==1){
            t=(t*a)%c;
        }
        b=b/2; //b>>1;
        a=(a*a)%c;
    }
    return t;
}
int main(void){
    printf("%d\n",fun(8,800,10));
    return 0;
}
```
