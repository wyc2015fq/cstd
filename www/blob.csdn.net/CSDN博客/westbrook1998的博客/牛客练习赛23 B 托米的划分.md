  题目描述  

  欧洲人托米非常喜欢数字，他经常在空闲时玩下面的游戏 

  对于一个数字 n, 托米会随性选中一个数 p, (1 ＜ p <=  n), 将 n 拆分成 ,v=n-u,并对 u,v 重复这个过程，直到他有了 n 个 1 

  1317 为了挑战托米，在每次托米进行划分时，会给托米奖励 u * v 的分数，托米希望你能帮他最大化他的得分。 

  输入描述: 

  第一行一个正整数 T下面 T 行每行一个正整数 n 

  输出描述: 

  对于每组数据，输出托米的最大得分 

  示例1 

  输入 

  复制 

  1 5 

  输出 

  复制 

  10 

  备注: 

  T≤ 104, n≤ 109
题意一开始读不太懂 

就是说给一个数n分割成两个数u和v 得到分数u*v 然后继续重复分割u和v 然后最大化分数 

显然分割的时候两个数越接近越好 

不过试了几个发现其实n对应的分数就是(n-1)!
代码：

```cpp
#include <cstdio>
#include <algorithm>
#include <cmath>
using namespace std;
double e=exp(1);
int main(void){
    int t;
    scanf("%d",&t);
    int n;
    while(t--){
        scanf("%d",&n);
        long long ans=0;
        for(int i=1;i<n;i++){
            ans+=i;
        }
        printf("%lld\n",ans);
    }
    return 0;
}
```
