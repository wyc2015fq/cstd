#### 1、Nginx Windows 32位系统的支持Windows系统版本

      目前所有的Windows Nginx 32位系统的版本都是基于Cygwin 构建于Windows 32位终极版，它们可以在windows vista、Windows XP、Windows 2000和Windows 7上工作，另外也支持其他相似的版本。（原文：All
 current 32-bit versions of Nginx for Windows are built with Cygwin on Microsoft Windows 7 Ultimate (32-bit) and work on 32-bit versions of Windows Vista, Windows XP, and Windows 2000, Windows 7, and other similar versions.）





#### 2、Nginx Windows 64位系统支持Windows系统版本



      同样的，目前所有的Windows Nginx 64位系统的版本都是基于Cygwin 构建于Windows 64位终极版，它们可以在windows vista、Windows XP、Windows 2000和Windows 7上工作，另外也支持其他相似的版本。（原文：Similarly,
 all current 64-bit versions of Nginx for Windows are built with Cygwin on Microsoft Windows 7 Ultimate (64-bit) and work on 64-bit versions of Windows Vista, Windows XP, and Windows 2000, Windows 7, and other similar versions.)

#### 3、相关描述

     其他Windows版本可能会兼容，但是我们没有测试过。（原文：Other versions of Windows may be compatible, but were not tested.）

     Cygwin被用于构建Windows Nginx，但是运行Windows Nginx对于一个完整的Cygwin
 安装却不是必须的。必要的是Cygwin 的文件中要包含Windows Nginx的安装包。（原文：Cygwin
 is used to build Nginx for Windows, but a full Cygwin installation is not required to run Nginx for Windows. The necessary Cygwin files are included in the Nginx for Windows setup package.）

      从Nginx的0.7.34版本开始，大多数默认的模块都被编译进去了。（原文：From Nginx 0.7.34 forward, most default modules are compiled in.）

      Windows Nginx 不能用于生产，但是却可以用于出于开发目的。（原文：Nginx for Windows should not be used for production, but can be used for development purposes.）
