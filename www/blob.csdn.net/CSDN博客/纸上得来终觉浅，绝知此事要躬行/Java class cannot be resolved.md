# Java class cannot be resolved - 纸上得来终觉浅，绝知此事要躬行 - CSDN博客

很多时候会出现比较奇怪的现象，那是Eclipse的编译问题。

如：自己写的类，在别的文件中被引用报错，maven打包时提示：** cannot be resolved**

如果代码无缘无故报错，那就是Eclipse的问题了，可以选择Project> clean一下项目。
