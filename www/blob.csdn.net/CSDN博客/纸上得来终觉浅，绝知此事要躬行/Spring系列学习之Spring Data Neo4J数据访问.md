# 概述

该项目为Neo4j图形数据库提供Spring Data支持，包括带注释的POJO，SD-Repositories和Neo4j-Template。

Spring Data Neo4j提供了高级功能，可将带注释的实体类映射到Neo4j图形数据库。模板编程模型等同于其他Spring模板，并构建了与图形交互的基础，也用于Spring Data存储库支持。 Spring Data Neo4j是Spring Data项目的核心部分，旨在为NoSQL数据库提供方便的数据访问。

## 特性
-     支持属性图（通过关系连接的节点，每个节点具有任意属性）
-     已注释的POJO实体的对象图形映射
-     广泛的Spring Data Commons Repositories支持，包括带注释和派生的finder方法
-     finder方法的任意结果预测
-     支持Neo4j因果集群的Cypher Graph Quey语言支持基于Neo4j-OGM库支持二进制（bolt），http和嵌入式传输连接到Neo4j可用Spring Boot
