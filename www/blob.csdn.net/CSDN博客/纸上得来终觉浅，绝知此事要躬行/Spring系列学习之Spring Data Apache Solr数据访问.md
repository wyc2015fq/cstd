# 概述

Apache Solr的Spring Data是Spring Data系列的一部分，可以从Spring应用程序轻松配置和访问Apache Solr Search Server。 它提供低级和高级抽象，以便与商店进行交互。

Solr特定功能的派生查询和注释允许轻松集成到现有应用程序中。

## 特性
-      具有多核支持的高级存储库抽象
-      Boost-，Facet-和Highlighting的注释
-      可自定义的类型映射和类型转换
-      Solr模板支持流畅的查询api
-      异常转换为Spring的可移植数据访问异常层次结构

## 快速开始



使用[Spring Initializr](https://start.spring.io/)引导您的应用程序。
