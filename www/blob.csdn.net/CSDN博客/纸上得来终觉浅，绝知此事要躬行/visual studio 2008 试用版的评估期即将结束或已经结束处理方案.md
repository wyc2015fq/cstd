莫名其妙，打开Visual Studio Team System 2008 Team Suite就弹出提示：Visual Studio 试用版的评估期 27 天后结束！
![visual studio 2008 试用版评估期已结束的解决方法 - 小妮子 - 午夜阳光](http://www.mzwu.com/pic/200904/026.gif)
百度了下说是要注册成正式版的就好了，打开"添加或删除程序"，很遗憾，我的名没有出现如下图所示的序列号输入框：

![visual studio 2008 试用版评估期已结束的解决方法 - 小妮子 - 午夜阳光](http://www.mzwu.com/pic/200904/027.gif)

这时需要在[http://cid-30c78d23997c4082.skydrive.live.com/browse.aspx/.Public/%e5%8e%9f%e5%88%9b%e8%bd%af%e4%bb%b6/VS2008](http://cid-30c78d23997c4082.skydrive.live.com/browse.aspx/.Public/%E5%8E%9F%E5%88%9B%E8%BD%AF%E4%BB%B6/VS2008) 去下载一个VS2008的升级插件，以管理员身份运行，输入如序列号PYHYP-WXB3B-B2CCM-V9DX9-VDY8T即可。

注：必须在显示了维护页之后，插件才能检测到需要打补丁的页。

![](https://img-blog.csdn.net/20140325093233828?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvYm9vbnlh/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)


最后完成升级维护。

![visual studio 2008 试用版评估期已结束的解决方法 - 小妮子 - 午夜阳光](http://www.mzwu.com/pic/200904/028.gif)

此法对于 visual  studio  2008 试用版的评估期即将结束或已经结束处理均有效，最容易出现的情况就是不能出现序列框。
