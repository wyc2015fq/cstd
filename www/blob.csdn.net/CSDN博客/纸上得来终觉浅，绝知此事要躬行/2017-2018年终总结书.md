# 2017-2018年终总结书 - 纸上得来终觉浅，绝知此事要躬行 - CSDN博客

      记忆经过时间的打磨都会被遗忘。 历史不能断代，年终必须总结。

      2017-5-30日，从四川XX科技有限公司离职，该公司主要从事物流仓储一类的业务和平台服务，离职原因是公司发不出工资。

      2017-5-31日，开始效力于上海某车联网公司，薪水上是一次飞跃，但是负责的东西也更深入，比如Mycat数据库中间件的二次开发。拿一份工资就要干这份工资匹配的事情，这也是天经地义的。

      2017-6-5日左右，飞上海总部学习和了解业务，在此待了一个多月，然后异地生活习惯主要是吃的不一样，而且住的房间画了鬼符休息不好，受不了了申请先回成都去了在家办公。

      2017-7月，在家办公实现Mycat二次开发功能，研究DevOps、Docker等等。

      2017-8月-9月，负责平台808协议业务接口开发，独立完成前后端，前端采用Vue Element实现，Nginx代理前后端负载均衡，前端与后台业务分离。

     2017-10月，开始研究C#音视频播放技术。

     2017-10-10，双十节那天，我和老婆领证了。

     2017-11月-2018-1月，C# 使用ffmpeg+SDL2实现音视频播放。

     2018-2-4_2018-2-7，休婚假两边办婚宴。下图接上，有重复：

       2018-3月-2018-6月，从事OCX网页端视频继承和接口发布。

       2018-4月_2018-5月，母亲患尿结石，在医院请了一周假陪护（独生子女家庭），一边照顾母亲一边照顾怀孕的老婆，我这么尽心尽力也是希望她将心比心能好好照顾下我老婆，但是她做完手术后就像变了一个人了。

       2018-7月-2018-8月，北京出差，OCX音视频播放北京交通部过检及现场修改和测试。

      2018-8-10日夜23：00，老婆羊水破裂，打去120由天府新区医院送医至成都市第一人民医院，次日10点14、15分别诞下两枚公主，请休产假一周在医院寸步不离照顾妻子。此后因为新生儿需要办理一些手续都是出院后办理的，所以隔三差五请个假，孩子们爷爷奶奶帮不上忙，姥姥姥爷在山东太远了也过不来家里有事。

       2018-11-3日，我们一家四口乘机从成都到郑州，老婆二舅开车到郑州接的我们，在成都没人关心过孩子们和我老婆，也是这辈子最让人心痛的一件事情了，从此我们的人生轨迹完全变了。通过家里对宝宝们和我老婆的态度，我心里也不好受还不如自己入赘女方家里呢。在我们最艰难的时候，仿佛被所有人抛弃了，我甚至绝望到开始不跟家里联系。我爸妈通过电话来的每一句问候，都是那么苍白无力，我只能无可奉告，离成都一个半小时的车程都不来看孩子无论如何都说不过去。也许这就是成长的痛吧，非得要承受的话我希望自己一个人能扛下来，为什么要让我老婆经历这些呢？她一个人远嫁四川，我认为家里应该给予更多的温暖，可是我们家这般作为让我很不理解。小孩出生的时候才7个多月，是早产儿，两个宝宝都住了保温箱，小的住了半个月，大的住了10天，因为没人照顾老婆月子没坐，也没有奶水喂小孩儿，都是靠奶粉。我希望记录下2018的每一个瞬间，因为每一个瞬间都让人痛彻心扉，有些东西不是过去了就过去了，它会成为心里的刺如鲠在喉。我每次面对我老婆，心里就莫名地愧疚，因为我欠她的情这辈子都还不清。在成都照顾宝宝的日子里，老婆照顾孩子经常饭都吃不上，累的要死，我白天要上班，偶尔宝宝发烧什么的中午还得往回赶，这种日子持续了两个多月，这种煎熬只有经历过的才知道。好在，孩子们的姥姥姥爷愿意照顾她们，他们也知道不能因为孩子耽误上班，而我成都的父母哪怕有为我想过一丁点儿也不至于此。北方的父母更多的是考虑孩子们过的怎么样，我们这边似乎都不怎么过问。我家装修房子，家具一件件往家弄而不是一次性搞定，结婚之前家里没问过我这边少什么东西。在我老婆整个怀孕期间，我妈在的时间不超过一个月。生孩子那天，头天晚上就给他们说羊水破了，让他们第二天早点过来，结果人家愣是下午一点钟才到，我老婆剖腹产生完孩子很虚也就没说什么。下午，我爸吃完饭就回去了，仿佛是在看别人家生孩子。因为孩子出生的时候就进了保温箱了，我爸也没再来成都看过她们，所以等孩子送走以后我爸都没见过她们。

       2018年11月工作，实现.net web core 跨平台音视频转发服务在docker的自动化shell脚本环境安装部署。

       2018年12月，互联网的寒冬开始席卷中国互联网行业，正如权力的游戏所说的那样Winter is coming.....，许多公司开始裁员求生。因为我的薪水高，再者公司技术方面倾斜更偏嵌入式了我Java爱莫能助，加之也因为我妈生病和老婆生孩子耽误太多，成了被裁的对象。这一年经历的暴风雨已经够多的了，那就让暴风来得更猛烈些吧，不经历风雨怎能见彩虹。

     2018年12月15日离职，当然是有赔偿的。因为孩子和老婆都到山东去了，我也没心思在成都找工作了，于是元旦过后就到了北京。因为对北京不是很熟，而且薪酬水平方面也不是很了解，生怕自己开高了。还有一点就是，我老婆说如果年前没找到工作就不用回家过年了，这一点更让我觉得不行啊，过年不回去在这里呆着算什么啊。后面有了offer于是果断地去了，虽然离我的要求有一定差距，但是年前不好找工作这是事实而且又是寒冬。为了两个小宝宝能有奶粉吃，也顾不得那么许多了，有事情做总比闲着无所事事要强。

      北漂们，同是天涯沦落人，相逢何必曾相识。 2018年乃多事存亡之秋，然而不死就会更强，2019再出发已是北漂人。之所以选择来北京，有两个原因：第一，离孩子们近；第二，老婆可以在北京参加培训然后就业（她找工作也碰了不少壁了，音乐专业不好找工作）。前面说我们的人生轨迹从此改变了，的确如此，因为我们走出了自己的舒适区。未来只有努力拼搏，奋力向前了。

     来一杯烈酒，敬2019！！！！！
