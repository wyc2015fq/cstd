IDEA连不上Github,连接错误，想要直接提交代码到Github上面我们需要使用Git的命令进行操作。

##### 克隆
`git clone ${your project respositry}`
##### 添加

添加一个文件：
`git add  ${your file that wants to commit}`
添加所有修改文件：

`git add  -A`

##### 提交
`git commit -m "commit message"    //#this can't ignored.`

##### 推送
`git push`至此，我们的代码就都提交到仓库了。
