# 概述

Spring Geode项目Spring Data的主要目标是使用Apache Geode构建高度可扩展的Spring驱动应用程序，以便更轻松地进行分布式数据管理。


Apache Geode的Spring Data为您的Apache Geode应用程序带来了Spring Framework的全部功能。
