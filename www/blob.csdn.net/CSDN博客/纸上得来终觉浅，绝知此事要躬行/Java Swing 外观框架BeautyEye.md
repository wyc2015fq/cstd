# 关于 BeautyEye L&F
|BeautyEye is a Java Swing cross-platform look and feel. 得益于Android平台的NinePatch技术，使得美化Swing变得容易。BeautyEye is free, you can study, learn, even for commercial use.|
|----|

## 兼容性


BeautyEye L&F可运行于java 1.5、1.6以及1.7之上，但推荐至少应运行在[java1.6.0_u10或更新的版本](http://www.java.com/zh_CN/download/)，因这些版本将能带来窗口透明特性，更重要的是Swing的性能提升。另附：[BeautyEye
 L&F兼容性测试结果(仅供参考)](http://code.google.com/p/beautyeye/wiki/Compatibility_test_results).

## 特征
- 

更好的兼容性，可运行于java 1.5、1.6、1.7，SUN的非公开API被移除？木有关系

- 

遵从当前主流审美，与时俱进

- 

跨平台

- 

使用NinePatch技术，用最少的代码实现最满意的外观，Synth、Nimbus都是浮云
