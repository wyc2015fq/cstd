## 题目

整数成绩最大化 

给出一个整数n，将n分解为至少两个整数之和，使得这些整数的乘积最大化，输出能够获得的最大的乘积。 

例如： 

2=1+1，输出1； 

10=3+3+4，输出36。
## 解法

代码

```cpp
#include<iostream>

using namespace std;

int main()
{
    int num,result;
    cin>>num;
    if(num<2)
        result=0;
    if(num==2)
        result=1;
    if(num==3)
        result=2;
    if(num>3)
    {
    int* dp=new int[num+1];

    dp[1]=1;
    dp[2]=2;
    dp[3]=3;

    for(int i=4;i<=num;i++)
    {
        int current_max=-1;
        for(int j=1;j<=i/2;j++)
        {
            if(dp[j]*dp[i-j]>current_max)
                current_max=dp[j]*dp[i-j];
        }
        dp[i]=current_max;
    }
        result=dp[num];

    }
    cout<<result<<endl;
    return 0;
}
```
