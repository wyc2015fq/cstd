Quest Central for DB2

   由于公司业务的需要，前一段时间学习和了解了一下DB2数据库知识。开始时使用的是DB2 V8.0软件，DB2中附带了许多辅助工具，比如：控制中心，开发中心等等。写SQL语句时是在命令编辑器中编辑与执行的，但是感觉不是很好使用，不象SQL SERVER2000中编辑与执行SQL语句简单明了，后来就使用了Quest Central for DB2 v4.9工具来进行DB2数据库的使用与维护十分方便，编辑与执行SQL语句也十分方便。现在就简单介绍一下Quest Central for DB2 v4.9功能。

      一款集多功能于一体的数据库管理工具。该软件在单一图形化界面中，集成了全面的数据库维护、管理和性能优化功能，成为首套在异构环境下，同时管理Oracle、DB2（包括分布式及大型机环境）和SQL Server数据库的领先解决方案。以Quest Spotlight技术为基础，提供一个针对各种数据库功能强大的单一控制界面，包括数据库管理（Database Administration）、SQL调校（SQL Tuning）、效能数据历史记录（Historical Performance Data）、空间管理（Space Management）及效能诊断（Performance Diagnostics）等工具。同时，本产品也提供针对各种数据库环境的知识库，以自动调校的方式协助DB2顺利的管理数据库 。

**Quest Central for DB2 components include:（Quest Central for DB2组件介绍）**
- [Complete Database Administration](http://www.quest.com/quest_central_for_db2/database_administration/database_administration.asp) — Eases the burden of maintaining, migrating and comparing database objects by providing a comprehensive graphical object management solution. Quickly create, drop or alter database objects to accommodate your business and application needs. 
- [SQL Tuning](http://www.quest.com/quest_central_for_db2/sql_tuning/sql--tuning.asp) — Makes SQL optimization easy with a complete SQL tuning lab — a robust environment that lets you work with multiple versions of a base SQL statement to easily improve queries. 
- [Quest SQL Optimizer for DB2](http://www.quest.com/quest_central_for_db2/sql_optimizer/sql_optimizer.asp) (Included with SQL Tuning for DB2 Linux, Unix and Windows) — Offers complete SQL performance optimization with proactive detection of inefficient SQL, automated SQL rewrite, and comprehensive virtual index advice and index impact analysis. Quest SQL Optimizer maximizes application performance and eliminates problematic SQL before users are affected. 
- [SQL Analysis](http://www.quest.com/quest_central_for_db2/sql_analysis/sql--analysis.asp) – Provides you with extensive historical SQL collection capabilities to pinpoint the SQL statements in need of tuning on DB2 UDB for LUW. Easily identify your most costly SQL statement, transaction, application program or DB2 user. 
- [Comprehensive Space Management](http://www.quest.com/quest_central_for_db2/space_management/space_management.asp) — Reorganize and restructure DB2 database growth with our space management capabilities. Avoid poor response time and eventually database failure that can be caused by unmanaged database growth. 
- [Spotlight on DB2](http://www.quest.com/quest_central_for_db2/performance_diagnostics/performance_diagnostics.asp) — Quickly identify and eliminate bottlenecks in your DB2 environment through an innovative graphical display that renders the DB2 process model. Detailed metrics available in the drilldown features provide you with the necessary information to resolve the problem.
