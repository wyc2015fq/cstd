       插入代码片段，比如Python代码，可以使用如下方式：

#+BEGIN_SRC python

def Demo():

   print("org-mode")



#+END_SRC

       这其实是插入了一个宏，关键字python说明嵌入的是python代码。代码片段分别有一个固定的开头或者结尾的标识符。

       这一对儿成对儿的标识符自然是可以自己敲上去，但是还有一个简单的方式：

       <sTAB

       输入上面的三个按键，会自动创建一个成对儿的片段模板。只是值得注意的是，在补全模式下，必须得使得补全功能不退出的情况下才会有自动创建的效果。或许，这算得上是一个小bug。
