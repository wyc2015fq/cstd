         这是Libc中的功能。

         代码如下：

```cpp
#include "stdio.h"
#include "locale.h"

int main(void)
{
    int a = 0xFF;

    a = getenv("Path");
    printf("%d\n",a);
    printf("%s",(unsigned char *)a);

    a = getenv("PATH");
    printf("%d\n",a);
    printf("%s",(unsigned char *)a);

    a = getenv("NOTEXIST");
    printf("%d\n",a);
    printf("%s",(unsigned char *)a);

    return 0;
}
```

         看得出，能够正确获取环境变量。但是有一点需要注意，环境变量的查找并不会区分大小写。

         Getenv函数传入参数是环境变量名称，如果存在则返回一个指向环境变量的字符串的地址。如果查找的环境变量不存在，返回0。
