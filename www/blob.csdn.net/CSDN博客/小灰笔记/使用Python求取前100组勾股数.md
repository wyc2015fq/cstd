      本来想采用穷举试探的方式来做这个算法，后来发现还是有点麻烦。从网络上找来了一种求解方法如下：

      当a为大于1的奇数2n+1时，b=2n^2+2n,c=2n^2+2n+1。实际上就是把a的[平方数](http://baike.sogou.com/lemma/ShowInnerLink.htm?lemmaId=1849728&ss_c=ssc.citiao.link)拆成两个连续[自然数](http://baike.sogou.com/lemma/ShowInnerLink.htm?lemmaId=213727&ss_c=ssc.citiao.link)。

      编写代码如下：

#!/usr/bin/python



for n in range(1,101):

      a = 2 * n +1

      b = 2 * (n** 2) + 2 * n

      c = b + 1

      # check theresult

      if a ** 2 +b ** 2 == c ** 2:

            print("a= %d, b = %d, c = %d" %(a,b,c))
