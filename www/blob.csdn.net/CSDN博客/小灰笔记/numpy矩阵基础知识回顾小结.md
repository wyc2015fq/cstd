简单学习了两个矩阵功能，eye和ones。试了一下相应的功能又查了一下线性代数的书籍，两个函数分别是单位矩阵和全1矩阵。




单位矩阵：

>>> a = np.eye(2)




全1矩阵

>>> b = np.ones((2,2)) * 2




查看矩阵的具体数值

>>> a

array([[ 1.,  0.],

       [ 0.,  1.]])

>>> b

array([[ 2.,  2.],

       [ 2.,  2.]])




矩阵的点乘

>>> a.dot(b)

array([[ 2.,  2.],

       [ 2.,  2.]])

>>> a.dot(b).dot(b)

array([[ 8.,  8.],

       [ 8.,  8.]])




再创建一个3*3的全1矩阵

>>> np.ones((3,3))

array([[ 1.,  1.,  1.],

       [ 1.,  1.,  1.],

       [ 1.,  1.,  1.]])
