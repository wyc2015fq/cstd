# AutoMagic自动化测试平台

AutoMagic 是一个基于WebUI的自动化管理平台。为什么叫AutoMagic呢？因为自动化（Automation）在执行起来的时候是一个很神奇的事情，它可以无人值守的模拟人的操作，就像魔术（Magic）一样。 所以我给她取名叫AutoMagic。
