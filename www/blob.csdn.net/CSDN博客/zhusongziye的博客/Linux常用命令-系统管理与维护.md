#### 系统管理与维护

ls 命令

> 
显示指定工作目录下的内容


语法格式
`ls [选项] [路径或文件]`
选项说明

```
# -a 显示所有档案及子目录，包含隐藏文件
# -d 只显示目录列表，不显示文件
# -l 除文件名称外，同时将文件或子目录的权限，使用者信息等详细列出
# -s 输出每个文件的大小
# -k 以k字节的形式显示文件大小
# -u 文件上次被访问的时间排序
# -t 以时间排序
# -o 显示除了组以外的详细信息
# -x 按列输出
# -r 对目录反向排序
# -q 用 ? 代替不可输出字符
# -m 横向输出文件名，并以“,”作为分隔符
# -S(大写) 以文件大小排序
# -R 列出所有子目录下的文件
# -C 按列输出
# -Q 把输出的文件名用 "双引号 " 括起来
# -F 在每个输出项后追加文件的类型标识符，具体含义：“*”表示具有可执行权限的普通文件，“/”表示目录，“@”表示符号链接，“|”表示命令管道FIFO，“=”表示sockets套接字。当文件为普通文件时，不输出任何标识符
```

`pwd`
cd命令

> 
改变当前工作目录


语法格式
`cd [目录名]`
选项说明

```
# cd [目录]： 切换到指定目录下，注意区分大小写
# cd 或 cd ~ ：返回当前用户默认工作目录
# cd ~[用户名]：返回指定用户默认目录下
# cd.. 或 cd../：返回到上级工作目录
# cd /：返回到根目录
```

date命令

> 
显示或者修改系统时间与日期。只有超级管理员才能使用date命令设置时间，普通用户只能用来显示时间


语法格式
`date [选项] 显示时间格式(以+开头, 后面接时间格式)`
命令选项及说明

> 
-s --set: 设置系统时间
-d --date：显示描述的日期


时间显示格式

```
%H：显示小时，范围 00~23 
%I ：显示小时，范围 01~12 
%k ：显示小时，范围 0~23 
%l ：显示小时，范围 1~12 
%M：显示分钟，范围 00~59
%S：显示秒，格式为 00~59
%p：显示是AM还是PM
%r ：显示时间，（hh:mm:ss）12小时
%T： 显示时间，（hh:mm:ss）24小时
%x：显示年份和日期，格式 mm/dd/yyyy
%X：显示时间的格式，相当于%H:%M:%S组合
%a：显示星期几 
%b或%B：显示月份，%b是简称，%B是全称
%m：显示月份，格式为 01~12
%Z：显示时区
%d：显示一个月的第几天
%D：显示年份和月份，格式 mm/dd/yy，yy = 年份最后2位
%Y或%y：显示年份，%Y显示完整年份，%y显示年份最后2位
%c：显示日期和时间格式，相同与“%a%d%b%Y%r%Z”的组合
```

su 命令

> 
作用是改变用户


语法格式
`su [选项] [用户名]`
选项

```
# -：加载相应用户下的环境变量
# -l：使目前的shell成为改变身份后用户默认的shell
# -c：改变身份运行一个指令后结束
# -m：改变用户身份，但是不改变环境变量
```

who 命令

> 
命令显示目前登录到系统的用户


语法格式
`who [选项] [file]`
选项

```
# -a：列出所有信息
# -b：列出系统最近启动的日期
# -l：列出所有可登陆的终端信息
# -m：仅列出关于当前终端的信息 等同于 who am i 和 who am I
# -q：列出在本地系统上的用户和用户数
# -r：显示当前系统的运行级别
# -s：仅显示名称，终端和时间字段信息 who默认输出，who 和 who -s等效
# -u：显示当前每个用户的用户名，登录终端，登录时间，活动和进程标识
# -T或-w：显示tty终端的状态
```

```
# -h：不显示输出信息的标题
```

`uptime`
last 命令

> 
列出目前与过去登入系统的用户相关信息


语法格式
`last [选项] [-n 显示列数]`
选项

```
# -a：把从何处登入系统的主机名或IP地址显示在最后一行
# -R：不显示登入系统的主机名和IP
# -x：显示系统关机，重新开机以及执行等级的改变信息
# -n<显示列数>或-<显示列数>：设置列出名单的显示列数
# -d：将显示的IP地址转换成主机名称
```

dmesg 命令

> 
显示开机信息


语法格式
`dmesg [选项]`
选项

```
# -c：显示开机信息后，清楚ring buffer信息
# -s：设置缓冲区大小，默认设置8192
# -n：设置记录信息的层级
```

free 命令

> 
显示系统内存状态，包括物理内存、虚拟内存、共享内存和系统缓存


语法格式
`free [选项] [-s (间隔描述)]`
选项

```
# -b：以字节为单位显示内存使用情况
# -m：以MB为单位显示内存使用情况
# -K：以KB为单位显示内存使用情况
# -t：显示内存总和列
# -s(间隔秒数)：根据指定的间隔秒数持续显示内存使用情况
# -o：不显示系统缓冲区列
```

ps 命令

> 
显示系统进程在瞬间的运行状态


语法格式
`ps [选项]`
选项（仅列出常用）

```
# a：显示所有用户进程
# -x：显示所有系统程序
# -u：显示使用者的名称和其实时间
# -f：详细显示程序执行的路径
# -c：只显示进程的名称
# -e：将除内核进程以外所有的进程信息写到标准输出
```

top 命令

> 
提供了对系统处理器状态的实时监控，能够实时显示系统中各个进程的资源占用状况。


语法格式
`top [选项]`
选项(只列出常用的)

```
# -d：指定每两次屏幕信息刷新之间的间隔
# -i：不显示闲置或者僵死的进程信息
# -c：显示进程的整个命令路径
# -s：使top命令在安全模式下运行
# -b：分屏显示输出信息，结合-n选项可以将屏幕信息输出到文档
# -n：top命令输出信息更新的次数，完成后退出
```
