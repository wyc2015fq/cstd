                   今天写到一个交互应用，由于平时使用对话框都是自己自定义设置。但是，突然用到对话框默认的那些图片。我顿时凌乱了！

                   然后在网上就收了一下，在这里总结供需要的朋友： 设置对话框默认图片是这样设置的，请看代码![吐舌头](http://static.blog.csdn.net/xheditor/xheditor_emot/default/tongue.gif)：




**在java代码中是这样使用的**





```java
Java Usage example:

myMenuItem.setIcon(android.R.drawable.ic_menu_save);
```



**在布局xml中是这样使用的**



```java
Resource Usage example:

android:icon="@android:drawable/ic_menu_save"
```
