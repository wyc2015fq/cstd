# 标题
- 
### 主标题:
【语法】
```bash
主标题文字 (空行) ==（两个及以上）
复制代码
```
【示例】
```bash
主标题文字（最高阶标题）
==
复制代码
```
# 主标题文字（最高阶标题）
【说明】相当于H1大纲标题效果，一般在首行应用和表示
- 
### 副标题：
【语法】
```bash
副标题文字 (空行) --(减号两个及以上)
复制代码
```
【示例】
```bash
主标题文字（第二阶标题）
--
复制代码
```
## 主标题文字（第二阶标题）
【说明】相当于H2大纲标题效果
- 
### H1~H6大纲标题:
【语法】
```bash
# (空格)文字
复制代码
```
1-6等级分别在文字前加对应数量#，后面的#为可选，不加效果也一样。
【示例】
```bash
# 大纲标题文字 H1（等同 主标题）
## 大纲标题文字 H2（等同 副标题）
### 大纲标题文字 H3
#### 大纲标题文字 H4
##### 大纲标题文字 H5
###### 大纲标题文字 H6
####### 大纲标题文字 H7（最多6阶）
复制代码
```
# 大纲标题文字 H1（等同 主标题）
## 大纲标题文字 H2（等同 副标题）
### 大纲标题文字 H3
#### 大纲标题文字 H4
##### 大纲标题文字 H5
###### 大纲标题文字 H6
####### 大纲标题文字 H7（最多6阶）
【说明】最多6个等级的大纲标题，后面加上同等数量“（空格）#”效果也一样

```bash

```
【示例】
```bash
_斜体_ 或  *斜体*
复制代码
```

- 
### 粗体：
【语法】
```bash
__粗体__ 或 **粗体** 在文字两边加 双下底线 或 双星号
复制代码
```
【示例】
```bash
__粗体__ 或 **粗体**
复制代码
```

### 删除线：

【示例】
```bash

复制代码
```


# 链接
- 
### 网页链接（内联风格）：
【语法】
```bash
[链接文字](http://链接地址)"（空格）鼠标悬停显示文字（可选）"
复制代码
```
"（空格）鼠标悬停显示文字"非必须项，在pc浏览器中能看到效果，手机移动端因无鼠标效果无法直接体现。
【示例】
```bash
这是去往[本人博客](https://pangao1990.github.io)的链接。
复制代码
```
这是去往[本人博客](https://link.juejin.im?target=https%3A%2F%2Fpangao1990.github.io)的链接。
- 
### 链接的快速定义：
【语法】
```bash
前面文档中引用链接的语法为：
[链接文字][id]
前后id要匹配一致，一般为数字，这样可以在文档的其他任意地方快速插入这个链接。
后面定义链接的语法为：
[id]:（空格）http://链接地址（空格）"鼠标悬停显示文字（可选）"
后面的语法必须在列首并单独一行，适合前期不知道网址，后面再加入的情况。
复制代码
```
【示例】
```bash
这是去往[本人博客][1]的链接。
 
 [1]: https://pangao1990.github.io
复制代码
```
这是去往[本人博客](https://link.juejin.im?target=https%3A%2F%2Fpangao1990.github.io)的链接。

# 列表
- 
### 有序列表：
【语法】
```bash
直接数字加英文句点加空格（最多三个空格）和文字
复制代码
```
【示例】
有序单层列表
```bash
520. 列表项1（第一项数字一定要为“520”）
 201314. 列表项2（其实我前面的标的数字是“201314”）
 20161130. 列表项3（其实我前面的标的数字是“20161130”
复制代码
```
- 列表项1（第一项数字一定要为“520”）
- 列表项2（其实我前面的标的数字是“201314”）
- 列表项3（其实我前面的标的数字是“20161130”
【说明】后面的不必在意数字从小到大的排序，显示时会自动排序，有智能的乱序纠正功能，非常贴心。数字前面加了空格，排序会变成罗马数字形式。
- 
### 无序列表:
【语法】

```bash
文字前面加 星号空格*（空格）或 短横空格-（空格）或 加号+（空格）表示
复制代码
```
【示例】
无序单层列表
```bash
- 列表项1
- 列表项2
复制代码
```
-

- 
列表项2

### 多重列表:
【语法】
```bash
单层列表项缩进1个tab就可以往后面叠加子层的列表
复制代码
```

```bash
复制代码
```

# 代码块
- 
### 行内（内联）代码
【语法】
```bash
使用`内容` 表示行内 代码块。
即 在文字左右使用反引号包含（键盘数字“1”前面的波浪号大写状态时能打出）。
复制代码
```
【示例】
```bash
我是一只`小鸭子`，咿呀咿呀哟。
复制代码
```
我是一只`小鸭子`，咿呀咿呀哟。
【说明】和`标签/关键词`的效果差不多，还可用于如按键`Ctrl+F4`，字母`B`，单词`F**K`、短句`Are you ok？`等诸多行的应用场景。能起到强调区分的作用。
- 
### 段落文本代码块:

```bash
复制代码
```
【示例】

```bash
(tab)这是一个代码块。
复制代码
```
