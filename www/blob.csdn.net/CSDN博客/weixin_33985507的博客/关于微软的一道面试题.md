原题是：求 1 + 2 + 3 + ... + n
 要求：不能用乘除法，不能用if,while, switch,case, 不能用(a < b ? a : b)三目运算符
关键点：利用&&运算的短路性质以及递归
关键代码：
```cpp
int fun3(int n, int *sum)
{
	int ret;
	ret = n&&fun3(n-1, sum);
	return (*sum += n);	
}
```
