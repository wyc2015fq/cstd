**工欲善其事，必先利其器**，苹果发布会即将来临，今天把我平时使用频率比较高的软件做一个汇总，不妨先看一看我的利器，也许对你有所帮助。
**1. Xcode 10.0 beta**
![2717693-5e12fe1fe760cade.png](https://upload-images.jianshu.io/upload_images/2717693-5e12fe1fe760cade.png)
image.png
吃饭工具，iOS 开发必备，目前使用的是**最新 beta 版**，支持暗黑色主题，使用舒服，一般是使用 beta 版开发，正式版提交构建版本，Android 开发工具 Android Studio 就不说了。
**2. Google Chrome Canary**
![2717693-075a36092789391c.png](https://upload-images.jianshu.io/upload_images/2717693-075a36092789391c.png)
image.png
Google 推出的一个特别版 Chrome，都称它为金丝雀版，其实就是**开发者版本**，使用一段时间后我觉得它的加载页面速度更快。
**3. iTerm2**
![2717693-291aab3478cc9ca1.png](https://upload-images.jianshu.io/upload_images/2717693-291aab3478cc9ca1.png)
image.png
绝对是 Mac OS 的**终端利器**，之前我一直使用的是 Mac OS 自带的终端，但系统自带的主题太有限，所以发现这款软件，不仅强大而且支持主题配置，主题列表： [https://github.com/robbyrussell/oh-my-zsh/wiki/themes，你可以挑选自己喜爱的主题进行配置。](https://github.com/robbyrussell/oh-my-zsh/wiki/themes%EF%BC%8C%E4%BD%A0%E5%8F%AF%E4%BB%A5%E6%8C%91%E9%80%89%E8%87%AA%E5%B7%B1%E5%96%9C%E7%88%B1%E7%9A%84%E4%B8%BB%E9%A2%98%E8%BF%9B%E8%A1%8C%E9%85%8D%E7%BD%AE%E3%80%82)
**4. VSCode**
![2717693-0451baa58d7507d0.png](https://upload-images.jianshu.io/upload_images/2717693-0451baa58d7507d0.png)
image.png
VS Code 全称 Visual Studio Code，是微软推出的一款免费的、开源的、高性能的、跨平台的、轻量级的代码编辑器，号称微软最好的开源软件作品。
VS Code 有一个扩展和主题市场，包含了数以千计质量不同的插件，它提供了 **Windows、macOS 和 Linux**版本，内置支持**JavaScript、TypeScript 和 Node.js**，通过丰富的扩展和插件生态，你可以使用 VS Code 高效的编写 **C++、C#、Java、Python、PHP、Go** 等主流编程语言。
在此之前我用 Sublime Text，现在我用 VS Code。
**5. SourceTree**
![2717693-7c09301d8323dc8f.png](https://upload-images.jianshu.io/upload_images/2717693-7c09301d8323dc8f.png)
image.png
免费的 Git**图形化管理工具、项目版本控制软件**，适用于 git 项目管理，使用它管理代码，基本够用。
**6. Sequel Pro**
![2717693-243d010fcb20676b.png](https://upload-images.jianshu.io/upload_images/2717693-243d010fcb20676b.png)
image.png
Mac 上的**MySQL 管理工具**，是我经常使用的工具，它支持包括索引在内的所有表管理功能，支持 MySQL 视图，可以同时使用多个窗口来操作多个数据库 /表。
**7. ForkLift**
![2717693-2c5e0246d9bea330.png](https://upload-images.jianshu.io/upload_images/2717693-2c5e0246d9bea330.png)
image.png
**远程 /本地文件管理利器**，我更看重它的本地文件管理器，很强大，平时配合 FileZilla 使用。
**8. Eagle**
![2717693-35052b976c497332.png](https://upload-images.jianshu.io/upload_images/2717693-35052b976c497332.png)
image.png
平时自己设计的内容太多，正好用它解决大量图片素材**「收藏、整理、查找」**的各种困扰,让你可以轻松管理各种图片，提升工作效率。
**9. Notion**
![2717693-79c219d3ac079560.png](https://upload-images.jianshu.io/upload_images/2717693-79c219d3ac079560.png)
image.png
是一款**「将笔记、知识库和任务管理无缝整合的协作平台」**，它是知识库、任务管理的神器，平时使用它轻松管理自己的 App 开发进度和任务。
**10. bear**
![2717693-4c13b50d571a1cb7.png](https://upload-images.jianshu.io/upload_images/2717693-4c13b50d571a1cb7.png)
image.png
颜值即正义，它是一款可以快速提升你**写作质量与效率的神奇工具**，在设计上精致与易用，功能上强大却不失便捷，满足了我对「记笔记」的设想和需求。
**11. MacDown**
![2717693-8d95b3ef2fabb2cc.png](https://upload-images.jianshu.io/upload_images/2717693-8d95b3ef2fabb2cc.png)
image.png
它是一款**MarkDown 编辑器工具**，提到 markdown 工具市面上不胜枚举，使用它的主要原因在于它的轻量、简洁。
**12. OmniGraffle**
![2717693-458faa638cdae3b6.png](https://upload-images.jianshu.io/upload_images/2717693-458faa638cdae3b6.png)
image.png
它是一款功能**强大的绘图软件**。可以用来绘制图表，**流程图**，组织结构图以及插图，也可以用来组织头脑中思考的信息，组织头脑风暴的结果，绘制心智图，作为样式管理器，或设计网页或 PDF 文档的原型，我使用它的主要用途是绘制流程图。
**13. Things**
![2717693-cf19e148c44b61a5.png](https://upload-images.jianshu.io/upload_images/2717693-cf19e148c44b61a5.png)
image.png
Things 是去年开始使用的，设计简洁大方，交互方式自然流畅，用它来管理自己的**待办事项**甚是舒心。
**14. MAMP**
![2717693-7008b0c0c8eedb37.png](https://upload-images.jianshu.io/upload_images/2717693-7008b0c0c8eedb37.png)
image.png
它是一款很棒的在 MAC 下运行的**网站集成环境软件**，功能强大，配置简单，PHP 本地调试我一般都是使用它。
**15. Postman**
![2717693-2b05fd78038e6ba1.png](https://upload-images.jianshu.io/upload_images/2717693-2b05fd78038e6ba1.png)
image.png
它是一个很强大的**API 调试、Http 请求的工具**，允许用户发送任何类型的 HTTP 请求，例如 GET、POST、HEAD、PUT、DELETE 等。
**16. Magnet**
![2717693-6835c82d6158f540.png](https://upload-images.jianshu.io/upload_images/2717693-6835c82d6158f540.png)
image.png
它是一款 Mac 上优秀的**窗口大小控制工具**，每当你想将内容从一个应用程序复制到另一个应用程序、并排比较文件或任何其它方式的多任务时，你需要所有窗口都能相应地排列。Magnet 让这一流程变得整洁便利。你只需一次拖动窗口到边沿，就能将任何窗口捕捉到你的左、右、上或下半个屏幕中。
**17. iHosts**
![2717693-b6dee15b617defd1.png](https://upload-images.jianshu.io/upload_images/2717693-b6dee15b617defd1.png)
image.png
这也是我经常使用的软件之一，如果你需要**修改 /etc/hosts 来切换开发环境**，甚至已经在用脚本、工具来编辑 Hosts，你真的需要试试 iHosts。
**18. 小历**
![2717693-802deb4d20ae2625.png](https://upload-images.jianshu.io/upload_images/2717693-802deb4d20ae2625.png)
image.png
小历是 macOS & iOS 平台上**小而美的日历**，macOS 上通过菜单栏和通知中心让用户查看日历十分便捷，我也是近期才发现，很好用。
**19. [pap.er](http://pap.er/)**
![2717693-188dbb6d85bc608b.png](https://upload-images.jianshu.io/upload_images/2717693-188dbb6d85bc608b.png)
image.png
自从换了新 Mac，一直在找一款**桌面壁纸软件**，直到发现了 [pap.er](http://pap.er/) ，小而精，每天享受来自全球新鲜精美的壁纸。
**20. Easy New File**
![2717693-61768bd578c3a20a.png](https://upload-images.jianshu.io/upload_images/2717693-61768bd578c3a20a.png)
image.png
它是一款小巧的 **Finder 扩展应用，可以为 Finder 右键菜单中添加「新建文件」**、「拷贝路径」、「打开终端」等功能选项，其中「新建文件」一项可支持许多格式文件的创建操作，比如纯文本、富文本、Word/Excel/PPT、Markdown、HTML、XML 以及各类编程脚本，我也是最近才使用，觉得不错。
**21. 幕布**
![2717693-8f1a8e605c35532f.png](https://upload-images.jianshu.io/upload_images/2717693-8f1a8e605c35532f.png)
image.png
极简大纲笔记 | 一键生成思维导图，它是一款**超好用的思维导图辅助工具**，大纲笔记与思维导图可以一键转换。
**22. iMazing**
![2717693-bfaedb7026da913d.png](https://upload-images.jianshu.io/upload_images/2717693-bfaedb7026da913d.png)
image.png
iMazing 是一款比**iTunes 好用很多很多的 iOS 设备管理软件**，你可以用来进行备份数据、备份应用（应用数据）、传输文件、数据迁移，甚至可以备份应用，就算下架也能再次安装。
**23. Sip**
![2717693-ff3b16b4b750a384.png](https://upload-images.jianshu.io/upload_images/2717693-ff3b16b4b750a384.png)
image.png
Mac 上一款**清爽简单的颜色选择工具**，它可以帮助你提取任意颜色，屏幕上的任何地方都可以获取像素颜色编码。
**24. Paste**
![2717693-5ba07171a4105d02.png](https://upload-images.jianshu.io/upload_images/2717693-5ba07171a4105d02.png)
image.png
Mac 平台上**简洁好用的剪贴板工具**，软件拥有如原生般美观的界面和各种好用的功能，足够应付你日常对于剪贴板工具的各种需求。
能够自动识别你复制的内容格式，包括文本、图像、链接、文件甚至是其它任何类型的格式，软件都可以帮你保留它们。
**25. IINA**
![2717693-77e209cfdc7111d5.png](https://upload-images.jianshu.io/upload_images/2717693-77e209cfdc7111d5.png)
image.png
最有可能变得完美的**macOS 「视频播放器」**，它是一款开源的视频播放器，支持 Touch Bar、兼容 MPV 脚本、几乎支持所有格式、网络播放、拥有优雅外观的视频播放器。
**26. GIF Brewery 3**
![2717693-bb759168c1f97ef9.png](https://upload-images.jianshu.io/upload_images/2717693-bb759168c1f97ef9.png)
image.png
是 Mac 上**超好用的动图制作应用**，支持从视频中截取片段制作动图，还支持调节长宽尺寸，制作出来的动图体积小画质高。
GIF Brewery 3 可以通过导入视频、从摄像头拍摄视频、桌面屏幕录制、网络视频链接四种方式获得需要制作 GIF 的视频源。
**27. Telegram Desktop**
![2717693-256bd62b345d6e76.png](https://upload-images.jianshu.io/upload_images/2717693-256bd62b345d6e76.png)
image.png
是一个**跨平台的实时通讯应用**，它大概是国外最有逼格的聊天软件，专为聊天而生，喜欢暗黑的主题。
**28. 微信多开**
![2717693-6b8bf9d059193965.png](https://upload-images.jianshu.io/upload_images/2717693-6b8bf9d059193965.png)
image.png
**WeChatTweak 是一款微信客户端插件**，仅支持 macOS，可实现阻止消息撤回、消息列表通知、系统通知、正常撤回自己发出的消息、**客户端无限多开**，具体的实现教程可自行 google 查看，地址： [https://github.com/Sunnyyoung/WeChatTweak-macOS。](https://github.com/Sunnyyoung/WeChatTweak-macOS%E3%80%82)
另外一种方法使用**weweChat**，下载地址： [https://github.com/trazyn/weweChat](https://github.com/trazyn/weweChat)
**29. HapticKey**
![2717693-f49fbc56713f5754.png](https://upload-images.jianshu.io/upload_images/2717693-f49fbc56713f5754.png)
image.png
可以为你的**Macbook Pro 带来 Touch Bar 触控反馈的工具**，HapticKey 运行后默认不会自动启用，你需要去开启它的反馈功能，除此以外，你还可以使用音效和屏幕闪烁反馈，我使用后觉得效果不好，音效和屏幕闪烁还是别启用了。
