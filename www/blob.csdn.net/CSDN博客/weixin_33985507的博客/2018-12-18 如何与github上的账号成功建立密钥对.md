接下来我们就可以在github上建立仓库，然后按照github给的指示构建项目
```
echo "# learning_pytorch" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin git@github.com:linnabrown/learning_pytorch.git
git push -u origin master
```
