```
int a,b,c;
__asm {
    
    mov eax,10
    mov a,eax
    add a,5
}
printf("%d",a);
```
__asm 是在C中插入汇编指令
mov eax ,10 是指 在寄存器eax 中生成 10常量
mov a, eax 是指将寄存器中的10赋值给变量 a
add a，5是指将变量a加5操作  最后a = 15
# 变量的赋值是先将常量从代码区中的常量符号表取值在寄存器eax中生成赋值给内存中的变量
