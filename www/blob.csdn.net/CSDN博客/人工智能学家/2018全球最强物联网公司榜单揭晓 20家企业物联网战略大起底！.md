IDG旗下杂志《NetWork World》近期公布了全球最强物联网公司名单。本文根据入选评语，对20家企业战略布局进行搜集整理，供业内人士参考！
根据Gartner预测，到2020年将有超过200亿台联网设备。全球第二大市场研究MarketsandMarkets表示，市场将从去年的1700亿美元，到2020年增值至5000亿美元。那么，谁将成为这个巨大和不断增长的市场中最大的玩家呢？
IDG是全球著名的信息技术、电信行业和消费科技市场咨询、顾问和活动服务专业提供商，其下属的《NetWork World》杂志近期公布了全球最强物联网公司名单，亚马逊、谷歌、ARM、高通、华为等位列其中。
具体公司名单如下（公司按字母顺序列出）：Amazon Web Services、ARM、AT&T、Ayla、Bosch、Cisco、Dell、Fujitsu、GE、Google、Hitachi、Huawei、IBM、Intel、Microsoft、Particle、Qualcomm、Samsung、SAP、Siemens.
**—  01  —**
**Amazon Web Services**
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/l75P11FFuiaXBILwtpHfwf5qWhgI9ia3rHbIYxicLxyoXPtLSMp6ziby572tnlZxNu824kD7vjguFuia3kUbZtNwbOQ/640?wx_fmt=png)
当一个项目足够复杂时，能够从中抽象出大部分的细节是一种非凡的能力。在2015年，亚马逊就开始了一系列的物联网服务，包括核心的物联网互连层，通过AWS Greengrass分析决策，并支持一个简单的按钮就可以快速订购快要用完的生活消耗品等。
AWS即Amazon Web Services，是亚马逊（Amazon）公司的云计算IaaS和PaaS平台服务。AWS面向用户提供包括弹性计算、存储、数据库、应用程序在内的一整套云计算服务，能够帮助企业降低IT投入成本和维护成本。AWS提供了一整套基础设施和应用程序服务，使几乎能够在云中运行一切应用程序：从企业应用程序和大数据项目，到社交游戏和移动应用程序。
其中，AWS IoT 解决方案是一个全托管的云平台，使互联设备可以轻松安全地与云应用程序及其他设备交互。AWS IoT 可支持数十亿台设备和数万亿条消息，并且可以对这些消息进行处理并将其安全可靠地路由至 AWS 终端节点和其他设备。AWS IoT 平台支持客户将设备连接到 AWS 服务和其他设备，保证数据和交互的安全，处理设备数据并对其执行操作，以及支持应用程序与即便处于离线状态的设备进行交互。
**—  02  —**
**ARM**
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/l75P11FFuiaXBILwtpHfwf5qWhgI9ia3rHE5kibibib0NPwSeJRsktcwjIoET4WPB9M3h8qQPicK5Wtg7nicGicbiaG9U3A/640?wx_fmt=png)
作为一家主要的半导体厂商，一个系列新的计算设备投入市场，对于ARM来说是一个好消息。为此，ARM还推出了一款名为“Project Trillium”的机器学习平台，旨在为物联网部署提供机器学习能力。
近期，Arm宣布了其ProjectTrillium项目，这是一套包括新的高度可扩展处理器的Arm IP组合，这些产品可以提供增强的机器学习(ML)和神经网络(NN)功能。当前的技术产品主要针对移动设备市场，将让全新的搭载机器学习功能的设备具有先进的计算能力，包括最先进的目标检测功能。
现在的机器学习技术通常是针对特定的设备类别或某个市场领域的需求。Arm的ProjectTrillium则通过提供最佳的可扩展性来改变这一点。虽然最初的发布是针对移动处理器的，但未来的Arm 机器学习产品系列将会按照性能需求满足不同应用场合的需求，包括从传感器、智能音箱，到移动设备、家庭娱乐以及其他领域的应用。
对于ARM在物联网上布局，可追溯到多年以前。早在2013年，ARM公司收购了芬兰物联网软件创业公司Sensinode，将继续推广和深化Sensinode公司的NanoStack、NanoRouter 和 NanoService三大产品。针对此消息，有媒体评论道：“看来手机和平板市场的统治地位还不能满足 ARM 的胃口，吞下整个物联网才是它的目标。
做为全球领先的IP供应商，近来意识到必须深耕新的物联网市场，特别是因为ARM的主要产品线便是处理器IP系列。事实上，ARM允许其客户针对物联网应用自由开发任何芯片（只要他们支付授权许可费和版税）。不过现在，ARM的市场营销人员做了大量认真的工作，从他们的角度定义物联网市场，并基于ARM Cortex-M处理器系列提出了针对物联网应用的解决方案。
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/zJgiblIeQ3uiayEpsWk8ViaUynwrUpXFvfQ5CQia7JaQoSHwWkYsibibJ3sicp4Hk3Bux3CtWDROiay78ZeJ3lqexXb96w/640?wx_fmt=png)
这是一种新型生态系统的构建，显然ARM深喑此道，甚至可以说，正是ARM公司在半导体行业内引入了生态系统的概念！一个生态系统的建立需要多个合作伙伴的参与，ARM定义了三种类别的合作伙伴：云端（提供服务）、合作伙伴（提供开发工具）和提供技术的硅合作伙伴。这三组成员构成了mbed合作伙伴生态系统（实际上，mbed是ARM为支持物联网应用而开发的操作系统）。

**AT&T**
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/l75P11FFuiaXBILwtpHfwf5qWhgI9ia3rHhWcRwMXQkcf65knQ5qKRFdiaVpz41thRfImxT67JXicGibvvJOvJnED3g/640?wx_fmt=png)
联网是“物联网”中非常关键的一部分。AT&T和其他主要无线运营商都希望他们的网络是大部分物联网业务的基础，特别是那些有需要进行远距离通信和多个站点之间的通信的客户。
10多年前，当苹果推出第一款iPhone手机时，其传感、定位、万物连接功能让AT&T意识到未来通信业将发生巨大变化，传感、物联将是未来发展的重点领域。为此，AT&T结合自身注重科研的传统，投入较多资源提前布局物联网、传感等新技术和新应用研究，在下一波科技浪潮来临前未雨绸缪，为核心科研能力掌控、创新突破做好准备。
目前，在美国的物联网市场，AT&T占市场份额高达43%，在全球财富1000强中，99%的企业正在应用AT&T为其提供的物联网服务。AT&T拥有科研实力很强的AT&T领先技术实验室、AT&TFoundry创新中心和AT&T车联工作室等机构。其中AT&T领先技术实验室的任务是针对当前行业、市场等的难点问题，通过发明颠覆性技术解决并推动未来的创新，其研究项目主要包括创建尖端先进的技术，用物联网建设中的数据分析、云服务、智能系统等对网络和服务质量进行管理。

**Ayla**
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/l75P11FFuiaXBILwtpHfwf5qWhgI9ia3rHm9bhodia7fn8YG2OBYciaBRNJKOMov3FwUVF0flt1YcvU2CcTxDtBsUQ/640?wx_fmt=png)
Ayla 是一家创业公司，但它已经拥有100个“大企业”的客户。他提供的敏捷型物联网平台，提供物联网数据的收集、采集和分析，便于企业应用于现有的作业流程。
Ayla敏捷物联网平台主要包含三大组件：Ayla嵌入式模块、Ayla云服务、以及Ayla应用程序库。这是一套完整的解决方案，让几乎任何设备都能够连接到云端和App；还包含了优质的工具和服务，让制造商可以对设备进行管理、配置以及分析。Ayla让制造商在打造物联网产品时，无需投资到网络堆栈开发、端到端安全、云服务以及其他基础技术工作，而是将资源投入到他们所擅长的专业领域中。
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/zJgiblIeQ3uiayEpsWk8ViaUynwrUpXFvfQEcZpHGwicWGWFVQ7s4CyVabeJiblM7xovyVhHtPTibsNt2ibDkcib1EAXicg/640?wx_fmt=png)
无论是之前发力智能家电领域，帮助制造商将产品接入云端，还是随着5G商用来临，和运营商一起协作深入更多垂直领域，Ayla 始终以服务为导向的架构能够灵活地适应市场需求，确保海量设备的安全畅通连接和管理，激活设备数据中蕴藏的巨大价值。

**Bosch**
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/l75P11FFuiaXBILwtpHfwf5qWhgI9ia3rHyY56STxyicGIUwBVvsnglM5yqtpgmw0dV3y4GibvQlKAhwNGUj8BK3yQ/640?wx_fmt=png)
目前，博世在物联网的浪潮中处于领先地位，在家庭自动化和连接家庭设备方面有许多优势。博世的物联网套件是一种支持云技术的软件平台，可用于开发物联网领域的各种应用。并且套件服务已与博世物联网市场集成一体，用户通过它能构建大量的物联网解决方案和项目。
博世物联网套件基于开放标准和源代码，是一个的灵活物联网平台。其云服务满足物联网项目的典型要求，软件开发人员可以在实施之前快速设置原型应用程序，在几分钟内完成部署，并在高可用性的情况下运行程序。此外，该平台的设备管理功能也可以单独使用或现场使用。博世市场领先的基于OSGi的ProSyst网关软件可广泛用于不同的领域，实现基于网关的众多物联网场景。
从2008年到2017年，在近十年时间里，博世软件创新共设计、开发并实施了250多个国际大型物联网项目，覆盖了交通、智慧城市、能源、制造业、农业、健康、智能家居和楼宇等领域。

像许多IT巨头一样，戴尔正在IT中间添加一个“O”，公司自2017年10月以来专门成立了物联网部门，并承诺花费10亿美元在研究新的物联网产品和解决方案上。戴尔的产品聚焦在所谓的“分布式核心”模型，将计算功能被从数据中心推到边缘设备上。
戴尔现有产品组合中的产品已经适用于物联网部署，或者即将用于物联网部署。例如，Dell EMC PowerEdge C系列服务器已经更新了批量训练和机器学习以支持这种分布式核心。Isilon和Elastic CloudStorage可用于保存物联网网络需要的大量存储，以及来自戴尔Pivotal子公司的Cloud Foundry平台即服务，可用于构建基于云的分析应用以支持智能设备的大型网络。
此外，在戴尔公布物联网战略时，还公布了针对物联网的开发项目：
> 
**-Project Nautilus：**一个实时分析和流式存储包，结合了开源Pravega软件定义存储、用于流分析的Apache Flink和戴尔PowerEdge服务器
**-Project Fire：**是VMware Pulse物联网产品线的超融合组件，可通过简化管理、本地处理和实时分析加速物联网部署
**-Project Iris：**来自戴尔的RSA Security LLC子公司，将提供针对物联网设备的可见性监控，同时开发检测异常行为的新机制
**-Project Worldwide Herd：**旨在开发一个分析平台，在那些无法整合或者集中的地理分散数据进行机器学习。

**GE**
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/l75P11FFuiaXBILwtpHfwf5qWhgI9ia3rHEibiao3I9F1iapd6MdUaTdjiazpbS6UvPspzVYbHckygaoEaBFbibt8ng5A/640?wx_fmt=png)
GE布局了物联网/工业4.0产业网络的底层，专注于下一代工业过程仪表。公司的产品是一个专业的IIoT Predix平台，可用于各种不同项目的框架。同时，也可以看作将工业生产过程自动化，从制造到预防维护等方面的解决方案。
Predix的主要组件包括：Predix Machine、Predix Cloud、Predix Services。
> 
**PredixMachine：**一个软件层，负责从工业资产收集数据并将其推送到Predix Cloud，以及运行本地应用程序（如边缘分析）。PredixMachine安装在网关，工业控制器和传感器上。
**PredixCloud：**一个全球性，安全的云基础架构，针对工业工作负载进行了优化，并符合卫生保健和航空等行业的严格监管标准。
**PredixServices：**提供开发人员可以用来构建，测试和运行工业互联网应用程序的工业服务。并提供一个微服务市场，开发人员可以发布他们自己的服务，并从第三方获取服务。Predix平台使客户和合作伙伴共同优化其工业业务流程。
近日，通用电气宣布推出基于物联网(IoT)全新的储能平台，首个推出的产品为1.2MW , 4MWh的储能单元(6 m长)。应用GE的Predix平台和Edgecontrols技术，该储能平台通过数据驱动方式帮助业主提高发电系统运行品质，可用于风电、太阳能、热能等可再生能源的补充，支持电网快速有功调节要求，有利于电网稳定，实现智能管控。

**Google**
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/l75P11FFuiaXBILwtpHfwf5qWhgI9ia3rHva2I2QZibKFjptLwEsHTXpTPMYMIo0oAraEwYFlg9gBO6ic0Td7w6RTg/640?wx_fmt=png)
谷歌的云服务平台正努力抓住物联网市场机遇，以其云计算核心产品为中心，谷歌管理的软件平台旨在为几乎所有的物联网项目提供基础服务。
2016年12月，谷歌第一次对外公布了物联网操作系统Android Things的开发者预览版本。新版操作系统将能够支持一系列物联网设备的计算平台。而其前身是2015年5月29日，谷歌I/O开发者大会上，谷歌宣布推出了全新的Brillo操作系统。Android Things除了继承 Brillo 的功能，还加入了 Android Studio、Android SDK、Google Play 服务以及 Google 云平台等 Android 开发者熟悉的工具和服务。任何 Android 开发者现在都可以利用 Android API 和 Google 服务轻松构建智能联网设备。
近期，谷歌云平台（GCP）推出了用于管理连接的电子设备的测试版服务CloudIoT Core，该服务能够让企业轻松安全连接、管理和接收数百万全球分散设备中的数据。
谷歌方面表示其云服务包括Google Cloud Dataflow，Big Query和Cloud Machine Learning Engine，将Cloud IoT Core与竞争对手的产品分开。谷歌的物联网生态系统版图包括，投资Nest、Android Things、Android Wear和Google Home，而本周发布的Google Cloud IoT Core将帮助谷歌跑步进入工业物联网。
Cloud IoT Core的推出将会是Google在物联网领域内与AmazonWeb Services和MicrosoftAzure等对手竞争的关键性变量。

2016年9月底时，SAP 决定拨出 20 亿欧元的专款发展物联网业务，用于投资开发从家居到汽车的所有设备都接入互联网的硬件产品。这20亿欧元将围绕以下三个方面逐步展开布局。
> 
**1.推出 SAP IoT 的产品线**
作为进军物联网的第一步，SAP将推出一个名为“SAPIoT”的产品线，将它把物联网设备收集到的数据和 SAP 的实时数据库 S/4 HANA 打通。SAP IoT将利用机器学习和自家的数据库S/4 HANA对物联网收集到的海量信息进行分析和整合。
**2.收购并购**
在过去的一年里，SAP先是花1.25亿美元买下了大数据创业公司Altiscale，然后又买下了意大利的PLAT.ONE和挪威的FedemTechnology这两家物联网公司。
Altiscale是一家Hadoop云服务提供商，其解决方案让客户无需在自己的数据中心增加大量的新服务器、存储器和网络基础设施也可获得Hadoop服务；PLAT.ONE是一个企业级M2M和物联网应用平台，而Fedem Technology公司则是一家进阶工程分析软件制造者。
SAP计划要把PLAT.ONE与其分析产品SAP HANA云端平台整合，进而推出如设备生命周期管理与链接，以及其他物联网相关服务。至于Fedem，SAP则打算用来强化物联网相关的分析机制。
在未来，这22亿美元的其中一部分将继续用于收购并购，尤其是那些适合SAP物联网战略的小型交易。
**3.物联网业务层面的合作**
除了投资并购和新产品线外，SAP还在物联网业务层面展开一系列的合作。目前，SAP 正在与西门子、戴尔、英特尔、T-Systems 和沃达丰等公司合作。
