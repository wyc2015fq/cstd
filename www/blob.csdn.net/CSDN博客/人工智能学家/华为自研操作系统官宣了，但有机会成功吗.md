# 华为自研操作系统官宣了，但有机会成功吗 - 人工智能学家 - CSDN博客

摘要：近日，据媒体报道，华为余承东在接受德国《世界报》采访时正式承认，华为自研操作系统确实存在，不仅支持手机和PC，而且已经研发了七年！
**“华为确实已经准备了一套自研的操作系统，但这套系统是 Plan B，是为了预防未来华为不能使用 Android 或 Windows 而做的。当然，华为还是更愿意与谷歌和微软的生态系统合作。”**
余承东如是说。
也就是说，早在2012年，美国司法部对华为和中兴通讯进行调查后，华为就开始着手构建自己的操作系统。
虽然之后的每一年业内都会传出华为研发操作系统一事，但华为从来没有承认过。
在余承东确认华为自研操作系统以后，华为官方发言人也对外表示：
**“华为的确有备用系统，但仅在 必要情况下 使用。说实话，我们并不想使用。我们全力支持合作伙伴的操作系统，我们也喜欢这些系统，客户同样很喜欢。Android 和 Windows 仍将是我们的首选。”**
诚然，华为早就有危机意识，所以在七年前就开始研发自由的操作系统。
如今，中美贸易局势愈发紧张，中兴被美国制裁、华为CFO孟晚舟事件已足以引起中国每一个科技从业者的担忧，这时候，华为宣布自研操作系统，也是给自己和外界一个信心。
不过，做操作系统容易，做生态难。前有诺基亚Symbian、三星Tizen、黑莓Blackberry、微软Windows Phone等诸多系统的前车之鉴，万一形势恶化，华为操作系统真能做起来吗？
小编摘取了业内人士的一些评论，希望给大家带来思考：
“华为自己的移动设备开发操作系统，首先将意味着国产操作系统的破局。但面临的最大的问题也是是否能够有友好的开发体系，让移动应用技术工作者在系统内开发出各种应用软件，以支持其获得用户使用的最重要的关键问题。”
——陈国庆（东南大学教授）
“这个事儿必须得分几种角度来看待。先说说从技术实现方向华为开发出新的操作系统完全是没有问题的。但是创新我们就只能拭目以待，现有的国产技术还真的不能强大到可以取代成熟的现有操作系统，内心依然是希望我们国产的核心技术能有创新，甚至是颠覆。如果真的要就此国产系统发行产品，我相信就华为的实力，再给他时间，说不定真的会给国人一个惊喜。
而华为宣布的本质是一个谈判外交的手段而已，大家可以并不那么在意。就是想表明其实我们也是有能力做到不依靠任何合作方的，借此来进一步进行谈判而已。”
——韦海同（北京美宅科技AI算法研究员）
“自建系统主要还是生态体系的构建，手机销售的成功与否与系统生态息息相关。国内厂商在智能机上发力不过十年，一开始大家靠的都是压硬件成本+使用免费系统起家。后续就是各自在系统优化上发力，试图抵消安卓的混乱性。但谷歌也看到了自家系统的弊端，在版本更新中已经越来越统一，越来越规范，越来越像iOS了。这就说明离收费越来越近，留给华为小米的时间也是越来越少了。”
——张旭（看懂经济评论作家）
华为自研操作系统何时初露真容，或许还要看中美贸易局势走向，以及华为是否真的准备好了？
如果华为真的发布自研操作系统，你会支持吗？
