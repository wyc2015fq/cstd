2017年11月1日，北京 —— 微软公司首席执行官萨提亚 ∙ 纳德拉在2017微软技术暨生态大会（Microsoft Tech Summit）上宣布，微软Azure Stack混合云解决方案正式进入中国，与全球硬件合作伙伴思科、戴尔、华为、联想合作推出的集成式系统将于2018年上半年正式商用。Azure Stack是Azure的延伸，能够将Azure云计算的灵活性和快节奏的技术创新带到混合云环境，在满足混合云部署的多样化需求的同时，推进企业业务的数字化转型。AzureStack在中国将助力客户打造基于由世纪互联运营的Microsoft
 Azure的混合云解决方案。

**真正混合应用创新**
Azure 和 Azure Stack 造就了业界唯一的真正一致的混合云平台，提供完全一致的应用开发体验，帮助客户塑造全新的混合云应用场景，并实现了pay-as-you-use根据用量收费的计价模式。我们的客户正在通过使用 Azure 和 Azure Stack 打造基于混合云的应用，如AVID、ABSA银行、ANZ银行、MKI、盛宝银行、斯伦贝谢等。
针对企业服务其客户，以及企业内部的业务应用，Azure和Azure  Stack带来了全新的应用场景：
**边缘和离线环境解决方案**
借助相同的应用逻辑，客户可通过Azure Stack在本地进行数据处理，之后将其转移到Azure进行进一步分析，以满足延迟和连接要求。我们发现有很多的环境都对这样的边缘应用场景有着需求，包括工厂车间、航船和矿井等。
**满足多种监管要求的云应用**
客户可以在Azure中开发和部署应用，并灵活地在Azure Stack上进行本地部署，以满足监管或政策要求，且无需更改代码。很多客户正寻求在不同环境下部署相同应用（如：一个全球性的审计或财务报告应用软件）的可能性，以满足业务和技术要求。Azure能够满足大多数要求，而Azure Stack则能够实现多地的本地部署。盛宝银行（Saxo Bank）正在计划充分利用Azure
 Stack实现灵活部署。  
**本地实现的云应用模式**
客户可以借助Azure web和移动服务、容器、无服务器和微服务架构来更新和扩展现有应用或开发新应用。在基于Azure的云上，和基于Azure Stack的本地环境中，用户可以使用一致的DevOps流程。我们看到了客户对于应用现代化的广泛兴趣，这其中也包括了核心的关键业务应用。日本三井物产公司（Mitsui）目前正使用Azure Stack和Azure规划自己的应用现代化路线图。

**Azure Stack集成系统**
通过与硬件合作伙伴思科、戴尔、华为、联想的合作，Azure Stack以集成系统的形式提供，将于2018年上半年正式商用。客户可以根据自己的实际需求，灵活选择不同硬件设备。所有系统均拆箱即用，并直接由硬件厂商提供端到端客户支持。Azure Stack最少需要由4台服务器组成小型生产系统，并可根据需求进行扩展。
**Azure Stack开发工具包（ASDK）**
ASDK是一个免费的单服务器部署工具，用于试用和概念验证。ASDK即日起可以在网络上下载，用户可以借助该工具包开发原型应用。在ASDK版本中，门户网站、Azure服务、DevOps工具和Marketplace内容与集成系统版本是一致的。因此，使用ASDK版本开发的应用可以部署到多服务器系统上。
