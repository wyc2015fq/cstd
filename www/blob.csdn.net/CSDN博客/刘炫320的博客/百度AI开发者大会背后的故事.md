# 1. 前言
最近几天的百度AI开发者大会正在风头上。我也稍微关注了一下，总的来讲，百度最近也知道处在紧要关头，因此接连发力，围绕着AI，分别创出了DUEROS嵌入式智能平台和Apollo无人驾驶体系。但是毕竟百度搜索出身，自然语言处理性质浓厚，就让我带着大家逛一逛，百度最近发布的有哪些有用的API/SDK吧！
# 2.无人驾驶
无人驾驶无非是现在最热的话题。大家都聚焦在李总的“违规驾驶”上，但是它所体现出来的[Apollo体系](http://apollo.auto/)，以及计划2020年实现无人驾驶的线路图，都表明这个是即将产生价值的，尤其是百度的无人车已经达到了无人驾驶的l4等级，因此即使未来没法做到完全无人化（无人监督）的自动驾驶，但是有人监督的自动驾驶还是完全有可能的（例如飞机和地铁）。
总的来讲，美国高速公路交通安全局（NHTSA）把自动化驾驶汽车分为 4 个级别（L0是纯手工驾驶，不算），分别是 L1 辅助驾驶阶段、L2 半无人驾驶阶段、L3 高度无人驾驶阶段和 L4 完全无人驾驶阶段。
在北汽百度无人车之前，即便是最先进的谷歌无人车亦未能实现L4级全天候全工况条件下的自动驾驶，即摒弃人为干预因素，车辆日常路面行驶完全自动化。
而美国汽车工程师协会（SAE）则把自动驾驶分成六级，从L0一直到L5，在L2、L3之间加了个“有条件的自动驾驶”。
所以无论怎么说，这条路算是崎岖的走了下来，未来如何发展，也是要看社会全体人员（政治、法律、伦理道德）的努力，而不仅仅是我们程序员们的任务。
# 2. DuerOS
[DuerOS](http://developer.dueros.baidu.com/openduer/main/index)这是一个并不太被大众熟悉的平台，它主要是以度秘这样移动式对话AI应用为主的嵌入式应用开发系统。当然我不太熟悉，我们就不多讲这里了，它展示的应用主要在智能音箱、冰箱、电视、手机等，相信未来也可以面向家庭智能中控，控制所有的灯光、门窗和家用电器。
科幻电影中的美好蓝图，就在我们脚下，我们每一个开发人员都在创造者历史和决定着未来。
# 4. AI开放平台
早在6.30号的时候，百度就已经偷偷地更新了它的[AI体系](http://ai.baidu.com/)。总的来讲，丰富了了很多AI应用，尤其是商业情景下的应用。我们这里来逛一逛，有哪些对于我们个人开发者有用的。
首当其冲的是自然语言处理方面的应用，不愧是搜索出身的。首当其冲的自然是[机器翻译](http://fanyi-api.baidu.com/api/trans/product/index)，百度提供了200万字一下免费的API接口，可以解决很大一部分问题。
另外一个就是自然语言处理的核心了，包括分词与词性标注、依存句法分析、DNN语言模型，以及最近非常火的WordEmbedding技术，也就是Word2vec词向量表示也被加入到了API中，另外语义相似度和短文本相似度以及情感分析和评论观点抽取等浅层语义分析也都有了相应的API。
另一大特色就是百度的[UNIT理解与交互技术](http://ai.baidu.com/tech/unit)这也是百度新添加的一个大特色，可以对于对话系统进行快速的理解和分析，能够进行智能客服、机器人、智能汽车等应用上。
当然[语音识别技术](http://ai.baidu.com/tech/speech)就不用提了，百度又新增加了一个叫做[知识图谱](http://ai.baidu.com/tech/kg/schema)的体系，用以在更高层面上理解世界。
当然我也对于舆情分析也有一定的涉及，当我看到了[百度舆情平台](http://di.baidu.com/product/yuqing)时，觉得还是挺有用的，尤其是对于舆情监控的需求的掌握，十分精准。当然这是基于百度多年来对于舆情的了解（百度贴吧、百度搜索、百度知道等产品）。
