# 1. 前言
作为科研工作的重要一环，论文写作是不得不面对的事情。尤其是像我们这种刚开始从事这方面工作的人来说，如何写一篇合格（注意不是优秀）的英文论文，就是一件非常头疼的事情。仅以我个人的观点，谈一谈英文论文写作中需要注意的事情。
# 2. 写作思路
如果能够直接使用英文写作，最好使用英文写作，因为中文中有很多表述不清楚或者表达方式与英文不同的情况，所以如果是先写中文后翻译成英文，有些地方就非常难以翻译。
另外，英语也是一种语言，同样可以理解世界，表达世界，反映世界的，不一定非要使用中文来从事。所以英文好的，通常是使用英文理解，英文表达和英文交互世界的。
# 3. 翻译
如果有专业人员帮助翻译，那当然是再好不过了，不过可能由于翻译质量、领域知识等存在一些问题，因此只能节省翻译时间，但并不能一次成型。
如果是自己翻译的话，除了我们常用的谷歌、百度、有道翻译外，还需要一些英语拼写检查工具，例如[word自带的英语拼写检查工具](http://jingyan.baidu.com/article/39810a238ca8e8b636fda69e.html)。但有可能这仍然不能满足我们的需求，所以这里我额外提供2种帮助英语拼写检查的工具，他们分别是[1checker](http://www.1checker.com/)和[grammarly](https://app.grammarly.com/)。
## 3.1 word英语拼写检查工具
事实上，即使按照上面所述的教程里的方法做了，仍然达不到一个非常满意的效果，word的英语拼写检查工具除了检查语法外，还可以检查风格上的错误，具体的设置如下： 
![这里写图片描述](https://img-blog.csdn.net/20170705100406806?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcXFfMzUwODIwMzA=/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/SouthEast)
前提是你是一篇英文文档，如果你是一篇中文文档，出现的拼写检查和语法时，就不是上面这种设置了。然后点击设置按钮，就会出现语法设置，这时候你就可以根据需要添加你要检查的语法和风格了。加的越多，则规则越强。
## 3.2 1checker和grammarly
1checker是一个第三方的英语拼写检查工具，用过之后感觉还可以。有客户端和word插件版。注册也比较简单，邮箱+密码就完成了。另外一个grammarly是chrome的一个插件，也有桌面版，免费版的功能比较少，付费的比较强大，有兴趣的话可以试一试。
# 4. 小结
一篇文章改个10遍8遍的都是正常的，要端正心态，最终才能够完成你的大作！
