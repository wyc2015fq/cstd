# 1. 字符串的创建
## 1.1 字符串的定义
在Python中，字符串的创建都十分的简单，相比较java等复杂的类型定义，python中对于类型的限制实在是少见。对于这一点，我想有这么几种解释，首先，在python诞生之初，其作者就认为类型并不需要再用显式的方式来进行确定，因为既然都已经经过单元测试，保证了编码的正确性，就不需要再声明一遍了。另外，python的类型并不是没有，而是动态类型，相对于java这种静态语言，python这种解释型语言，都是动态语言，它和大名鼎鼎的LISP语言一样，充满了灵活性，但是牺牲了可读性和运行效率，这也是没有办法的事情，鱼和熊掌不可兼得。

大家可以看到的是，对于一个字符串，它可以有很多种方法：单引号的可以，双引号的可以，甚至是三引号的，也可以，那么它们又有什么区别呢？首先，单引号是python里最基本的字符串，因为系统里默认的字符串就是单引号，从系统的输出就可以看出。但是双引号的作用是什么呢?如果字符串中出现了单引号，我们会像C一样里面出现双引号一样，使用转义字符，但是如果有很多的话，这时候使用双引号可以很方便的让单引号以普通方式出现，但是如果要在字符串中出现双引号，那么是时候该祭出最强大招，三引号！，三引号是不可能在正常的语句中出现的，如果你一定要出现，那么，请使用转义字符吧。

## 3.2 字符串的特殊输入
另外稍微提一下几个小的注意点， 
一、如果是一个长文本，可以使用三引号来让文本分行输入。 
二、如果有路径，则可以

这样，即在路径字符串上使用r，表示是原始字符串，这样不仅仅是路径，包括引号或者其他特殊字符都会原封不动的输出。 
 三、在python2中，如果想输出中文，请 
```

```

# 4. 字符串的常用方法
## 1. find()方法
还记得当初判断成员资格的方法吗？in，没错，就是它，不过它只能判断一个字符存不存在于字符串中，返回的是一个bool值，那么要查找一个子字符串就要用到find()了，返回的是第一个索引值，如果找不到就返回为-1,这个在java等语言中也有提到。这样，我们就可以做一个简易的垃圾过滤器了。只需要拿好字典，然后遍历，即可！当然，我们以后用到布隆过滤器或者是贝叶斯过滤器，那种就很高级了。

## 3. translate()方法和replace()方法
replace()方法我们再熟悉不过了，就是word中的查找并替换功能。但是translate就比较特殊了，虽然它只能处理单个字符，但是可以同时进行多个替换。当它和maketrans函数相结合的时候就特别的强大。
## 4. join()方法和split()方法
这两个方法十分重要，并且互为逆方法。Split在Java和C#中均有，你可以指定一个分割字符串，把字符串分割成一个列表。这个并不多讲，我们接下来讲解join()方法，join方法正好相反，可以把一个列表中各个元素根据你所要求的连接字符连成一个字符串，但是需要注意的是，所有的元素必须是字符串才行。
## 5.  strip()方法
如果你熟悉C#的话，那么这个方法你一定不陌生，他就相当于是C#中的trim()方法，目的就是把字符串首和尾的空格去掉。
当然以上所讲的只是常用的一些方法，字符串的方法是十分庞大的，需要用的时候可以自行查阅即可。
