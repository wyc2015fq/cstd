# 物理学四大神兽，除了“薛定谔的猫”, 你还知道哪几个？ - 算法与数学之美 - CSDN博客
2018年11月01日 21:21:12[算法与数学之美](https://me.csdn.net/FnqTyr45)阅读数：593
物理学是一门研究物质运动最一般规律和物质基本结构的学科。分为纯物理学和多学科物理学，其中的纯物理学又分为：1.经典力学；2.热力学和统计力学；3.电磁学；4.相对论；5.量子力学。
如果有深入学过物理学的朋友应该知道，上面的许多分支都是非常晦涩难懂的知识。
不过在科学家在解释一些难懂的知识时，就会用上一些恰当的比喻。
在科学的历史上，就流传着非常有趣的“物理学四大神兽”。
这篇文章我们就来看看”物理四大神兽“是何方神圣。
#  1. 薛定谔的猫 
这只猫，不管是在物理学教材书，还是平时看一些科普知识文章总会看到，应该有很多人都知道吧。
这只猫把物理学扩展到了一个更大的范围中，不过科学家折腾了快一个世纪，也没有确切的定论。
1935年，奥地利物理学家薛定谔，针对量子力学的边界问题，提出了著名的思想实验"薛定谔的猫"，该问题直指量子力学在微观和宏观的过渡边界，把微观的不确定性带到了宏观世界。
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/951TjTgiabkzSZUkibqiaw2iarRmeYFnsmkTncJr6JSA0A6x5tDR7Rh4FLoSQdY15ew8w39zFY9ITLhUxjkL1c8ywQ/640?wx_fmt=png)
该实验简单地说：一只猫被关在一个密闭无窗的盒子里，盒子里有一些放射性物质。一旦放射性物质衰变，有一个装置就会使锤子砸碎毒药瓶，将猫毒死。反之，衰变未发生，猫便能活下来。
其中概率取决于量子随机。
薛定谔设计了这个可怕的实验来挖苦量子理论荒唐的一面，按照量子论支持者的解释，在打开盒子看猫之前，这只猫非生非死，而是处在典型的量子态，即活与不活叠加的离奇状态。
这个实验不可谓不出名，我记得刘慈欣写的《球状闪电》这本书就运用了类似的理论，甚至加入了自己的理解。
#  2. 芝诺乌龟 
古希腊数学家芝诺（前490年～前425年），针对运动的不可分性提出了著名的"芝诺悖论"，
说：一只乌龟从起点走到终点，要先走完1/2路程，再走完剩下的1/2路程，继续走完剩下的1/2路程……一直重复下去，永远走不完。
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/951TjTgiabkzSZUkibqiaw2iarRmeYFnsmkTFJUprkhxOhPu3UYQKqTibxcoeamHaI2ibQEeiamxAU7wJJxheNibukkAbw/640?wx_fmt=png)
这个故事的延伸，还有阿基里斯追龟等等，本质上都是一样的。芝诺悖论蕴含了微积分的思想。
《庄子·天下篇》中也提到：
> 
“一尺之棰，日取其半，万世不竭。”
芝诺与庄子悖论的区别为芝诺悖论一定时间内行走的距离不变（即速度不变），而庄子时间不变，这段时间里的工作却越来越少（速度越来越慢），可以看出芝诺限制了时间，而庄子的理论可以使时间为无穷大。
#  3. 拉普拉斯兽 
法国物理学家拉普拉斯，是号称"将上帝赶出宇宙"的人，他提出的拉普拉斯方程广泛应用于各个领域。
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/951TjTgiabkzSZUkibqiaw2iarRmeYFnsmkTK1t8bxZibhNXm2fljYYP7Vh6Kv2FV53Eq4NgicsBNKWFevC3LldkjOqA/640?wx_fmt=png)
他还是经典力学的坚定拥护者，也是决定论的支持者，基于经典力学，拉普拉斯提出：如果有一个智者，知道我们宇宙之初，所有粒子的状态，那么他就可以根据所有的定律，推算出宇宙任何时间的状态。
这个智者，就被人们调侃为拉普拉斯兽。
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/951TjTgiabkzSZUkibqiaw2iarRmeYFnsmkTia2jSSp1vLfvDkvf7y6Ze83WwmHAZPXLab8XRt9sGsSnkAVefEc9zdw/640?wx_fmt=png)
#  4. 麦克斯韦妖 
麦克斯韦妖是大物理学家麦克斯韦，在1871年，针对热力学提出的一个假想实验。为了说明违反热力学第二定律的可能性而设想的。
![640?wx_fmt=png](https://ss.csdn.net/p?https://mmbiz.qpic.cn/mmbiz_png/951TjTgiabkzSZUkibqiaw2iarRmeYFnsmkTKxiaUsoa9pS4HicP7sKRlGJQ2OrnleWTDwjQI5oKqL5tQvELAdGRJUrA/640?wx_fmt=png)
当时麦克斯韦意识到自然界存在着与熵增加相拮抗的能量控制机制。但他无法清晰地说明这种机制。他只能诙谐地假定一种“妖”，能够按照某种秩序和规则把作随机热运动的微粒分配到一定的相格里。麦克斯韦妖是耗散结构的一个雏形。
可以简单的这样描述，一个绝热容器被分成相等的两格，中间是由“妖”控制的一扇小“门”，容器中的空气分子作无规则热运动时会向门上撞击，“门”可以选择性的将速度较快的分子放入一格，而较慢的分子放入另一格，这样，其中的一格就会比另外一格温度高，可以利用此温差，驱动热机做功。这是第二类永动机的一个范例。
#  总结 
物理学中除了实用型的知识外，其他的理论型知识大多都是比较晦涩难懂的，科学家能够用上各种比喻来解释这些知识，让本来无聊的物理学变得非常有趣，也不失为一个理科生的“浪漫”。
