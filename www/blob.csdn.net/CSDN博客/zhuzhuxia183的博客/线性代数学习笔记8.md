# 线性相关性
- 
对于向量组来说，线性相关是指：存在不全为0的集合$${c_1,c_2,c_3,……c_n}$$使得
$$ \left\{\begin{aligned}c_1x_1+c_2x_2 + c_3x_3+……+c_nx_n & = & 0 \end{aligned}\right.$$

- 
反之，如果不存在，就说明该向量组线性无关


# AX = 0 非零解：
- 当矩阵A 存在自由变量时，一定有非零解，所以当未知数>方程数时，即，m<n时，一定有非零解，对应着矩阵A <n，即矩阵A 的列向量组线性相关
- 当矩阵A 不存在自由变量时，就没有非零解，即矩阵A 的列向量组线性无关
- 即 当 rank(A) = n 时，即不存在自由变量，即一定线性无关，反之线性无关=>rank(A) = n

# 向量组生成空间，
- 向量组生成空间：是指这些向量的线性组合结果

ps: 向量组线性相关性不确定

# 空间的基
- 向量组生成空间的一组基：

一群线性无关的向量，满足可以构成之前的空间
- 基的两个特点：

1、线性无关

2、他们构成整个子空间
所以可以认为空间的基包含里空间的所有有效信息，且基不是唯一的

# Rn 空间的基
- 由n个向量构成基

这n个向量构成n*n 的矩阵，可知rank(A) = n,基可以化简得到单位矩阵，即满足
$$E A = I$$

所有有 A 是可逆矩阵！- Rn空间基虽然不唯一，但是这些基都有相同的特征:

1、数目: n  => m = r

2、 线性无关 => n = r

所有我们有Rn空间的基对应的矩阵是一个m = n 的满秩方阵
# 空间的维度

空间基向量的个数为空间的维度

# 矩阵的列向量组的线性相关性与 矩阵零空间的关系
- 矩阵A 的秩 为 r,我们有矩阵A 主列数目 = 矩阵A 的列空间维数

矩阵A 零空间维数 = n -r
- 矩阵的列向量组的线性无关=>  r = n,线性有关，则有r<n
