# 矩阵的列空间
- 矩阵的列空间是指：由矩阵的各列进行线性组合的之后形成的空间，这个是满足空间的定义的
- 比如列向量是4维的，所以矩阵的各列构成的列空间是R4的一个子空间
- 那么有，子空间与子空间，即不同子空间，取并，如，过原点直线和过原点平面在不覆盖的情况下，取并之后，不能构成一个空间
- 不同子空间之间取交，是可以，构成一个空间，是由于，逻辑是 KE1 + KE 2是属于第一个子空间，也是属于第二个子空间，所以KE1 + KE2 是属于两个子空间的交，所以两个子空间的交的向量的排列组合 还是在两个子空间的交里面，所以两个子空间的交还是 属于 一个空间
- AX = b  <=> b是属于A 的列空间，即如果b是A 的各列的线性组合时，有解，当b 如果不是A 的各列线性组合时，无解
- 矩阵各列之间，是否是线性相关的，这点也很重要，即是不是缺少某列，不影响矩阵的列空间大小

# 矩阵的零空间
- 定义是指满足 AX = 0 的所有 X 的解，首先这里有一个理解是说，为什么可以认为他的解是一个空间，也就是说X 的所有解的线性组合还是 X 的解。

证明如下：

AX1 = 0 ,AX2 = 0 =>A(KX1 + KX2) = 0- 同时还有，如果AX = b的解，一定不能构成一个空间，当b 不为0时。因为不过原点，记住，空间 一定包含原点

# 总结：矩阵的列空间、零空间是构成子空间的两种手段

1、矩阵的列空间，是直接通过定义，通过矩阵各列的线性组合 构成的 集合，自然满足空间的定义，针对的矩阵的 b

2、矩阵的零空间: 是求 AX = 0 的X 的解，发现 其解所在的集合是满足空间的定义,针对的是矩阵的 AX = 0 的X

3、矩阵的列空间是直接从定义出发，矩阵的零空间是求解之后，发现恰好满足条件

4、一个是有解情况下b构成的集合，一个是指 x 的解
