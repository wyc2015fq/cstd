# 子空间投影引入-向量的投影

我们首先以R2维空间中为例，一个向量投影b到另外一个向量a，含义为： 在a所在的空间中，找到离b最近的点，即垂直点，所以距离 我们也可以认为是b 距离 b到a 的投影之间的误差,对于投影 到的向量p = xa，由于垂直，我们有
$$a^Te=0   $$

$$ a^T(b-p)=0$$

$$ a^T(b-xa)=0$$

$$xa^Ta = a^Tb$$

$$x = a^Tb/a^Ta$$

$$所以有 投影的向量  p = ax= a \times (a^Tb/a^Ta)$$

我们向量e 来记录这个距离值，易知 e与 a空间垂直，成e为误差

这里我们引入投影矩阵的概念，即一个投影矩阵作用到一个向量，结果为投影的向量

由于 $$ p  =  (a a^T/a^Ta) b$$

所以有投影矩阵为
$$ P =  (a a^T/a^Ta),有 p = Pb$$
# 分析投影矩阵P

1、由$$ P =  (a a^T/a^Ta)$$由于 $$a^Ta$$ 是一个数值，在一维情况下。分析 $$aa^T结果是在 a 的各列的一个组合$$

所以 c( P )是在a的列空间里

2、rank( P ) =1,为一列 * 一行

3、对称的，且投影一次 = 投影 n次，即
$$ P^T = P,P^n = P(n>=1)$$
# 转入高维情况下：
- 为什么需要投影？

是在$$应用 AX = b 可能无解的情况下$$我们想要求出近似解。由于b 不在 A 的列空间中，然后我们需要找到A列空间中距离b 最近的向量，即
$$A x^‘  = p,p 是b在A列空间上面的投影$$- 我们继续分析，加深A 列空间是一个二维的空间，有a1,a2 两个基，我们如何来记录这个投影操作，即投影矩阵是多少，投影结果p为多少？

假定$$A ={\left[ \begin{array}{ccc}a1  & a2\end{array} \right ]}$$

有 $$ p = x_1^‘a1 + x_2^‘$$并且由于投影涉及到了垂直操作，寻找正交量，有 b - p 与 a1 是垂直，有 b - p 与 a2 是垂直的，即
$$ \left\{\begin{aligned}a_1^T (b-Ax^‘)=0     \\a_2^T (b-Ax^‘)=0  \end{aligned}\right.$$=>

$$ {\left[ \begin{array}{ccc}a_1^T \\ a_2^T\end{array} \right ]} (b-Ax^‘) = {\left[ \begin{array}{ccc}0\\ 0\end{array} \right ]} $$=>

$$A^T(b-Ax^‘) = 0$$=>

$$A^T e = 0$$

所以有 误差e 位于A 转置的零空间中，即左零空间。左零空间与列空间正交，e 垂直于A的列空间

=>
$$A^TAx^‘ = A^Tb$$=>
$$x^‘ = (A^TA)^- A^T b$$=>

$$p = Ax^‘ = A(A^TA)^- A^Tb$$=>

$$投影矩阵P = A (A^TA)^- A^T$$

研究投影矩阵，有分析和一维的区别，一维是$$P =  (a a^T/a^Ta)$$

1、其次如果A 是可逆方阵，所以有
$$P = AA^-(A^T)^-A^T = I$$

这个说明如果A是可逆的时候，说明这个时候是绝对有解的时候，不需要投影，只需要用一个单位矩阵相乘，即投影过后就是原向量

2、如果不是可逆方阵，就不可以分解
同样满足
$$P^T = P,P^n = P(n>=1)$$

# 最小二乘法

通过最小二乘法来拟合曲线，如（1，1）、（2，2）、（3，2）三个点来拟合一条直线，我们假设 b = C + Dt，所以有
$$ \left\{\begin{aligned}C+ D = 1     \\C + 2D = 2 \\C + 3D = 2\end{aligned}\right.$$

所以有$$ {\left[ \begin{array}{ccc}1 & 1\\1 & 2\\1 & 3\\\end{array} \right ]}\times  {\left[ \begin{array}{ccc}C\\D \\\end{array} \right ]} = {\left[ \begin{array}{ccc}1  \\2\\2\\\end{array} \right ]}$$

这个方程是无解，所以考虑最小二乘法来求解，需要投影- 
如果b在A 列空间中，有Pb = b

证明：
$$由于b在A 列空间中，有b = AX,在Pb = A (A^TA)^- A^TAx $$$$=>pb = Ax$$- 
如果b 在A列空间中，Pb = 0

证明
$$由于b垂直于A列空间中，所以有b在A 的左零空间，所以$$$$ A^Tb = 0，所以$$$$=>Pb = 0$$- 
可以认为我们的投影向量是把原向量b 在列空间和1做零空间两个空间进行分解，有b = p + e,即愿向量 = 投影向量 + 误差向量

有 e = (I-P)b

- 
最小二乘法的思想为：想要计算得到的x 符合最右，即$$ ||AX-b||^2最小，即 ||e||^2最小，即距离最小，所以有垂直时候距离最小，即进行投影的时候是最小的 $$

所以我们求解出来结构为 【1\2,2\3】,所以也可以计算出来e = [-1\6,2\6,-1\6],所以根绝 b = e + p，所以也就可以计算出来投影向量，我们有 e不仅是与 p 垂直（正交）的，而且是垂直于 A 的列空间的，与列向量也是垂直（正交的）
