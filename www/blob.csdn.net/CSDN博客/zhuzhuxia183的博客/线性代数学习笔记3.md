# E1 E2 的逆：

```
E1 E2 的逆 为 E2-1 E1 -1
形象比喻 如果先脱了裤子，再拖袜子，穿的时候，就是先穿袜子，再穿裤子
```

# A 转置的逆 和 A 的逆的转置

首先提到了 如果对于 AA(-1) = I ,通过同时取转置，可以发现A 转置的逆 和 A 的逆的转置是想等的

即 转置、逆两种操作对于单个矩阵来说，顺序是可以变换的

矩阵相乘的转置公式 的理解：

上一节讲矩阵乘法的时候讲了四种解释，包括将行看成整体乘以矩阵以及将列看成整体等。而转置是将行转为列， 那么应该想到，如果要和原来的乘法一样的话，那么矩阵顺序要换一下。 有一种，左乘看行，右乘看列的思想。

# 消元的目的，只是为了更加正确的理解矩阵的概念

```
A = LU 是最基础的矩阵分解
```

当然 EA = U 是进行消元的初始形态，但是我们可以很快的把 EA = U => A = LU,因为我们发现如果没有行变换， L 矩阵可以直接通过矩阵行变化的参数直接写入

消元意味着是 行变换，意味着解线性方程组时，优先考虑行变换是对的，但是线性代数并不是只是行变换
