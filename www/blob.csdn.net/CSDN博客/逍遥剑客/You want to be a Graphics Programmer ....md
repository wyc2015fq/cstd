Author: [Wolfgang Engel](http://diaryofagraphicsprogrammer.blogspot.com)
I regularly receive e-mails with the question what kind of books I recommend if someone wants to become a graphics programmer. Here is my current list (maybe some of you guys can add to this list?):
First of all math is required:
- [Vector Calculus](http://www.amazon.com/Vector-Calculus-Jerrold-E-Marsden/dp/0716749920/ref=sr_1_1?ie=UTF8&s=books&qid=1258320777&sr=8-1)
- [Vector Calculus, Linear Algebra, and Differential Forms](http://www.amazon.com/Vector-Calculus-Linear-Algebra-Differential/dp/0971576653/ref=sr_1_2?ie=UTF8&s=books&qid=1258320878&sr=8-2) I have the 1999 version of this book
- [Computer Graphics Mathematical First Steps](http://www.amazon.com/Computer-Graphics-Mathematical-First-Steps/dp/0135995728/ref=sr_1_1?ie=UTF8&s=books&qid=1258321011&sr=1-1)
- [Mathematics for Computer Graphics](http://www.amazon.com/Mathematics-Computer-Graphics-John-Vince/dp/1846280346/ref=sr_1_2?ie=UTF8&s=books&qid=1258321048&sr=1-2)
For a general knowledge in programming the CPU:
- [Write Great Code Volume 1: Understanding the Machine](http://www.amazon.com/Write-Great-Code-Understanding-Machine/dp/1593270038/ref=sr_1_1?ie=UTF8&s=books&qid=1258321123&sr=1-1)
For a better knowledge on how to program the GPU:
- [DirectX documentation](http://msdn.microsoft.com/en-us/directx/default.aspx)
- [NVIDIA GPU Programming Guide](http://developer.nvidia.com/object/gpu_programming_guide.html)
- [ATI GPU Programming Guide](http://developer.amd.com/media/gpu_assets/ATI_Radeon_HD_2000_programming_guide.pdf)
To learn about how to program certain effects in an efficient way:
- [ShaderX - ShaderX7](http://www.amazon.com/s/ref=nb_ss?url=search-alias%3Dstripbooks&field-keywords=ShaderX&x=0&y=0)
- [GPU Gems - GPU Gems 3](http://developer.nvidia.com/page/home.html)
- [GPU Pro](http://www.akpeters.com/product.asp?ProdCode=4728) and [GPU Pro Blog](http://gpupro.blogspot.com/)
To start learning DirectX 10 API + Shader Programming:
- [Introduction to 3D Programming with DirectX 10](http://www.amazon.com/Introduction-3D-Game-Programming-DirectX/dp/1598220535/ref=sr_1_1?ie=UTF8&s=books&qid=1258330039&sr=8-1)
- [Programming Vertex, Geometry and Pixel Shaders](http://wiki.gamedev.net/index.php/D3DBook:Book_Cover)
To start learning OpenGL & OpenGL ES:
- [Khronos group](http://www.khronos.org/)
For general overview:
- [Real-Time Rendering](http://www.amazon.com/Real-Time-Rendering-Third-Tomas-Akenine-Moller/dp/1568814240/ref=sr_1_1?ie=UTF8&s=books&qid=1258330141&sr=1-1)
- [Fundamentals of Computer Graphics (this one also belongs in the math section)](http://www.amazon.com/Fundamentals-Computer-Graphics-Peter-Shirley/dp/1568814690/ref=sr_1_1?ie=UTF8&s=books&qid=1258330205&sr=1-1)
To get started with C:
- [C Programming Language](http://www.amazon.com/Programming-Language-2nd-Brian-Kernighan/dp/0131103628/ref=sr_1_1?ie=UTF8&s=books&qid=1258330464&sr=1-1)
To learn C++
- [C++ for Game Developers](http://www.amazon.com/C-Game-Programmers-Development/dp/1584504528/ref=sr_1_2?ie=UTF8&s=books&qid=1258331002&sr=8-2)
- [C++ Cookbook](http://www.amazon.com/Cookbook-Cookbooks-OReilly-Ryan-Stephens/dp/0596007612/ref=sr_1_1?ie=UTF8&s=books&qid=1258331060&sr=1-1)
- there is a long list of more advanced C++ books ...
