**工作地点：**北京/深圳/上海




# 移动端视觉算法工程师




**岗位职责**



1. 负责移动端的视觉算法研发。工作内容包括但不限于：目标检测、目标分割、特效迁移、目标跟踪、视频序列结果融合；

2. 负责视频序列算法效果优化，序列结果平滑与稳定。




**岗位要求**



1. 计算机视觉、深度学习、机器视觉等领域的硕士或者博士；

2. 精通 CV、深度学习 CNN/RNN 相关算法，精通一个以上相关开发工具 PaddlePaddle/Caffe/Tensorflow/PyTorch；

3. 熟悉移动端目标检测、目标分割和目标跟踪的模型设计、训练、调优，或熟悉 GAN、style transfer、super-resolution 等端到端图像编辑模型的设计和训练；

4. 熟悉视频序列分析算法者优先（OpticalFlow，FlowNet），熟悉嵌入式模型设计、压缩、裁剪者优先；

5. 熟悉 Linux 操作系统，熟练掌握 C++，熟练掌握一门脚本语言（Python、Shell 等）；

6. 在计算机视觉国际顶级会议或者期刊（CVPR，ICCV，ECCV，NIPS）上发表论文、及有相关专利者优先；

7. 在公开数据集或者国际竞赛，如 WebVision, MSCOCO, VOC, KITTI, Cityscapes 上取得优异成绩者优先；

8. 有较强的语言表达、沟通能力和分析、解决问题的能力。




# Android高级研发工程师




**岗位职责**



1. 根据产品需求完成公司多媒体 SDK 设计及代码编写，单元测试和维护工作；

2. 负责安卓平台 OpenGL 开发；

3. 负责优化多媒体 SDK 的模块结构，流程，逻辑，性能；

4. 集成算法团队的研发并使之实用化及工程化。




**岗位要求**



1. 本科及以上学历，计算机、通信等相关专业； 

2. 具有扎实的编程功底，良好的设计能力和编程习惯； 

3. 3 年以上面向对象程序设计与开发经验，2 年以上安卓实际开发经验，1 年以上图像处理经验或图像处理相关项目； 

4. 精通 openGL 开发，熟练 shader 编写及优化；有实际 openGL 开发经验者优先；

5. 熟悉 Cmake 框架，精通 Java ；熟悉 JAVA 应用开发技术，多线程编程等； 

6. 熟悉常用的设计模式、数据结构及算法；有较好的产品意识； 

7、积极乐观，责任心强，工作认真细致，具有良好的团队沟通与协作能力。
