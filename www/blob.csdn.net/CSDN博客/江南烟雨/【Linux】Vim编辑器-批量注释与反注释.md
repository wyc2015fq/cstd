# vim编辑器---批量注释与反注释
在使用vim编写代码的时候，经常需要用到批量注释与反注释一段代码。下面简要介绍其操作。
## 方法一 块选择模式
### **插入注释：**
用v进入virtual模式
![](https://img-my.csdn.net/uploads/201301/10/1357782623_5116.png)
用上下键选中需要注释的行数
![](https://img-my.csdn.net/uploads/201301/10/1357782673_8455.png)
按Control+v（win下面ctrl+q）进入列模式
![](https://img-my.csdn.net/uploads/201301/10/1357782699_7841.png)
按大些“I”进入插入模式，输入注释符“#”或者是"//"，然后立刻按下ESC（两下）
![](https://img-my.csdn.net/uploads/201301/10/1357782731_5309.png)
### **取消注释：**
Ctrl + v 进入块选择模式，选中你要删除的行首的注释符号，注意// 要选中两个，选好之后按d即可删除注释
![](https://img-my.csdn.net/uploads/201301/10/1357782805_9136.png)![](https://img-my.csdn.net/uploads/201301/10/1357782826_8711.png)
## 方法二 替换命令
### **批量注释：**
使用下面命令在指定的行首添加注释：
:起始行号,结束行号s/^/注释符/g
### **取消注释：**
:起始行号,结束行号s/^注释符//g
### **实例演示：**
在27 - 30行添加 // 注释
:27,30s#^#//#g
![](https://img-my.csdn.net/uploads/201301/10/1357782953_5893.png)![](https://img-my.csdn.net/uploads/201301/10/1357782973_2833.png)
在27 - 30行删除 // 注释
:27,30s#^//##g
![](https://img-my.csdn.net/uploads/201301/10/1357783052_7774.png)![](https://img-my.csdn.net/uploads/201301/10/1357783061_6776.png)
在10 - 20行添加 # 注释
:10,20s/^/#/g
在10 - 20行删除 # 注释
:10,20s/^/#/g
注意例子中正则的分割符使用的是相反的符号，如果匹配// 那么使用 #作分隔符这样不需要对/作转义处理，节省输入次数。
