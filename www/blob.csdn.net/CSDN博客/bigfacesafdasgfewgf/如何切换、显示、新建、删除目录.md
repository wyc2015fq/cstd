    在Linux下，基础的操作包含如何切换目录、显示当前目录、新建一个新的目录、删除一个空的目录。




# 1. 切换目录 cd




. 代表此层目录

.. 代表上一层目录

- 代表前一个工作目录

~ 代表『目前用户身份』所在的主文件夹

~account 代表 account 这个用户的主文件夹(account 是个账号名称)

**注意：根目录的上一层（..）与根目录自己（.）是同一个目录。**




# 2. 显示当前目录 pwd




pwd可以显示当前目录路径，加上-p参数，就可以获取正确的目录，而不是以连接文件的路径来显示的。




![](https://img-blog.csdn.net/20150430082328811?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcHVxdXRvZ2V0aGVy/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)





# 3. 新建一个新的目录 mkdir




mkdir可以新建一个目录.

加了-p参数，可以创建多层目录。例如：mkdir -p test1/test2/test3

加了-m参数，可以设置目录的属性，例如：mkdir -m 711 test2  如果不加-m，那么系统默认755的权限。




![](https://img-blog.csdn.net/20150430082512163?watermark/2/text/aHR0cDovL2Jsb2cuY3Nkbi5uZXQvcHVxdXRvZ2V0aGVy/font/5a6L5L2T/fontsize/400/fill/I0JBQkFCMA==/dissolve/70/gravity/Center)





# 4. 删除一个空的目录 rmdir




rmdir只能删除空目录！

如果有多个目录嵌套，需要在rmdir后面加上-p参数。

rm也有和rmdir类似的功能，rm -r test也可以删除test下的所有目录，-r是递归的意思。
