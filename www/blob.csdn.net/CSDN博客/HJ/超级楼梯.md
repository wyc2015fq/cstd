hdu  2041  超级楼梯
Problem Description 
有一楼梯共M级，刚开始时你在第一级，若每次只能跨上一级或二级，要走上第M级，共有多少种走法？
Input 
输入数据首先包含一个整数N，表示测试实例的个数，然后是N行数据，每行包含一个整数M（1<=M<=40）,表示楼梯的级数。
Output 
对于每个测试实例，请输出不同走法的数量
Sample Input 
2 
2 
3
Sample Output 
1 
2
乍一看，情况蛮复杂的？？？ 
不过要是经过仔细分析就会发现本质：
如果要从一级走到M级，那么你到达M级只能从M-1级或者M-2级上去，因为题目的核心语句就是，你只能一次上一级或者一次上两级。
因此如果设走到M级的方法数为F(M)，那么走到M-1级的方法数为F(M-1)，同理，走到M-2级的方法数为F(M-2)。
于是就有递推公式
```
F(M)=F(M-1)+F(M-2)
```
因为抓住了这一点，所以代码就简单咯。
```cpp
#include<stdio.h>
int main()
{
    int i,j,a[45],n,m;
    a[0]=0;a[1]=1;
    for(i=2;i<41;i++)
        a[i]=a[i-1]+a[i-2];  //核心公式
    scanf("%d",&n);
    while(n--)
    {
         scanf("%d",&m);
         printf("%d\n",a[m]);
    }
    return 0;
}
```
