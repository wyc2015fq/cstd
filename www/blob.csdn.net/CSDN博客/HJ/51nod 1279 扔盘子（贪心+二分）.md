有一口井，井的高度为N，每隔1个单位它的宽度有变化。现在从井口往下面扔圆盘，如果圆盘的宽度大于井在某个高度的宽度，则圆盘被卡住（恰好等于的话会下去）。 
盘子有几种命运：1、掉到井底。2、被卡住。3、落到别的盘子上方。 
盘子的高度也是单位高度。给定井的宽度和每个盘子的宽度，求最终落到井内的盘子数量。 
![这里写图片描述](http://img.51nod.com/upfile/000fbd24/08d196ad7e66babb0000000000000004.png)
如图井和盘子信息如下： 
井：5 6 4 3 6 2 3 
盘子：2 3 5 2 4
最终有4个盘子落在井内。 
本题由 @javaman 翻译。 
Input
第1行：2个数N, M中间用空格分隔，N为井的深度，M为盘子的数量(1 <= N, M <= 50000)。 
第2 - N + 1行，每行1个数，对应井的宽度Wi(1 <= Wi <= 10^9)。 
第N + 2 - N + M + 1行，每行1个数，对应盘子的宽度Di(1 <= Di <= 10^9)
Output
输出最终落到井内的盘子数量。

4

Output示例

贪心维护最小值，得到一个递减数组，然后逆序排一下，当盘子落下来的时候，二分查找一下即可。
```cpp
#include<iostream>
#include<algorithm>
using namespace std;
int a[50005],b[50005],p,q;
int find(int x)          //二分查找 
{
    int l=p,r=q;
    while(l<=r)
    {
        int mid=(l+r)/2;
        if(x<=a[mid])
          r=mid-1;
        else
          l=mid+1;
    }
    return l<=q?p=l+1:0;      //如果查找到了，把区间缩小，并且返回1，否则返回0 
}
int main()
{
    int n,m;
    while(cin>>n>>m)
    {
        cin>>a[0];
        for(int i=1;i<n;i++)
        {
            cin>>a[i];
            if(a[i]>a[i-1])      //维护最小值 
              a[i]=a[i-1];
        }
        sort(a,a+n);        //逆序排列 
        for(int i=0;i<m;i++)
          cin>>b[i];
        int x,cnt=0;
        p=0,q=n-1;
        for(int i=0;i<m;i++)
        {
            if(!find(b[i]))   //二分查找第一个大于或等于x的值  如果查找不到，说明盘子已经无法掉了如井中 
              break;
            cnt++;
        }
        cout<<cnt<<endl;
    }
    return 0;
}
```
