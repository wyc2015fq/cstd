# hdu 2897 邂逅明下（巴什博弈） - HJ - CSDN博客

[http://acm.hdu.edu.cn/showproblem.php?pid=2897](http://acm.hdu.edu.cn/showproblem.php?pid=2897)
必胜点：n%(p+q)的值在(p,p+q)中,或者n%(p+q)==0
剩下的就是必败点了。
```cpp
#include<iostream>
using namespace std;
int main()
{
    int n,p,q;
    while(cin>>n>>p>>q)
    {
        int k=n%(p+q);
        if(k>p&&k<p+q||k==0)
          cout<<"WIN"<<endl;
        else
          cout<<"LOST"<<endl;
    }
    return 0;
}
```
