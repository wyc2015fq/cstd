一个十进制整数被叫做权势二进制，当他的十进制表示的时候只由0或1组成。例如0，1，101，110011都是权势二进制而2，12，900不是。
当给定一个n的时候，计算一下最少要多少个权势二进制相加才能得到n。
Input
单组测试数据。 
第一行给出一个整数n (1<=n<=1,000,000)
Output
输出答案占一行。
Input示例
9
Output示例
9
解题思路：
你会发现1、11、111、1111、。。。因此只要确定每一位上面的数字即可。
```cpp
#include<iostream>
using namespace std;
int main()
{
    int n;
    while(cin>>n)
    {
        int ans=0;
        while(n)
        {
            ans=max(ans,n%10);
            n/=10;
        }
        cout<<ans<<endl;
    }
    return 0;
}
```
