hdu  1870  愚人节的礼物
Description 
四月一日快到了，Vayko想了个愚人的好办法——送礼物。嘿嘿，不要想的太好，这礼物可没那么简单，Vayko为了愚人，准备了一堆盒子，其中有一个盒子里面装了礼物。盒子里面可以再放零个或者多个盒子。假设放礼物的盒子里不再放其他盒子。 
用()表示一个盒子，B表示礼物，Vayko想让你帮她算出愚人指数，即最少需要拆多少个盒子才能拿到礼物。 
Input 
本题目包含多组测试，请处理到文件结束。  
每组测试包含一个长度不大于1000,只包含’(‘,’)’和’B’三种字符的字符串，代表Vayko设计的礼物透视图。  
你可以假设，每个透视图画的都是合法的。 
Output 
对于每组测试，请在一行里面输出愚人指数。
Sample Input 
((((B)()))()) 
(B)
Sample Output 
4 
1
很明显，这个题目用栈来实现。
```cpp
#include<iostream>
#include<stack>
#include<cstring>
using namespace std;
int main()
{
    char a[1005];
    stack<int> s;
    while(cin>>a)
    {
        int len=strlen(a);
        for(int i=0;i<len;i++)
        {
            if(a[i]=='(')      //判断入栈
              s.push(1);
            else if(a[i]==')')    //抵消出栈
              s.pop();
            else
              break;         //遇上B则跳出循环
        }
        int i=0;
        while(!s.empty())
        {
            s.pop();       //循环计算栈里面单括号的个数
            i++;
        }
        cout<<i<<endl;
    }
    return 0;
}
```
