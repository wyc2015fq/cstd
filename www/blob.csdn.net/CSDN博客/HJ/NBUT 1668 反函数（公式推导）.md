Description
已知arctan(1/s)=arctan(2/p)+arctan(3/q);
现在告诉你p和q,求pq-2qs-3ps的值为多少
Input 
第一行是一个整数t(不多于100000组)代表下面有t组数据, 
每组数据两个正整数p与q(不大于2147483648并且p*q!=6)
Output 
输出pq-2qs-3ps的值如果答案有小数保留两位
Sample Input
1 
4 6
Sample Output
6
解析：本题其实考的是数学，考查三角函数的转换。
首先，等式两边加上tan，得到
```
1/s=tan(arctan(2/p)+arctan(3/q));
```
根据正切两角和展开公式展开
```
1/s=(2/p+3/q)/(1-(2/p)*(3/q);
```
化简后得到结果如下
```
1/s=(2*q+3*p)/(p*q-6);
```
两边再取倒数
```
s=(p*q-6)/(2*q+3*p);
```
再合并
```
p*q-2*q*s-3*p*s=6;
```
因此，无论输入的p、q值为多少，输出的结果永远都是6！！！
代码如下：
```cpp
#include<iostream>
#include<cstdio>
using namespace std;
int main()
{
    int t,a,b;
    scanf("%d",&t);
    while(t--)
    {
        scanf("%d%d",&a,&b);
        printf("6\n");      //数学公式得到结果恒为6
    }
    return 0;
}
```
