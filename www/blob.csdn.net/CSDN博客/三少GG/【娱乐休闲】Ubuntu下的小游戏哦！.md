**1.SuperTux第三版—-Planet Penguin Racer **
sagi planetpenguin-racer
![planetpenguinracer.png](http://linuxdesktop.cn/wp-content/uploads/2007/05/planetpenguinracer.thumbnail.png)哈
 哈，可爱的Tux又出场了，这是是玩起了滑雪，这个不光是画面，声音效果，都不错，最关键的是，这个游戏的操作感非常不错，很爽快。速度飞快。游戏本身是 一个 3d 企鹅滑雪游戏，不过这和一般的滑雪游戏不同，他并不是着重于平衡，躲避障碍等，而是着重于 fly——飞行。从很多关卡的名字，到最后的统计中包括了你飞行的时间，无时无刻不在提醒你这一点。
**2。linux下面的百战天虫—-Wormux**
sudo apt-get install wormux
![wormux.png](http://linuxdesktop.cn/wp-content/uploads/2007/05/wormux.thumbnail.png)百战天虫大家都应该玩过或者听过吧，这帮可爱的虫子跑到了Linux下，主角全部变成了Linux里的吉祥物大战了，有BSD的小魔鬼、有Linux的企鹅、有KDE的恐龙、有Firefox的狐狸、有Thunderbird的雷鸟，成了吉祥物大战了。
不过老实说，我还真没有弄得太懂这个游戏，好像没有与电脑对战的模式，还是我没找到？
**.linux下面的雷电—-Chromium**
sudo apt-get install chromium
![chromium.png](http://linuxdesktop.cn/wp-content/uploads/2007/05/chromium.thumbnail.png)鼠标操作、画面优秀、战斗爽快。还等什么，赶紧装了进入战斗吧。
比雷电要进化多了，哈哈。
不过这游戏也太变态了吧，反正我每次玩不到两分钟就玩完。
**4.linux下面的泡泡龙—-frozen-bubble**
sudo apt-get install frozen-bubble
画面可爱，适合和女朋友一起玩，和wormux一样，也是对战不错的选择。
这个Linux版泡泡龙的主角仍然是Linux的吉祥物Tux，不过，嘿嘿，这两个Tux可真像两个呆瓜，哈哈。
**5.拯救Tux**
sudo apt-get install pingus
![pingus.png](http://linuxdesktop.cn/wp-content/uploads/2007/05/pingus.thumbnail.png)Tux还是主角了，现在不但是一个Tux，而是一群Tux，哈哈。是纯解密性的游戏。曾经在windows下也玩过类似的游戏，在幻想游戏里。
迷题都不算难，轻松的益智游戏。
**6.街机游戏—-恐龙**
[下载安装包](http://laf163.googlepages.com/klkd_0.1-1_i386.deb)
sudo apt-get install xmame-x
dpkg -i klkd_0.1-1_i386.deb
![klkd.png](http://linuxdesktop.cn/wp-content/uploads/2007/05/klkd.thumbnail.png) 看到下面那张图，应该就不用多说什么了，经过那个年代的人一定会装来玩的，呵呵。
特别提醒的是操作方式：
按Shift + Pageup放大窗口
按Shift + Pagedown缩小窗口
按Tab进入键盘设置
游戏者1:
按5加铜板
按1开始游戏
游戏者2:
按6加铜板
按2开始游戏

#### [安装Linux版跑跑卡丁车----SuperTuxKart](http://www.oceanboo.cn/read.php/80.htm)
http://www.oceanboo.cn/read.php/80.htm
上次给大家介绍过[安装Linux版超级马里奥----SuperTux](http://www.oceanboo.cn/read.php/59.htm)，这次更升级，推荐大家一个Linux版的跑跑卡丁车-----SuperTuxKart。
SuperTuxKart是一款3D的卡丁车赛车游戏，主角当然还是GNU/Linux 的吉祥物企鹅Tux，游戏仿照跑跑卡丁车的风格，可以获取武器等作战，画面相当漂亮，是一款居家旅游、休闲娱乐之必备佳品。
进入日志查看详细，获取这款优秀的游戏了。开始游戏吧。
sudo apt-get install supertuxkart
