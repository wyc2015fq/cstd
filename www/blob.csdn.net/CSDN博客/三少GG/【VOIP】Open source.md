## SIP Proxies
- [Mini-SIP-Proxy](http://blog.csdn.net/zrjdds/article/details/wiki/view/Mini-SIP-Proxy) A very tiny perl POE based SIP proxy
- [MjServer](http://www.mjsip.org/) cross-platform SIP proxy/registrar/redirect, written in java, based on MjSip stack
- [MySIPSwitch](http://blog.csdn.net/zrjdds/article/details/wiki/view/MySIPSwitch) SIP Proxy server which allows using multiple
 SIP accounts with a single SIP login
- [NethidPro3.0.6](http://www.vonets.com/service/show_download.asp?ID=90) Opensource Sip Encryption Bridge: www.vonets.com
- [Net-SIP](http://blog.csdn.net/zrjdds/article/details/wiki/view/Net-SIP) A Perl SIP framework that includes a stateless proxy
- [JAIN-SIP Proxy](http://blog.csdn.net/zrjdds/article/details/wiki/view/NIST+SIP)
- [OpenJSIP](http://code.google.com/p/openjsip) Opensource distributed standalone SIP proxy, SIP registrar, SIP location service
 run by Java VM. Based on [NIST SIP](http://blog.csdn.net/zrjdds/article/details/wiki/view/NIST+SIP) and derived
 from JAIN-SIP Proxy.
- [OpenSBC](http://blog.csdn.net/zrjdds/article/details/wiki/view/OpenSBC): MPL licensed SIP proxy/registrar/B2BUA with NAT traversal
 and ENUM
- [OpenSER](http://blog.csdn.net/zrjdds/article/details/wiki/view/OpenSER): GPL SIP Server with TLS support - renamed to [Kamailio](http://blog.csdn.net/zrjdds/article/details/wiki/view/Kamailio)
- [OpenSIPS](http://www.opensips.org/) forked from OpenSER.
- [partysip](http://blog.csdn.net/zrjdds/article/details/wiki/view/Partysip) SIP proxy server
- [SaRP](http://blog.csdn.net/zrjdds/article/details/wiki/view/SaRP) SIP and RTP Proxy in Perl
- [sipd](http://blog.csdn.net/zrjdds/article/details/wiki/view/sipd) SIP Proxy
- [SIP Express Router](http://blog.csdn.net/zrjdds/article/details/wiki/view/SIP+Express+Router) (SER): **the** SIP
 router/proxy/jack-in-all-trades from IPtel.org
- [Siproxd](http://blog.csdn.net/zrjdds/article/details/wiki/view/Siproxd) SIP and RTP Proxy
- [SIPVicious](http://blog.csdn.net/zrjdds/article/details/wiki/view/SIPVicious) tool suite: tools for auditing sip devices
- [sipX](http://blog.csdn.net/zrjdds/article/details/wiki/view/sipX) The SIP PBX for Linux: Complete, native SIP PBX solution from [SIPfoundry](http://blog.csdn.net/zrjdds/article/details/wiki/view/SIPfoundry)
- [Vocal](http://blog.csdn.net/zrjdds/article/details/wiki/view/VOCAL) SIP softswitch with H.323 and MGCP translators for non-SIP endpoints
- [Yxa](http://blog.csdn.net/zrjdds/article/details/wiki/view/Yxa) Written in the Erlang programming language

- [MSRP Library](http://www.sipsimpleclient.com/) - MSRP protocol (RFC4975) and its relay extension (RFC4976) written in Python
- [NIST SIP](http://blog.csdn.net/zrjdds/article/details/wiki/view/NIST+SIP) Various SIP appications and tools
 in Java
- [Open Sip Stack](http://www.opensipstack.org/) MPL licensed SIP stack with ENUM, Presence (XMPP/SIMPLE) and NAT traversal. Reference
 implementation of Session Border Controller (OpenSBC) available.
- [oSIP Library](http://blog.csdn.net/zrjdds/article/details/wiki/view/oSIP+Library) SIP Library
- [OSP client protocol stack](http://www.transnexus.com/OSP%20Toolkit/OSP%20Toolkit.htm) and [SIPfoundry](http://blog.csdn.net/zrjdds/article/details/wiki/view/SIPfoundry)
- [PhClickDial](http://www.mbdsys.com/opensource/phclickdial) - [Verona](http://www.mbdsys.com/opensource/verona) based
 Active/X plugin for IE allowing ClickToDial functionallity
- [PJSIP](http://blog.csdn.net/zrjdds/article/details/wiki/view/PJSIP): Small footprint, high performance, and ultra-portable SIP stack

- [reSIProcate](http://blog.csdn.net/zrjdds/article/details/wiki/view/reSIProcate) SIP stack and sample Application from [SIPfoundry](http://blog.csdn.net/zrjdds/article/details/wiki/view/SIPfoundry)

- [SIP SIMPLE client SDK](http://www.sipsimpleclient.com/) - High level middleware on top of SIP, RTP, MSRP and XCAP protocols
- [Twisted](http://blog.csdn.net/zrjdds/article/details/wiki/view/Twisted) Python protocol stacks and applications includes SIP support

- [Vovida SIP](http://blog.csdn.net/zrjdds/article/details/wiki/view/VOVIDA+SIP) Vovida SIP stack
- [XCAP Library](http://www.sipsimpleclient.com/) - XCAP client library written in Python

## Ekiga
Ekiga (formely known as GnomeMeeting) is an open source SoftPhone, Video Conferencing and Instant Messenger application over the Internet.
It supports HD sound quality and video up to DVD size and quality.
It is interoperable with many other standard compliant softwares, hardwares and service providers as it uses both the major telephony standards (SIP and H.323).
Main Features of the Ekiga Softphone Version 3.2 in a nutshell
- Ease of use with a modern Graphical User Interface.
- Audio and Video free calls through the internet.
- Free Instant Messaging through the internet with Presence support.
- Audio (and video) calls to landlines and cell phones with support to the cheapest service providers.
- High Definition Sound (wideband) and Video Quality up to DVD quality (high framerate, state of the art quality codec and frame size).
- Free of choice of the service provider.
- SMS to cell phones if the service provider supports it (like the default provider).
- Standard Telephony features support like Call Hold, Call Transfer, Call Forwarding, DTMF.
- Remote and Local Address Book support: Remote Address Book support with authentification using the standard LDAP technology, Local Address support in Gnome (Evolution).
- Multi platform: Windows and GNU/Linux
- Wide interoperability: Ekiga use the main deployed stantards for telephony protocols (SIP and H.323) and has been tested with a wide range of softphones, hardphones, PBX and service providers.
[http://www.ekiga.org/](http://www.ekiga.org/)
License: GPL
