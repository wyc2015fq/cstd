# 分享12款最佳的Bootstrap设计工具
发表于8小时前| 1999次阅读| 来源CSDN|
5 条评论| 作者夏梦竹
[bootstrap](http://www.csdn.net/tag/bootstrap/news)[web应用](http://www.csdn.net/tag/web%E5%BA%94%E7%94%A8/news)[设计师](http://www.csdn.net/tag/%E8%AE%BE%E8%AE%A1%E5%B8%88/news)[开发者](http://www.csdn.net/tag/%E5%BC%80%E5%8F%91%E8%80%85/news)[framework](http://www.csdn.net/tag/framework/news)
**摘要：**设计师总会渴望有一些新奇有趣的设计工具来提高工作效率。本文我们将分享12款最佳的Bootstrap设计工具，帮助你做出特别清爽同时又专业美观的页面。
设计师总会渴望有一些新奇有趣的设计工具来提高工作效率，而Bootstrap就是您的不二选择。2013年Bootstrap得到了广泛普及，它是开发者较为常用的框架之一，本文我们将分享12款最佳的Bootstrap设计工具，帮助你做出特别清爽同时又专业美观的页面。
### 12. [Bootstrap Designer](http://www.bootstrapdesigner.com/)
![](http://cms.csdnimg.cn/article/201403/13/532157577b3bb.jpg)
Bootstrap Designer是一款在线运行工具，无需下载和安装。使用该工具你可以创建出令人惊叹的HTML5模板。 
### 11. [Get Kickstrap](http://getkickstrap.com/)
![](http://cms.csdnimg.cn/article/201403/13/5321578b2543f.jpg)
如果你正在寻找与Bootsrap相匹配的Web技术，那你可以试试这款Get Kickstrap。该工具非常先进，无需任何后台即可运行数据库来驱动Web应用程序。
### 10. [Bootply](http://bootply.com/)
![](http://cms.csdnimg.cn/article/201403/13/5321579a1cd4a.jpg)
通过Bootply可以非常轻松地基于Bootstrap来进行设计、开发原型、扩展和测试等。Bootply可与其他流行的Bootstrap插件、微框架和库进行整合。 
### 9. [Bootstrap Button Generator](http://blog.koalite.com/bbg/)
![](http://cms.csdnimg.cn/article/201403/13/532157b296106.jpg)
通过Bootstrap Button Generator，你只需输入你想指定给新按钮的样式的CSS类，并简单调整图标的颜色，即可获得新的图标，然后复制代码，粘贴在任意你想要的地方即可。
### 8. [Easel](https://www.easel.io/)
![](http://cms.csdnimg.cn/article/201403/13/532157c1c429f.jpg)
这款工具可在你的浏览器中运行，具有高保真的特性，帮助开发者和设计师提供真实的Web元素。 
### 7. [Layoutit](http://www.layoutit.com/)
![](http://cms.csdnimg.cn/article/201403/13/532157d45eaf0.jpg)
 Layoutit是一个在线工具， 它可以简单而又快速搭建Bootstrap响应式布局，操作基本是使用拖动方式来完成，很简单，而元素都是基于Bootstrap框架集成的， 所以这工具很适合Web设计师和前端开发人员使用，快捷方便。 
### 6. [Bootswatch](http://bootswatch.com/)
![](http://cms.csdnimg.cn/article/201403/13/532157e943107.jpg)
该工具开源，极易安装。
### 5. [Boottheme](http://www.boottheme.com/)
![](http://cms.csdnimg.cn/article/201403/13/5321597b3a34b.jpg)
这是一款在线主题生成器，拖拽式的设计、所见即所得。
### 4. [Custom Font Tool](http://fontcustom.com/)
![](http://cms.csdnimg.cn/article/201403/13/532158776bb21.jpg)
该工具允许你在命令行中创建个性化字体。
### 3. [WordPress Twitter Bootstrap CSS](http://wordpress.org/plugins/wordpress-bootstrap-css/)
![](http://cms.csdnimg.cn/article/201403/13/53215893e420a.jpg)
在WordPress中包含最新的Twitter Bootstrap Javascript库。
### 2. [Bootmetro](http://aozora.github.io/bootmetro/)
![](http://cms.csdnimg.cn/article/201403/13/532158a237db5.jpg)
这是一款非常灵活且简单的框架，能为你带来类似Windows 8的视觉体验。 
### 1. [Flat UI Free–Framework and Bootstrap Theme Design](http://designmodo.com/flat-free/)
![](http://cms.csdnimg.cn/article/201403/13/532158b49a570.jpg)
这款工具拥有扁平化风格，最好的部分是其组件包括PSD文件；同时还可创建免费的UI。
