**有用资料：**
**[ubuntu 无线网络已禁用 wireless is disabled](http://hi.baidu.com/futrueboy/blog/item/79da6b33021ebde31b4cff54.html)**
我的Ubuntu的版本是
chq@chq-Lenovo:/etc/wicd$ cat /etc/issue
Ubuntu 11.04 \n \l
我的解决方案是：
1. 先连接上有线网络
2. 安装wicd管理器：sudo apt-get install wicd
3. 卸载了ubuntu自带的管理器network-manager： sudo apt-get remove network-manager network-manager-gnome
4. 重新启动: reboot
5. 在系统工具里 wicd network manager (我是直接在search中输入wicd)：点击运行就能进行有线网络和无线网络管理了
6. 这样应该可以了，当然最好查看一下: sudo rfkill list 看看无线网络设备有没有被禁用
chq@chq-Lenovo:~$ sudo rfkill list
[sudo] password for chq: 
0: hci0: Bluetooth
    Soft blocked: no
    Hard blocked: no
1: acer-wireless: Wireless LAN
    Soft blocked: yes
    Hard blocked: no
2: phy0: Wireless LAN
    Soft blocked: yes
    Hard blocked: no
chq@chq-Lenovo:~$ sudo rfkill unblock all
chq@chq-Lenovo:~$ sudo rfkill list
0: hci0: Bluetooth
    Soft blocked: no
    Hard blocked: no
1: acer-wireless: Wireless LAN
    Soft blocked: yes
    Hard blocked: no
2: phy0: Wireless LAN
    Soft blocked: no
    Hard blocked: no
有的话(只要有yes的话)，使用sudo rfkill unblock all打开，然后和第4步是一样的 打开wicd network manager
如果每次开机你查看rfkill list 的时候 wireless都是block的，最好在启动应用程序里加入
