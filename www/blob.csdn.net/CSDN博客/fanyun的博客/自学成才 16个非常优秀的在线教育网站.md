# 自学成才 16个非常优秀的在线教育网站 - fanyun的博客 - CSDN博客
2018年05月27日 21:59:21[老樊Lu码](https://me.csdn.net/fanyun_01)阅读数：1108
软件开发是一项非常复杂且有趣的工作，开发者需要在规定的期限内完成需求开发，并且把产品交付给客户。与此同时，开发者也不断学习各种编程知识，了解最新的技术动态。当然，学习的途径有很多种，比如购买书籍或者看视频。
在线资源最大的优势就是不断更新，取经去糟。本文提供16个非常好的在线学习资源提供给大家，帮助大家在假期温习旧知识的同时，不断学习新技术。
**1.[JavaScript-Garden](http://bonsaiden.github.io/JavaScript-Garden/)**
JavaScript -Garden主要提供JavaScript 学习资源。 对于如何避免常见的错误，难以发现的问题，以及性能问题和不好的实践给出建议。
![](http://img.ptcms.csdn.net/article/201502/05/54d32145133d0.jpg)
**2.[Goto and Learn](http://gotoandlearn.com/)**
Goto and Learn主要提供HTML5、游戏开发等方面的学习资源。
![](http://img.ptcms.csdn.net/article/201502/05/54d321c92731c.jpg)
**3.[General Assembly](https://dash.generalassemb.ly/)**
General Assembly主要教大家如何使用HTML+CSS+JavaScript制作网站，并且提供各种教学视频。
![](http://img.ptcms.csdn.net/article/201502/05/54d3229069da8.jpg)
**4.[Replit](http://repl.it/languages)**
Replit上面有非常丰富学习资源，开发者可以任意选择语言来学习，比如Ruby、PHP、Python、JavaScript等。
![](http://img.ptcms.csdn.net/article/201502/05/54d327e287493.jpg)
**5.[Codewars](http://www.codewars.com/)**
开发者可以在Codewars上面学习各种编程语言，Codewars的奖励系统激励程序员们像打游戏升级一样地做习题。
![](http://img.ptcms.csdn.net/article/201502/05/54d32ab94b9cf.jpg)
**6.[Learn to Code HTML & CSS](http://learn.shayhowe.com/html-css/)**
顾名思义，Learn to Code HTML & CSS是一个提供HTML和CSS学习资源的平台，尤其适合初学者，它可以给你带来简单且非常全面的学习指导。
![](http://img.ptcms.csdn.net/article/201502/05/54d32ba27ff3d.jpg)
**7.[Node School](http://nodeschool.io/)**
Node School是一个开源平台，教大家学习各种Web编程技能。
![](http://img.ptcms.csdn.net/article/201502/05/54d32c11e3ede.jpg)
**8.[Pluralsight](http://www.pluralsight.com/)**
PluralSight软件开发教育平台，已发展多年。Pluralsight提供的课程类型包括：C#、jQuery、iOS与Android、Java、 Ruby以及包括设计、社会化媒体、NoSQL、商业智能和Salesforce等。除了个人用户外，Pluralsight还提供为企业的培训服务。
![](http://img.ptcms.csdn.net/article/201502/05/54d32ce499f68.jpg)
**9.[Programmr](http://www.programmr.com/)**
Programmr是非常流行的一个在线学习网站，也是程序员门评估、提高和展示他们编程技能的实验室。
![](http://img.ptcms.csdn.net/article/201502/05/54d32eac4e7f2.jpg)
**10.[Team Tree House](http://teamtreehouse.com/)**
Treehouse所提供的课程类别涵盖甚广，不仅有Web开发、移动应用开发（iOS、Android）等的编程类课程教学，还有设计（移动应用、网站等）及业务管理等课程内容。
![](http://img.ptcms.csdn.net/article/201502/05/54d32f22a84a5.jpg)
**11.[Online Courses](http://www.onlinecollegecourses.com/2009/10/28/100-incredible-open-courses-for-the-ultimate-tech-geek/)**
Online Courses上面有不少数据库学习资源。
![](http://img.ptcms.csdn.net/article/201502/05/54d330ac6398b.jpg)
**12.[Ontwik](http://ontwik.com/)**
开发者在Ontwik上很容易分享和发现最好和最新的会议类、面试类信息。
![](http://img.ptcms.csdn.net/article/201502/05/54d331af51db9.jpg)
**13.[Lynda](http://www.lynda.com/)**
Lynda.com是由一对名为Bruce Heavin和Lynda Weinman的夫妇于上世纪90年代初所创立，而最初，只是因为Heavin想让自己可爱的作家妻子获得满足感，却不曾想到有一天，它会成为深受全球用户所喜爱的网络教育平台。
![](http://img.ptcms.csdn.net/article/201502/05/54d333f58818e.jpg)
**14.[Eloquent Javascript](http://eloquentjavascript.net/)**
这是一本有关JavaScript编程的在线书籍，里面的例子都可调试。
![](http://img.ptcms.csdn.net/article/201502/05/54d3349bd3623.jpg)
**15.[Course Hero](https://www.coursehero.com/)**
Course Hero上面提供的学习资源非常丰富，并且根据人群进行分类，提供相应的学习资源。
![](http://img.ptcms.csdn.net/article/201502/05/54d335b58f6c5.jpg)
**16.[Processing](https://processing.org/)**
Processing是一门编程语言、开发环境和在线社区。
![](http://img.ptcms.csdn.net/article/201502/05/54d3366fd2b50.jpg)
