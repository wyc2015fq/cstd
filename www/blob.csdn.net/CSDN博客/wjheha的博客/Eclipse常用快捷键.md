***Eclipse常用快捷键***：
- alt + / :        进行代码的提示(sysout、main)
- ctrl + 1:       为错误代码给出纠正方案
- ctrl + shift + o :      组织导入，导入其他类的包
- ctrl + d :      删除当前行代码
- ctrl + / :       使用单行注释
- ctrl + h :    强力搜索
- ctrl + alt +↓：    复制当前行代码
- ctrl + shift + L :    全部快捷键列表
