注意事项： 0 <= N <= 1000，1 <= M <= 1000

输入第一行输入一个 字符串 A，第二行输入一个字符串 B ，最后输出一个整数使得A 重复整数次后，B 是 A 重复后的字符串的子串。
样例输入
A = abcd  
B = cdabcdab
样例输出
3
