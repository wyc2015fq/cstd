# 和为零的子矩阵
给定一个整数矩阵，请编写一个函数找出一个子矩阵，使得其数字之和等于0。输出答案时，请返回左上数字和右下数字的坐标。
**格式**：
输入每一行输入一个 二维整数数组戴代表一个矩阵，输出每一行输出使得子矩阵数字之和为 0 的左上数字和右下数字的坐标组成的数组。
**样例输入**
[ 
  [ 1，5，7 ]， 
  [ 3，7，-8 ]， 
  [ 4，-8，9 ]， 
]
**样例输出**
[ (1，1)，(2，2) ]
