首先，假设有一个元组或者列表
a = (1,2,3,4)
b = [1,2,3,4]
则a[::-1]和b[::-1]的含义是将元组或列表的内容翻转
a[::-1]  # 结果为(4,3,2,1)
b[::-1]  #结果为[4,3,2,1]
返回结果是4321，那么问题就来了[::-1]表示的是从头到尾，步长为-1
`Sequence[start:end:step]`b = a[i:j:s]这种格式呢，i,j与上面的一样，但s表示步进，缺省为1.
所以a[i:j:1]相当于a[i:j]。当s<0时，i缺省时，默认为-1. j缺省时，默认为-len(a)-1
