对于矩阵的结构，肯定要声明行宽和列高，但是变量的名字到底是起Row，Col好还是起Height，Width好呢，这个问题可能仁者见仁智者见智啦，我偶然发现OpenCV处理这个问题的方法有些意思，分享给大家



```cpp
union
    {
        int rows;
        int height;
    };

    union
    {
        int cols;
        int width;
    };
```
大家以后再弄类似的东西就可以参考啦
