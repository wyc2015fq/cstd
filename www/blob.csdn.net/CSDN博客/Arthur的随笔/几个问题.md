2. 求一个字符串中第一次出现的唯一字符
答： 两种做法，要不把字符第一次出现的坐标也存入hash表，要不构造一个256长度的数组，用字符的ascii值作下标，把对应的数组值递增加一，最后再扫描一遍字符串把第一个对应数组中值为1的返回
3. tcp close_wait状态的产生
答：客户端的主动关闭连接会导致服务端产生CLOSE_WAIT, 服务端的主动关闭连接会导致客户端产生CLOSE_WAIT

5. 求一个整数二进制表达式中1的个数
答： 
```cpp
unsigned int FindOneInNumber_02(unsigned int x)
{
    unsigned int n;
    for(n=0; x; n++)
        x &= x-1;
    return n;
}
```
