# ubuntu安装谷歌浏览器 - 心纯净，行致远 - CSDN博客





2019年04月11日 16:39:09[吉大秦少游](https://me.csdn.net/zhanshen112)阅读数：38








打开该网站http://www.ubuntuchrome.com/

找到自己合适的谷歌浏览器版本后下载

打开中终端输入解压命令来解压文件

sudo dpkg -i google-chrome-stable_current_amd64.deb

如果在解压的时候发生错误可以在终端尝试输入

sudo apt-get -f install

安装成功后在次尝试解压文件

以上成功后可以在search your computer 中输入goo就可以发现谷歌浏览器已经安装完成，打开即可以使用
