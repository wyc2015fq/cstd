1047 编程团体赛 （20 分）

编程团体赛的规则为：每个参赛队由若干队员组成；所有队员独立比赛；参赛队的成绩为所有队员的成绩和；成绩最高的队获胜。

现给定所有队员的比赛成绩，请你编写程序找出冠军队。

### 输入格式：

输入第一行给出一个正整数 N（≤10​4​​），即所有参赛队员总数。随后 N 行，每行给出一位队员的成绩，格式为：`队伍编号-队员编号 成绩`，其中`队伍编号`为 1 到 1000 的正整数，`队员编号`为 1 到 10 的正整数，`成绩`为 0 到 100 的整数。

### 输出格式：

在一行中输出冠军队的编号和总成绩，其间以一个空格分隔。注意：题目保证冠军队是唯一的。

### 输入样例：

```
6
3-10 99
11-5 87
102-1 0
102-3 100
11-9 89
3-2 61
```

### 输出样例：

```
11 176
```


**思路：**

这道题相对简单，用结构体存储`：队伍编号-队员编号 成绩，然后用一个一维数组的下标存储队伍编号，对应的value存储队员成绩的和，比较即可。`

**代码：**

```cpp
//PAT1047V1
#include <stdio.h>

int main(){
	int n,i,j,m=0;
	scanf("%d",&n);
	int sum[10000]={0};
	struct BiSai{
		int tm;	//team
		int mb;	//member
		int sc;	//score
	}bs[n];
	for(i=0;i<n;i++){
		scanf("%d-%d %d",&bs[i].tm,&bs[i].mb,&bs[i].sc);
		sum[bs[i].tm]+=bs[i].sc;
		if(sum[m]<sum[bs[i].tm])
			m=bs[i].tm;
	}
	printf("%d %d",m,sum[m]); 
}
```
