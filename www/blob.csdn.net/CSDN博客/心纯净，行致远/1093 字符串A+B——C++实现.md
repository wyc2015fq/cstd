# 题目

> 
1093 字符串A+B （20 分）

给定两个字符串 A 和 B，本题要求你输出 A+B，即两个字符串的并集。要求先输出 A，再输出 B，但**重复的字符必须被剔除**。

### 输入格式：

输入在两行中分别给出 A 和 B，均为长度不超过 10​6​​的、由可见 ASCII 字符 (即码值为32~126)和空格组成的、由回车标识结束的非空字符串。

### 输出格式：

在一行中输出题面要求的 A 和 B 的和。

### 输入样例：

```
This is a sample test
to show you_How it works
```

### 输出样例：

```
This ampletowyu_Hrk
```


# 算法 

将两个字符串s1,s2拼接在一起是s3，然后开一个128个元素的int型数组，并初始化为0，凡是输出过的都+1或者置1，然后判断将要输出的元素s[i]对应的a[s[i]]是否为0,为0则输出，否则不输出即可。

# 代码

```cpp
#include <iostream>
using namespace std;
int main(){
	int a[128]={0};
	string s1,s2,s3;
	getline(cin,s1);
	getline(cin,s2);
	s3=s1+s2;
	for(int i=0;i<s3.length()  ;i++){
		if(a[s3[i]]==0){
			cout<<s3[i];
			a[s3[i]]++;
		}
	}
	return 0;
}
```
