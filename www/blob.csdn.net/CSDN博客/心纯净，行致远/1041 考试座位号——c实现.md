1041 考试座位号 （15 point(s)）

每个 PAT 考生在参加考试时都会被分配两个座位号，一个是试机座位，一个是考试座位。正常情况下，考生在入场时先得到试机座位号码，入座进入试机状态后，系统会显示该考生的考试座位号码，考试时考生需要换到考试座位就座。但有些考生迟到了，试机已经结束，他们只能拿着领到的试机座位号码求助于你，从后台查出他们的考试座位号码。

### 输入格式：

输入第一行给出一个正整数 N（≤1000），随后 N 行，每行给出一个考生的信息：`准考证号 试机座位号 考试座位号`。其中`准考证号`由 14 位数字组成，座位从 1 到 N 编号。输入保证每个人的准考证号都不同，并且任何时候都不会把两个人分配到同一个座位上。

考生信息之后，给出一个正整数 M（≤N），随后一行中给出 M 个待查询的试机座位号码，以空格分隔。

### 输出格式：

对应每个需要查询的试机座位号码，在一行中输出对应考生的准考证号和考试座位号码，中间用 1 个空格分隔。

### 输入样例：

```
4
10120150912233 2 4
10120150912119 4 1
10120150912126 1 3
10120150912002 3 2
2
3 4
```

### 输出样例：

```
10120150912002 2
10120150912119 1
```


# 思路：

这道题很简单，直接用一个结构体存储即可。注意由于准考证号是14位数，可以用long long int 存储，如果用字符数组也行，但是还要转换成数字，比较麻烦。

# 代码：

```cpp
//PAT1041V1
#include <stdio.h>

int main(){
	int n,i,j;	scanf("%d",&n);
	struct Inf{
		long long int a;
		int b;
		int c;
	}inf[n];
	for(i=0;i<n;i++){
		scanf("%lld %d %d",&inf[i].a,&inf[i].b,&inf[i].c);
	}
	int k,tmp=0;	scanf("%d",&k);
	for(i=0;i<k;i++){
		scanf("%d",&tmp);
		for(j=0;j<n;j++){
			if(tmp==inf[j].b){
				printf("%lld %d\n",inf[j].a,inf[j].c);
			}
		}
	}
}
```
