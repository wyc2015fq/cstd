# 题目

> 
1082 射击比赛 （20 point(s)）

本题目给出的射击比赛的规则非常简单，谁打的弹洞距离靶心最近，谁就是冠军；谁差得最远，谁就是菜鸟。本题给出一系列弹洞的平面坐标(x,y)，请你编写程序找出冠军和菜鸟。我们假设靶心在原点(0,0)。

### 输入格式：

输入在第一行中给出一个正整数 N（≤ 10 000）。随后 N 行，每行按下列格式给出：

```
ID x y
```

其中 `ID` 是运动员的编号（由 4 位数字组成）；`x` 和 `y` 是其打出的弹洞的平面坐标(`x`,`y`)，均为整数，且 0 ≤ |`x`|, |`y`| ≤ 100。题目保证每个运动员的编号不重复，且每人只打 1 枪。

### 输出格式：

输出冠军和菜鸟的编号，中间空 1 格。题目保证他们是唯一的。

### 输入样例：

```
3
0001 5 7
1020 -1 3
0233 0 -1
```

### 输出样例：

```
0233 0001
```


# 算法

使用结构体存储输入，用4个double来记录id和距离靶心的距离，两个用来记录最近的，两个用来记录最远的，符合要求更新就可以。最后输出。

# 代码

```cpp
#include <iostream>
#include <cmath>
using namespace std;
struct node{
  double id,x,y;
};
int main(){
  int N;  cin>>N;
  double a1=0,a2=1000,a3=0,a4=0;
  node play[N];
  for(int i=0;i<N;i++){
      cin>>play[i].id>>play[i].x>>play[i].y;
      if(a2>sqrt(pow(play[i].x,2)+pow(play[i].y,2))){
          a1=play[i].id;
          a2=sqrt(pow(play[i].x,2)+pow(play[i].y,2));
      }
      if(a4<sqrt(pow(play[i].x,2)+pow(play[i].y,2))){
          a3=play[i].id;
          a4=sqrt(pow(play[i].x,2)+pow(play[i].y,2));
      }
  }
  printf("%04.0lf %04.0lf",a1,a3);
  return 0;
}
```
