 离散数据有时不易观察出数据的内在规律，为此有时需要把离散数据进行展示来发现隐藏在背后数据的规律。展示的方式多样，具有时序特点的数据可以按照时间进行展示，具有范围和重复性的数据可以按照不同颜色同一坐标系进行对比展示等等。

下面以今天学习的简单绝对值函数进行展示。

题目：图形化展示离散绝对值函数y=|x|



```
>> n=(-10:10)';
>> y=abs(n);
>> plot(n,y,'r*') 
>> axis equal
>> grid on
>> xlabel('n')
```
