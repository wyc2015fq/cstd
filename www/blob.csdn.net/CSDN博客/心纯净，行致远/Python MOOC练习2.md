# 4、同心圆函数

```python
'''
Name： concentric circles function
function：draw a series of  concentric circles
usage:target(innerR,gap,pencolor,number)
'''
from turtle import *
def target(innerR,gap,number):
    for i in range(number):
        circle(innerR+i*gap)
        penup()
        goto(0,-gap*(i+1))
        pendown()
```
