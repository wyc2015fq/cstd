1036 跟奥巴马一起编程 （15 分）

美国总统奥巴马不仅呼吁所有人都学习编程，甚至以身作则编写代码，成为美国历史上首位编写计算机代码的总统。2014 年底，为庆祝“计算机科学教育周”正式启动，奥巴马编写了很简单的计算机代码：在屏幕上画一个正方形。现在你也跟他一起画吧！

### 输入格式：

输入在一行中给出正方形边长 N（3≤N≤20）和组成正方形边的某种字符 C，间隔一个空格。

### 输出格式：

输出由给定字符 C 画出的正方形。但是注意到行间距比列间距大，所以为了让结果看上去更像正方形，我们输出的行数实际上是列数的 50%（四舍五入取整）。

### 输入样例：

```
10 a
```

### 输出样例：

```
aaaaaaaaaa
a        a
a        a
a        a
aaaaaaaaaa
```


思路：这道题很简单。可以将打印的图形分成3部分：第一行/中间部分/最后一行。小难点是四舍五入的计算方法，实际上就是一个条件表达式就可以判断，判断的核心是根据小数部分是否大于等于0.5,或者取余的整数是否大于等于1。[整数的四舍五入——c实现](https://blog.csdn.net/zhanshen112/article/details/83615846)

代码：

```cpp
//PAT1036V1
#include <stdio.h>

int main(){
	int n;
	int i,j;
	char a;
	scanf("%d %c",&n,&a);
	int k=(n-2*(n/2))>=1?n/2+1:n/2;	//四舍五入
	for(i=0;i<n;i++){	//第一行 
		printf("%c",a);
	} 
	for(i=0;i<k-2;i++){	//第二行 
		printf("\n%c",a);
		for(j=0;j<n-2;j++)
			printf(" ");
		printf("%c",a);
	}
	printf("\n");
	for(i=0;i<n;i++){	//第三行 
		printf("%c",a);
	} 
	return 0;	
}
```
