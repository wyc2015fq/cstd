题目：绘制9边形，体会自变量排列对绘图的重要性以及如何画好多边形。

代码如下：



```
>> clear
>> n=9;
>> t=0:2*pi/n:2*pi;
>> x=sin(t);y=cos(t);
>> tt=reshape(t,2,(n+1)/2)   %将t的10个数值排成2*5的二维数组 

tt =

         0    1.3963    2.7925    4.1888    5.5851
    0.6981    2.0944    3.4907    4.8869    6.2832

>> tt=flipud(tt)   %%将二维数组上下两行调换

tt =

    0.6981    2.0944    3.4907    4.8869    6.2832
         0    1.3963    2.7925    4.1888    5.5851

>> tt=tt(:)    获得变序排列的自变量，注意先按行排，再依次按列排

tt =

    0.6981
         0
    2.0944
    1.3963
    3.4907
    2.7925
    4.8869
    4.1888
    6.2832
    5.5851
 
>> xx=sin(tt);yy=cos(tt); 
>> subplot(1,2,1),plot(x,y)  %正常排序下的图形 
>> axis equal on,shg,title('(1)正常排序图形')   %shg为显示图形窗命令
>> subplot(1,2,2),plot(xx,yy)
>> axis equal on,shg,title('(2)变序排列的图形')
```
