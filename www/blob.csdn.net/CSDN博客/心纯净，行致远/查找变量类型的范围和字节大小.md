编程时经常需要知道各种字符型变量、整型变量的范围和占用的字节大小，各种编译器可能会略有区别，因此可以通过将头文件包含进来进行查找。

如下：


```cpp
#include <stdio.h> 
#include <limits.h>

main()
{
	int a;
	printf("The value of INT_MAX is %i\n", INT_MAX);	
	printf("The value of INT_MIN is %i\n", INT_MIN);	
	printf("An int takes %d bytes\n", sizeof(int));
	printf("The value of LONG_MIN is %i\n", LONG_MIN);	
	printf("The value of ULLONG_MAX is %i\n", ULLONG_MAX);	
	printf("An LONG_LONG_MAX takes %d bytes\n", sizeof(LONG_LONG_MAX));

}
```
