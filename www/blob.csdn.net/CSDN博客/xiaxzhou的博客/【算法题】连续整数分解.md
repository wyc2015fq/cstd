判断整数x是否可以表示为n个连续的正整数之和


由 


$x = m+(m+1)+...+(m-n+1)$

可得 


$m=\frac{\frac{(2*x)}{n}-n+1}{2}$

若m为整数则可分解。
```
bool fun(int x,int n)
{
    float m = ((2.0*x) / n - n + 1) / 2.0;
    if (m==(int)m)
    {
        return true;
    }
    else
    {
        return false;
    }

}
```
