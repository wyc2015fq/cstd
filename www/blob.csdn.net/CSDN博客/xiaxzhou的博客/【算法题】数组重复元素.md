判断数组是否存在重复元素，已知数组有n个元素，元素范围为1~n 。

- 暴力循环
- 排序后遍历
- 哈希表或者数组
- 交换归位，直到出现冲突

**方法4代码如下：**

```
bool IsReapeat(int * a,int len)
{
    register int j(0);
    for(int i = 0; i < len; ++i)
    {
        j = a[i];
        if (j-1 == i)
        {
            continue;
        }
        if (a[i] == a[j-1])
        {
            return 1;
        }
        swap(a[i],a[j-1]);
        --i;
    }
    return -1;
}
```
