## 有一个类里面只有一个私有int类型成员变量，也有一个类对象，怎么访问这个变量

```
class A
{
public:
    A(int lhs) :i(lhs){  };
    virtual void  func(){};
private:
    int i;
};


int main()
{
    A a(1);
    int * b = (int *)&a;
    cout << *(b+1) << endl;
    return 0;
}
```

内存结构： 

虚函数表指针 4字节 

int成员变量   4字节
